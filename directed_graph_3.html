What is a Directed Graph?
Understand the fundamental building blocks: vertices, directed edges, and how direction changes everything.
BEGINNER
ЁЯУЦ
Theory
A directed graph (digraph) is a set of vertices (nodes) connected by directed edges (arcs). Unlike an undirected graph, each edge has a specific direction тАФ it goes FROM one vertex TO another.

Formally: G = (V, E) where V is a set of vertices and E is a set of ordered pairs (u, v) meaning "there is an edge from u to v".

Key difference: In a directed graph, edge (AтЖТB) does NOT imply edge (BтЖТA).
V = {A, B, C, D}
E = {(AтЖТB), (AтЖТC), (BтЖТD), (CтЖТD)}
|V| = 4 vertices
|E| = 4 directed edges

Incidence Matrix
The primary representation we use тАФ a 2D matrix where rows are vertices and columns are edges.
BEGINNER
ЁЯУЦ
Theory
In an Incidence Matrix M of size V ├Ч E:

M[v][e] = +1 if vertex v is the source (tail) of edge e
M[v][e] = -1 if vertex v is the destination (head) of edge e
M[v][e] = 0 if vertex v is not connected to edge e

This lets us encode direction explicitly. Each column (edge) has exactly one +1 and one -1.

C Code: Full Incidence Matrix Structure
#include <stdio.h>
#include <string.h>

#define MAX_V 10
#define MAX_E 20

/* Incidence matrix: M[vertex][edge] */
int  M[MAX_V][MAX_E];   /* +1, -1, or 0             */
int  W[MAX_E];          /* Weight of each edge       */
char V[MAX_V][20];      /* Vertex names              */
int  nV = 0, nE = 0;    /* Vertex and edge counts    */

void initGraph() {
    memset(M, 0, sizeof(M));
    memset(W, 0, sizeof(W));
    nV = 0; nE = 0;
}

void printMatrix() {
    printf("Incidence Matrix:\n");
    printf("     ");
    for (int e = 0; e < nE; e++) printf(" e%d ", e);
    printf("\n");
    for (int v = 0; v < nV; v++) {
        printf("%-5s", V[v]);
        for (int e = 0; e < nE; e++)
            printf(" %2d ", M[v][e]);
        printf("\n");
    }
}

int main() {
    initGraph();
    /* We'll add vertices and edges in the next sections */
    return 0;
}

int insertVertex(char *name) {

void deleteVertex(int vIdx) {
    if (vIdx < 0 || vIdx >= nV) { printf("Invalid!\n"); return; }

        strcpy(V[v], V[v+1]);
        for (int e = 0; e < nE; e++)
            M[v][e] = M[v+1][e];

    nV--;
    printf("Vertex deleted. Vertices left: %d\n", nV);
}

/* Search a vertex by name, return its index or -1 */
int searchVertex(char *name) {
    for (int v = 0; v < nV; v++)
        if (strcmp(V[v], name) == 0) return v;
    return -1;   /* Not found */
}

ЁЯУЦ
Theory: Inserting a Directed Edge
To add an edge from vertex u to vertex v with weight w:

Set M[u][nE] = +1 (u is the source)
Set M[v][nE] = -1 (v is the destination)
Store W[nE] = weight, then increment nE

C Code: Insert / Delete / Search Edge
/* Insert directed edge from u to v with weight w */
int insertEdge(int u, int v, int w) {
    if (nE >= MAX_E) { printf("Edge limit!\n"); return -1; }
    if (u < 0 || u >= nV || v < 0 || v >= nV) {
        printf("Invalid vertices!\n"); return -1;
    }
    M[u][nE] = 1;     /* u is source  (+1) */
    M[v][nE] = -1;    /* v is dest    (-1) */
    W[nE]    = w;      /* store weight      */
    return nE++;       /* return edge index */
}

/* Delete an edge by its column index */
void deleteEdge(int eIdx) {
    if (eIdx < 0 || eIdx >= nE) { printf("Invalid!\n"); return; }
    /* Shift columns left to fill the gap */
    for (int e = eIdx; e < nE - 1; e++) {
        W[e] = W[e+1];
        for (int v = 0; v < nV; v++)
            M[v][e] = M[v][e+1];
    }
    nE--;
}

/* Search for an edge from u to v, return edge index */
int searchEdge(int u, int v) {
    for (int e = 0; e < nE; e++)
        if (M[u][e] == 1 && M[v][e] == -1)
            return e;   /* Found! */
    return -1;           /* Not found */
}

Breadth First Search (BFS)
Explore a directed graph level by level using a queue. Visit all neighbors before going deeper.
INTERMEDIATE
ЁЯУЦ
Theory
BFS starts at a source vertex and explores all its neighbors first, then their neighbors, and so on тАФ level by level.

1. Initialize queue 2. Enqueue source 3. Mark visited 4. Dequeue & explore neighbors
In the incidence matrix: for vertex v, neighbors reachable by directed edge are found where M[v][e] == +1 (v is source of edge e), then find the vertex w where M[w][e] == -1.

C Code: BFS on Incidence Matrix
/* BFS using a simple queue array */
void bfs(int start) {
    int visited[MAX_V] = {0};
    int queue[MAX_V], front = 0, rear = 0;

    /* Step 1: Enqueue start vertex */
    queue[rear++] = start;
    visited[start] = 1;

    printf("BFS order: ");

    while (front < rear) {
        /* Step 2: Dequeue */
        int v = queue[front++];
        printf("%s ", V[v]);

        /* Step 3: Find all edges where v is source */
        for (int e = 0; e < nE; e++) {
            if (M[v][e] == 1) {     /* v is source of edge e */
                /* Find destination vertex */
                for (int w = 0; w < nV; w++) {
                    if (M[w][e] == -1 && !visited[w]) {
                        visited[w] = 1;
                        queue[rear++] = w;
                    }
                }
            }
        }
    }
    printf("\n");
}

Depth First Search (DFS)
Explore as deep as possible before backtracking. Uses a stack (or recursion).
INTERMEDIATE
ЁЯУЦ
Theory
DFS dives deep into one path until it can't go further, then backtracks to explore other paths. Two variants:

Recursive DFS: Use the call stack тАФ each recursive call goes deeper.
Iterative DFS (Stack): Manually manage a stack тАФ push neighbors, pop and visit.

C Code: Recursive DFS + Stack DFS
int visited[MAX_V];

/* Recursive DFS */
void dfsRecursive(int v) {
    visited[v] = 1;
    printf("%s ", V[v]);

    /* Explore all edges where v is source */
    for (int e = 0; e < nE; e++) {
        if (M[v][e] == 1) {          /* v is source */
            for (int w = 0; w < nV; w++) {
                if (M[w][e] == -1 && !visited[w])
                    dfsRecursive(w); /* Recurse deeper */
            }
        }
    }
}

/* Iterative DFS using a stack */
void dfsIterative(int start) {
    int stack[MAX_V], top = -1;
    int vis[MAX_V] = {0};
    stack[++top] = start;

    while (top >= 0) {
        int v = stack[top--];   /* Pop */
        if (vis[v]) continue;
        vis[v] = 1;
        printf("%s ", V[v]);

        for (int e = 0; e < nE; e++) {
            if (M[v][e] == 1) {
                for (int w = 0; w < nV; w++)
                    if (M[w][e] == -1 && !vis[w])
                        stack[++top] = w; /* Push neighbor */
            }
        }
    }
}

Degree Operations
Calculate in-degree, out-degree, total degree, and find max/min degree vertices.
BEGINNER
ЁЯУЦ
Theory
In a directed graph, each vertex has two degree types:

Out-degree: Number of edges leaving vertex v тЖТ count M[v][e] == +1
In-degree: Number of edges entering vertex v тЖТ count M[v][e] == -1
Total degree: In-degree + Out-degree

/* Out-degree of vertex v */
int outDegree(int v) {
    int count = 0;

/* In-degree of vertex v */

    for (int e = 0; e < nE; e++)

int totalDegree(int v) {
    return inDegree(v) + outDegree(v);
}

/* Vertex with max degree */
int maxDegreeVertex() {
    int maxV = 0, maxD = totalDegree(0);
    for (int v = 1; v < nV; v++) {
        int d = totalDegree(v);
        if (d > maxD) { maxD = d; maxV = v; }
    }
    return maxV;
}

/* Vertex with min degree */
int minDegreeVertex() {
    int minV = 0, minD = totalDegree(0);
    for (int v = 1; v < nV; v++) {
        int d = totalDegree(v);
        if (d < minD) { minD = d; minV = v; }
    }
    return minV;
}

Path Operations
Find paths, shortest paths, all paths, check reachability, and calculate path costs.
INTERMEDIATE
ЁЯУЦ
Theory
A path in a directed graph is a sequence of vertices where each consecutive pair has a directed edge between them.

Find Path: Use DFS to find any path from u to v.
Shortest Path: Use BFS (unweighted) or Dijkstra (weighted).
All Paths: Use DFS with backtracking to enumerate every path.
Reachability: Can we reach v from u at all? (Run BFS/DFS from u).
Path Cost: Sum of edge weights along a path.

C Code: Find Path & Reachability
int path[MAX_V], pathLen = 0;
int vis2[MAX_V];

/* Find any path from src to dst using DFS */
int findPath(int v, int dst) {
    vis2[v] = 1;
    path[pathLen++] = v;
    if (v == dst) return 1;   /* Found! */

    for (int e = 0; e < nE; e++) {
        if (M[v][e] == 1) {        /* v is source */
            for (int w = 0; w < nV; w++) {
                if (M[w][e] == -1 && !vis2[w])
                    if (findPath(w, dst)) return 1;
            }
        }
    }
    path[--pathLen] = 0;  /* Backtrack */
    return 0;
}

/* Reachability check */
int isReachable(int src, int dst) {
    memset(vis2, 0, sizeof(vis2));
    pathLen = 0;
    return findPath(src, dst);
}

/* Calculate cost of current path */
int pathCost() {
    int cost = 0;
    for (int i = 0; i < pathLen - 1; i++) {
        int u = path[i], v = path[i+1];
        for (int e = 0; e < nE; e++)
            if (M[u][e] == 1 && M[v][e] == -1)
                cost += W[e];
    }
    return cost;
}

Connectivity Operations
Check strong/weak connectivity and find strongly connected components (SCCs).
ADVANCED
ЁЯУЦ
Theory
Strongly Connected: For every pair (u, v), there is a directed path from u to v AND from v to u.

Weakly Connected: If you ignore edge directions, the graph is connected (undirected sense).

Strongly Connected Components (SCCs): Maximal subgraphs that are strongly connected. Found using Kosaraju's Algorithm:
1. DFS, record finish order 2. Transpose graph 3. DFS in reverse finish order
ЁЯТ╗
C Code: Strong Connectivity Check
/* Check if all vertices reachable from vertex 0 */
int allReachableFrom(int src) {
    int vis[MAX_V] = {0};
    int stack[MAX_V], top = -1;
    stack[++top] = src; vis[src] = 1;
    while (top >= 0) {
        int v = stack[top--];
        for (int e = 0; e < nE; e++)
            if (M[v][e] == 1)
                for (int w = 0; w < nV; w++)
                    if (M[w][e] == -1 && !vis[w])
                    { vis[w] = 1; stack[++top] = w; }
    }
    for (int v = 0; v < nV; v++)
        if (!vis[v]) return 0; /* Not all reached */
    return 1;
}

/* Strong connectivity: reachable both ways */
int isStronglyConnected() {
    /* Check forward reachability from vertex 0 */
    if (!allReachableFrom(0)) return 0;

    /* Transpose: swap +1 and -1 in M */
    for (int v = 0; v < nV; v++)
        for (int e = 0; e < nE; e++)
            M[v][e] = -M[v][e];  /* Reverse directions */

    int result = allReachableFrom(0);

    /* Restore original */
    for (int v = 0; v < nV; v++)
        for (int e = 0; e < nE; e++)
            M[v][e] = -M[v][e];

    return result;
}

Real-Life Analogy

A strongly connected city means you can travel by bus from ANY neighborhood to ANY other neighborhood (possibly with transfers). Weakly connected means there's at least one path if you could also go the wrong way on buses. SCCs are groups of neighborhoods with direct two-way bus connections.
Cycle Detection
Detect and count cycles in directed graphs. Essential for deadlock detection and dependency resolution.
INTERMEDIATE
ЁЯУЦ
Theory
A cycle exists when you can start at a vertex, follow directed edges, and return to the same vertex.

Detection method: DFS with 3 states:
WHITE (0): Not visited yet
GRAY (1): Currently in DFS stack (being explored)
BLACK (2): Fully processed
If we reach a GRAY vertex, we found a cycle!
ЁЯСБя╕П

C Code: Cycle Detection (DFS Color Method)
/* 0=WHITE, 1=GRAY, 2=BLACK */
int color[MAX_V];

int dfsHasCycle(int v) {
    color[v] = 1;  /* Mark GRAY: on DFS stack */

    for (int e = 0; e < nE; e++) {
        if (M[v][e] == 1) {       /* v is source */
            for (int w = 0; w < nV; w++) {
                if (M[w][e] == -1) {
                    if (color[w] == 1)    /* GRAY = cycle! */
                        return 1;
                    if (color[w] == 0)    /* WHITE = explore */
                        if (dfsHasCycle(w)) return 1;
                }
            }
        }
    }
    color[v] = 2;  /* Mark BLACK: done */
    return 0;
}

int detectCycle() {
    memset(color, 0, sizeof(color));
    for (int v = 0; v < nV; v++)
        if (color[v] == 0)
            if (dfsHasCycle(v)) return 1;
    return 0;
}

Real-Life Analogy
ЁЯФТ Deadlock in Operating Systems
Cycles in directed graphs model deadlocks тАФ Process A waits for Process B, Process B waits for Process C, Process C waits for A. They're all stuck in a circle! The OS detects this cycle to break the deadlock and recover.
Topological Sort
Order vertices such that every directed edge goes from earlier to later. Basis for task scheduling and dependency resolution.
INTERMEDIATE
ЁЯУЦ
Theory
Topological sort only exists for DAGs (Directed Acyclic Graphs тАФ no cycles).

Kahn's Algorithm (BFS-based):
1. Compute in-degree of all vertices 2. Enqueue all 0 in-degree vertices 3. Dequeue, add to result 4. Reduce in-degree of neighbors

C Code: Kahn's Topological Sort
void topologicalSort() {
    int indeg[MAX_V] = {0};

    /* Step 1: Compute in-degrees */
    for (int v = 0; v < nV; v++)
        for (int e = 0; e < nE; e++)
            if (M[v][e] == -1) indeg[v]++;

    /* Step 2: Queue vertices with in-degree 0 */
    int queue[MAX_V], front = 0, rear = 0;
    for (int v = 0; v < nV; v++)
        if (indeg[v] == 0) queue[rear++] = v;

    printf("Topological order: ");
    int count = 0;

    /* Step 3: Process */
    while (front < rear) {
        int v = queue[front++];
        printf("%s ", V[v]); count++;

        /* Reduce in-degree of all neighbors */
        for (int e = 0; e < nE; e++) {
            if (M[v][e] == 1) {
                for (int w = 0; w < nV; w++)
                    if (M[w][e] == -1) {
                        indeg[w]--;
                        if (indeg[w] == 0)
                            queue[rear++] = w;
                    }
            }
        }
    }
    if (count != nV) printf("\nCycle detected! Sort impossible.\n");
    else printf("\n");
}

C Code: Dijkstra's Algorithm
#define INF 99999

void dijkstra(int src) {
    int dist[MAX_V], done[MAX_V] = {0};
    for (int i = 0; i < nV; i++) dist[i] = INF;
    dist[src] = 0;

    for (int iter = 0; iter < nV; iter++) {
        /* Find min distance unvisited vertex */
        int u = -1;
        for (int v = 0; v < nV; v++)
            if (!done[v] && (u == -1 || dist[v] < dist[u]))
                u = v;

        if (dist[u] == INF) break;
        done[u] = 1;

        /* Relax outgoing edges from u */
        for (int e = 0; e < nE; e++) {
            if (M[u][e] == 1) {
                for (int v = 0; v < nV; v++) {
                    if (M[v][e] == -1) {
                        int newDist = dist[u] + W[e];
                        if (newDist < dist[v])
                            dist[v] = newDist;  /* Relax! */
                    }
                }
            }
        }
    }

    printf("Shortest from %s:\n", V[src]);
    for (int v = 0; v < nV; v++)
        printf("  тЖТ %s: %d\n", V[v],
               dist[v] == INF ? -1 : dist[v]);
}

C Code: Bellman-Ford Algorithm
void bellmanFord(int src) {
    int dist[MAX_V];
    for (int i = 0; i < nV; i++) dist[i] = INF;
    dist[src] = 0;

    /* Relax all edges (V-1) times */
    for (int iter = 0; iter < nV - 1; iter++) {
        for (int e = 0; e < nE; e++) {
            int u = -1, v = -1;
            for (int k = 0; k < nV; k++) {
                if (M[k][e] ==  1) u = k;
                if (M[k][e] == -1) v = k;
            }
            if (u != -1 && dist[u] != INF)
                if (dist[u] + W[e] < dist[v])
                    dist[v] = dist[u] + W[e];
        }
    }

    /* Check for negative-weight cycles */
    for (int e = 0; e < nE; e++) {
        int u = -1, v = -1;
        for (int k = 0; k < nV; k++) {
            if (M[k][e] ==  1) u = k;
            if (M[k][e] == -1) v = k;
        }
        if (u != -1 && dist[u] != INF)
            if (dist[u] + W[e] < dist[v])
                printf("Negative weight cycle!\n");
    }
}

/* Floyd-Warshall: All-pairs shortest path */
void floydWarshall() {
    int d[MAX_V][MAX_V];
    /* Initialize distance matrix */
    for (int i = 0; i < nV; i++)
        for (int j = 0; j < nV; j++)
            d[i][j] = (i == j) ? 0 : INF;

    for (int e = 0; e < nE; e++) {
        int u = -1, v = -1;
        for (int k = 0; k < nV; k++) {
            if (M[k][e] ==  1) u = k;
            if (M[k][e] == -1) v = k;
        }
        if (u != -1) d[u][v] = W[e];
    }

    /* Main DP loop */
    for (int k = 0; k < nV; k++)
        for (int i = 0; i < nV; i++)
            for (int j = 0; j < nV; j++)
                if (d[i][k] != INF && d[i][k] + d[k][j] < d[i][j])
                    d[i][j] = d[i][k] + d[k][j];
}

/* Transpose graph: reverse all edges in-place */
void transposeGraph() {
    for (int v = 0; v < nV; v++)
        for (int e = 0; e < nE; e++)
            M[v][e] = -M[v][e];  /* Flip +1 and -1 */
    printf("Graph transposed.\n");
}

/* Bipartite check using BFS 2-coloring */
int isBipartite() {
    int colorArr[MAX_V];
    memset(colorArr, -1, sizeof(colorArr));
    int queue[MAX_V], front = 0, rear = 0;
    colorArr[0] = 1;
    queue[rear++] = 0;

    while (front < rear) {
        int v = queue[front++];
        for (int e = 0; e < nE; e++) {
            if (M[v][e] != 0) {
                for (int w = 0; w < nV; w++) {
                    if (M[w][e] != 0 && w != v) {
                        if (colorArr[w] == -1) {
                            colorArr[w] = 1 - colorArr[v];
                            queue[rear++] = w;
                        } else if (colorArr[w] == colorArr[v])
                            return 0;  /* Not bipartite */
                    }
                }
            }
        }
    }
    return 1;
}

/* Greedy Graph Coloring */
void greedyColoring() {
    int col[MAX_V];
    memset(col, -1, sizeof(col));
    col[0] = 0;

    for (int v = 1; v < nV; v++) {
        int available[MAX_V] = {1};
        for (int e = 0; e < nE; e++) {
            if (M[v][e] != 0)
                for (int w = 0; w < nV; w++)
                    if (w != v && M[w][e] != 0 && col[w] != -1)
                        available[col[w]] = 0;
        }
        for (col[v] = 0; !available[col[v]]; col[v]++);
        printf("  %s тЖТ color %d\n", V[v], col[v]);
    }
}

/* Build residual capacity matrix from weights */
int cap[MAX_V][MAX_V];
void buildCapacityMatrix() {
    memset(cap, 0, sizeof(cap));
    for (int e = 0; e < nE; e++) {
        int u = -1, v = -1;
        for (int k = 0; k < nV; k++) {
            if (M[k][e] ==  1) u = k;
            if (M[k][e] == -1) v = k;
        }
        if (u != -1) cap[u][v] = W[e];  /* Capacity = weight */
    }
}
