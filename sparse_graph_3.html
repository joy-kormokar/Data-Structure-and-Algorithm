01

Introduction to Sparse Graphs

Theory

What is a Graph?

A graph is a collection of vertices (nodes) connected by edges (links). Graphs model real-world networks: roads, friendships, internet, circuits.

Graph G = (V, E) where V = set of vertices, E = set of edges

What is a Sparse Graph?

A graph is sparse when it has very few edges compared to the maximum possible edges. If a graph has V vertices, maximum edges = V×(V-1)/2. A sparse graph has edges much less than this maximum.

Sparse condition: |E| ≪ |V|² — the number of edges is much less than V squared.
🌍 Real-Life Analogy

Imagine a country with 1000 cities. A dense graph = every city has a direct highway to every other city (500,000 roads!). A sparse graph = each city only connects to 3–4 nearby cities (≈3000 roads). Most real road networks are sparse!

C Code

Basic Graph Structure in C

sparse_graph.cC
#include <stdio.h>
#include <stdlib.h>

#define MAX_V 10  /* maximum vertices */
#define MAX_E 20  /* maximum edges */

/* Incidence Matrix: rows=vertices, cols=edges */
/* cell[v][e] = 1 if vertex v is incident to edge e, else 0 */
struct SparseGraph {
    int matrix[MAX_V][MAX_E]; /* incidence matrix */
    int numV;                  /* number of vertices */
    int numE;                  /* number of edges */
    char vertexLabel[MAX_V];   /* vertex names: A,B,C... */
};

void initGraph(struct SparseGraph *g) {
    g->numV = 0;
    g->numE = 0;
    for (int i = 0; i < MAX_V; i++)
        for (int j = 0; j < MAX_E; j++)
            g->matrix[i][j] = 0;  /* start empty */
}

int main() {
    struct SparseGraph g;
    initGraph(&g);
    printf("Sparse Graph initialized!\n");
    return 0;
}

Incidence Matrix Representation

Theory

What is an Incidence Matrix?

An incidence matrix is a 2D array where:

  • Rows represent Vertices
  • Columns represent Edges
  • Cell [v][e] = 1 if vertex v is endpoint of edge e
  • Cell [v][e] = 0 if not connected
  • For directed graphs: +1 (tail/source), -1 (head/destination)
📋 Real-Life Analogy

Think of a bus route schedule. Rows = bus stops (vertices), Columns = bus routes (edges). A cell is "1" if that bus stop is on that route. Most stops are only on a few routes — making it sparse!

Insert Vertex

Theory

How Insert Vertex Works

Adding a vertex means adding a new row to the incidence matrix. The new row starts with all zeros — the vertex has no edges yet.

1
Check if maximum vertex limit is reached.
2
Add a new row at position numV in the matrix.
3
Initialize all cells of that row to 0.
4
Store the vertex label and increment numV.
🏙️ Real-Life Analogy

Adding a new city to a map. The city exists but has no roads yet. We add a new row to our bus route table, all empty.

void insertVertex(struct SparseGraph *g, char label) {
    /* Step 1: Check overflow */
    if (g->numV >= MAX_V) {
        printf("Error: Max vertices reached!\n");
        return;
    }
    /* Step 2: Store label */
    g->vertexLabel[g->numV] = label;
    /* Step 3: Fill new row with 0s */
    for (int e = 0; e < MAX_E; e++)
        g->matrix[g->numV][e] = 0;
    /* Step 4: Increment count */
    g->numV++;
    printf("Vertex '%c' inserted. Total: %d\n", label, g->numV);
}

Delete Vertex

Theory

How Delete Vertex Works

Deleting a vertex means removing its row AND removing all edges that touch this vertex (removing those columns too).

1
Find the vertex by its label or index.
2
Identify all edges incident to this vertex (columns with value 1 in that row).
3
Delete those edge columns by shifting remaining columns left.
4
Delete the vertex row by shifting rows up. Decrement numV and numE.
C Code
delete_vertex.cC
void deleteVertex(struct SparseGraph *g, int vIdx) {
    if (vIdx < 0 || vIdx >= g->numV) {
        printf("Error: Invalid vertex!\n");
        return;
    }
    /* Find & delete edges of this vertex */
    int e = 0;
    while (e < g->numE) {
        if (g->matrix[vIdx][e] != 0) {
            /* Shift columns left to remove edge e */
            for (int v = 0; v < g->numV; v++)
                for (int j = e; j < g->numE - 1; j++)
                    g->matrix[v][j] = g->matrix[v][j+1];
            g->numE--;
        } else e++;
    }
    /* Shift rows up to remove vertex row */
    for (int v = vIdx; v < g->numV - 1; v++) {
        g->vertexLabel[v] = g->vertexLabel[v+1];
        for (int j = 0; j < g->numE; j++)
            g->matrix[v][j] = g->matrix[v+1][j];
    }
    g->numV--;
    printf("Vertex deleted. Remaining: %d\n", g->numV);
}
Use Case
🗺️ Real-Life Use Case

A city closes down an airport. Remove the airport vertex. All flight routes (edges) to/from that airport must also be removed from the route table.

Linear scan through the vertex labels array. Compare each label to the target. Return index if found, -1 if not.

Looking up a city by name in a phone book. Scan each entry until found.

08

Insert Edge

Theory

How Insert Edge Works

Adding an edge means adding a new column to the incidence matrix. Both endpoint vertices get a 1 in that column.

1
Check if max edges reached.
2
Verify both endpoints (u, v) exist in the graph.
3
Create new column at position numE.
4
Set matrix[u][numE] = 1 and matrix[v][numE] = 1.
5
Increment numE.
🛣️ Real-Life Analogy

Building a new highway between two cities. Add a new column to the route table. Mark both cities as "1" on that route.

Visualization
C Code
insert_edge.cC
void insertEdge(struct SparseGraph *g, int u, int v) {
    if (g->numE >= MAX_E) {
        printf("Max edges reached!\n");
        return;
    }
    if (u < 0 || v < 0 || u >= g->numV || v >= g->numV) {
        printf("Invalid vertex index!\n");
        return;
    }
    /* New column index = numE */
    g->matrix[u][g->numE] = 1;  /* mark vertex u */
    g->matrix[v][g->numE] = 1;  /* mark vertex v */
    g->numE++;
    printf("Edge %c-%c inserted. Total edges: %d\n",
            g->vertexLabel[u], g->vertexLabel[v], g->numE);
}
/* Time: O(1) | Space: O(1) extra */

Delete Edge

Theory

Remove the column corresponding to edge e. Shift all columns after it one position to the left. Decrement numE.

C Code
delete_edge.cC
void deleteEdge(struct SparseGraph *g, int eIdx) {
    if (eIdx < 0 || eIdx >= g->numE) {
        printf("Invalid edge index!\n");
        return;
    }
    /* Shift columns left from eIdx onwards */
    for (int v = 0; v < g->numV; v++)
        for (int j = eIdx; j < g->numE - 1; j++)
            g->matrix[v][j] = g->matrix[v][j+1];
    g->numE--;
    printf("Edge E%d deleted. Total edges: %d\n", eIdx, g->numE);
}
/* Time: O(V × E) — shift operation */
🚧 Real-Life Analogy

A road is permanently closed. Remove that route column from the schedule. All remaining route columns shift to fill the gap.

Search Edge

Theory

To check if edge (u,v) exists: scan all columns. An edge exists in column e if matrix[u][e]=1 AND matrix[v][e]=1.

C Code
search_edge.cC
int searchEdge(struct SparseGraph *g, int u, int v) {
    for (int e = 0; e < g->numE; e++) {
        /* Both endpoints must have 1 in this column */
        if (g->matrix[u][e] == 1 && g->matrix[v][e] == 1) {
            printf("Edge %c-%c found at E%d\n",
                    g->vertexLabel[u], g->vertexLabel[v], e);
            return e;  /* return edge index */
        }
    }
    printf("Edge %c-%c NOT found!\n", g->vertexLabel[u], g->vertexLabel[v]);
    return -1;
}
/* Time: O(E) */

Breadth First Search (BFS)

Theory

What is BFS?

BFS explores the graph level by level, starting from a source vertex. It uses a Queue (First-In-First-Out) to track vertices to visit next.

1
Queue Initialization: Create a queue, enqueue the starting vertex, mark it visited.
2
Visit Node: Dequeue a vertex, print/process it.
3
Mark Visited: Keep a visited[] array to avoid revisiting vertices.
4
Traverse Adjacent Nodes: Use incidence matrix to find neighbors — for each edge of current vertex, find the other endpoint and enqueue if unvisited.
🌊 Real-Life Analogy

Like ripples in water. Drop a stone at the start. All 1-hop neighbors discovered first, then 2-hop, etc. Used in GPS (shortest path in unweighted roads), social network friend suggestions.

Animated BFS
// Click Run BFS to start
C Code
bfs.cC
#define MAX_Q 100
int queue[MAX_Q], front = 0, rear = 0;

void enqueue(int v) { queue[rear++] = v; }
int  dequeue()      { return queue[front++]; }
int  isEmpty()      { return front == rear; }

/* Find neighbor of vertex v via edge e */
int getNeighbor(struct SparseGraph *g, int v, int e) {
    for (int u = 0; u < g->numV; u++)
        if (u != v && g->matrix[u][e] == 1)
            return u;
    return -1;
}

void BFS(struct SparseGraph *g, int start) {
    int visited[MAX_V] = {0};
    front = rear = 0;            /* reset queue */
    enqueue(start);
    visited[start] = 1;
    printf("BFS: ");
    while (!isEmpty()) {
        int v = dequeue();
        printf("%c ", g->vertexLabel[v]);
        /* Visit all unvisited neighbors via incidence matrix */
        for (int e = 0; e < g->numE; e++) {
            if (g->matrix[v][e] == 1) {
                int nb = getNeighbor(g, v, e);
                if (nb != -1 && !visited[nb]) {
                    visited[nb] = 1;
                    enqueue(nb);
                }
            }
        }
    }
    printf("\n");
}
/* Time: O(V + E) | Space: O(V) for visited + queue */

Depth First Search (DFS)

Theory

What is DFS?

DFS dives as deep as possible along one path before backtracking. Uses a Stack (or recursion).

Recursive DFS

Uses the call stack itself for backtracking. Elegant and simple.

Iterative (Stack) DFS

Explicit stack. Better for very deep graphs (avoids stack overflow).

🗺️ Real-Life Analogy

Exploring a maze. Go as far as you can in one direction. Hit a dead end? Backtrack to the last junction and try another path. Used in solving puzzles, topological sorting, detecting cycles.

Animated DFS
// Click Run DFS to start
C Code
dfs.cC
int visited_dfs[MAX_V];

/* Recursive DFS */
void DFS_Recursive(struct SparseGraph *g, int v) {
    visited_dfs[v] = 1;
    printf("%c ", g->vertexLabel[v]);
    for (int e = 0; e < g->numE; e++) {
        if (g->matrix[v][e] == 1) {
            int nb = getNeighbor(g, v, e);
            if (nb != -1 && !visited_dfs[nb])
                DFS_Recursive(g, nb); /* recurse deeper */
        }
    }
    /* Backtracking happens automatically here */
}

/* Stack-based Iterative DFS */
void DFS_Stack(struct SparseGraph *g, int start) {
    int stack[MAX_V], top = -1;
    int vis[MAX_V] = {0};
    stack[++top] = start;
    while (top >= 0) {
        int v = stack[top--];
        if (vis[v]) continue;
        vis[v] = 1;
        printf("%c ", g->vertexLabel[v]);
        for (int e = 0; e < g->numE; e++) {
            if (g->matrix[v][e] == 1) {
                int nb = getNeighbor(g, v, e);
                if (nb != -1 && !vis[nb])
                    stack[++top] = nb;
            }
        }
    }
}
/* Time: O(V + E) | Space: O(V) */

Sparse Analysis Operations

Theory

Key Analysis Operations

Edge Density: density = 2|E| / (|V| × (|V|-1)) for undirected graphs. Range: 0 (empty) to 1 (complete).

Sparse Condition: A graph is typically called sparse when density < 0.5 or |E| < |V| × log|V|.

Missing Edge Detection: Count total possible edges minus actual edges.

Graph Compression: Use compressed formats (like CSR — Compressed Sparse Row) to save memory.

📊 Real-Life Analogy

Internet routing tables. Billions of nodes but each router only connects to a handful of others. Extremely sparse — using dense representations would waste terabytes of memory.

Interactive Density Calculator
C Code
sparse_analysis.cC
/* Edge Density: how full is the graph? */
float edgeDensity(struct SparseGraph *g) {
    int maxE = g->numV * (g->numV - 1) / 2;
    if (maxE == 0) return 0.0;
    return (float) g->numE / maxE;
}

/* Check if sparse (density < 0.5) */
int isSparse(struct SparseGraph *g) {
    return edgeDensity(g) < 0.5;
}

/* Count missing edges */
int missingEdges(struct SparseGraph *g) {
    int maxE = g->numV * (g->numV - 1) / 2;
    return maxE - g->numE;
}

void sparseReport(struct SparseGraph *g) {
    printf("Vertices: %d\n", g->numV);
    printf("Edges: %d\n", g->numE);
    printf("Density: %.2f%%\n", edgeDensity(g)*100);
    printf("Sparse: %s\n", isSparse(g) ? "YES" : "NO");
    printf("Missing edges: %d\n", missingEdges(g));
}

Connectivity Operations

Theory

Connectivity Concepts

Connected Graph: Every vertex is reachable from every other vertex.

Connected Components: Groups of vertices where each group is internally connected but not connected to other groups.

Isolated Vertex: A vertex with degree 0 — no edges touch it.

Reachability: Can vertex u reach vertex v? (using BFS/DFS)

Weak Connectivity: In directed graphs — connected if we ignore edge directions.

🏝️ Real-Life Analogy

Islands and bridges. A connected graph = you can reach any island from any other. Connected components = separate island groups. Isolated vertex = an island with no bridges at all.

C Code
connectivity.cC
/* Check if the entire graph is connected using BFS */
int isConnected(struct SparseGraph *g) {
    int vis[MAX_V] = {0};
    /* BFS from vertex 0 */
    BFS_mark(g, 0, vis);
    for (int v = 0; v < g->numV; v++)
        if (!vis[v]) return 0; /* some vertex unreachable */
    return 1;
}

/* Count connected components */
int connectedComponents(struct SparseGraph *g) {
    int vis[MAX_V] = {0}, count = 0;
    for (int v = 0; v < g->numV; v++) {
        if (!vis[v]) {
            DFS_mark(g, v, vis); /* visit whole component */
            count++;
        }
    }
    printf("Components: %d\n", count);
    return count;
}

/* Detect isolated vertices (degree 0) */
void findIsolated(struct SparseGraph *g) {
    for (int v = 0; v < g->numV; v++) {
        int deg = 0;
        for (int e = 0; e < g->numE; e++)
            if (g->matrix[v][e] != 0) deg++;
        if (deg == 0)
            printf("Isolated vertex: %c\n", g->vertexLabel[v]);
    }
}

Path Operations

Theory

Path Operations Overview

Find Path: Find ANY path between two vertices (DFS-based).

Shortest Path: Find the path with minimum edges between two vertices (BFS-based, for unweighted graphs).

All Paths: Find all possible paths using backtracking DFS.

Reachability Check: Can u reach v? Run BFS/DFS from u, check if v is visited.

Path Length: Count edges in the path.

🗺️ Real-Life Analogy

Google Maps. "Find path" = any route. "Shortest path" = fewest turns. "All paths" = every possible route. Used in GPS, network routing, game AI pathfinding.

C Code
path_ops.cC
/* Shortest path (BFS — fewest edges) */
int shortestPath(struct SparseGraph *g, int src, int dst) {
    int dist[MAX_V], vis[MAX_V] = {0};
    int q[MAX_V], f = 0, r = 0;
    for (int i = 0; i < MAX_V; i++) dist[i] = -1;
    dist[src] = 0; vis[src] = 1; q[r++] = src;
    while (f < r) {
        int v = q[f++];
        if (v == dst) {
            printf("Shortest path %c→%c = %d edges\n",
                g->vertexLabel[src], g->vertexLabel[dst], dist[v]);
            return dist[v];
        }
        for (int e = 0; e < g->numE; e++) {
            if (g->matrix[v][e] == 1) {
                int nb = getNeighbor(g, v, e);
                if (nb != -1 && !vis[nb]) {
                    vis[nb] = 1;
                    dist[nb] = dist[v] + 1;
                    q[r++] = nb;
                }
            }
        }
    }
    printf("No path found!\n");
    return -1;
}

/* Find ALL paths using DFS + backtracking */
void allPaths(struct SparseGraph *g, int cur, int dst,
              int *path, int pLen, int *vis) {
    vis[cur] = 1; path[pLen++] = cur;
    if (cur == dst) {
        for (int i = 0; i < pLen; i++)
            printf("%c ", g->vertexLabel[path[i]]);
        printf("\n");
    } else {
        for (int e = 0; e < g->numE; e++)
            if (g->matrix[cur][e] == 1) {
                int nb = getNeighbor(g, cur, e);
                if (nb != -1 && !vis[nb])
                    allPaths(g, nb, dst, path, pLen, vis);
            }
    }
    vis[cur] = 0; /* backtrack */
}
/* Shortest Path Time: O(V + E) */

Cycle Operations

Theory

Cycle Operations

Detect Cycle: During DFS, if we visit a vertex that's already in the current path → cycle found.

Count Cycles: Back edges in DFS tree = number of independent cycles.

Check Tree Structure: A tree has exactly V-1 edges and no cycles. If |E| = |V|-1 AND no cycle → it's a tree.

Forest Validation: A forest = multiple disconnected trees. No cycles allowed.

🔄 Real-Life Analogy

Detecting circular dependencies in software packages. If package A depends on B, B depends on C, C depends on A → that's a cycle → deadlock!

C Code
cycle_ops.cC
int hasCycle_help(struct SparseGraph *g, int v, int par, int *vis) {
    vis[v] = 1;
    for (int e = 0; e < g->numE; e++) {
        if (g->matrix[v][e] == 1) {
            int nb = getNeighbor(g, v, e);
            if (nb == -1) continue;
            if (!vis[nb]) {
                if (hasCycle_help(g, nb, v, vis))
                    return 1;  /* cycle found deeper */
            } else if (nb != par) {
                return 1;  /* back edge = cycle! */
            }
        }
    }
    return 0;
}

int detectCycle(struct SparseGraph *g) {
    int vis[MAX_V] = {0};
    for (int v = 0; v < g->numV; v++)
        if (!vis[v] && hasCycle_help(g, v, -1, vis)) {
            printf("Cycle detected!\n");
            return 1;
        }
    printf("No cycle found.\n");
    return 0;
}

/* Check Tree: V-1 edges and connected and no cycle */
int isTree(struct SparseGraph *g) {
    return (g->numE == g->numV - 1)
           && isConnected(g)
           && !detectCycle(g);
}
/* Time: O(V + E) */

Degree Operations

Theory

Degree of a Vertex

The degree of a vertex = the number of edges incident to it = the count of non-zero values in its row in the incidence matrix.

  • Find Degree: Count 1s in vertex's row
  • Maximum Degree: Vertex with most edges
  • Minimum Degree: Vertex with fewest edges
  • Average Degree: 2|E| / |V| (each edge contributes to 2 vertices)
  • Degree Distribution: How many vertices have each degree value
✈️ Real-Life Analogy

Airport hub analysis. Degree = number of routes from that airport. Maximum degree = the busiest hub (e.g., Heathrow). Minimum degree = small regional airport with 1-2 routes.

Theory — Minimum Spanning Tree

Minimum Spanning Tree (MST)

An MST connects all vertices with minimum total edge weight and no cycles. For an unweighted sparse graph, the MST = BFS/DFS tree using V-1 edges.

Kruskal's Algorithm: Sort edges by weight. Add edges one by one using Union-Find (skip if cycle forms).

⚡ Real-Life Analogy

Connecting cities with minimum cable length. MST gives cheapest wiring. Used in network design, electricity grids, water pipelines.

/* Union-Find (Disjoint Set Union) */
int parent[MAX_V], rank_uf[MAX_V];

void initUF(int n) {
    for (int i = 0; i < n; i++) { parent[i]=i; rank_uf[i]=0; }
}

int find(int x) {
    if (parent[x] != x) parent[x] = find(parent[x]); /* path compression */
    return parent[x];
}

int unite(int a, int b) {
    int ra = find(a), rb = find(b);
    if (ra == rb) return 0; /* same component = cycle! */
    if (rank_uf[ra] < rank_uf[rb]) parent[ra] = rb;
    else if (rank_uf[ra] > rank_uf[rb]) parent[rb] = ra;
    else { parent[rb] = ra; rank_uf[ra]++; }
    return 1;
}

/* Kruskal's MST (unweighted: all weights=1) */
void kruskalMST(struct SparseGraph *g) {
    initUF(g->numV);
    int mstEdges = 0;
    printf("MST Edges:\n");
    for (int e = 0; e < g->numE; e++) {
        int u = -1, v = -1;
        for (int i = 0; i < g->numV; i++)
            if (g->matrix[i][e] == 1)
                { if (u == -1) u = i; else v = i; }
        if (u != -1 && v != -1 && unite(u, v)) {
            printf("  %c - %c\n", g->vertexLabel[u], g->vertexLabel[v]);
            mstEdges++;
            if (mstEdges == g->numV - 1) break;
        }
    }
    printf("MST complete: %d edges\n", mstEdges);
}
/* Time: O(E log E) for sorting + O(E α(V)) for union-find */
Graph Compression

Graph Compression — CSR Format

For large sparse graphs, the incidence matrix wastes memory. Compressed Sparse Row (CSR) stores only non-zero entries.

graph_compression.cC
/* CSR: Compressed Sparse Row format for adjacency */
struct CSR {
    int rowPtr[MAX_V+1]; /* start index of each vertex's edges */
    int colIdx[MAX_E*2]; /* neighbor vertex indices */
    int nnz;             /* number of non-zeros */
};

void convertToCSR(struct SparseGraph *g, struct CSR *csr) {
    csr->nnz = 0;
    for (int v = 0; v < g->numV; v++) {
        csr->rowPtr[v] = csr->nnz;
        for (int e = 0; e < g->numE; e++) {
            if (g->matrix[v][e] == 1) {
                int nb = getNeighbor(g, v, e);
                if (nb != -1) csr->colIdx[csr->nnz++] = nb;
            }
        }
    }
    csr->rowPtr[g->numV] = csr->nnz;
    printf("CSR uses %d ints vs %d (matrix)\n",
            csr->nnz, g->numV * g->numE);
}
/* Memory savings huge for sparse graphs! */
Efficient Traversal

Efficient Traversal for Sparse Graphs

For incidence matrix traversal, each vertex scans all E edges. For sparse graphs where E ≈ V, this is O(V) per vertex. Using adjacency list would be O(degree) per vertex — but with incidence matrix we can optimize by skipping zero columns early using a sorted edge list per vertex.

Key insight: Sparse + Incidence Matrix → iterate over edges (columns), not adjacency. For each edge column, extract both endpoints in O(V) time.

Putting It All Together

complete_sparse_graph.cC — Full Program
#include <stdio.h>
#include <stdlib.h>
#define MAX_V 10
#define MAX_E 20

struct SparseGraph {
    int  matrix[MAX_V][MAX_E];
    int  numV, numE;
    char vertexLabel[MAX_V];
};

int getNeighbor(struct SparseGraph *g,int v,int e){
    for(int u=0;u<g->numV;u++) if(u!=v&&g->matrix[u][e])return u;
    return -1;
}

int main() {
    struct SparseGraph g = {.numV=0,.numE=0};
    /* 1. Initialize */
    for(int i=0;i<MAX_V;i++) for(int j=0;j<MAX_E;j++) g.matrix[i][j]=0;

    /* 2. Add vertices A, B, C, D */
    char labels[] = {'A','B','C','D'};
    for(int i=0;i<4;i++) {
        g.vertexLabel[g.numV] = labels[i];
        g.numV++;
    }

    /* 3. Add edges: A-B(E0), B-C(E1), C-D(E2), A-D(E3) */
    int edges[][2] = {{0,1},{1,2},{2,3},{0,3}};
    for(int i=0;i<4;i++){
        g.matrix[edges[i][0]][g.numE] = 1;
        g.matrix[edges[i][1]][g.numE] = 1;
        g.numE++;
    }

    /* 4. Display matrix */
    printf("\nIncidence Matrix:\n   E0 E1 E2 E3\n");
    for(int v=0;v<g.numV;v++){
        printf("%c: ",g.vertexLabel[v]);
        for(int e=0;e<g.numE;e++) printf(" %d  ",g.matrix[v][e]);
        printf("\n");
    }

    /* 5. BFS Traversal */
    int vis[MAX_V]={0},q[MAX_V],f=0,r=0;
    q[r++]=0; vis[0]=1;
    printf("\nBFS: ");
    while(f<r){
        int v=q[f++];
        printf("%c ",g.vertexLabel[v]);
        for(int e=0;e<g.numE;e++){
            if(g.matrix[v][e]){
                int nb=getNeighbor(&g,v,e);
                if(nb!=-1&&!vis[nb]){vis[nb]=1;q[r++]=nb;}
            }
        }
    }
    printf("\n");

    /* 6. Density report */
    int maxE = g.numV*(g.numV-1)/2;
    printf("Density: %.1f%% - %s\n",
        100.0*g.numE/maxE, g.numE < maxE/2 ? "SPARSE" : "DENSE");
    return 0;
}
