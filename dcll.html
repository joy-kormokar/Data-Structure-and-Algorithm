Doubly Circular Linked List

A Doubly Circular Linked List (DCLL) is a data structure where each node has three parts: a prev pointer, a data field, and a next pointer. Unlike a regular list, the last node's next points back to the first node, and the first node's prev points to the last node — forming a perfect circle. You can traverse in both directions infinitely.

18Operation Groups
🧩 Node Structure & Memory Layout ▼
A DCLL node contains 3 fields:

1. prev — Pointer to the previous node (or tail if this is head)
2. data — The actual value stored
3. next — Pointer to the next node (or head if this is tail)

The list is managed by a single head pointer. If head is NULL, the list is empty. In a single-node list, both head->next and head->prev point to head itself.
🎡 Real-Life Analogy
Think of a merry-go-round (carousel) at a fair. Each seat is a node. You can move forward (next) or backward (prev) to reach any seat. The last seat connects back to the first — you never "fall off" — it's truly circular! A DCLL works exactly like this. 
/* Node structure for Doubly Circular Linked List */
struct Node {
    int data;               // stores the value
    struct Node* prev;     // points to previous node
    struct Node* next;     // points to next node
};

/* Global head pointer (NULL means empty list) */
struct Node* head = NULL;

/* Helper: create a new node */
struct Node* createNode(int val) {
    struct Node* newNode = (struct Node*) malloc(sizeof(struct Node));
    newNode->data = val;
    newNode->prev = newNode;  // points to itself (circular)
    newNode->next = newNode;  // points to itself (circular)
    return newNode;
}

Creation Operations
Section 01 · 2 subtopics
1.1 Create Node malloc ▼
Creating a node means allocating memory on the heap using malloc(). The new node's prev and next initially point to itself, satisfying the circular property even for a lone node.
💡 Analogy
Like buying a single train bogie. It has two couplers (prev/next) but both currently hook onto itself since there are no other bogies yet. 
#include <stdio.h>
#include <stdlib.h>

struct Node {
    int data;
    struct Node* prev;
    struct Node* next;
};

struct Node* createNode(int val) {
    struct Node* n = (struct Node*)malloc(sizeof(struct Node));
    if (n == NULL) {
        printf("Memory allocation failed!\n");
        exit(1);
    }
    n->data = val;
    n->prev = n;  // self-link
    n->next = n;  // self-link
    return n;
}

int main() {
    struct Node* node = createNode(42);
    printf("Created node with data = %d\n", node->data);
    printf("node->next points to itself: %s\n",
           node->next == node ? "YES" : "NO");
    return 0;
}
1.2 Initialize Circular List head = NULL ▼
Initialization simply means setting the head pointer to NULL. A NULL head signals an empty list. All operations check for this condition first. We can also wrap this in an initList() function.
head = NULL
struct Node* head = NULL;  // empty list

void initList() {
    head = NULL;
}

int isEmpty() {
    return (head == NULL);
}

int main() {
    initList();
    if (isEmpty())
        printf("List is empty.\n");
    return 0;
}

Insertion Operations
Section 02 · 3 subtopics
2.1 Insert at Beginning O(1) ▼
Steps:
1. Create a new node
2. If list is empty → node points to itself, head = node
3. Otherwise: newNode.next = head, newNode.prev = head.prev (tail), head.prev.next = newNode, head.prev = newNode
4. Update head = newNode
🎬 Analogy
Like jumping to the front of a circular queue. Everyone reshuffles: the person who was last now holds your hand behind, and the old front person is now behind you. 
void insertBeginning(int val) {
    struct Node* newNode = createNode(val);
    if (head == NULL) {
        head = newNode;  // first node, self-circular
        return;
    }
    struct Node* tail = head->prev;  // find tail
    newNode->next = head;
    newNode->prev = tail;
    tail->next   = newNode;
    head->prev   = newNode;
    head = newNode;  // update head
}
2.2 Insert at End O(1) ▼
Since head->prev always gives us the tail (O(1)), inserting at end is efficient:
1. tail = head->prev
2. newNode.prev = tail, newNode.next = head
3. tail.next = newNode, head.prev = newNode
(head does NOT change)
🚂 Analogy
Attaching a new bogie to the end of a circular train. The last bogie connects to the new one, and the new one connects back to the engine (head). 
void insertEnd(int val) {
    struct Node* newNode = createNode(val);
    if (head == NULL) {
        head = newNode;
        return;
    }
    struct Node* tail = head->prev;
    tail->next    = newNode;
    newNode->prev = tail;
    newNode->next = head;
    head->prev    = newNode;
    // head stays the same
}
2.3 Insert at Specific Position O(n) ▼
Traverse to position (pos-1), then link the new node between current and current.next. Position 1 = beginning. Position = n+1 = end. 
void insertAtPosition(int val, int pos) {
    if (pos == 1) { insertBeginning(val); return; }
    struct Node* cur = head;
    for (int i = 1; i < pos - 1; i++) {
        cur = cur->next;
        if (cur == head) {  // wrapped around
            printf("Position out of range\n");
            return;
        }
    }
    struct Node* newNode = createNode(val);
    struct Node* nextNode = cur->next;
    newNode->next  = nextNode;
    newNode->prev  = cur;
    cur->next      = newNode;
    nextNode->prev = newNode;
}

Deletion Operations
Section 03 · 4 subtopics
3.1 Delete from Beginning O(1) ▼
1. If list has only one node → free it, head = NULL
2. Otherwise: tail = head->prev, newHead = head->next, update links, free old head
3. Important: tail.next must now point to newHead, and newHead.prev must point to tail 
void deleteBeginning() {
    if (head == NULL) { printf("Empty list!\n"); return; }
    if (head->next == head) {   // single node
        free(head);
        head = NULL;
        return;
    }
    struct Node* tail    = head->prev;
    struct Node* newHead = head->next;
    tail->next    = newHead;
    newHead->prev = tail;
    free(head);
    head = newHead;
}
3.2 Delete from End O(1) ▼
Since tail = head->prev (O(1)), we get the tail directly. We find newTail = tail->prev, fix links, free tail. 
void deleteEnd() {
    if (head == NULL) { printf("Empty list!\n"); return; }
    if (head->next == head) {
        free(head); head = NULL; return;
    }
    struct Node* tail    = head->prev;
    struct Node* newTail = tail->prev;
    newTail->next = head;
    head->prev    = newTail;
    free(tail);
}
3.3 Delete at Specific Position O(n) ▼ 
void deleteAtPosition(int pos) {
    if (head == NULL) return;
    if (pos == 1) { deleteBeginning(); return; }
    struct Node* cur = head;
    for (int i = 1; i < pos; i++) {
        cur = cur->next;
        if (cur == head) { printf("Position out of range\n"); return; }
    }
    cur->prev->next = cur->next;
    cur->next->prev = cur->prev;
    free(cur);
}
3.4 Delete by Value O(n) ▼
Traverse the circular list until you find a node with the matching value (or return to head without finding it). Then unlink that node. 
void deleteByValue(int val) {
    if (head == NULL) return;
    struct Node* cur = head;
    do {
        if (cur->data == val) {
            if (cur == head) { deleteBeginning(); return; }
            cur->prev->next = cur->next;
            cur->next->prev = cur->prev;
            free(cur);
            return;
        }
        cur = cur->next;
    } while (cur != head);
    printf("Value %d not found!\n", val);
}

Traversal Operations
Section 04 · 3 subtopics
4.1 Forward Traversal head → tail ▼
Start at head, keep moving cur = cur->next until we return to head. Use a do...while loop because we must visit head at least once. 
void forwardTraversal() {
    if (head == NULL) { printf("Empty list!\n"); return; }
    struct Node* cur = head;
    printf("Forward: ");
    do {
        printf("%d ", cur->data);
        cur = cur->next;
    } while (cur != head);  // stop when back at head
    printf("\n");
}
4.2 Backward Traversal tail → head ▼
Start at head->prev (the tail), keep moving cur = cur->prev until we return to tail. This is a unique advantage of DCLL — no need to reverse the list! 
void backwardTraversal() {
    if (head == NULL) return;
    struct Node* tail = head->prev;  // start from tail
    struct Node* cur  = tail;
    printf("Backward: ");
    do {
        printf("%d ", cur->data);
        cur = cur->prev;
    } while (cur != tail);
    printf("\n");
}
4.3 Circular Traversal (n rounds) infinite loop guard ▼
Traverse the list exactly n full rounds. Use a counter: total nodes to visit = n × list_size. Useful in round-robin scheduling.
🎯 Use Case
CPU round-robin scheduling — each process gets a turn, then cycles back. The OS traverses the process list circularly with a fixed time quantum. 
void circularTraversal(int rounds) {
    if (head == NULL) return;
    struct Node* cur = head;
    int count = 0;
    int n = countNodes();   // get total nodes
    int total = rounds * n;
    printf("Circular (%d rounds): ", rounds);
    while (count < total) {
        printf("%d ", cur->data);
        cur = cur->next;
        count++;
    }
    printf("\n");
}

Searching Operations
Section 05 · 2 subtopics
5.1 Search by Value O(n) ▼
Linear search: traverse from head, compare each node's data. Return position if found, -1 if not found (after returning to head). 
int searchByValue(int val) {
    if (head == NULL) return -1;
    struct Node* cur = head;
    int pos = 1;
    do {
        if (cur->data == val) return pos;
        cur = cur->next;
        pos++;
    } while (cur != head);
    return -1;  // not found
}
5.2 Search by Position O(n) ▼ 
int searchByPosition(int pos) {
    if (head == NULL || pos < 1) return -1;
    struct Node* cur = head;
    for (int i = 1; i < pos; i++) {
        cur = cur->next;
        if (cur == head) return -1;  // out of range
    }
    return cur->data;
}

Update Operations
Section 06 · 1 subtopic
6.1 Update Node Data O(n) ▼
Traverse to the target position (or find by value), then simply assign cur->data = newVal. No pointer changes needed — just a data overwrite. 
void updateNode(int pos, int newVal) {
    if (head == NULL) { printf("Empty!\n"); return; }
    struct Node* cur = head;
    for (int i = 1; i < pos; i++) {
        cur = cur->next;
        if (cur == head) { printf("Position out of range\n"); return; }
    }
    int old = cur->data;
    cur->data = newVal;
    printf("Updated position %d: %d -> %d\n", pos, old, newVal);
}

Counting Operations
Section 07 · 1 subtopic
7.1 Count Total Nodes O(n) ▼
Use do...while to traverse the circular list once. Increment a counter for each node visited. Stop when we return to head. 
int countNodes() {
    if (head == NULL) return 0;
    int count = 0;
    struct Node* cur = head;
    do {
        count++;
        cur = cur->next;
    } while (cur != head);
    return count;
}

int main() {
    insertEnd(10); insertEnd(20); insertEnd(30);
    printf("Total nodes: %d\n", countNodes());  // Output: 3
    return 0;
}

Sorting Operations
Section 08 · 2 subtopics
8.1 Ascending Sort (Bubble Sort) O(n²) ▼
Bubble sort on a DCLL: swap only the data values (not pointers) between adjacent nodes. Repeat until no swaps in a full pass. The circular structure requires tracking when we've done a full loop. 
void sortAscending() {
    if (head == NULL || head->next == head) return;
    int swapped;
    struct Node* cur;
    struct Node* tail = head->prev;
    do {
        swapped = 0;
        cur = head;
        while (cur->next != head) {
            if (cur->data > cur->next->data) {
                /* swap data only */
                int tmp      = cur->data;
                cur->data    = cur->next->data;
                cur->next->data = tmp;
                swapped = 1;
            }
            cur = cur->next;
        }
    } while (swapped);
}
8.2 Descending Sort O(n²) ▼
Same as ascending but change the comparison: swap when cur->data < cur->next->data. 
void sortDescending() {
    if (head == NULL || head->next == head) return;
    int swapped;
    struct Node* cur;
    do {
        swapped = 0;
        cur = head;
        while (cur->next != head) {
            if (cur->data < cur->next->data) {  // reversed
                int tmp         = cur->data;
                cur->data       = cur->next->data;
                cur->next->data = tmp;
                swapped = 1;
            }
            cur = cur->next;
        }
    } while (swapped);
}

Reversing Operations
Section 09 · 1 subtopic
9.1 Reverse Entire Circular List O(n) ▼
For every node in the list, swap its prev and next pointers. Then update head = head->next (which was the old tail before the swap).
🔄 Analogy
Like reversing the direction of a circular train track. Every coupling that was "forward" becomes "backward". The new engine (head) is the old caboose (tail). 
void reverseList() {
    if (head == NULL || head->next == head) return;
    struct Node* cur  = head;
    struct Node* temp = NULL;
    do {
        /* swap prev and next for current node */
        temp      = cur->prev;
        cur->prev = cur->next;
        cur->next = temp;
        cur = cur->prev;  // move forward (was next)
    } while (cur != head);
    /* head->next is now old head->prev = old tail */
    head = head->next;
}

Rotation Operations
Section 10 · 3 subtopics
10.1 / 10.2 / 10.3 Clockwise / Anti-clockwise / Rotate by k O(k) ▼
Rotation in a DCLL is just moving the head pointer!

• Clockwise (right): head = head->prev (head moves backward by 1, so the list appears shifted right)
• Anti-clockwise (left): head = head->next (head moves forward by 1)
• Rotate by k: repeat k times. No actual node reordering needed — just pointer update!
🎠 Analogy
Like rotating a circular conveyor belt. The belt doesn't change shape — you just change which item is considered "front" by moving the reference point. 
/* Clockwise: last element becomes first */
void rotateClockwise() {
    if (head == NULL) return;
    head = head->prev;  // tail becomes new head
}

/* Anti-clockwise: first element goes to end */
void rotateAntiClockwise() {
    if (head == NULL) return;
    head = head->next;  // second becomes new head
}

/* Rotate by k positions (clockwise) */
void rotateByK(int k) {
    if (head == NULL) return;
    int n = countNodes();
    k = k % n;  // avoid full rotations
    for (int i = 0; i < k; i++)
        head = head->prev;
}

Merging Operations
Section 11 · 2 subtopics
11.1 Merge Two Circular Lists O(1) ▼
Join two DCLLs into one. Steps:
1. tail1 = head1->prev, tail2 = head2->prev
2. tail1->next = head2, head2->prev = tail1
3. tail2->next = head1, head1->prev = tail2
Result: one big circular list with head = head1 
struct Node* mergeTwoLists(struct Node* h1, struct Node* h2) {
    if (h1 == NULL) return h2;
    if (h2 == NULL) return h1;
    struct Node* tail1 = h1->prev;
    struct Node* tail2 = h2->prev;
    tail1->next = h2;    // connect end of list1 to start of list2
    h2->prev    = tail1;
    tail2->next = h1;    // connect end of list2 to start of list1
    h1->prev    = tail2;
    return h1;           // head1 is the new head
}
11.2 Merge Sorted Circular Lists O(n+m) ▼
Both lists are sorted. We need to produce one merged sorted DCLL. Approach: convert both to linear (break circular link temporarily), merge like merge-sort, then restore circular links. 
/* Helper: break circular link temporarily */
void breakCircular(struct Node* h) {
    if (h) { h->prev->next = NULL; h->prev = NULL; }
}

struct Node* mergeSortedLists(struct Node* h1, struct Node* h2) {
    breakCircular(h1); breakCircular(h2);
    struct Node* result = NULL;
    struct Node* tail   = NULL;
    while (h1 && h2) {
        struct Node* pick;
        if (h1->data <= h2->data) { pick = h1; h1 = h1->next; }
        else                       { pick = h2; h2 = h2->next; }
        pick->next = pick->prev = pick;
        if (!result) { result = pick; tail = pick; }
        else {
            tail->next   = pick; pick->prev = tail;
            pick->next   = result; result->prev = pick;
            tail = pick;
        }
    }
    /* append remaining nodes */
    struct Node* rest = h1 ? h1 : h2;
    while (rest) {
        struct Node* nxt = rest->next;
        rest->next = rest->prev = rest;
        tail->next = rest; rest->prev = tail;
        rest->next = result; result->prev = rest;
        tail = rest; rest = nxt;
    }
    return result;
}

Concatenation Operations
Section 12 · 1 subtopic
12.1 Concatenate Two Circular Lists O(1) ▼
Concatenation is identical to merging two unsorted lists — connect tail1 to head2 and tail2 to head1. The difference from "merge" is conceptual: concat just appends list2 at end of list1 without sorting.
🚂 Analogy
Coupling two separate circular train loops into one big loop. Disconnect one link on each loop, join them, and you have a single large circular track. 
struct Node* concatenate(struct Node* h1, struct Node* h2) {
    if (!h1) return h2;
    if (!h2) return h1;
    struct Node* tail1 = h1->prev;
    struct Node* tail2 = h2->prev;
    /* Break h1's circle and bridge to h2 */
    tail1->next = h2;
    h2->prev    = tail1;
    /* Close circle back to h1 */
    tail2->next = h1;
    h1->prev    = tail2;
    return h1;
}

Splitting Operations
Section 13 · 3 subtopics
13.1 Split into Two Circular Lists O(n) ▼
Split at a given node (splitNode). After split: list1 = head to splitNode, list2 = splitNode->next to tail. Fix circular links for both. 
void splitAtNode(struct Node* splitNode,
               struct Node** h1, struct Node** h2) {
    *h1 = head;
    *h2 = splitNode->next;
    struct Node* tail = head->prev;
    /* Close list1 circle */
    splitNode->next = *h1;
    (*h1)->prev     = splitNode;
    /* Close list2 circle */
    tail->next    = *h2;
    (*h2)->prev   = tail;
}
13.2 Split at Middle O(n) ▼
Use the slow/fast pointer technique: slow moves 1 step, fast moves 2 steps. When fast reaches the tail, slow is at the midpoint. Then split there. 
void splitAtMiddle(struct Node** h1, struct Node** h2) {
    if (!head || head->next == head) { *h1 = head; *h2 = NULL; return; }
    struct Node* slow = head;
    struct Node* fast = head;
    while (fast->next != head && fast->next->next != head) {
        slow = slow->next;
        fast = fast->next->next;
    }
    splitAtNode(slow, h1, h2);
}
13.3 Split at Position O(n) ▼ 
void splitAtPosition(int pos, struct Node** h1, struct Node** h2) {
    struct Node* cur = head;
    for (int i = 1; i < pos; i++) {
        cur = cur->next;
        if (cur == head) { printf("Out of range\n"); return; }
    }
    splitAtNode(cur, h1, h2);
}

Copying Operations
Section 14 · 2 subtopics
14.1 Shallow Copy pointer copy ▼
A shallow copy just copies the head pointer. Both the original and copy point to the same nodes in memory. Modifying one affects the other. Useful only when you need a second reference. 
/* Shallow copy — both heads point to SAME nodes */
struct Node* shallowCopy() {
    return head;  // just returns the same pointer!
}
/* WARNING: changes via copy affect the original too */
14.2 Deep Copy O(n) ▼
A deep copy creates brand new nodes with the same data and links them identically. The copy is completely independent of the original. 
struct Node* deepCopy() {
    if (!head) return NULL;
    struct Node* newHead = NULL;
    struct Node* cur = head;
    do {
        struct Node* copy = createNode(cur->data);
        if (!newHead) {
            newHead = copy;
        } else {
            /* attach to end of new list */
            struct Node* tail = newHead->prev;
            tail->next   = copy;
            copy->prev   = tail;
            copy->next   = newHead;
            newHead->prev = copy;
        }
        cur = cur->next;
    } while (cur != head);
    return newHead;  // independent copy
}

Loop / Cycle Operations
Section 15 · 3 subtopics
15.1 Detect Circular Nature verify design ▼
In a proper DCLL, tail->next == head always. We verify this by traversing to the tail and checking the link. Also check head->prev == tail. 
int detectCircular() {
    if (!head) return 1;  // empty = trivially circular
    struct Node* cur = head;
    while (cur->next != head) {
        cur = cur->next;
        if (cur == NULL) return 0;  // broken link!
    }
    /* cur is now the tail */
    return (cur->next == head && head->prev == cur);
}
15.2 Verify Proper Circular Link integrity check ▼
A deeper verification: for every node, check that node->next->prev == node. This confirms bidirectional integrity across the whole list. 
int verifyLinks() {
    if (!head) return 1;
    struct Node* cur = head;
    do {
        if (cur->next->prev != cur) {
            printf("Link broken at node %d!\n", cur->data);
            return 0;
        }
        cur = cur->next;
    } while (cur != head);
    printf("All links are intact.\n");
    return 1;
}
15.3 Remove Circular Connection linearize ▼
Sometimes we want to temporarily convert a DCLL to a regular doubly linked list (linear). We break the circular link by setting tail->next = NULL and head->prev = NULL. 
void removeCircularConnection() {
    if (!head) return;
    struct Node* tail = head->prev;
    tail->next  = NULL;  // break forward circular link
    head->prev  = NULL;  // break backward circular link
    /* Now it's a regular doubly linked list */
}

void restoreCircularConnection(struct Node* tail) {
    tail->next  = head;
    head->prev  = tail;
}

Duplicate Handling
Section 16 · 1 subtopic
16.1 Remove Duplicate Nodes O(n²) ▼
For each node, traverse the rest of the list and delete any node with the same data value. Uses two pointers: outer (current) and inner (checker).
📚 Use Case
Like deduplicating a playlist — if the same song appears twice, remove the second occurrence, keeping the first. Useful in browser history or music queues. 
void removeDuplicates() {
    if (!head || head->next == head) return;
    struct Node* outer = head;
    do {
        struct Node* inner = outer->next;
        while (inner != head) {
            struct Node* nextInner = inner->next;
            if (inner->data == outer->data) {
                inner->prev->next = inner->next;
                inner->next->prev = inner->prev;
                free(inner);
            }
            inner = nextInner;
        }
        outer = outer->next;
    } while (outer != head);
}

Special Operations
Section 17 · 5 subtopics
17.1 Find Middle Node slow/fast pointer ▼
Use slow and fast pointer technique (Floyd's): slow moves 1 step, fast moves 2 steps. When fast reaches the tail, slow is at the middle. 
struct Node* findMiddle() {
    if (!head) return NULL;
    struct Node* slow = head;
    struct Node* fast = head;
    while (fast->next != head && fast->next->next != head) {
        slow = slow->next;
        fast = fast->next->next;
    }
    printf("Middle node: %d\n", slow->data);
    return slow;
}
17.2 Find Intersection O(n+m) ▼
Two lists may share nodes (same memory address). Convert both to linear, find length difference, advance longer list, then walk both until pointers meet. 
struct Node* findIntersection(struct Node* h1, struct Node* h2) {
    /* linearize both lists first */
    struct Node* t1 = h1->prev; t1->next = NULL;
    struct Node* t2 = h2->prev; t2->next = NULL;
    int len1 = 0, len2 = 0;
    struct Node* c = h1; while(c) { len1++; c=c->next; }
    c = h2; while(c) { len2++; c=c->next; }
    struct Node* p1 = h1, *p2 = h2;
    while (len1 > len2) { p1 = p1->next; len1--; }
    while (len2 > len1) { p2 = p2->next; len2--; }
    while (p1 != p2) { p1 = p1->next; p2 = p2->next; }
    /* restore circular links */
    t1->next = h1; t2->next = h2;
    return p1;  // intersection node (or NULL)
}
17.3 Swap Nodes data swap ▼
The simplest approach: swap only the data of the two nodes at given positions. This avoids the complex pointer re-linking. 
void swapNodes(int pos1, int pos2) {
    int val1 = searchByPosition(pos1);
    int val2 = searchByPosition(pos2);
    if (val1 == -1 || val2 == -1) {
        printf("Invalid positions!\n"); return;
    }
    updateNode(pos1, val2);
    updateNode(pos2, val1);
    printf("Swapped positions %d and %d\n", pos1, pos2);
}
17.4 Palindrome Check bidirectional ▼
A DCLL's bidirectional nature makes palindrome checking elegant: use two pointers — front starting at head and back starting at tail. Compare as they move toward each other.
🪞 Analogy
Like reading "RACECAR" from both ends simultaneously. If both readers always see the same letter, it's a palindrome. 
int isPalindrome() {
    if (!head) return 1;
    struct Node* front = head;
    struct Node* back  = head->prev;  // tail
    int n = countNodes();
    for (int i = 0; i < n / 2; i++) {
        if (front->data != back->data) return 0;
        front = front->next;
        back  = back->prev;
    }
    return 1;  // it's a palindrome!
}
17.5 Josephus Problem Simulation classic algorithm ▼
The Josephus Problem: n people sit in a circle. Every k-th person is eliminated. DCLL is the perfect data structure for this — we can traverse circularly and delete nodes as people are eliminated. The last remaining node is the survivor.
⚔️ Historical Context
Based on a story by Flavius Josephus (1st century). 41 Jewish soldiers in a cave agreed to die in a circle rather than surrender. Josephus used math to find the safe seat. DCLL simulates this perfectly! 
int josephus(int n, int k) {
    /* Build circular list 1..n */
    struct Node* h = NULL;
    for (int i = 1; i <= n; i++) {
        if (!h) { h = createNode(i); }
        else {
            struct Node* tail = h->prev;
            struct Node* newN = createNode(i);
            tail->next = newN; newN->prev = tail;
            newN->next = h;   h->prev    = newN;
        }
    }
    struct Node* cur = h;
    while (cur->next != cur) {
        for (int i = 1; i < k; i++) cur = cur->next;
        printf("Eliminated: %d\n", cur->data);
        struct Node* toDelete = cur;
        cur->prev->next = cur->next;
        cur->next->prev = cur->prev;
        cur = cur->next;
        free(toDelete);
    }
    int survivor = cur->data;
    free(cur);
    return survivor;
}

int main() {
    printf("Survivor: %d\n", josephus(7, 3));  // Output: 4
}

Memory Management Operations
Section 18 · 2 subtopics
18.1 Destroy Circular List free all nodes ▼
Free every node's memory and set head = NULL. Must be careful not to follow pointers after freeing. Break the circular link first to simplify traversal.
🗑️ Analogy
Like demolishing every seat on a merry-go-round one by one, until there's nothing left. After demolition, the "head" reference is set to NULL so you don't accidentally use freed memory (dangling pointer). 
void destroyList() {
    if (!head) return;
    struct Node* cur  = head;
    struct Node* tail = head->prev;
    tail->next = NULL;  // break circular link
    while (cur != NULL) {
        struct Node* temp = cur;
        cur = cur->next;
        free(temp);  // release memory
    }
    head = NULL;  // avoid dangling pointer
    printf("List destroyed successfully.\n");
}
18.2 Clear Entire List same as destroy ▼
Clear is functionally identical to destroy for a DCLL — all nodes are freed and head is set to NULL. In some implementations, clear() frees data but keeps the head node alive; destroy() frees everything including the list object itself. 
/* clearList: free all nodes but keep head pointer variable */
void clearList() {
    destroyList();  // same operation for basic DCLL
    /* head is now NULL, ready to be reused */
}

/* Complete program with cleanup */
int main() {
    insertEnd(10);
    insertEnd(20);
    insertEnd(30);
    forwardTraversal();   // 10 20 30
    clearList();
    printf("After clear: isEmpty = %d\n", isEmpty()); // 1
    return 0;
}
