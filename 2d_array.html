SECTION 01.1Create / Initialize
// The very first step: how to declare and fill a 2D array in C

📖 What is a 2D Array?

THEORY
CONCEPT
A 2D array is a table of values arranged in rows and columns. Think of it as an array of arrays.

In memory, C stores all values row by row (row-major order) in a single contiguous block.

Syntax: datatype name[rows][cols];

  • Rows → horizontal direction (↕)
  • Columns → vertical direction (↔)
  • Index starts at 0, ends at n-1
ANALOGY
🪑

Imagine a movie theatre. Every seat is identified by its row number and seat number.

Row 2, Seat 5 → arr[2][5]

All seats are arranged in a perfect grid — that's your 2D array!

// Insert row at position pos
// Shift rows down first, then insert

void insertRow(int arr[][4], int *rows,
               int pos, int newRow[]) {
    int i, j;
    // Shift rows downward
    for(i = *rows - 1; i >= pos; i--)
        for(j = 0; j < 4; j++)
            arr[i+1][j] = arr[i][j];

    // Insert the new row
    for(j = 0; j < 4; j++)
        arr[pos][j] = newRow[j];

    (*rows)++; // increase row count
}

// Usage:
int newRow[] = {100,200,300,400};
insertRow(arr, &rows, 0, newRow); // at start
insertRow(arr, &rows, 1, newRow); // at middle
insertRow(arr, &rows, rows, newRow); // at end
🪑

Like adding a new row of seats to a theatre. To add it in the middle, everyone behind that row must move back one row. Shifting happens from the bottom up to avoid data loss.

void insertCol(int arr[][5], int rows,
               int *cols, int pos,
               int newCol[]) {
    int i, j;
    // Shift columns rightward
    for(i = 0; i < rows; i++)
        for(j = *cols - 1; j >= pos; j--)

}

SECTION 02.3Element Insertion
// Insert specific elements at given positions within the matrix

➕ Insert Specific / Multiple Elements

CODE + THEORY
THEORY
In a 2D array, element insertion means placing a value at a specific [row][col] position. Unlike 1D arrays, you choose the exact cell.

Two common patterns:
  • Insert Specific Element: Set arr[r][c] = value
  • Insert Multiple: Loop over a list of (r,c,value) tuples and assign each

⚠️ No shifting needed — you're directly writing to an exact coordinate.
C CODE
// Insert specific element
arr[1][3] = 42;

// Insert multiple elements
int positions[][3] = {
    {0,0,5},  // row,col,value
    {1,2,8},
    {2,3,15}
};
int n = 3;
for(int k=0; k<n; k++) {
    int r = positions[k][0];
    int c = positions[k][1];
    int v = positions[k][2];
    arr[r][c] = v;
}
📬

Like addressing a letter to seat A-5 in a stadium. You know exactly which row (A) and which column (5) — no need to move anyone else.

// Delete row at position pos
void deleteRow(int arr[][4],
               int *rows, int pos) {
    int i, j;
    // Shift rows upward
    for(i = pos; i < *rows - 1; i++)

SECTION 03.2Column Deletion
// Remove a column by shifting remaining columns leftward

🗑️ Delete Column

CODE + THEORY
THEORY
Deleting a column means shifting all columns to the right of the deleted position leftward by one, for every row.

Steps:
  • For each row i
  • For each col j from pos to cols-2
  • arr[i][j] = arr[i][j+1]
  • Decrease column count
C CODE
void deleteCol(int arr[][5], int rows,
               int *cols, int pos) {
    int i, j;
    for(i = 0; i < rows; i++)
        for(j = pos; j < *cols-1; j++)
            arr[i][j] = arr[i][j+1];
    (*cols)--;
}

// Delete first column
deleteCol(arr, rows, &cols, 0);
// Delete middle column
deleteCol(arr, rows, &cols, 2);
// Delete last column
deleteCol(arr, rows, &cols, cols-1);
📊

In Excel, when you delete a column, all columns to its right automatically shift left. Same idea — but we do it manually in C using a loop.

SECTION 03.3Element Deletion
// Delete a specific element or reset its value to zero

🗑️ Delete Specific Element / Reset

CODE
THEORY
In a fixed-size 2D array, you can't truly "remove" a single element (the array stays the same size). Instead:

  • Reset to 0 — Mark the cell as empty using a sentinel value like 0 or -1
  • Delete Specific — Search for the value and reset it

This is common in sparse matrix representation and game boards where a cell gets cleared.
C CODE
// Reset element at [r][c] to 0
arr[1][2] = 0;

// Delete specific value (search + reset)
void deleteElement(int arr[][4],
    int rows, int cols, int val) {
    for(int i=0; i<rows; i++)
        for(int j=0; j<cols; j++)
            if(arr[i][j] == val)
                arr[i][j] = 0; // reset
}

// Reset all elements to 0
memset(arr, 0, sizeof(arr));
🎮

In a Minesweeper game, when you clear a cell it doesn't disappear — it just becomes blank (value = 0). The grid size stays the same.

// Linear search — single value
void searchSingle(int arr[][4],
    int rows, int cols, int target) {
    for(int i=0; i<rows; i++) {
        for(int j=0; j<cols; j++) {
            if(arr[i][j] == target) {
                printf("Found at [%d][%d]\n",i,j);
                return; // stop at first
            }
        }
    }
    printf("Not Found\n");
}

// Search ALL occurrences
void searchAll(int arr[][4],
    int rows, int cols, int target) {
    int found = 0;
    for(int i=0; i<rows; i++)
        for(int j=0; j<cols; j++)
            if(arr[i][j]==target) {
                printf("[%d][%d] ",i,j);
                found++;
            }
    printf("\nTotal: %d\n", found);
}
🔍

Like searching for a specific word in a book by reading every word line by line. You start from the top-left and go row by row — simple but works on any matrix. Time: O(rows × cols).

SECTION 04.2Specialized Search
// Efficient searching in sorted matrices

⚡ Search in Sorted / Row-wise / Column-wise Matrix

CODE + THEORY
THEORY — Staircase Search
If every row is sorted and every column is sorted, we can search in O(rows + cols) instead of O(rows × cols).

Algorithm (Start at top-right corner):
  • If current == target → Found!
  • If current > target → move left (col--)
  • If current < target → move down (row++)
Row-wise Sorted Search (Binary Search per Row)
// Binary search in a sorted row
int binarySearch(int arr[], int n, int t) {
    int lo=0, hi=n-1;
    while(lo <= hi) {
        int mid=(lo+hi)/2;
        if(arr[mid]==t) return mid;
        else if(arr[mid]<t) lo=mid+1;
        else hi=mid-1;
    }
    return -1;
}
Staircase Search (Row+Col Sorted)
// Start at top-right corner
void staircaseSearch(int arr[][4],
    int rows, int cols, int target) {
    int r=0, c=cols-1;
    while(r < rows && c >= 0) {
        if(arr[r][c] == target) {
            printf("Found [%d][%d]\n",r,c);
            return;
        }
        if(arr[r][c] > target) c--;
        else r++;
    }
    printf("Not found\n");
}
🧭

Imagine a city grid where block numbers increase going right and going down. If you're looking for block 50 and you're at block 80, go left. If you're at block 30, go down. You never need to backtrack!

SECTION 05.3Pattern Traversal
// Spiral, Zigzag, Boundary, Wave, Diagonal

🌀 Spiral Traversal

INTERACTIVE
VISUALIZATION
Watch the spiral order
C CODE — Spiral
void spiralPrint(int arr[][4],
                 int rows, int cols) {
    int top=0, bot=rows-1;
    int left=0, right=cols-1;
    while(top <= bot && left <= right) {
        for(int c=left;c<=right;c++)
            printf("%d ",arr[top][c]);
        top++;
        for(int r=top;r<=bot;r++)
            printf("%d ",arr[r][right]);
        right--;
        if(top <= bot) {
            for(int c=right;c>=left;c--)
                printf("%d ",arr[bot][c]);
            bot--;
        }
        if(left <= right) {
            for(int r=bot;r>=top;r--)
                printf("%d ",arr[r][left]);
            left++;
        }
    }
}

↕️ Zigzag / Boundary / Wave / Diagonal Traversal

CODE
Zigzag Traversal
for(int i=0; i<rows; i++) {
    if(i%2==0) // even row: L→R
        for(int j=0;j<cols;j++)
            printf("%d ",arr[i][j]);
    else // odd row: R→L
        for(int j=cols-1;j>=0;j--)
            printf("%d ",arr[i][j]);
}
Boundary Traversal
// Top row, Right col, Bottom row, Left col
for(int j=0;j<cols;j++) printf("%d ",arr[0][j]);
for(int i=1;i<rows;i++) printf("%d ",arr[i][cols-1]);
for(int j=cols-2;j>=0;j--) printf("%d ",arr[rows-1][j]);
for(int i=rows-2;i>0;i--) printf("%d ",arr[i][0]);
Wave Traversal (col alternates)
for(int j=0;j<cols;j++) {
    if(j%2==0)
        for(int i=0;i<rows;i++)
            printf("%d ",arr[i][j]);
    else
        for(int i=rows-1;i>=0;i--)
            printf("%d ",arr[i][j]);
}
Diagonal Traversal (main diag)
// Main diagonal (top-left → bottom-right)
for(int i=0; i<rows; i++)
    printf("%d ", arr[i][i]);

// Anti-diagonal (top-right → bottom-left)
for(int i=0; i<rows; i++)
    printf("%d ", arr[i][cols-1-i]);
🐍

Spiral = Peeling an onion layer by layer. Zigzag = A snake slithering. Boundary = Walking around the perimeter of a room. Diagonal = A chess bishop's path.

SECTION 06.2Reshaping
// Flatten to 1D, Convert 1D to 2D, Resize Matrix

📐 Flatten / Unflatten / Resize

CODE + THEORY
Flatten 2D → 1D
// 3×4 matrix → 12-element array
int flat[12];
int k = 0;
for(int i=0; i<3; i++)
    for(int j=0; j<4; j++)
        flat[k++] = arr[i][j];

// Index formula: k = i*cols + j
Convert 1D → 2D
// 12-element array → 3×4 matrix
int k = 0;
for(int i=0; i<3; i++)
    for(int j=0; j<4; j++)
        arr[i][j] = flat[k++];

// Row = k/cols, Col = k%cols
Resize Matrix (copy to larger)
// Copy 3×4 into 5×6 (zero-padded)
int big[5][6] = {0};
for(int i=0; i<3; i++)
    for(int j=0; j<4; j++)
        big[i][j] = arr[i][j];
ANALOGY
📦

Flatten = Taking items out of a grid storage and placing them in a single long conveyor belt. Unflatten = Sorting conveyor belt items back into shelves. The key formula: row = index ÷ cols, col = index % cols.

🧠 Static vs Dynamic Memory Allocation

CODE + THEORY
THEORY
Static Allocation: Size decided at compile time. Memory lives on the stack. Fast, but inflexible — you must know the size upfront.

Dynamic Allocation: Size decided at runtime. Memory lives on the heap. Flexible, but you must manually free it.

In C, use malloc() / calloc() to allocate and free() to release.
Static Memory
// Fixed at compile time
int arr[3][4]; // Stack memory
// No need to free!
// ⚠️ Size cannot change at runtime
Dynamic Memory Allocation
// 2D array using pointer-to-pointer
int **arr;
int rows=3, cols=4;

// Allocate rows
arr = (int**)malloc(rows * sizeof(int*));

// Allocate each row's columns
for(int i=0; i<rows; i++)
    arr[i]=(int*)malloc(cols*sizeof(int));

// Use it: arr[i][j] = value;

// Free memory (REQUIRED!)
for(int i=0; i<rows; i++)
    free(arr[i]);
free(arr);

// Resize: calloc for zero-initialized
arr[i] = (int*)realloc(arr[i],
    newCols * sizeof(int));

Static: Renting a fixed apartment — you know the exact room count when you move in. Dynamic: Booking hotel rooms one by one as guests arrive. Flexible, but you must check out (free) when done — or you'll have a memory leak!

struct Node {

struct Node* matToList(
    int arr[][4], int r, int c) {
    struct Node *head=NULL, *tail=NULL;
    for(int i=0;i<r;i++) {
        for(int j=0;j<c;j++) {
            struct Node *n =
                malloc(sizeof(struct Node));
            n->val = arr[i][j];
            n->next = NULL;
            if(!head) head=tail=n;

        }
    }
    return head;

// BFS to find shortest path
// 4 directions: up, down, left, right
int dr[] = {-1,1,0,0};
int dc[] = {0,0,-1,1};

int bfsShortestPath(int grid[][6],
    int sr, int sc, int er, int ec) {
    int visited[6][6] = {0};
    int queue[100][3]; // [row,col,dist]
    int head=0, tail=0;
    queue[tail][0]=sr; queue[tail][1]=sc;
    queue[tail++][2]=0;
    visited[sr][sc]=1;
    while(head < tail) {
        int r=queue[head][0];
        int c=queue[head][1];
        int d=queue[head++][2];
        if(r==er && c==ec) return d;
        for(int k=0;k<4;k++) {
            int nr=r+dr[k], nc=c+dc[k];
            if(nr>=0 && nc>=0 &&
               nr<6 && nc<6 &&
               !grid[nr][nc] &&
               !visited[nr][nc]) {
                visited[nr][nc]=1;
                queue[tail][0]=nr;
                queue[tail][1]=nc;
                queue[tail++][2]=d+1;
            }
        }
    }
    return -1; // no path
}

    for(int j=1;j<=n;j++) {

            dp[i][j] = dp[i-1][j-1]+1;
