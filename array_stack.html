01 Push Operation Beginner ▼
📖 Concept Overview
Push is the operation of adding an element to the TOP of the stack.

Think of it like placing a plate on top of a pile — the new plate always goes on top. Every time you push, the stack grows by one element from the top.

Key Rules:
  • You can only add to the TOP
  • If the stack is full → it causes Stack Overflow
  • After push, top pointer increases by 1
🍽️Real-life Analogy: Imagine a cafeteria stack of plates. Each time a clean plate is added, it goes on TOP. You can't place a plate in the middle — only the top!

💻 C Code
C Language
#include <stdio.h>
#define MAX 5   /* maximum size of stack */

int stack[MAX];  /* array to store stack elements */
int top = -1;    /* -1 means stack is empty */

/* Push function: adds element to top of stack */
void push(int value) {
    /* First check: is stack already full? */
    if (top == MAX - 1) {
        printf("Stack Overflow! Cannot push %d\n", value);
        return;  /* exit the function */
    }

    top++;              /* move top pointer up by 1 */
    stack[top] = value; /* place value at new top position */
    printf("Pushed: %d  (top = %d)\n", value, top);
}

int main() {
    push(10);  /* add 10 to stack */
    push(20);  /* add 20 on top of 10 */
    push(30);  /* add 30 on top of 20 */
    return 0;
}
Pushed: 10 (top = 0)

02 Pop Operation Beginner ▼
📖 Concept Overview
Pop is the operation of removing the TOP element from the stack and returning it.

Just like taking the top plate off a pile — you always remove from the top.

Key Rules:
  • Remove from TOP only
  • If stack is empty → Stack Underflow
  • After pop, top decreases by 1
📚 Removing the top book from a stack of books. You can only take the topmost one — you can't pull from the middle!

💻 C Code
C Language
#include <stdio.h>
#define MAX 5

int stack[MAX];
int top = -1;

void push(int value) {
    if (top == MAX - 1) { printf("Overflow!\n"); return; }
    stack[++top] = value;
}

/* Pop function: removes and returns top element */
int pop() {
    /* Check: is stack empty? */
    if (top == -1) {
        printf("Stack Underflow! Stack is empty\n");
        return -1;  /* return -1 as error indicator */
    }

    int removed = stack[top]; /* save top value */
    top--;                    /* move top pointer down */
    printf("Popped: %d  (top = %d)\n", removed, top);
    return removed;           /* return removed value */
}

int main() {
    push(10); push(20); push(30);
    pop();  /* removes 30 */
    pop();  /* removes 20 */
    return 0;
}
Popped: 30 (top = 1)
Popped: 20 (top = 0)

isEmpty checks whether the stack has zero elements.

Condition: top == -1 → stack is empty.

📖 Concept Overview
Traverse means visiting and printing all elements from top to bottom (or bottom to top).

We loop from top down to 0 to display top → bottom order.
👁️ Like reading a stack of papers from top to bottom, one by one, without removing them.

    if (top == -1) {
        printf("Stack is empty!\n");
        return;

    printf("Stack (Top → Bottom): ");

08 Search Beginner ▼
📖 Concept Overview
Search finds whether a given element exists in the stack and returns its position from the top.

We scan from top → 0. Position 1 = top element. Time: O(n) in worst case.

💻 C Code
C Language
#include <stdio.h>
#define MAX 5

int stack[MAX];
int top = -1;

/* search: find position of element from top */
int search(int key) {
    int i, pos = 1;  /* pos 1 = top element */
    for (i = top; i >= 0; i--) {
        if (stack[i] == key) {
            printf("Found %d at position %d from top\n", key, pos);
            return pos;
        }
        pos++;
    }
    printf("%d not found in stack\n", key);
    return -1;
}

int main() {
    stack[++top]=10; stack[++top]=20; stack[++top]=30;
    search(20);  /* 20 is at position 2 from top */
    search(99);  /* not in stack */
    return 0;
}

10 Copy / Clone Intermediate ▼
📖 Concept Overview
Copy / Clone creates a duplicate of the entire stack.

Both the array contents and the top value are copied. Modifying the copy does NOT affect the original.
📋 Like photocopying a document — you get an identical copy, and writing on the copy doesn't change the original.
💻 C Code
C Language
#include <stdio.h>
#define MAX 5

int original[MAX], copy[MAX];
int origTop = -1, copyTop = -1;

/* clone: copy all elements from original to copy */
void cloneStack() {
    int i;
    copyTop = origTop;  /* copy the top index too */
    for (i = 0; i <= origTop; i++) {
        copy[i] = original[i];  /* copy each element */
    }
    printf("Stack cloned! Copy has %d elements\n", copyTop+1);
}

int main() {
    original[++origTop]=10;
    original[++origTop]=20;
    original[++origTop]=30;
    cloneStack();
    copy[++copyTop] = 99; /* modify copy — original unchanged */
    printf("Original top: %d, Copy top: %d\n", origTop, copyTop);
    return 0;
}
Stack cloned! Copy has 3 elements
Original top: 2, Copy top: 3
11 Reverse Stack Intermediate ▼
📖 Concept Overview
Reverse Stack flips the order of elements so that the bottom becomes the top and vice versa.

Strategy: Pop all elements into a temporary array, then push them back in reverse order.
🔄 Like flipping a stack of pancakes upside down — the bottom one is now on top!

💻 C Code
C Language
#include <stdio.h>
#define MAX 10

int stack[MAX];
int top = -1;

/* reverse: reverse the entire stack using temp array */
void reverseStack() {
    int temp[MAX];
    int i, n = top + 1;  /* total elements */

    /* step 1: copy to temp array (top to bottom) */
    for (i = 0; i < n; i++) {
        temp[i] = stack[top - i];
    }
    /* step 2: copy back in reversed order */
    for (i = 0; i < n; i++) {
        stack[i] = temp[i];
    }
    printf("Stack reversed!\n");
}

void display() {
    int i;
    for (i = top; i >= 0; i--) printf("%d ", stack[i]);
    printf("\n");
}

int main() {
    stack[++top]=1; stack[++top]=2; stack[++top]=3; stack[++top]=4;
    printf("Before: "); display();
    reverseStack();
    printf("After : "); display();
    return 0;
}

12 Swap Top Elements Intermediate ▼
📖 Concept Overview
Swap Top Elements exchanges the top two items of the stack.

Elements at position top and top-1 are swapped using a temp variable.
💻 C Code
C Language
#include <stdio.h>
#define MAX 5

int stack[MAX];
int top = -1;

/* swap: exchange the top two elements */
void swapTop() {
    if (top < 1) {       /* need at least 2 elements */
        printf("Need at least 2 elements to swap!\n");
        return;
    }
    int temp = stack[top];       /* save top */
    stack[top] = stack[top-1];  /* put second on top */
    stack[top-1] = temp;        /* put old top below */
    printf("Swapped top two elements!\n");
}

int main() {
    stack[++top]=10; stack[++top]=20; stack[++top]=30;
    printf("Before: top=%d, second=%d\n", stack[top], stack[top-1]);
    swapTop();
    printf("After : top=%d, second=%d\n", stack[top], stack[top-1]);
    return 0;
}

13 Sort Stack Intermediate ▼
📖 Concept Overview
Sort Stack arranges elements in ascending or descending order.

We can sort by popping all elements into an array, sorting the array, then pushing back. The simplest approach uses selection sort on the internal array.
🎮 Sort Visualization
Click Random Stack to begin!
💻 C Code
C Language
#include <stdio.h>
#define MAX 10

int stack[MAX];
int top = -1;

/* sortStack: sort using bubble sort on the array */
void sortStack() {
    int i, j, temp;
    int n = top + 1;
    /* bubble sort the underlying array */
    for (i = 0; i < n-1; i++) {
        for (j = 0; j < n-i-1; j++) {
            if (stack[j] > stack[j+1]) {
                /* swap adjacent elements */
                temp = stack[j];
                stack[j] = stack[j+1];
                stack[j+1] = temp;
            }
        }
    }
    printf("Stack sorted! Top is now the largest.\n");
}

int main() {
    stack[++top]=30; stack[++top]=10;
    stack[++top]=50; stack[++top]=20;
    sortStack();
    int i;
    printf("Sorted (top to bottom): ");
    for(i=top;i>=0;i--) printf("%d ", stack[i]);
    return 0;
}
Stack sorted! Top is now the largest.
Sorted (top to bottom): 50 30 20 10
14 Merge Stacks Intermediate ▼
📖 Concept Overview
Merge Stacks combines two stacks into one.

Strategy: Copy all elements of Stack 1 into a new stack, then push all elements of Stack 2 on top.
🫂 Like combining two piles of papers into one — stack 1 goes in first, then stack 2 on top.
💻 C Code
C Language
#include <stdio.h>
#define MAX 20

int s1[MAX], s2[MAX], merged[MAX];
int t1 = -1, t2 = -1, tm = -1;

/* merge: combine s1 and s2 into merged[] */
void mergeStacks() {
    int i;
    /* copy stack1 (bottom to top order preserved) */
    for (i = 0; i <= t1; i++) {
        merged[++tm] = s1[i];
    }
    /* push stack2 on top of stack1 */
    for (i = 0; i <= t2; i++) {
        merged[++tm] = s2[i];
    }
    printf("Merged! Total elements: %d\n", tm+1);
}

int main() {
    s1[++t1]=1; s1[++t1]=2; s1[++t1]=3;
    s2[++t2]=4; s2[++t2]=5;
    mergeStacks();
    int i;
    printf("Merged stack (top→btm): ");
    for(i=tm;i>=0;i--) printf("%d ", merged[i]);
    return 0;
}
Merged! Total elements: 5
Merged stack (top→btm): 5 4 3 2 1
15 Split Stack Intermediate ▼
📖 Concept Overview
Split Stack divides one stack into two halves.

First half goes into Stack 1, second half into Stack 2. Useful when a stack grows too large.
💻 C Code
C Language
#include <stdio.h>
#define MAX 10

int original[MAX]={1,2,3,4,5,6};
int top = 5;  /* 6 elements: index 0..5 */
int s1[MAX], s2[MAX];
int t1 = -1, t2 = -1;

/* split: divide into two halves */
void splitStack() {
    int n = top + 1;
    int half = n / 2;
    int i;
    /* first half into s1 */
    for (i = 0; i < half; i++) {
        s1[++t1] = original[i];
    }
    /* second half into s2 */
    for (i = half; i < n; i++) {
        s2[++t2] = original[i];
    }
    printf("Split: s1 has %d elements, s2 has %d\n", t1+1, t2+1);
}

int main() {
    splitStack();
    return 0;
}

16 Recursive Stack Operations Advanced ▼
📖 Concept Overview
Recursive Stack Operations use recursion (function calling itself) to reverse or sort the stack — without using extra arrays!

The call stack itself acts as temporary storage. Elegant but uses more memory.
🪆 Like Russian nesting dolls — each doll opens, reveals another, all the way to the smallest, then closes back. Each function call is a "doll."
💻 C Code — Recursive Reverse
C Language
#include <stdio.h>
#define MAX 10

int stack[MAX];
int top = -1;

void push(int v) { stack[++top] = v; }
int  pop()      { return stack[top--]; }

/* insertAtBottom: inserts element at very bottom */
void insertAtBottom(int val) {
    if (top == -1) {  /* stack empty → just push */
        push(val);
        return;
    }
    int temp = pop();   /* hold top element */
    insertAtBottom(val); /* recurse until empty */
    push(temp);          /* put back on top */
}

/* reverseRecursive: reverses by inserting at bottom */
void reverseRecursive() {
    if (top == -1) return; /* base case: empty */
    int temp = pop();     /* remove top */
    reverseRecursive();   /* reverse the rest */
    insertAtBottom(temp); /* put popped at bottom */
}

int main() {
    push(1); push(2); push(3); push(4);
    reverseRecursive();
    int i;
    printf("Reversed: ");
    for(i=top;i>=0;i--) printf("%d ", stack[i]);
    return 0;
}
Reversed: 1 2 3 4

17 Evaluate Expressions (Postfix / Prefix) Advanced ▼
📖 Concept Overview
Stacks are used to evaluate Postfix (Reverse Polish Notation) and Prefix expressions efficiently.

Postfix: Operators come AFTER operands → 3 4 + 2 * = (3+4)*2 = 14
Prefix: Operators come BEFORE operands → * + 3 4 2 = 14

Why not Infix? Infix needs bracket handling and precedence rules. Postfix/Prefix are unambiguous — no brackets needed!
🧮 Postfix is how calculators actually work internally! Press 3, 4, +, and it pushes 3 and 4, then pops both and pushes 7.
🎮 Postfix Evaluator
Enter a postfix expression (e.g. 3 4 + 2 *)
Enter postfix and evaluate!
💻 C Code — Postfix Evaluation
C Language
#include <stdio.h>
#include <ctype.h>   /* for isdigit() */
#define MAX 20

int stack[MAX];
int top = -1;

void push(int v) { stack[++top] = v; }
int  pop()      { return stack[top--]; }

/* evaluatePostfix: evaluate expression like "3 4 + 2 *" */
int evaluatePostfix(char* expr) {
    int i = 0;
    while (expr[i] != '\0') {
        if (expr[i] == ' ') { i++; continue; } /* skip spaces */

        if (isdigit(expr[i])) {
            push(expr[i] - '0'); /* convert char to int, push */
        } else {
            int b = pop(); /* second operand */
            int a = pop(); /* first operand */
            switch (expr[i]) {
                case '+': push(a + b); break;
                case '-': push(a - b); break;
                case '*': push(a * b); break;
                case '/': push(a / b); break;
            }
        }
        i++;
    }
    return pop(); /* final result */
}

int main() {
    char expr[] = "3 4 + 2 *"; /* (3+4)*2 = 14 */
    printf("Result: %d\n", evaluatePostfix(expr));
    return 0;
}
Result: 14

18 Expression Conversion Advanced ▼
📖 Concept Overview
There are three notations for expressions:

Infix: A + B * C — human-friendly, needs precedence rules
Postfix: A B C * + — machine-friendly, no brackets needed
Prefix: + A * B C — used in compilers

Stack is used for all conversions. The key is operator precedence: * / before + -.

💻 C Code — Infix to Postfix
C Language
#include <stdio.h>
#include <string.h>
#define MAX 50

char stack[MAX];  /* stack for operators */
int top = -1;

void push(char c) { stack[++top] = c; }
char pop()        { return stack[top--]; }
char peek()       { return top==-1 ? '#' : stack[top]; }

/* precedence: returns priority of operators */
int prec(char c) {
    if(c=='*'||c=='/') return 2;
    if(c=='+'||c=='-') return 1;
    return 0;
}

/* infixToPostfix: converts infix expr to postfix */
void infixToPostfix(char* infix) {
    int i = 0;
    printf("Postfix: ");
    while (infix[i] != '\0') {
        char c = infix[i];
        if ((c>='A'&&c<='Z')||(c>='a'&&c<='z')||(c>='0'&&c<='9')) {
            printf("%c", c);  /* operand → print directly */
        } else if (c == '(') {
            push(c);            /* push '(' */
        } else if (c == ')') {
            while (peek() != '(') printf("%c", pop());
            pop(); /* remove '(' */
        } else {
            while (prec(peek()) >= prec(c)) printf("%c", pop());
            push(c);
        }
        i++;
    }
    while (top != -1) printf("%c", pop());
    printf("\n");
}

int main() {
    infixToPostfix("A+B*C");  /* output: ABC*+ */
    return 0;
}
Postfix: ABC*+

19 Balanced Parentheses Checking Intermediate ▼
📖 Concept Overview
Check if brackets in an expression are properly matched and nested.

Works for: ( ) { } [ ]

Algorithm:
1. Scan left to right
2. Open brackets → push onto stack
3. Close bracket → pop and check if it matches
4. If mismatch or stack not empty at end → UNBALANCED
🎯 Like a mirror — every opening bracket must have its matching closing bracket in the right order!
🎮 Interactive Checker
Enter expression and click Check!
💻 C Code
C Language
#include <stdio.h>
#include <string.h>
#define MAX 50

char stack[MAX];
int top = -1;

void push(char c) { stack[++top] = c; }
char pop()        { return top==-1 ? '#' : stack[top--]; }

/* isMatching: returns 1 if open-close pair matches */
int isMatching(char open, char close) {
    return (open=='('&&close==')') ||
           (open=='{'&&close=='}') ||
           (open=='['&&close==']');
}

/* checkBalance: checks if brackets are balanced */
int checkBalance(char* expr) {
    int i;
    top = -1; /* reset stack */
    for (i = 0; expr[i] != '\0'; i++) {
        char c = expr[i];
        if (c=='('||c=='{'||c=='[') {
            push(c);  /* opening bracket: push */
        } else if (c==')'||c=='}'||c==']') {
            char open = pop();
            if (!isMatching(open, c)) return 0; /* mismatch! */
        }
    }
    return top == -1; /* balanced if stack is empty */
}

int main() {
    printf("{[()]} : %s\n", checkBalance("{[()]}") ? "BALANCED" : "UNBALANCED");
    printf("([)]   : %s\n", checkBalance("([)]")   ? "BALANCED" : "UNBALANCED");
    return 0;
}

20 Undo / Redo Operations Advanced ▼
📖 Concept Overview
Undo/Redo is one of the most practical real-world uses of stacks!

How it works:
• Undo Stack: Each action you take is pushed here
• Redo Stack: When you undo, the action moves to the redo stack

Ctrl+Z = pop from undo stack, push to redo stack
Ctrl+Y = pop from redo stack, push back to undo stack
📝 Like a text editor — every keystroke is pushed onto the undo stack. Ctrl+Z pops the last action and undoes it!

💻 C Code
C Language
#include <stdio.h>
#include <string.h>
#define MAX 10
#define LEN 50

char undoStack[MAX][LEN]; /* undo stack of strings */
char redoStack[MAX][LEN]; /* redo stack of strings */
int uTop = -1, rTop = -1;

/* doAction: perform action and push to undo */
void doAction(char* action) {
    strcpy(undoStack[++uTop], action);
    rTop = -1; /* clear redo when new action done */
    printf("Done: %s\n", action);
}

/* undo: move top of undo to redo */
void undo() {
    if (uTop == -1) { printf("Nothing to undo!\n"); return; }
    strcpy(redoStack[++rTop], undoStack[uTop]);
    printf("Undone: %s\n", undoStack[uTop--]);
}

/* redo: move top of redo back to undo */
void redo() {
    if (rTop == -1) { printf("Nothing to redo!\n"); return; }
    strcpy(undoStack[++uTop], redoStack[rTop]);
    printf("Redone: %s\n", redoStack[rTop--]);
}

int main() {
    doAction("Type Hello");
    doAction("Bold Text");
    undo();   /* undo Bold */
    redo();   /* redo Bold */
    return 0;
}

21 Backtracking (DFS & Maze Solving) Advanced ▼
📖 Concept Overview
Stacks power two famous algorithms:

DFS (Depth-First Search): Explore a graph as deep as possible before backtracking. Uses a stack to track the path.

Maze Solving: Try a path — if dead end, backtrack (pop from stack) and try another.

The Stack stores: nodes/cells visited on the current path. Pop = backtrack.
🧭 Like exploring a cave with string — you leave a trail. When you hit a dead end, you follow the string back (pop from stack) and try a different tunnel.

C Language
#include <stdio.h>
#define V 5    /* number of vertices */
#define MAX 50

/* Adjacency matrix: adj[i][j]=1 means edge i-j */
int adj[V][V] = {
    {0,1,1,0,0},  /* node 0 connects to 1,2 */
    {1,0,0,1,0},  /* node 1 connects to 0,3 */
    {1,0,0,1,1},  /* node 2 connects to 0,3,4 */
    {0,1,1,0,0},
    {0,0,1,0,0}
};

int stackArr[MAX], top = -1;
int visited[V] = {0};  /* 0=not visited */

void push(int v) { stackArr[++top] = v; }
int  pop()       { return stackArr[top--]; }

/* dfs: iterative DFS using stack */
void dfs(int start) {
    push(start);  /* push starting node */
    while (top != -1) {
        int node = pop();
        if (visited[node]) continue;
        visited[node] = 1;
        printf("Visit node %d\n", node);
        int j;
        for (j = V-1; j >= 0; j--) {  /* push neighbors */
            if (adj[node][j] && !visited[j])
                push(j);
        }
    }
}

int main() {
    printf("DFS from node 0:\n");
    dfs(0);
    return 0;
}
