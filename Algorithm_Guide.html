TOPIC 01

Function, Recursion & GCD (Euclidean Algorithm)

Recursion = A function that calls itself. Every recursive function has:
• Base Case — stops the recursion (no more calls)
• Recursive Case — breaks the problem into a smaller version

GCD (Euclidean Algorithm): gcd(a, b) = gcd(b, a % b) until b = 0
📊 Recursion Tree — gcd(48, 18)
gcd(48, 18) └── gcd(18, 12) 48 % 18 = 12 └── gcd(12, 6) 18 % 12 = 6 └── gcd(6, 0) 12 % 6 = 0 └── return 6 ✅ BASE CASE
💻 C Code
#include <stdio.h>

// Recursive GCD using Euclidean Algorithm
int gcd(int a, int b) {
    if (b == 0)           // Base Case: stop when b is 0
        return a;
    return gcd(b, a % b);  // Recursive Case
}

// Factorial using recursion
int factorial(int n) {
    if (n == 0) return 1;     // Base Case
    return n * factorial(n - 1); // Recursive Case
}

int main() {
    printf("GCD(48, 18) = %d\n", gcd(48, 18));   // Output: 6
    printf("5! = %d\n", factorial(5));           // Output: 120
    return 0;
}

Asymptotic Notation & Complexity Analysis

Describes how runtime grows as input size (n) increases.
• O (Big-O) → Worst case upper bound
• Ω (Omega) → Best case lower bound
• Θ (Theta) → Exact (tight) bound — best = worst

TOPIC 03

Brute Force, Searching & Sorting Algorithms

Brute Force = Try all possibilities (simple but slow).
Linear Search = Check each element one by one → O(n)
Binary Search = Divide sorted array in half each time → O(log n)
Bubble, Selection, Insertion Sort → All O(n²) basic sorts
📊 Binary Search — Find 7 in [1,3,5,7,9,11,13]
Array: [ 1 3 5 7 9 11 13 ] Index: 0 1 2 3 4 5 6 Step 1: mid = 3 → arr[3]=7 == target ✅ FOUND at index 3 Bubble Sort — [5,3,8,1,2]: Pass 1: [3,5,1,2,8] → largest bubbles up Pass 2: [3,1,2,5,8] Pass 3: [1,2,3,5,8] ✅ Sorted Selection Sort — find min each pass: [5,3,8,1,2] → min=1 → swap → [1,3,8,5,2] [1,3,8,5,2] → min=2 → swap → [1,2,8,5,3] ... Insertion Sort — insert in correct position: [5] → [3,5] → [3,5,8] → [1,3,5,8] → [1,2,3,5,8] ✅
💻 C Code
#include <stdio.h>

/* ── Linear Search ── */
int linearSearch(int arr[], int n, int key) {
    for (int i = 0; i < n; i++)
        if (arr[i] == key) return i;
    return -1;
}

/* ── Binary Search (array must be sorted) ── */
int binarySearch(int arr[], int lo, int hi, int key) {
    while (lo <= hi) {
        int mid = lo + (hi - lo) / 2;
        if (arr[mid] == key) return mid;
        else if (arr[mid] < key) lo = mid + 1;
        else hi = mid - 1;
    }
    return -1;
}

/* ── Bubble Sort ── */
void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n-1; i++)
        for (int j = 0; j < n-i-1; j++)
            if (arr[j] > arr[j+1]) {
                int t = arr[j]; arr[j] = arr[j+1]; arr[j+1] = t;
            }
}

/* ── Selection Sort ── */
void selectionSort(int arr[], int n) {
    for (int i = 0; i < n-1; i++) {
        int minIdx = i;
        for (int j = i+1; j < n; j++)
            if (arr[j] < arr[minIdx]) minIdx = j;
        int t = arr[i]; arr[i] = arr[minIdx]; arr[minIdx] = t;
    }
}

/* ── Insertion Sort ── */
void insertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i], j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j+1] = arr[j]; j--;
        }
        arr[j+1] = key;
    }
}

int main() {
    int arr[] = {5, 3, 8, 1, 2}, n = 5;
    bubbleSort(arr, n);
    for(int i=0;i<n;i++) printf("%d ",arr[i]); // 1 2 3 5 8
    return 0;
}
TOPIC 04

Divide and Conquer — Merge Sort & Quick Sort

Divide → Split problem into sub-problems
Conquer → Solve sub-problems recursively
Combine → Merge results

Merge Sort: Always O(n log n) | Quick Sort: Avg O(n log n), Worst O(n²)
Master Theorem: T(n)=aT(n/b)+f(n) → determines complexity
📊 Merge Sort — [38,27,43,3]
[38, 27, 43, 3] / \ [38, 27] [43, 3] / \ / \ [38] [27] [43] [3] \ / \ / [27, 38] [3, 43] \ / [3, 27, 38, 43] ✅ Quick Sort Partition (pivot=last element): [3, 6, 8, 10, 1, 2, 1] pivot=1 → elements < 1 left | elements > 1 right
💻 C Code — Merge Sort & Quick Sort
#include <stdio.h>

/* ────── MERGE SORT ────── */
void merge(int arr[], int l, int m, int r) {
    int i, j, k;
    int n1 = m-l+1, n2 = r-m;
    int L[n1], R[n2];
    for(i=0;i<n1;i++) L[i]=arr[l+i];
    for(j=0;j<n2;j++) R[j]=arr[m+1+j];
    i=0; j=0; k=l;
    while(i<n1 && j<n2)
        arr[k++] = (L[i]<=R[j]) ? L[i++] : R[j++];
    while(i<n1) arr[k++]=L[i++];
    while(j<n2) arr[k++]=R[j++];
}

void mergeSort(int arr[], int l, int r) {
    if(l < r) {
        int m = l + (r-l)/2;
        mergeSort(arr, l, m);
        mergeSort(arr, m+1, r);
        merge(arr, l, m, r);
    }
}

/* ────── QUICK SORT ────── */
int partition(int arr[], int lo, int hi) {
    int pivot = arr[hi], i = lo - 1;
    for(int j=lo; j<hi; j++)
        if(arr[j] < pivot) {
            i++;
            int t=arr[i]; arr[i]=arr[j]; arr[j]=t;
        }
    int t=arr[i+1]; arr[i+1]=arr[hi]; arr[hi]=t;
    return i+1;
}

void quickSort(int arr[], int lo, int hi) {
    if(lo < hi) {
        int pi = partition(arr, lo, hi);
        quickSort(arr, lo, pi-1);
        quickSort(arr, pi+1, hi);
    }
}

int main() {
    int a[] = {38,27,43,3}, n=4;
    mergeSort(a, 0, n-1);
    for(int i=0;i<n;i++) printf("%d ",a[i]); // 3 27 38 43
    return 0;
}
TOPIC 05

Greedy Approach — Coin Change, Knapsack, Huffman

Greedy = Always pick the locally best option at each step.
✅ Works for: Coin Change (standard), Huffman Coding, Fractional Knapsack
❌ Fails for: 0/1 Knapsack (use DP instead)
📊 Coin Change Greedy — Amount=41, coins=[25,10,5,1]
Amount = 41 Step 1: Use 25 → remaining = 16 [25] Step 2: Use 10 → remaining = 6 [25,10] Step 3: Use 5 → remaining = 1 [25,10,5] Step 4: Use 1 → remaining = 0 [25,10,5,1] ✅ 4 coins Huffman Coding — Frequencies: A=5, B=9, C=12, D=13 Build min-heap → merge smallest two repeatedly: A(5)+B(9)=14 → C(12)+D(13)=25 → 14+25=39 A:110 B:111 C:10 D:0
💻 C Code — Coin Change & Fractional Knapsack
#include <stdio.h>
#include <stdlib.h>

/* ── Greedy Coin Change ── */
void coinChange(int coins[], int n, int amount) {
    // coins[] must be sorted descending
    for(int i=0; i<n && amount>0; i++) {
        int count = amount / coins[i];
        amount -= count * coins[i];
        if(count) printf("Coin %d × %d\n", coins[i], count);
    }
}

/* ── Fractional Knapsack ── */
typedef struct { int w, v; double ratio; } Item;

int cmp(const void* a, const void* b) {
    Item* x=(Item*)a; Item* y=(Item*)b;
    return (y->ratio > x->ratio) ? 1 : -1;
}

double fractionalKnapsack(Item items[], int n, int W) {
    for(int i=0;i<n;i++)
        items[i].ratio = (double)items[i].v / items[i].w;
    qsort(items, n, sizeof(Item), cmp);

    double total = 0;
    for(int i=0;i<n;i++) {
        if(W >= items[i].w) { total+=items[i].v; W-=items[i].w; }
        else { total += items[i].ratio * W; break; }
    }
    return total;
}

int main() {
    int coins[]={25,10,5,1};
    coinChange(coins, 4, 41);

    Item items[]={{10,60},{20,100},{30,120}};
    printf("Max value = %.2f\n", fractionalKnapsack(items, 3, 50)); // 240.0
    return 0;
}
TOPIC 06

Dynamic Programming — Fibonacci, Coin Change, 0/1 Knapsack

DP = Break into overlapping sub-problems, store results to avoid re-computation.
• Memoization = Top-down (recursive + cache)
• Tabulation = Bottom-up (fill table iteratively)
Key insight: Optimal substructure + Overlapping subproblems
📊 Fibonacci DP Table & 0/1 Knapsack
Fibonacci Tabulation: index: 0 1 2 3 4 5 6 7 dp: 0 1 1 2 3 5 8 13 0/1 Knapsack — weights=[1,3,4,5], values=[1,4,5,7], W=7 w→ 0 1 2 3 4 5 6 7 i=0: [0, 1, 1, 1, 1, 1, 1, 1] i=1: [0, 1, 1, 4, 5, 5, 5, 5] i=2: [0, 1, 1, 4, 5, 6, 6, 9] i=3: [0, 1, 1, 4, 5, 7, 8, 9] ↑ answer = 9 ✅
💻 C Code
#include <stdio.h>
#include <string.h>
#define MAX 1000

/* ── Fibonacci: Memoization ── */
int memo[MAX];
int fib(int n) {
    if(n <= 1) return n;
    if(memo[n] != -1) return memo[n]; // cached
    return memo[n] = fib(n-1) + fib(n-2);
}

/* ── Coin Change: DP (min coins) ── */
int coinChangeDP(int coins[], int n, int amount) {
    int dp[amount+1];
    for(int i=0;i<=amount;i++) dp[i]=amount+1; // init large
    dp[0] = 0;
    for(int i=1;i<=amount;i++)
        for(int j=0;j<n;j++)
            if(coins[j] <= i)
                dp[i] = (dp[i]<dp[i-coins[j]]+1) ? dp[i] : dp[i-coins[j]]+1;
    return dp[amount] > amount ? -1 : dp[amount];
}

/* ── 0/1 Knapsack ── */
int knapsack(int W, int wt[], int val[], int n) {
    int dp[n+1][W+1];
    for(int i=0;i<=n;i++) {
        for(int w=0;w<=W;w++) {
            if(i==0 || w==0) dp[i][w]=0;
            else if(wt[i-1]<=w)
                dp[i][w] = (val[i-1]+dp[i-1][w-wt[i-1]] > dp[i-1][w])
                           ? val[i-1]+dp[i-1][w-wt[i-1]] : dp[i-1][w];
            else dp[i][w]=dp[i-1][w];
        }
    }
    return dp[n][W];
}

int main() {
    memset(memo, -1, sizeof(memo));
    printf("Fib(10) = %d\n", fib(10));  // 55

    int coins[]={1,5,6,9};
    printf("Min coins for 11 = %d\n", coinChangeDP(coins,4,11)); // 2

    int wt[]={1,3,4,5}, val[]={1,4,5,7};
    printf("Knapsack max = %d\n", knapsack(7,wt,val,4));  // 9
    return 0;
}
TOPIC 07

Advanced DP — LCS & LIS

LCS: Longest Common Subsequence of two strings (not necessarily contiguous)
LIS: Longest Increasing Subsequence within one array
📊 LCS: "ABCBDAB" vs "BDCABA" | LIS: [10,9,2,5,3,7,101,18]
LCS Table (X="ABCB", Y="BCB"): "" B C B "" 0 0 0 0 A 0 0 0 0 B 0 1 1 1 C 0 1 2 2 B 0 1 2 3 ← LCS length = 3 → "BCB" LIS: [10, 9, 2, 5, 3, 7, 101, 18] dp: [ 1, 1, 1, 2, 2, 3, 4, 4] LIS length = 4 → [2, 3, 7, 101]
💻 C Code
#include <stdio.h>
#include <string.h>

/* ── LCS ── */
int lcs(char* X, char* Y, int m, int n) {
    int dp[m+1][n+1];
    for(int i=0;i<=m;i++) {
        for(int j=0;j<=n;j++) {
            if(i==0 || j==0) dp[i][j]=0;
            else if(X[i-1]==Y[j-1]) dp[i][j]=dp[i-1][j-1]+1;
            else dp[i][j] = (dp[i-1][j]>dp[i][j-1]) ? dp[i-1][j]:dp[i][j-1];
        }
    }
    return dp[m][n];
}

/* ── LIS ── */
int lis(int arr[], int n) {
    int dp[n], ans=1;
    for(int i=0;i<n;i++) dp[i]=1;
    for(int i=1;i<n;i++) {
        for(int j=0;j<i;j++)
            if(arr[j]<arr[i] && dp[j]+1>dp[i])
                dp[i]=dp[j]+1;
        if(dp[i]>ans) ans=dp[i];
    }
    return ans;
}

int main() {
    char X[]= "ABCBDAB", Y[]= "BDCABA";
    printf("LCS = %d\n", lcs(X,Y,strlen(X),strlen(Y))); // 4

    int arr[]={10,9,2,5,3,7,101,18};
    printf("LIS = %d\n", lis(arr,8)); // 4
    return 0;
}

TOPIC 09

Graph Traversal — BFS & DFS

BFS (Breadth-First): Level by level using a Queue → finds shortest path in unweighted graphs
DFS (Depth-First): Go deep using a Stack/Recursion → explores fully before backtracking
📊 BFS vs DFS on same graph (start=0)
Graph: 0 ─ 1 ─ 3 └── 2 ─ 4 BFS (Queue): Visit level-by-level Queue: [0] → visit 0 Queue: [1,2] → visit 1, visit 2 Queue: [3,4] → visit 3, visit 4 Order: 0 → 1 → 2 → 3 → 4 DFS (Recursive): Go as deep as possible Visit 0 → go to 1 → go to 3 → backtrack → go to 2 → go to 4 → backtrack Order: 0 → 1 → 3 → 2 → 4
💻 C Code
#include <stdio.h>
#include <string.h>
#define V 5

int adj[V][V], deg[V], visited[V];

/* ── BFS ── */
void bfs(int start) {
    int queue[V], front=0, rear=0;
    memset(visited,0,sizeof(visited));
    visited[start]=1;
    queue[rear++]=start;
    printf("BFS: ");
    while(front < rear) {
        int node=queue[front++];
        printf("%d ",node);
        for(int i=0;i<deg[node];i++) {
            int nb=adj[node][i];
            if(!visited[nb]) { visited[nb]=1; queue[rear++]=nb; }
        }
    }
    printf("\n");
}

/* ── DFS (Recursive) ── */
void dfs(int node) {
    visited[node]=1;
    printf("%d ",node);
    for(int i=0;i<deg[node];i++)
        if(!visited[adj[node][i]])
            dfs(adj[node][i]);
}

void addEdge(int u, int v) {
    adj[u][deg[u]++]=v; adj[v][deg[v]++]=u;
}

int main() {
    addEdge(0,1); addEdge(0,2);
    addEdge(1,3); addEdge(2,4);
    bfs(0);                            // 0 1 2 3 4
    memset(visited,0,sizeof(visited));
    printf("DFS: "); dfs(0);          // 0 1 3 2 4
    return 0;
}

Graph Applications — Cycle Detection, Topological Sort, SCC

• Cycle Detection: DFS with color marking (white/gray/black)
• Topological Sort: Linear order of a DAG (Directed Acyclic Graph) using DFS stack
• SCC: Strongly Connected Components via Kosaraju's (2 DFS passes)
📊 Topological Sort — DAG
DAG: 5 → 0 ← 4 ↓ ↓ 2 → 3 → 1 Topological Order (one valid): [5, 4, 2, 3, 1, 0] → Each node appears before its dependents ✅ Cycle Detection — Undirected Graph: DFS from 0: visit 0→1→2→3 3's neighbor = 1 (already visited, not parent) → CYCLE ✅
💻 C Code — Topological Sort (DFS-based)
#include <stdio.h>
#include <string.h>
#define V 6

int adj[V][V], deg[V];
int visited[V], stack[V], top=-1;

void topoUtil(int u) {
    visited[u] = 1;
    for(int i=0;i<deg[u];i++)
        if(!visited[adj[u][i]])
            topoUtil(adj[u][i]);
    stack[++top] = u; // push after all neighbors done
}

void topologicalSort() {
    memset(visited,0,sizeof(visited));
    for(int i=0;i<V;i++)
        if(!visited[i]) topoUtil(i);
    printf("Topo Sort: ");
    while(top>=0) printf("%d ",stack[top--]);
}

/* ── Cycle Detection (Undirected) ── */
int dfsCycle(int u, int parent) {
    visited[u]=1;
    for(int i=0;i<deg[u];i++) {
        int v=adj[u][i];
        if(!visited[v]) { if(dfsCycle(v,u)) return 1; }
        else if(v != parent) return 1; // visited and not parent = cycle
    }
    return 0;
}

void addEdge(int u, int v) { adj[u][deg[u]++]=v; }

int main() {
    addEdge(5,0); addEdge(5,2); addEdge(4,0);
    addEdge(4,1); addEdge(2,3); addEdge(3,1);
    topologicalSort(); // 5 4 2 3 1 0
    return 0;
}
TOPIC 11

Minimum Spanning Tree — Kruskal's & Prim's

MST: Connects all vertices with minimum total edge weight, no cycles.
• Kruskal's: Sort edges by weight, add if no cycle (uses Union-Find) → O(E log E)
• Prim's: Grow MST from a vertex, always pick min-weight edge → O(V²)
📊 Kruskal's on weighted graph
Edges sorted by weight: (0-1,1), (1-2,2), (0-3,3), (1-3,5), (2-3,4) Step 1: Add (0-1, w=1) → MST=[0-1] Step 2: Add (1-2, w=2) → MST=[0-1,1-2] Step 3: Add (0-3, w=3) → MST=[0-1,1-2,0-3] Step 4: Skip (1-3, w=5) → would form cycle Total weight = 1+2+3 = 6 ✅
💻 C Code — Kruskal's Algorithm
#include <stdio.h>
#include <stdlib.h>
#define MAXE 100

typedef struct { int u, v, w; } Edge;
Edge edges[MAXE];
int parent[MAXE], rank_[MAXE];

int find(int x) {
    if(parent[x]!=x) parent[x]=find(parent[x]); // path compression
    return parent[x];
}

void unite(int a, int b) {
    int ra=find(a), rb=find(b);
    if(rank_[ra]<rank_[rb]) parent[ra]=rb;
    else if(rank_[ra]>rank_[rb]) parent[rb]=ra;
    else { parent[rb]=ra; rank_[ra]++; }
}

int cmpEdge(const void* a, const void* b) {
    return ((Edge*)a)->w - ((Edge*)b)->w;
}

void kruskal(int V, int E) {
    for(int i=0;i<V;i++) { parent[i]=i; rank_[i]=0; }
    qsort(edges,E,sizeof(Edge),cmpEdge);
    int total=0;
    printf("MST Edges:\n");
    for(int i=0;i<E;i++) {
        int pu=find(edges[i].u), pv=find(edges[i].v);
        if(pu!=pv) {
            printf("%d--%d (w=%d)\n",edges[i].u,edges[i].v,edges[i].w);
            total+=edges[i].w;
            unite(pu,pv);
        }
    }
    printf("Total weight = %d\n",total);
}

int main() {
    edges[0]={0,1,1}; edges[1]={1,2,2};
    edges[2]={0,3,3}; edges[3]={2,3,4};
    edges[4]={1,3,5};
    kruskal(4,5);
    return 0;
}
TOPIC 12

Single Source Shortest Path — Dijkstra & Bellman-Ford

Dijkstra's: Greedy, uses min-heap → O((V+E) log V). Only for non-negative weights.
Bellman-Ford: Relaxes all edges V-1 times → O(VE). Handles negative weights & detects negative cycles.
📊 Dijkstra — source=0

#include <stdio.h>
#include <limits.h>
#define V 5
#define INF INT_MAX

/* ── Dijkstra's Algorithm ── */
int minDist(int dist[], int visited[]) {
    int min=INF, idx=-1;
    for(int v=0;v<V;v++)
        if(!visited[v] && dist[v]<min) { min=dist[v]; idx=v; }
    return idx;
}

void dijkstra(int graph[][V], int src) {
    int dist[V], vis[V];
    for(int i=0;i<V;i++) { dist[i]=INF; vis[i]=0; }
    dist[src]=0;
    for(int c=0;c<V-1;c++) {
        int u=minDist(dist,vis); vis[u]=1;
        for(int v=0;v<V;v++)
            if(!vis[v] && graph[u][v] && dist[u]!=INF
               && dist[u]+graph[u][v]<dist[v])
                dist[v]=dist[u]+graph[u][v];
    }
    for(int i=0;i<V;i++)
        printf("0→%d = %d\n",i,dist[i]);
}

/* ── Bellman-Ford ── */
typedef struct { int u,v,w; } BEdge;

void bellmanFord(BEdge edges[], int E, int src) {
    int dist[V];
    for(int i=0;i<V;i++) dist[i]=INF;
    dist[src]=0;
    for(int i=0;i<V-1;i++)         // Relax V-1 times
        for(int j=0;j<E;j++)
            if(dist[edges[j].u]!=INF &&
               dist[edges[j].u]+edges[j].w < dist[edges[j].v])
                dist[edges[j].v]=dist[edges[j].u]+edges[j].w;
    // Check negative cycle
    for(int j=0;j<E;j++)
        if(dist[edges[j].u]+edges[j].w<dist[edges[j].v])
            { printf("Negative cycle!\n"); return; }
    for(int i=0;i<V;i++) printf("dist[%d]=%d\n",i,dist[i]);
}

int main() {
    int g[V][V]={{0,4,8,0,0},{4,0,0,1,0},
                 {8,0,0,2,0},{0,1,2,0,9},{0,0,0,9,0}};
    dijkstra(g, 0);
    return 0;
}
TOPIC 13

All Pairs Shortest Path — Floyd-Warshall

Finds shortest path between every pair of vertices using DP.
Complexity: O(V³) time, O(V²) space.
Key idea: dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j]) for each intermediate node k.
📊 Floyd-Warshall — 4 vertices
Initial dist matrix (∞=no edge): 0 1 2 3 0 [ 0, 3, ∞, 7 ] 1 [ 8, 0, 2, ∞ ] 2 [ 5, ∞, 0, 1 ] 3 [ 2, ∞, ∞, 0 ] After k=0 (use vertex 0 as intermediate): dist[1][3] = min(∞, dist[1][0]+dist[0][3]) = min(∞,8+7)=15 After all k: 0 1 2 3 0 [ 0, 3, 5, 6 ] 1 [ 5, 0, 2, 3 ] 2 [ 3, 6, 0, 1 ] 3 [ 2, 5, 7, 0 ] ✅
💻 C Code — Floyd-Warshall
#include <stdio.h>
#include <limits.h>
#define V 4
#define INF 99999

void floydWarshall(int graph[][V]) {
    int dist[V][V];

    // Initialize dist[][] from input graph
    for(int i=0;i<V;i++)
        for(int j=0;j<V;j++)
            dist[i][j]=graph[i][j];

    // Try each vertex as intermediate k
    for(int k=0;k<V;k++)
        for(int i=0;i<V;i++)
            for(int j=0;j<V;j++)
                if(dist[i][k]+dist[k][j] < dist[i][j])
                    dist[i][j]=dist[i][k]+dist[k][j];

    // Print result matrix
    printf("All Pairs Shortest Distance:\n");
    for(int i=0;i<V;i++) {
        for(int j=0;j<V;j++)
            dist[i][j]==INF ? printf(" ∞ ") : printf("%3d ",dist[i][j]);
        printf("\n");
    }
}

int main() {
    int graph[V][V] = {
        {  0,   3, INF,   7},
        {  8,   0,   2, INF},
        {  5, INF,   0,   1},
        {  2, INF, INF,   0}
    };
    floydWarshall(graph);
    return 0;
}
