Introduction to Disconnected Graph

What is a Disconnected Graph?
A graph is a set of vertices (nodes) connected by edges.

A graph is called disconnected when there is at least one pair of vertices with no path between them. The graph is split into separate groups called connected components.

In a connected graph, every vertex can reach every other vertex. In a disconnected graph, some vertices are isolated or grouped in clusters with no link to other clusters.
🌍 Real-Life Analogy
Imagine several islands. Each island has roads connecting its own cities, but there are no bridges between islands. Cities on the same island can reach each other, but cities on different islands cannot. This is a disconnected graph!

Adjacency Matrix Representation

How Adjacency Matrix Works
An adjacency matrix is a 2D array adj[V][V] where V is the number of vertices.

Rule: adj[i][j] = 1 if there is an edge from vertex i to vertex j, else 0.

For an undirected graph, the matrix is symmetric: adj[i][j] = adj[j][i].

For a disconnected graph, some rows/columns will be entirely 0 (isolated vertices), or groups of rows/columns will only connect among themselves.
📊 Real-Life Analogy
Think of a table where rows and columns are people. A "1" means those two people know each other. If person A and person B have 0s everywhere between their groups, they are in different social circles — a disconnected graph!

/* Adjacency Matrix for Disconnected Graph in C */
#include <stdio.h>
#define MAX 10   /* Maximum number of vertices */

int adj[MAX][MAX];  /* Adjacency Matrix (global, auto-initialized to 0) */
int V = 0;          /* Number of vertices */

/* Initialize graph with n vertices */
void initGraph(int n) {
    V = n;
    int i, j;
    for (i = 0; i < V; i++)
        for (j = 0; j < V; j++)
            adj[i][j] = 0;   /* All zeros = no edges */
}

/* Print the adjacency matrix */
void printMatrix() {
    int i, j;
    printf("  ");
    for (i = 0; i < V; i++) printf("%d ", i);
    printf("\n");
    for (i = 0; i < V; i++) {
        printf("%d ", i);
        for (j = 0; j < V; j++)
            printf("%d ", adj[i][j]);
        printf("\n");
    }
}

int main() {
    initGraph(5);   /* 5 vertices: 0,1,2,3,4 */
    /* Component 1: 0-1-2 */
    adj[0][1] = adj[1][0] = 1;
    adj[1][2] = adj[2][1] = 1;
    /* Component 2: 3-4 (disconnected from 0,1,2) */
    adj[3][4] = adj[4][3] = 1;
    printMatrix();
    return 0;
}

Vertex Operations

Insert Vertex Beginner
Adding a vertex means increasing V by 1 and setting the new row and column in the adjacency matrix to all zeros (no edges yet). The new vertex is initially isolated — a separate component.
🏙️ Analogy
Adding a new city to the map. It has no roads yet — it's its own little island!

/* Insert Vertex */
void insertVertex() {
    if (V >= MAX) {
        printf("Graph is full!\n");
        return;
    }
    int i;
    for (i = 0; i <= V; i++) {
        adj[V][i] = 0;  /* New row: all zeros */
        adj[i][V] = 0;  /* New column: all zeros */
    }
    V++;  /* Increase vertex count */
    printf("Vertex %d inserted. Total: %d\n", V-1, V);
}

/* Delete Vertex (last vertex for simplicity) */
void deleteVertex(int v) {
    int i, j;
    for (i = v; i < V - 1; i++)
        for (j = 0; j < V; j++)
            adj[i][j] = adj[i+1][j];  /* Shift rows up */
    for (j = v; j < V - 1; j++)
        for (i = 0; i < V - 1; i++)
            adj[i][j] = adj[i][j+1];  /* Shift columns left */
    V--;
    printf("Vertex %d deleted. Total: %d\n", v, V);
}

/* Search Vertex — just check if it exists */
int searchVertex(int v) {
    return (v >= 0 && v < V) ? 1 : 0;
}

/* Update Vertex — rename via label array */
char labels[MAX][20];
void updateVertex(int v, char* newLabel) {
    if (searchVertex(v))
        strcpy(labels[v], newLabel);
}

/* Count Vertices */
int countVertices() { return V; }

/* Display all vertices */
void displayVertices() {
    int i;
    printf("Vertices: ");
    for (i = 0; i < V; i++) printf("%d ", i);
    printf("\n");
}

Edge Operations

Insert / Delete / Search / Update / Count / Display Edges Beginner
An edge connects two vertices. In the adjacency matrix:
• Insert edge (u,v): Set adj[u][v] = 1 and adj[v][u] = 1 (undirected)
• Delete edge (u,v): Set both back to 0
• Search edge (u,v): Check if adj[u][v] == 1
• Count edges: Count 1s in upper triangle (divide by 2 for undirected)
• Display edges: Print all pairs where adj[i][j] == 1
🛣️ Analogy
Each edge is a road between two cities. Insert = build a road. Delete = remove it. Search = check if a road exists. Count = count total roads.

/* Edge Operations for Disconnected Graph */

/* Insert Edge (undirected) */
void insertEdge(int u, int v) {
    if (u < 0 || u >= V || v < 0 || v >= V) {
        printf("Invalid vertices!\n"); return;
    }
    adj[u][v] = 1;
    adj[v][u] = 1;  /* Undirected: both directions */
    printf("Edge (%d-%d) inserted.\n", u, v);
}

/* Delete Edge */
void deleteEdge(int u, int v) {
    if (!adj[u][v]) { printf("Edge not found!\n"); return; }
    adj[u][v] = adj[v][u] = 0;
    printf("Edge (%d-%d) deleted.\n", u, v);
}

/* Search Edge */
int searchEdge(int u, int v) {
    return adj[u][v];  /* 1=exists, 0=not exists */
}

/* Update Edge weight (for weighted graph) */
void updateEdge(int u, int v, int weight) {
    if (adj[u][v]) {
        adj[u][v] = adj[v][u] = weight;
        printf("Edge (%d-%d) updated to %d.\n", u, v, weight);
    }
}

/* Count Edges */
int countEdges() {
    int count = 0, i, j;
    for (i = 0; i < V; i++)
        for (j = i + 1; j < V; j++)  /* Upper triangle only */
            if (adj[i][j]) count++;
    return count;
}

/* Display all Edges */
void displayEdges() {
    int i, j;
    printf("Edges: ");
    for (i = 0; i < V; i++)
        for (j = i + 1; j < V; j++)
            if (adj[i][j]) printf("(%d-%d) ", i, j);
    printf("\n");
}
05

Breadth First Search (BFS)

BFS — Queue, Visit, Multi-Component Intermediate
BFS explores a graph level by level using a Queue. Starting from a vertex:
1. Enqueue the start vertex and mark it visited
2. Dequeue a vertex, print it, enqueue all unvisited neighbors
3. Repeat until queue is empty

For a disconnected graph, a single BFS won't visit all vertices. We must loop through all vertices and start BFS from each unvisited vertex — this ensures every component is traversed (Multi-Component BFS).
🏄 Analogy
Imagine spreading water on a floor. Water spreads level by level from the source. But on disconnected islands, water can't cross the gap — so you must pour water on each island separately!

C
/* BFS for Disconnected Graph */
#include <stdio.h>

int visited[MAX];
int queue[MAX], front = -1, rear = -1;

/* Queue helpers */
void enqueue(int v) { queue[++rear] = v; }
int  dequeue()    { return queue[++front]; }
int  isEmpty()    { return front == rear; }

/* BFS from a single source */
void bfs(int start) {
    front = rear = -1;
    enqueue(start);
    visited[start] = 1;
    while (!isEmpty()) {
        int v = dequeue();
        printf("%d ", v);
        int i;
        for (i = 0; i < V; i++) {
            if (adj[v][i] == 1 && !visited[i]) {
                visited[i] = 1;
                enqueue(i);
            }
        }
    }
}

/* Multi-Component BFS — covers ALL vertices */
void bfsAll() {
    int i, comp = 0;
    for (i = 0; i < V; i++) visited[i] = 0;
    for (i = 0; i < V; i++) {
        if (!visited[i]) {
            printf("Component %d: ", ++comp);
            bfs(i);
            printf("\n");
        }
    }
    printf("Total components: %d\n", comp);
}

Depth First Search (DFS)

DFS — Recursive, Stack, Backtracking, Multi-Component Intermediate
DFS dives as deep as possible into the graph before backtracking. It uses a stack (or recursion).

Algorithm:
1. Mark the current vertex as visited, print it
2. Recurse on all unvisited neighbors
3. Backtrack when no unvisited neighbors remain

For disconnected graphs, loop through all vertices and call DFS on each unvisited one (Multi-Component DFS).
🗺️ Analogy
Exploring a maze: always go as far as possible in one direction. Hit a dead end? Backtrack and try another path. For disconnected islands, you must start fresh on each island.

/* DFS — Recursive approach */
void dfsRecursive(int v) {
    visited[v] = 1;
    printf("%d ", v);
    int i;
    for (i = 0; i < V; i++)
        if (adj[v][i] && !visited[i])
            dfsRecursive(i);  /* Go deeper */
    /* Backtracking happens automatically when function returns */
}

/* DFS — Iterative with Stack */
void dfsStack(int start) {
    int stack[MAX], top = -1, i;
    stack[++top] = start;
    visited[start] = 1;
    while (top >= 0) {
        int v = stack[top--];
        printf("%d ", v);
        for (i = V - 1; i >= 0; i--)
            if (adj[v][i] && !visited[i]) {
                visited[i] = 1;
                stack[++top] = i;
            }
    }
}

/* Multi-Component DFS */
void dfsAll() {
    int i, comp = 0;
    for (i = 0; i < V; i++) visited[i] = 0;
    for (i = 0; i < V; i++) {
        if (!visited[i]) {
            printf("Component %d: ", ++comp);
            dfsRecursive(i);
            printf("\n");
        }
    }
}

Connectivity Operations

Check Disconnected, Count Components, Find Components, Isolated Vertices, Reachability, Weak/Strong Intermediate
Check Disconnected: Run BFS/DFS from vertex 0. If not all vertices are visited → graph is disconnected.

Count Connected Components: Count how many times we start a fresh BFS/DFS in multi-component traversal.

Find Connected Components: Label each vertex with its component number.

Isolated Vertex: A vertex with degree 0 — no edges at all.

Reachability: Check if vertex u can reach vertex v using BFS/DFS.

Weak Connectivity: In a directed graph, treat all edges as undirected — is it connected?

Strong Connectivity: Every vertex can reach every other vertex (directed graph).
🌐 Analogy
Internet subnets: devices in the same subnet talk freely; devices in different subnets need a router (bridge). Isolated devices are like computers with no network card.

/* Connectivity Operations */

/* Check if graph is disconnected */
int isDisconnected() {
    int i;
    for (i = 0; i < V; i++) visited[i] = 0;
    bfs(0);
    for (i = 0; i < V; i++)
        if (!visited[i]) return 1;  /* Found unvisited = disconnected */
    return 0;
}

/* Count connected components */
int countComponents() {
    int i, count = 0;
    for (i = 0; i < V; i++) visited[i] = 0;
    for (i = 0; i < V; i++)
        if (!visited[i]) { bfs(i); count++; }
    return count;
}

/* Find component of each vertex */
int compLabel[MAX];
void findComponents() {
    int i, comp = 0;
    for (i = 0; i < V; i++) visited[i] = -1;
    for (i = 0; i < V; i++)
        if (visited[i] == -1) {
            bfsLabel(i, comp++);
        }
}

/* Detect isolated vertices (degree 0) */
void detectIsolated() {
    int i, j, deg;
    for (i = 0; i < V; i++) {
        deg = 0;
        for (j = 0; j < V; j++) deg += adj[i][j];
        if (deg == 0) printf("Vertex %d is isolated\n", i);
    }
}

/* Reachability: can u reach v? */
int canReach(int u, int v) {
    int i;
    for (i = 0; i < V; i++) visited[i] = 0;
    bfs(u);
    return visited[v];
}

Component Operations

Traverse, Label, Merge, Separate, Size Analysis Intermediate
Traverse Each Component: Use multi-component BFS/DFS to visit each independently.

Label Components: Assign each vertex a component ID (0, 1, 2, ...).

Merge Components: Add an edge between two vertices from different components.

Separate Components: Remove all edges crossing between two groups.

Component Size Analysis: Count vertices (or edges) in each component to find the largest/smallest.
🧩 Analogy
Jigsaw puzzle pieces: each connected piece is a component. Merging = connecting two pieces. Separating = breaking them apart. Size = how big each cluster is.

/* Component Operations */

/* Label each vertex with its component ID */
int compID[MAX];
void bfsLabel(int start, int id) {
    front = rear = -1;
    enqueue(start); visited[start] = 1; compID[start] = id;
    while (!isEmpty()) {
        int v = dequeue(), i;
        for (i = 0; i < V; i++)
            if (adj[v][i] && !visited[i]) {
                visited[i] = 1; compID[i] = id;
                enqueue(i);
            }
    }
}

/* Merge two components by adding an edge */
void mergeComponents(int u, int v) {
    insertEdge(u, v);
    printf("Merged components of %d and %d\n", u, v);
}

/* Separate components by removing edges between two groups */
void separateComponents(int compA, int compB) {
    int i, j;
    for (i = 0; i < V; i++)
        for (j = 0; j < V; j++)
            if (compID[i]==compA && compID[j]==compB)
                adj[i][j] = adj[j][i] = 0;
}

/* Component Size Analysis */
void componentSizeAnalysis() {
    int i, numComp = countComponents();
    int sizes[MAX] = {0};
    /* Relabel after counting */
    for (i = 0; i < V; i++) sizes[compID[i]]++;
    for (i = 0; i < numComp; i++)
        printf("Component %d: %d vertices\n", i, sizes[i]);
}

Path Operations

Find Path, Shortest Path, All Paths, Reachable/Unreachable Intermediate
Find Path (u→v): Use DFS, tracking the path array. Stop when you reach v.

Shortest Path: Use BFS — BFS naturally finds the shortest path in unweighted graphs (fewest edges).

All Paths: Use recursive DFS with backtracking to find every path from u to v.

Reachable Path Check: BFS from u — if v is visited, it's reachable.

Unreachable Node Detection: After BFS from u, any vertex still unvisited is unreachable from u (may be in a different component).
🗺️ Analogy
GPS navigation: Find a route (path), the quickest route (shortest path), all possible routes (all paths). No GPS signal = unreachable node (different island)!

/* Path Operations */
int path[MAX], pathLen = 0;

/* Find ANY path from src to dst using DFS */
int findPath(int src, int dst) {
    path[pathLen++] = src;
    visited[src] = 1;
    if (src == dst) {
        int i;
        for (i = 0; i < pathLen; i++) printf("%d ", path[i]);
        return 1;
    }
    int i;
    for (i = 0; i < V; i++)
        if (adj[src][i] && !visited[i])
            if (findPath(i, dst)) return 1;
    pathLen--;  /* Backtrack */
    return 0;
}

/* Shortest Path using BFS (unweighted) */
int shortestPath(int src, int dst) {
    int dist[MAX], parent[MAX], i;
    for (i = 0; i < V; i++) { dist[i] = -1; visited[i] = 0; }
    dist[src] = 0; parent[src] = -1;
    front = rear = -1;
    enqueue(src); visited[src] = 1;
    while (!isEmpty()) {
        int v = dequeue();
        for (i = 0; i < V; i++)
            if (adj[v][i] && !visited[i]) {
                visited[i] = 1; dist[i] = dist[v]+1;
                parent[i] = v; enqueue(i);
            }
    }
    return dist[dst];  /* -1 if unreachable */
}

/* Unreachable node detection */
void findUnreachable(int src) {
    int i;
    for (i = 0; i < V; i++) visited[i] = 0;
    bfs(src);
    printf("Unreachable from %d: ", src);
    for (i = 0; i < V; i++)
        if (!visited[i]) printf("%d ", i);
}

Cycle Operations

Detect, Count, Check Cyclic Components, Remove Cycle Intermediate
A cycle is a path that starts and ends at the same vertex.

Detect Cycle (undirected): During DFS, if we visit a neighbor that is already visited and is NOT the parent, a cycle exists.

Count Cycles: Count components where cycles are detected.

Cyclic Component: A component is cyclic if edges > vertices - 1 (i.e., has a cycle).

Remove Cycle: Find the back-edge creating the cycle and delete it.
🔄 Analogy
A one-way street system where you can drive in a loop back to where you started. Detecting cycles is like finding traffic roundabouts. Removing them means converting those circles into dead-end streets.
C
/* Cycle Detection (Undirected Graph) */
int dfsHasCycle(int v, int parent) {
    visited[v] = 1;
    int i;
    for (i = 0; i < V; i++) {
        if (adj[v][i]) {
            if (!visited[i]) {
                if (dfsHasCycle(i, v)) return 1;
            } else if (i != parent) {
                return 1;  /* Back-edge found = cycle! */
            }
        }
    }
    return 0;
}

/* Check if graph (or any component) has a cycle */
int detectCycle() {
    int i;
    for (i = 0; i < V; i++) visited[i] = 0;
    for (i = 0; i < V; i++)
        if (!visited[i])
            if (dfsHasCycle(i, -1)) return 1;
    return 0;
}

/* Check if a specific component (rooted at start) is cyclic */
int isCyclicComponent(int start) {
    int i; for (i=0;i<V;i++) visited[i]=0;
    return dfsHasCycle(start, -1);
}

/* Remove a cycle edge (the back-edge that causes the cycle) */
int cycleU = -1, cycleV = -1;
int dfsFindCycleEdge(int v, int par) {
    visited[v] = 1;
    int i;
    for (i = 0; i < V; i++) {
        if (adj[v][i] && !visited[i]) {
            if (dfsFindCycleEdge(i, v)) return 1;
        } else if (adj[v][i] && i != par) {
            cycleU = v; cycleV = i;
            return 1;
        }
    }
    return 0;
}
void removeCycle() {
    int i; for(i=0;i<V;i++) visited[i]=0;
    if (dfsFindCycleEdge(0,-1))
        adj[cycleU][cycleV] = adj[cycleV][cycleU] = 0;
}

Degree Operations

Degree, In-Degree, Out-Degree, Max/Min Degree Beginner
The degree of a vertex is the number of edges connected to it.

• Undirected graph: Degree = sum of that row in the adjacency matrix
• Directed graph — In-Degree: Sum of that column
• Directed graph — Out-Degree: Sum of that row
• Max/Min Degree: Find the vertex with highest/lowest degree
• Isolated vertex: Degree = 0
📱 Analogy
In a social network, degree = number of friends. In-degree = how many follow you. Out-degree = how many you follow. The most popular person has the highest in-degree!

Structural Operations

Size, Order, Density, Sparse/Dense, Forest Validation, Component Analysis Intermediate
Order (Graph Order): Total number of vertices (V).

Size (Graph Size): Total number of edges (E).

Density: Ratio of actual edges to maximum possible edges: D = 2E / (V*(V-1))

Sparse: Density close to 0 — few edges. Dense: Density close to 1 — many edges.

Forest Validation: A disconnected graph is a forest if it has no cycles and is acyclic (each component is a tree).

Component Analysis: Analyze structure of each component (size, edges, cyclic/acyclic).
🌲 Analogy
A forest is a group of trees — each tree is connected (no cycles), but different trees don't connect. Dense graph = a city with roads everywhere. Sparse = remote villages with very few roads.
C
/* Structural Operations */

/* Graph Order = number of vertices */
int graphOrder() { return V; }

/* Graph Size = number of edges */
int graphSize() { return countEdges(); }

/* Graph Density (0 to 1, undirected) */
float graphDensity() {
    int E = graphSize();
    if (V <= 1) return 0.0;
    return (float)(2 * E) / (V * (V - 1));
}

/* Sparse/Dense check */
void checkDensity() {
    float d = graphDensity();
    printf("Density: %.2f\n", d);
    if (d < 0.5) printf("Graph is SPARSE\n");
    else        printf("Graph is DENSE\n");
}

/* Forest Validation: disconnected + acyclic = Forest */
int isForest() {
    /* A forest has E = V - numComponents */
    int E = graphSize();
    int C = countComponents();
    return (E == V - C) ? 1 : 0;
}

/* Component Analysis */
void componentAnalysis() {
    int numComp = countComponents();
    printf("Components: %d\n", numComp);
    printf("Forest: %s\n", isForest() ? "YES" : "NO");
    checkDensity();
}

Advanced Algorithms

Union-Find / DSU (Disjoint Set Union) Advanced
Union-Find is a data structure that efficiently tracks which component each vertex belongs to.

Find(v): Returns the root/representative of v's component.
Union(u,v): Merges the components of u and v.

Uses path compression (shortcut pointers to root) and union by rank (attach smaller tree under larger) for near O(1) per operation.
🏫 Analogy
School clubs: each student belongs to one club. Union = two clubs merge. Find = which club does this student belong to? Path compression = direct link to the president instead of going through every member.
C
/* Union-Find / Disjoint Set Union (DSU) */
int parent[MAX], rnk[MAX];

/* Initialize: each vertex is its own component */
void dsuInit() {
    int i;
    for (i = 0; i < V; i++) { parent[i] = i; rnk[i] = 0; }
}

/* Find root with PATH COMPRESSION */
int find(int x) {
    if (parent[x] != x)
        parent[x] = find(parent[x]);  /* Compress path */
    return parent[x];
}

/* Union by rank */
void unionSets(int u, int v) {
    int pu = find(u), pv = find(v);
    if (pu == pv) return;  /* Already same component */
    if (rnk[pu] < rnk[pv]) parent[pu] = pv;
    else if (rnk[pu] > rnk[pv]) parent[pv] = pu;
    else { parent[pv] = pu; rnk[pu]++; }
}

/* Build DSU from adjacency matrix */
void buildDSU() {
    int i, j;
    dsuInit();
    for (i = 0; i < V; i++)
        for (j = i+1; j < V; j++)
            if (adj[i][j]) unionSets(i, j);
}

/* Count components using DSU */
int dsuCountComponents() {
    buildDSU();
    int i, count = 0;
    for (i = 0; i < V; i++)
        if (find(i) == i) count++;
    return count;
}
Strongly Connected Components (Kosaraju's Algorithm) Advanced
In a directed graph, a Strongly Connected Component (SCC) is a maximal group of vertices where every vertex can reach every other vertex in the group.

Kosaraju's Algorithm (2 DFS passes):
1. Run DFS on original graph, store vertices in finish-order stack
2. Transpose the graph (reverse all edge directions)
3. Run DFS on transposed graph in reverse finish order — each DFS tree is one SCC
✈️ Analogy
Cities with one-way flights: an SCC is a set of cities where you can fly from any city to any other city (possibly via stops). If two cities are in different SCCs, you can't make a round-trip between them.
C
/* Kosaraju's SCC Algorithm */
int stk[MAX], stkTop = -1;

/* Pass 1: DFS and push to stack by finish time */
void dfs1(int v) {
    visited[v] = 1;
    int i;
    for (i = 0; i < V; i++)
        if (adj[v][i] && !visited[i]) dfs1(i);
    stk[++stkTop] = v;  /* Push on finish */
}

/* Pass 2: DFS on transposed graph */
int adjT[MAX][MAX];
void dfs2(int v) {
    visited[v] = 1;
    printf("%d ", v);
    int i;
    for (i = 0; i < V; i++)
        if (adjT[v][i] && !visited[i]) dfs2(i);
}

void kosarajuSCC() {
    int i, j;
    for (i = 0; i < V; i++) visited[i] = 0;
    for (i = 0; i < V; i++)
        if (!visited[i]) dfs1(i);  /* Pass 1 */
    /* Transpose graph */
    for (i = 0; i < V; i++)
        for (j = 0; j < V; j++)
            adjT[i][j] = adj[j][i];
    for (i = 0; i < V; i++) visited[i] = 0;
    int scc = 0;
    while (stkTop >= 0) {
        int v = stk[stkTop--];
        if (!visited[v]) {
            printf("SCC %d: ", ++scc);
            dfs2(v); printf("\n");
        }
    }
}
Graph Coloring (Greedy) Advanced
Graph Coloring assigns colors to vertices so no two adjacent vertices share the same color, using the fewest colors possible.

Greedy Coloring: For each vertex, assign the smallest color not used by any neighbor. This is per-component — disconnected components can reuse the same color set independently.
📅 Analogy
Scheduling exams: vertices = subjects, edges = students taking both subjects, colors = time slots. No two exams a student takes can be at the same time. Disconnected groups can share time slots!
C
/* Greedy Graph Coloring */
int color[MAX];

void greedyColoring() {
    int i, j;
    int available[MAX];
    for (i = 0; i < V; i++) color[i] = -1;
    color[0] = 0;  /* Assign first color to first vertex */
    for (i = 1; i < V; i++) {
        /* Mark colors used by neighbors */
        for (j = 0; j < V; j++) available[j] = 1;
        for (j = 0; j < V; j++)
            if (adj[i][j] && color[j] != -1)
                available[color[j]] = 0;  /* Neighbor uses this color */
        /* Find first available color */
        for (j = 0; j < V; j++)
            if (available[j]) { color[i] = j; break; }
    }
    for (i = 0; i < V; i++)
        printf("Vertex %d -> Color %d\n", i, color[i]);
}
Spanning Forest (Kruskal's / Prim's per component) Advanced
A Spanning Forest of a disconnected graph is a collection of spanning trees, one per connected component.

For each component, a spanning tree is a subgraph that connects all vertices of that component with the minimum number of edges (V_comp - 1 edges) and no cycles.

Algorithm: Run BFS/DFS per component, recording only the tree edges (edges that first discover a new vertex).
🌲 Analogy
Build the minimum road network to connect every city within each island — but don't build bridges between islands (different components). The result is a forest of minimum spanning trees.
C
/* Spanning Forest using BFS tree edges */
int forestEdgeU[MAX], forestEdgeV[MAX], forestEdgeCount = 0;

void bfsSpanTree(int start) {
    front = rear = -1;
    enqueue(start); visited[start] = 1;
    while (!isEmpty()) {
        int v = dequeue(), i;
        for (i = 0; i < V; i++) {
            if (adj[v][i] && !visited[i]) {
                visited[i] = 1;
                /* Record tree edge */
                forestEdgeU[forestEdgeCount] = v;
                forestEdgeV[forestEdgeCount] = i;
                forestEdgeCount++;
                enqueue(i);
            }
        }
    }
}

void buildSpanningForest() {
    int i; forestEdgeCount = 0;
    for (i = 0; i < V; i++) visited[i] = 0;
    for (i = 0; i < V; i++)
        if (!visited[i]) bfsSpanTree(i);
    printf("Spanning Forest Edges:\n");
    for (i = 0; i < forestEdgeCount; i++)
        printf("  %d - %d\n", forestEdgeU[i], forestEdgeV[i]);
}
