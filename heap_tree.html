What is a Heap Tree? Beginner
Understanding the foundation before writing a single line of code
🏥

Real-Life Analogy: Hospital Emergency Room

Imagine an ER where patients are treated by severity, not arrival time. The most critical patient (max heap) is always at the front. When one leaves, the next most critical takes their place automatically. That's exactly how a max heap works — the "most important" element always stays at the top.

🌳

Complete Binary Tree

A heap is stored as a complete binary tree — all levels filled left to right.

📦

Array Storage

The entire tree lives inside a simple array. No pointers needed!

📜

Heap Property

Every parent is ≥ its children (max heap) or ≤ its children (min heap).

⚡

O(log n) Operations

Insert and delete in logarithmic time — extremely fast even for large data.

▲ Max Heap
  • Root = Maximum element
  • Parent ≥ Both children always
  • Used in: Heap Sort, Priority Queue
  • Top element: arr[0] = largest
▼ Min Heap
  • Root = Minimum element
  • Parent ≤ Both children always
  • Used in: Dijkstra's Algorithm, Scheduling
  • Top element: arr[0] = smallest
📐 Array Index Formulas (0-based index)
For node at index i:
→ Parent index: floor((i - 1) / 2)
→ Left child index: (2 * i) + 1
→ Right child index: (2 * i) + 2

Create Heap Basic
Initialize the heap structure — set up the array, size, and choose heap type
🏗️

Analogy: Building a Waiting Room

Creating a heap is like setting up a waiting room. You decide the maximum capacity (MAX_SIZE), put up a sign saying "Max Priority" or "Min Priority", and start with zero people (size = 0). The room (array) is empty but ready!

📋 Sub-Operations

  • Initialize array with MAX_SIZE
  • Set heap size to 0
  • Choose heap type (Max / Min)
  • Allocate memory for array

💡 Key Points

An empty heap is valid. We track size separately from capacity. The array is pre-allocated. Both max and min heaps use the same array structure — only the comparison changes!

heap_create.c C Language 
/* === CREATE HEAP: Initialize a max heap === */

#define MAX_SIZE 100  /* Maximum elements heap can hold */

/* Structure to represent our heap */
struct Heap {
    int arr[MAX_SIZE];  /* Array to store heap elements */
    int size;           /* Current number of elements */
    int heapType;       /* 1 = Max Heap, 0 = Min Heap */
};

/* Function to create and initialize a heap */
void createHeap(struct Heap *h, int type) {
    h->size = 0;       /* Empty heap to start */
    h->heapType = type; /* Set heap type */
    /* Array memory is already reserved by the struct */
}

/* In main: */
int main() {
    struct Heap myHeap;
    createHeap(&myHeap, 1); /* 1 = Max Heap */

    printf("Heap created! Type: %s, Size: %d\n",
           myHeap.heapType ? "Max Heap" : "Min Heap",
           myHeap.size);
    return 0;
}

Insert Element Basic
Add a new value to the heap while maintaining the heap property
🎟️

Analogy: Joining a Priority Queue

A new patient enters the ER. They first sit at the last chair available (end of array). Then they're assessed — if more critical than the person ahead of them, they swap seats. They keep swapping until they reach the right priority level. This is "Bubble Up"!

1
Add at End

Place new element at arr[size] and increment size by 1.

2
Find Parent

Parent of node at index i is at index (i-1)/2.

3
Compare with Parent

If new element > parent (for max heap), they violate heap property.

4
Swap if Needed

Swap the new element with parent. Move index up to parent position.

5
Repeat

Continue until element is ≤ parent (max heap) or reaches root (index 0).

insert_element.c C Language 
/* === INSERT ELEMENT into Max Heap === */

/* Swap two integers using a temp variable */
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

/* Bubble Up: move element up until heap is valid */
void bubbleUp(struct Heap *h, int index) {
    int parent = (index - 1) / 2; /* Parent index formula */

    /* Keep swapping while current > parent (max heap) */
    while (index > 0 && h->arr[index] > h->arr[parent]) {
        swap(&h->arr[index], &h->arr[parent]);
        index = parent;              /* Move up */
        parent = (index - 1) / 2; /* Recalculate parent */
    }
}

/* Insert element into heap */
void insert(struct Heap *h, int value) {
    if (h->size >= MAX_SIZE) {
        printf("Heap is full! Cannot insert %d\n", value);
        return;
    }
    /* Step 1: Add at end */
    h->arr[h->size] = value;
    h->size++;

    /* Step 2-5: Bubble Up to fix heap property */
    bubbleUp(h, h->size - 1);

    printf("Inserted %d. Heap size: %d\n", value, h->size);
}

Delete Root Intermediate
Remove the root node while restoring the heap property using Heapify Down
🏆

Analogy: Champion Leaves, Last Player Fills In

The champion (root) wins and exits. The last person in line is moved to champion's spot temporarily. Then we run a "tournament bracket" — they compete down the tree, swapping with the stronger child at each level, until they find the right position. This is Heapify Down (Bubble Down).

1
Save Root Value

Store arr[0] to return it later. This is the element we're deleting.

2
Replace Root with Last

Put the last element (arr[size-1]) into the root position (arr[0]).

3
Decrease Size

Reduce heap size by 1 to "remove" the last element.

4
Heapify Down

Compare root with its children. Swap with the larger child (max heap) if needed.

5
Repeat Until Fixed

Continue swapping downward until heap property is restored.

delete_root.c C Language 
/* === DELETE ROOT from Max Heap (Heapify Down) === */

void heapifyDown(struct Heap *h, int index) {
    int largest = index;           /* Assume current is largest */
    int left  = 2 * index + 1;    /* Left child index */
    int right = 2 * index + 2;   /* Right child index */

    /* Check if left child is larger */
    if (left < h->size && h->arr[left] > h->arr[largest])
        largest = left;

    /* Check if right child is even larger */
    if (right < h->size && h->arr[right] > h->arr[largest])
        largest = right;

    /* If largest is not current, swap and recurse down */
    if (largest != index) {
        swap(&h->arr[index], &h->arr[largest]);
        heapifyDown(h, largest); /* Recurse on affected subtree */
    }
}

/* Delete the root element */
int deleteRoot(struct Heap *h) {
    if (h->size == 0) {
        printf("Heap is empty!\n");
        return -1;
    }
    int root = h->arr[0];         /* Step 1: Save root */
    h->arr[0] = h->arr[h->size-1]; /* Step 2: Move last to root */
    h->size--;                     /* Step 3: Shrink heap */
    heapifyDown(h, 0);           /* Step 4-5: Fix heap property */

    printf("Deleted root: %d\n", root);
    return root;
}

Extract Max / Extract Min Intermediate
Access and remove the top priority element — the heart of priority queue operations
🎰

Analogy: The Jackpot Machine

Extract Max is like pulling the top prize from a jackpot machine — you get the best item, and the machine instantly reorganizes the remaining prizes so the next best is always ready at the top. You always know exactly which slot has the highest prize without searching!

⬆️ Extract Max (from Max Heap)

  • Access arr[0] — always the maximum
  • Remove root (same as Delete Root)
  • Reheapify to restore heap
  • Return the extracted maximum value

⬇️ Extract Min (from Min Heap)

  • Access arr[0] — always the minimum
  • Same process, different heap type
  • Reheapify using min comparison
  • Return the extracted minimum value
extract_max_min.c C Language 
/* === EXTRACT MAX from Max Heap === */

int extractMax(struct Heap *h) {
    if (h->size == 0) {
        printf("Error: Heap is empty!\n");
        return -1;
    }
    /* Root always holds the maximum */
    int maxVal = h->arr[0];

    /* Remove root (reuses deleteRoot logic) */
    h->arr[0] = h->arr[h->size - 1];
    h->size--;
    heapifyDown(h, 0);  /* Restore heap property */

    printf("Extracted Max: %d\n", maxVal);
    return maxVal;  /* Return the extracted value */
}

/* === EXTRACT MIN from Min Heap === */
/* Note: heapifyDownMin uses < instead of > in comparisons */

void heapifyDownMin(struct Heap *h, int index) {
    int smallest = index;
    int left  = 2 * index + 1;
    int right = 2 * index + 2;

    if (left < h->size && h->arr[left] < h->arr[smallest])
        smallest = left;
    if (right < h->size && h->arr[right] < h->arr[smallest])
        smallest = right;

    if (smallest != index) {
        swap(&h->arr[index], &h->arr[smallest]);
        heapifyDownMin(h, smallest);
    }
}

int extractMin(struct Heap *h) {
    if (h->size == 0) { printf("Heap empty!\n"); return -1; }
    int minVal = h->arr[0];  /* Root = minimum */
    h->arr[0] = h->arr[h->size - 1];
    h->size--;
    heapifyDownMin(h, 0);
    return minVal;
}

Peek Root Basic
View the top element without removing it — the fastest heap operation!
👁️

Analogy: Checking Who's First in Line

Peek is like walking to the front of a queue and looking at who's next — without actually taking them out. You just look. Nobody moves. The queue stays the same. In a heap, arr[0] is always the highest priority, and peeking costs nothing!

Heapify Intermediate
The core algorithm — fix heap property by comparing parent and children, then swapping
⚖️

Analogy: The Referee's Job

Think of heapify as a referee checking if the tournament bracket is fair. At each level, the referee checks: "Is the parent stronger than both children?" If not, swap the parent with the stronger child, then re-check that position. This cascades down the tree until everything is correct.

⬆️ Heapify Up (Bubble Up)

  • Used after Insert
  • Starts from leaf, goes to root
  • Compares with parent
  • Stops when parent is larger

⬇️ Heapify Down (Bubble Down)

  • Used after Delete Root
  • Starts from root, goes to leaf
  • Compares with both children
  • Swaps with larger child (max heap)
heapify.c C Language 
/* === HEAPIFY: Both directions === */

/* ── HEAPIFY UP: Fix after insert (Recursive) ── */
void heapifyUp(struct Heap *h, int index) {
    /* Base case: reached root */
    if (index == 0) return;

    int parent = (index - 1) / 2;

    /* If current node > parent, swap and recurse */
    if (h->arr[index] > h->arr[parent]) {
        swap(&h->arr[index], &h->arr[parent]);
        heapifyUp(h, parent);  /* Recurse upward */
    }
}

/* ── HEAPIFY DOWN: Fix after delete (Iterative) ── */
void heapifyDownIter(struct Heap *h, int index) {
    while (1) {
        int largest = index;
        int left  = 2 * index + 1;
        int right = 2 * index + 2;

        if (left  < h->size && h->arr[left]  > h->arr[largest])
            largest = left;
        if (right < h->size && h->arr[right] > h->arr[largest])
            largest = right;

        if (largest == index) break; /* Heap is valid, stop */

        swap(&h->arr[index], &h->arr[largest]);
        index = largest;  /* Move to child position */
    }
}

build_heap.c C Language 
/* === BUILD HEAP from existing array (Bottom-Up) === */
/* This is MORE efficient than inserting one by one! */
/* Time: O(n)  vs  O(n log n) for repeated insert */

void buildHeap(struct Heap *h, int *input, int n) {
    /* Copy input array into heap */
    int i;
    for (i = 0; i < n; i++) {
        h->arr[i] = input[i];
    }
    h->size = n;

    /* Start from last non-leaf node and go up */
    /* Last non-leaf = (n/2) - 1 */
    for (i = n / 2 - 1; i >= 0; i--) {
        heapifyDown(h, i);  /* Fix subtree rooted at i */
    }

    printf("Heap built from %d elements!\n", n);
}

/* Usage Example: */
int data[] = {3, 1, 6, 5, 2, 4}; /* Unsorted */
int n = sizeof(data) / sizeof(data[0]);
buildHeap(&myHeap, data, n);
/* Result: arr[0] = 6 (the maximum) */

Traverse Heap Basic
Visit and display all elements — heaps are naturally stored for level-order traversal
📺

Analogy: Reading a Family Tree Top-Down

Traversal is like reading a family tree starting from the grandparent, then all their children, then grandchildren — level by level. Since a heap is stored as an array, level-order traversal is just printing arr[0] through arr[size-1] in order!

traverse_heap.c C Language 
/* === TRAVERSE HEAP: Print all elements === */

/* Level-order traversal: just print the array! */
void traverseHeap(struct Heap *h) {
    if (isEmpty(h)) {
        printf("Heap is empty!\n");
        return;
    }
    int i;
    printf("Heap elements (level-order): ");
    for (i = 0; i < h->size; i++) {
        printf("%d ", h->arr[i]);
    }
    printf("\n");
}

/* Display as tree structure (visual) */
void displayTree(struct Heap *h) {
    int i = 0, level = 1, count = 0;
    while (i < h->size) {
        printf("Level %d: ", level);
        while (count < level && i < h->size) {
            printf("%d  ", h->arr[i++]);
            count++;
        }
        printf("\n");
        level *= 2;  /* Each level has 2x nodes */
        count = 0;
    }
}

/* Output example for [90,75,80,55,60,65,70]:
   Level 1: 90

Analogy: Looking for a Person in a Theater

Heaps guarantee the best seat is always in row 1, but there's no guarantee about where everyone else sits. To find a specific person, you have to walk row by row (linear search). Unlike a sorted array, you can't "skip" rows — you must check each seat.

/* === SEARCH ELEMENT in Heap (Linear Search) === */

int searchElement(struct Heap *h, int target) {
    int i;
    for (i = 0; i < h->size; i++) {

Update Key Intermediate
Change a value at a given index and restore heap property in the right direction
✏️

Analogy: Changing a Patient's Priority

A patient's condition changes. If they got worse (value increased in max heap), they move toward the front — bubble up. If they got better (value decreased), they move toward the back — bubble down. The system automatically adjusts their position!

update_key.c C Language 
/* === UPDATE KEY at given index === */

void updateKey(struct Heap *h, int index, int newVal) {
    if (index < 0 || index >= h->size) {
        printf("Invalid index!\n");
        return;
    }
    int oldVal = h->arr[index];
    h->arr[index] = newVal;  /* Modify value */

    if (newVal > oldVal) {
        /* Value increased → bubble up (for max heap) */
        heapifyUp(h, index);
    } else {
        /* Value decreased → bubble down */
        heapifyDown(h, index);
    }
    printf("Updated index %d: %d → %d\n", index, oldVal, newVal);
}

Increase Key Intermediate
Increase a node's value — always requires Heapify Up to maintain max heap property
📈

Analogy: Employee Gets a Promotion

An employee gets promoted (value increases). They now outrank some managers above them, so they swap positions upward until they reach their correct rank in the hierarchy. This is Heapify Up.

increase_key.c C Language 
/* === INCREASE KEY: value goes up → heapify up === */

void increaseKey(struct Heap *h, int index, int newVal) {
    if (index < 0 || index >= h->size) {
        printf("Invalid index!\n");
        return;
    }
    if (newVal < h->arr[index]) {
        printf("Error: New value must be >= old value!\n");
        return;  /* Can't decrease in an increase operation */
    }
    h->arr[index] = newVal;  /* Modify value */
    heapifyUp(h, index);     /* Bubble UP to fix property */

    printf("Key at index %d increased to %d\n", index, newVal);
}

   → Must move UP the tree (heapify up) */

Replace Element Advanced
Remove an old value and insert a new one, then restore heap property
🔄

Analogy: Replacing a Team Member

A team member leaves and a new person joins in their exact seat. Depending on whether the new person is stronger or weaker than those around them, they'll move up or down the corporate ladder. The team structure adjusts automatically.

replace_element.c C Language 
/* === REPLACE ELEMENT: Remove old, insert new === */

void replaceElement(struct Heap *h, int oldVal, int newVal) {
    /* Step 1: Find the old value */
    int index = searchElement(h, oldVal);

    if (index == -1) {
        printf("Value %d not found!\n", oldVal);
        return;
    }

    int oldValue = h->arr[index];
    /* Step 2: Insert new value at that position */
    h->arr[index] = newVal;

    /* Step 3: Heapify based on change direction */
    if (newVal > oldValue) {
        heapifyUp(h, index);   /* New > Old → go up */
    } else if (newVal < oldValue) {
        heapifyDown(h, index); /* New < Old → go down */
    }
    /* If equal, no reheapify needed */
    printf("Replaced %d with %d\n", oldVal, newVal);
}

Merge Heaps Advanced
Combine two heaps into one valid heap — merge arrays and rebuild from scratch
🤝

Analogy: Two Companies Merging

Two companies with their own hierarchies merge. All employees join one building (combine arrays), but the org chart is now a mess. To fix it, you run the Build Heap algorithm — reorganize everyone bottom-up into a proper new hierarchy.

merge_heaps.c C Language 
/* === MERGE TWO HEAPS into one === */

void mergeHeaps(struct Heap *result,
                struct Heap *h1,
                struct Heap *h2) {
    int i, totalSize;

    /* Step 1: Check capacity */
    totalSize = h1->size + h2->size;
    if (totalSize > MAX_SIZE) {
        printf("Merged heap exceeds capacity!\n");
        return;
    }

    /* Step 2: Combine both arrays into result */
    for (i = 0; i < h1->size; i++)
        result->arr[i] = h1->arr[i];
    for (i = 0; i < h2->size; i++)
        result->arr[h1->size + i] = h2->arr[i];

    result->size = totalSize;

    /* Step 3: Rebuild heap (heapify merged array) */
    for (i = totalSize / 2 - 1; i >= 0; i--)
        heapifyDown(result, i);

    printf("Merged %d + %d = %d elements\n",
           h1->size, h2->size, totalSize);
}

Heap Sort Advanced
Sort an array using heap operations — O(n log n) guaranteed, in-place!
🏅

Analogy: Ranking Athletes by Medal

Build the heap from all athletes. The best athlete (gold) exits first — placed at the end of the results list. The next best becomes root, exits next (silver), placed second-to-last. Repeat until all are ranked. The array fills from right to left with sorted values!

1
Build Max Heap

Convert the input array into a valid max heap using Bottom-Up Heapify. O(n)

2
Swap Root with Last

The root (maximum) is swapped with the last element in the heap. The max is now in its sorted position.

3
Shrink Heap Size

Reduce heap size by 1. The sorted portion grows from right to left.

4
Heapify Remaining

Call Heapify Down on the new root to restore heap property on the smaller heap.

5
Repeat Until Done

Repeat steps 2–4 until only one element remains. Array is now sorted ascending!

/* === HEAP SORT: O(n log n), in-place === */

/* Heapify for heap sort (works on subarray) */
void heapifySortHelper(int arr[], int n, int i) {
    int largest = i;
    int left  = 2 * i + 1;
    int right = 2 * i + 2;

    if (left  < n && arr[left]  > arr[largest]) largest = left;
    if (right < n && arr[right] > arr[largest]) largest = right;

    if (largest != i) {
        int temp = arr[i];
        arr[i] = arr[largest];
        arr[largest] = temp;
        heapifySortHelper(arr, n, largest);
    }
}

void heapSort(int arr[], int n) {
    int i;
    /* Step 1: Build max heap from array */
    for (i = n / 2 - 1; i >= 0; i--)
        heapifySortHelper(arr, n, i);

    /* Steps 2-5: Extract elements one by one */
    for (i = n - 1; i > 0; i--) {
        /* Swap root (max) with last element */
        int temp = arr[0];
        arr[0] = arr[i];
        arr[i] = temp;
        /* Heapify the reduced heap */
        heapifySortHelper(arr, i, 0);
    }
}

/* Usage: */
int data[] = {64, 25, 12, 22, 11};
heapSort(data, 5);
/* Result: {11, 12, 22, 25, 64} ascending */

Delete Specific Element Advanced
Remove any element (not just the root) by increasing it to max, then extracting
🎯

Analogy: VIP Removal from Queue

To remove a specific person from the middle of a priority queue, make them the highest priority (increase key to infinity), then remove them from the front as normal. Alternatively, find their spot, put the last person there, and let them bubble up or down to the right position.

delete_specific.c C Language 
/* === DELETE SPECIFIC ELEMENT from heap === */
/* Method: Replace with last node, then heapify */

void deleteSpecific(struct Heap *h, int value) {
    /* Step 1: Find element */
    int index = searchElement(h, value);
    if (index == -1) {
        printf("Element %d not found!\n", value);
        return;
    }

    int oldVal = h->arr[index];

    /* Step 2: Replace with last element */
    h->arr[index] = h->arr[h->size - 1];
    h->size--;  /* Step 3: Shrink heap */

    if (index < h->size) {
        /* Step 4: Heapify up OR down depending on new value */
        heapifyUp(h, index);   /* Try going up first */
        heapifyDown(h, index); /* Then try going down */
        /* Only one direction will actually swap */
    }
    printf("Deleted element: %d\n", oldVal);
}

Convert Min Heap to Max Heap Advanced
Transform the heap type by rebuilding — no extra memory needed!
🔀

Analogy: Flipping the Rules

Imagine a tournament where the weakest player was always at the top (min heap). Now we change the rules: strongest at top (max heap). We don't reassign people — we just re-run the entire tournament with the new rules. Same people, same array — different heap!

🎮 Convert Min Heap → Max Heap
Build a min heap first, then convert! 🔄
convert_heap.c C Language 
/* === CONVERT MIN HEAP TO MAX HEAP === */
/* Key insight: The array is the same — just re-heapify! */

/* Heapify down for MAX heap */
void maxHeapifyDown(int arr[], int n, int i) {
    int largest = i;
    int left  = 2 * i + 1;
    int right = 2 * i + 2;

    if (left  < n && arr[left]  > arr[largest]) largest = left;
    if (right < n && arr[right] > arr[largest]) largest = right;

    if (largest != i) {
        int temp = arr[i];
        arr[i] = arr[largest];
        arr[largest] = temp;
        maxHeapifyDown(arr, n, largest);
    }
}

void convertMinToMax(struct Heap *h) {
    int i;
    /* Rebuild heap using max-heap rules */
    for (i = h->size / 2 - 1; i >= 0; i--) {
        maxHeapifyDown(h->arr, h->size, i);
    }
    h->heapType = 1; /* Update heap type to Max */
    printf("Converted to Max Heap! Root: %d\n", h->arr[0]);
}

Clone Heap Basic
Create an exact copy of a heap — duplicate both array contents and size
📋

Analogy: Photocopying a Document

Cloning a heap is like photocopying an entire document — every element, every position, preserved exactly. The clone is independent: changing one won't affect the other. Useful when you want to experiment without destroying the original heap.

clone_heap.c C Language 
/* === CLONE HEAP: Make an exact copy === */

void cloneHeap(struct Heap *dest, struct Heap *src) {
    int i;

    /* Step 1: Copy heap size and type */
    dest->size     = src->size;
    dest->heapType = src->heapType;

    /* Step 2: Copy each element one by one */
    for (i = 0; i < src->size; i++) {
        dest->arr[i] = src->arr[i];
    }
    /* No reheapify needed — structure is preserved! */
    printf("Heap cloned! %d elements copied.\n", dest->size);
}

Clear Heap Basic
Remove all elements and reset the heap to its initial empty state
🗑️

Analogy: Clearing the Whiteboard

Clearing a heap is like erasing an entire whiteboard. You don't erase each letter one by one — you just wipe the whole thing clean and set the count to zero. The board (array) still exists; it's just ready for fresh content. In C, we simply set size = 0!

clear_heap.c C Language 
/* === CLEAR HEAP: Remove all elements === */

/* For static array (stack-allocated): */
void clearHeap(struct Heap *h) {
    h->size = 0; /* Just reset size — O(1)! */
    /* Old data still in array but size=0 means */
    /* heap treats it all as non-existent */
    printf("Heap cleared! Size is now 0.\n");
}

/* Optional: Also zero out the array (for security) */
void clearHeapSecure(struct Heap *h) {
    int i;
    for (i = 0; i < h->size; i++) {
        h->arr[i] = 0;  /* Zero out each element */
    }
    h->size = 0;
    printf("Heap securely cleared!\n");
}

/* For DYNAMIC arrays (malloc): */
void freeHeap(int **arr, int *size) {
    free(*arr);    /* Free allocated memory */
    *arr  = NULL;  /* Prevent dangling pointer */
    *size = 0;
    printf("Dynamic heap freed!\n");
}
