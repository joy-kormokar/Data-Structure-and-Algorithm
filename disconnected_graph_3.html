Foundation

What is a Disconnected Graph?

A graph is a collection of vertices (nodes) connected by edges (links). A graph is called disconnected when there exists at least one pair of vertices with no path between them. The graph breaks into 2 or more connected components.

🔗

Connected Graph

You can reach every vertex from any starting point. One big component.

🔀

Disconnected Graph

Has ≥2 isolated clusters. No path bridges them. Focus of this dashboard.

⚫

Isolated Vertex

A vertex with degree 0. Belongs to its own trivial component.

🌍 Real-World Analogy

Imagine a country split by an ocean. City A, B, C are on one island (connected). City D, E are on another (another component). You can drive A→B→C, but can't drive from island 1 to island 2 — that's a disconnected graph.

Representation

Incidence Matrix

An Incidence Matrix is a 2D array where:

  • Rows represent vertices (V)
  • Columns represent edges (E)
  • Cell value = 1 if vertex is an endpoint of the edge, 0 otherwise
  • For directed graphs: +1 (source), -1 (destination)

Graph Structure

Vertices: V0, V1, V2, V3, V4
Edges: e0=(V0,V1), e1=(V1,V2), e2=(V3,V4)
Note: V0-V2 and V3-V4 are disconnected!

Incidence Matrix

V\Ee0e1e2
V0100
V1110
V2010
V3001
V4001

Notice: rows for V3, V4 have all zeros in e0 and e1 columns — they are in a different component.

Structure in C

C — Incidence Matrix Structure
/* Incidence Matrix for Disconnected Graph */
#define MAX_V 10   /* max vertices */
#define MAX_E 20   /* max edges    */

int inc[MAX_V][MAX_E];  /* incidence matrix: rows=vertices, cols=edges */
int V = 0;              /* current number of vertices */
int E = 0;              /* current number of edges    */

/* Initialize the matrix with all zeros */
void initGraph() {
    int i, j;
    for (i = 0; i < MAX_V; i++)
        for (j = 0; j < MAX_E; j++)
            inc[i][j] = 0;
    V = 0; E = 0;
}
🧩 Analogy

Think of the incidence matrix like a classroom attendance sheet. Rows are students (vertices), columns are club meetings (edges). A 1 means that student attended that meeting. If student D and E never share any meeting column with A, B, C — they're in a "disconnected group".

Vertex Operations

Insert Vertex

Adding a vertex appends a new row of zeros to the incidence matrix. The new vertex has no edges — it starts as an isolated node (its own disconnected component).

▸ Insert Vertex Animation
Click "Insert Vertex" to add a new isolated node.
C — Insert Vertex
/* Insert a new vertex — adds a new row of zeros */
void insertVertex() {
    if (V >= MAX_V) {
        printf("Graph is full! Cannot add more vertices.\n");
        return;
    }
    int j;
    for (j = 0; j < MAX_E; j++)
        inc[V][j] = 0;  /* new row is all zeros (no edges) */
    V++;  /* increase vertex count */
    printf("Vertex V%d inserted. Total vertices: %d\n", V-1, V);
}
🏙️ Analogy

Like building a new island with no bridges yet — it's completely isolated until you add edges (bridges) to connect it to other islands.

Delete Vertex

Deleting vertex v removes its row AND all edge columns where that vertex was an endpoint. Then shift remaining rows up.

C — Delete Vertex
/* Delete vertex at index 'v' */
void deleteVertex(int v) {
    if (v < 0 || v >= V) {
        printf("Invalid vertex index!\n");
        return;
    }
    int i, j, col = 0;
    /* Remove all edges incident to vertex v */
    while (col < E) {
        if (inc[v][col] != 0) {
            /* Shift all columns left to remove this edge */
            for (i = 0; i < V; i++)
                for (j = col; j < E - 1; j++)
                    inc[i][j] = inc[i][j+1];
            E--;
        } else col++;
    }
    /* Shift rows up to remove vertex row */
    for (i = v; i < V - 1; i++)
        for (j = 0; j < E; j++)
            inc[i][j] = inc[i+1][j];
    V--;
    printf("Vertex %d deleted. Remaining vertices: %d\n", v, V);
}
💡 Analogy

Demolishing a city: you remove the city AND all roads connected to it. Neighboring cities may now become isolated — creating new disconnected components.

    printf("Vertex V%d FOUND!\n", v);

Update Vertex

In an incidence matrix, vertex data is implicitly its row index. "Updating" means relabeling — we can swap rows to reorder vertices.

C — Swap/Update Vertex Rows
/* Swap vertex positions (relabel by swapping rows) */
void updateVertex(int v1, int v2) {
    if (v1 < 0 || v1 >= V || v2 < 0 || v2 >= V) {
        printf("Invalid vertex indices!\n");
        return;
    }
    int j, temp;
    for (j = 0; j < E; j++) {
        temp = inc[v1][j];
        inc[v1][j] = inc[v2][j];
        inc[v2][j] = temp;
    }
    printf("Swapped vertex V%d and V%d.\n", v1, v2);
}

Edge Operations

Insert Edge

Adding edge (u, v) creates a new column in the matrix. Set inc[u][E] = 1 and inc[v][E] = 1 then increment E.

▸ Insert Edge Interactive
From: To:
Select vertices and click Insert Edge.
C — Insert Edge
void insertEdge(int u, int v) {
    if (u < 0 || u >= V || v < 0 || v >= V) {
        printf("Invalid vertices!\n");
        return;
    }
    if (E >= MAX_E) {
        printf("Edge limit reached!\n");
        return;
    }
    /* New column E: mark both endpoints with 1 */
    inc[u][E] = 1;
    inc[v][E] = 1;
    E++;  /* increment edge count */
    printf("Edge (V%d, V%d) inserted as e%d.\n", u, v, E-1);
}
🌉 Analogy

Building a bridge between two islands. Now those islands are part of the same component — the graph becomes less disconnected!

Delete Edge

Remove column e by shifting all subsequent columns left. E decreases by 1.

C — Delete Edge
void deleteEdge(int e) {
    if (e < 0 || e >= E) {
        printf("Invalid edge index!\n");
        return;
    }
    int i, j;
    /* Shift all columns after e to the left */
    for (i = 0; i < V; i++)
        for (j = e; j < E - 1; j++)
            inc[i][j] = inc[i][j+1];
    E--;
    printf("Edge e%d deleted. Remaining edges: %d\n", e, E);
}

Search Edge

Find if edge between vertices u and v exists by scanning all columns for a column with both inc[u][j] and inc[v][j] == 1.

C — Search Edge
int searchEdge(int u, int v) {
    int j;
    for (j = 0; j < E; j++) {
        if (inc[u][j] == 1 && inc[v][j] == 1) {
            printf("Edge (V%d, V%d) found as e%d.\n", u, v, j);
            return j;  /* return edge index */
        }
    }
    printf("Edge (V%d, V%d) NOT found.\n", u, v);
    return -1;
}

Update Edge

Change the endpoints of an existing edge: clear old endpoints (set to 0), set new ones (set to 1).

C — Update Edge
void updateEdge(int e, int newU, int newV) {
    if (e < 0 || e >= E) {
        printf("Edge e%d not found!\n", e);
        return;
    }
    int i;
    /* Clear the entire column for edge e */
    for (i = 0; i < V; i++) inc[i][e] = 0;
    /* Set new endpoints */
    inc[newU][e] = 1;
    inc[newV][e] = 1;
    printf("Edge e%d updated to (V%d, V%d).\n", e, newU, newV);
}

Traversal Operations

Breadth First Search (BFS)

BFS explores a graph level by level using a queue. Start from a vertex, visit all neighbors, then their neighbors, and so on. For a disconnected graph, run BFS from every unvisited vertex to cover all components.

▸ BFS Visualization
Click "Run BFS" to watch it traverse all components.

Queue Initialization

C — Queue for BFS
#define QMAX 100
int queue[QMAX], front = 0, rear = 0;

void  enqueue(int x) { queue[rear++] = x; }
int   dequeue()      { return queue[front++]; }
int   isEmpty()      { return front == rear; }
void  resetQueue()   { front = rear = 0; }

/* Build adjacency from incidence matrix for BFS */
int adjacent(int u, int v) {
    int j;
    for (j = 0; j < E; j++)
        if (inc[u][j] == 1 && inc[v][j] == 1) return 1;
    return 0;
}

BFS from a Single Source

C — BFS Single Component
int visited[MAX_V];

void BFS(int start) {
    resetQueue();
    enqueue(start);
    visited[start] = 1;
    printf("BFS: V%d ", start);

    while (!isEmpty()) {
        int u = dequeue();
        int v;
        for (v = 0; v < V; v++) {
            if (!visited[v] && adjacent(u, v)) {
                visited[v] = 1;
                enqueue(v);
                printf("V%d ", v);
            }
        }
    }
    printf("\n");
}

Multi-Component BFS

Loop through all vertices. If unvisited, start a new BFS — each restart = a new disconnected component.

C — Full Graph BFS (handles disconnection)
void fullBFS() {
    int i;
    int comp = 1;
    for (i = 0; i < V; i++) visited[i] = 0;

    for (i = 0; i < V; i++) {
        if (!visited[i]) {
            printf("--- Component %d ---\n", comp++);
            BFS(i);
        }
    }
}
🏊 Analogy

BFS is like dropping a stone in water — ripples expand outward in rings (levels). In a disconnected graph, you drop the stone in each separate pond to explore them all.

Depth First Search (DFS)

DFS goes as deep as possible before backtracking. It uses a stack (or recursion). For disconnected graphs, run from every unvisited vertex.

▸ DFS Visualization
Click "Run DFS" to watch depth-first exploration.

Recursive DFS

C — Recursive DFS
void DFS_Recursive(int u) {
    visited[u] = 1;
    printf("V%d ", u);
    int v;
    for (v = 0; v < V; v++) {
        if (!visited[v] && adjacent(u, v))
            DFS_Recursive(v);  /* go deeper */
    }
    /* backtracking happens automatically on return */
}

Stack-Based DFS

C — Stack DFS (Iterative)
int stk[MAX_V], top = -1;
void push(int x) { stk[++top] = x; }
int  pop()        { return stk[top--]; }
int  stackEmpty() { return top == -1; }

void DFS_Stack(int start) {
    push(start);
    while (!stackEmpty()) {
        int u = pop();
        if (visited[u]) continue;
        visited[u] = 1;
        printf("V%d ", u);
        int v;
        for (v = V-1; v >= 0; v--)
            if (!visited[v] && adjacent(u, v))
                push(v);
    }
    printf("\n");
}

Multi-Component DFS

C — Full DFS (Disconnected Graph)
void fullDFS() {
    int i, comp = 1;
    for (i = 0; i < V; i++) visited[i] = 0;

    for (i = 0; i < V; i++) {
        if (!visited[i]) {
            printf("--- Component %d (DFS) ---\n", comp++);
            DFS_Recursive(i);
            printf("\n");
        }
    }
}
🌊 Analogy

DFS is like exploring a maze — go straight ahead until you hit a wall, then backtrack and try another path. Each isolated section of the maze = a separate component you must enter fresh.

Connectivity Operations

Check Disconnected Graph

Run BFS/DFS from vertex 0. If all vertices aren't visited at the end → the graph is disconnected.

▸ Connectivity Checker
Click "Check Connectivity" to test the graph.
C — Is Graph Disconnected?
int isDisconnected() {
    int i;
    for (i = 0; i < V; i++) visited[i] = 0;
    BFS(0);  /* start from vertex 0 */

    for (i = 0; i < V; i++) {
        if (!visited[i]) {
            printf("Graph is DISCONNECTED. V%d unreachable from V0.\n", i);
            return 1;  /* 1 = disconnected */
        }
    }
    printf("Graph is CONNECTED.\n");
    return 0;
}

Count Connected Components

C — Count Components
int countComponents() {
    int i, count = 0;
    for (i = 0; i < V; i++) visited[i] = 0;
    for (i = 0; i < V; i++) {
        if (!visited[i]) {
            BFS(i);  /* each BFS call covers one component */
            count++;
        }
    }
    printf("Total connected components: %d\n", count);
    return count;
}

Find Connected Components

Assign each vertex a component label (integer ID). Vertices in the same component share the same label.

C — Label Each Component
int compLabel[MAX_V];  /* component ID for each vertex */

void findComponents() {
    int i;
    for (i = 0; i < V; i++) {
        visited[i] = 0;
        compLabel[i] = -1;
    }
    int label = 0;
    for (i = 0; i < V; i++) {
        if (!visited[i]) {
            printf("Component %d: ", label);
            /* BFS and label simultaneously */
            resetQueue();
            enqueue(i); visited[i]=1; compLabel[i]=label;
            while(!isEmpty()){
                int u=dequeue(), v;
                printf("V%d ",u);
                for(v=0;v<V;v++)
                    if(!visited[v]&&adjacent(u,v)){
                        visited[v]=1; compLabel[v]=label;
                        enqueue(v);
                    }
            }
            printf("\n"); label++;
        }
    }
}

Detect Isolated Vertices

An isolated vertex has no edges. In the incidence matrix, its entire row is all zeros.

C — Find Isolated Vertices
void detectIsolated() {
    int i, j, isolated;
    printf("Isolated vertices: ");
    int found = 0;
    for (i = 0; i < V; i++) {
        isolated = 1;
        for (j = 0; j < E; j++) {
            if (inc[i][j] != 0) { isolated = 0; break; }
        }
        if (isolated) { printf("V%d ", i); found = 1; }
    }
    if (!found) printf("None");
    printf("\n");
}

Reachability Check

Can vertex src reach vertex dst? Run BFS from src, check if dst is visited.

C — Reachability Check
int canReach(int src, int dst) {
    int i;
    for (i = 0; i < V; i++) visited[i] = 0;
    BFS(src);  /* BFS from source */
    if (visited[dst]) {
        printf("V%d CAN reach V%d.\n", src, dst);
        return 1;
    }
    printf("V%d CANNOT reach V%d (different component).\n", src, dst);
    return 0;
}
📡 Analogy

Like checking if two cities have a road connection. If they're on different islands, no matter how many roads you follow, you can never get from one to the other.

Connectivity Restoration

Add minimum edges to make a disconnected graph fully connected. For k components, you need at least k-1 edges — connecting component representatives.

C — Restore Connectivity
void restoreConnectivity() {
    int i, repr[MAX_V], rCount = 0;
    for (i = 0; i < V; i++) visited[i] = 0;

    /* Collect one representative vertex per component */
    for (i = 0; i < V; i++) {
        if (!visited[i]) {
            repr[rCount++] = i;
            BFS(i);
        }
    }
    /* Connect each representative to the previous */
    for (i = 1; i < rCount; i++) {
        printf("Adding bridge edge: V%d — V%d\n", repr[i-1], repr[i]);
        insertEdge(repr[i-1], repr[i]);
    }
    printf("Graph is now connected!\n");
}

Weak Connectivity Check

A directed graph is weakly connected if treating all edges as undirected results in a connected graph. Use the same BFS ignoring direction.

C — Weak Connectivity (Directed Graph)
/* For directed: inc[u][e] = +1 (source), inc[v][e] = -1 (dest) */
int weaklyAdjacent(int u, int v) {
    int j;
    for (j = 0; j < E; j++)
        if (inc[u][j] != 0 && inc[v][j] != 0)
            return 1;  /* share an edge regardless of direction */
    return 0;
}

int isWeaklyConnected() {
    int i;
    for (i=0;i<V;i++) visited[i]=0;
    /* BFS using weaklyAdjacent instead of adjacent */
    resetQueue(); enqueue(0); visited[0]=1;
    while(!isEmpty()){
        int u=dequeue(),v;
        for(v=0;v<V;v++)
            if(!visited[v]&&weaklyAdjacent(u,v))
            { visited[v]=1; enqueue(v); }
    }
    for(i=0;i<V;i++) if(!visited[i]) return 0;
    return 1;
}

Strong Connectivity Check

A directed graph is strongly connected if every vertex is reachable from every other vertex. Use Kosaraju's approach: two DFS passes.

C — Strong Connectivity (Kosaraju's Idea)
/* Step 1: DFS on original graph, record finish order */
int finishStack[MAX_V], fTop = -1;

void dfs1(int u) {
    visited[u] = 1;
    int v;
    for(v=0;v<V;v++)
        if(!visited[v] && inc[u][v]>0) dfs1(v);
    finishStack[++fTop] = u;
}

/* Step 2: DFS on transposed graph */
void dfs2(int u) {
    visited[u] = 1; printf("V%d ", u);
    int v;
    for(v=0;v<V;v++)
        if(!visited[v] && inc[v][u]>0) dfs2(v);
}

int isStronglyConnected() {
    int i; fTop=-1;
    for(i=0;i<V;i++) visited[i]=0;
    for(i=0;i<V;i++) if(!visited[i]) dfs1(i);
    for(i=0;i<V;i++) visited[i]=0;
    int comps=0;
    while(fTop>=0) {
        int u=finishStack[fTop--];
        if(!visited[u]){ dfs2(u); comps++; printf("\n"); }
    }
    return comps==1;
}
Component Operations

Traverse Each Component

Run separate BFS/DFS for each disconnected component, treating each as an independent subgraph.

C — Traverse Each Component
void traverseComponents() {
    int i, c = 1;
    for(i=0;i<V;i++) visited[i]=0;
    for(i=0;i<V;i++){
        if(!visited[i]){
            printf("\n=== Component %d ===\n", c++);
            DFS_Recursive(i);
        }
    }
}

Label Components

Assign integer IDs to components. All vertices in component 0 get label 0, component 1 gets label 1, etc.

C — Label Components
int label[MAX_V];
void labelComponents() {
    int i, id = 0;
    for(i=0;i<V;i++){ visited[i]=0; label[i]=-1; }
    for(i=0;i<V;i++){
        if(!visited[i]){
            /* BFS with labeling */
            resetQueue(); enqueue(i);
            visited[i]=1; label[i]=id;
            while(!isEmpty()){
                int u=dequeue(),v;
                for(v=0;v<V;v++)
                    if(!visited[v]&&adjacent(u,v)){
                        visited[v]=1; label[v]=id; enqueue(v);
                    }
            }
            id++;
        }
    }
    for(i=0;i<V;i++)
        printf("V%d → Component %d\n", i, label[i]);
}

Merge Components

To merge two components, simply insert an edge between one vertex from each component.

C — Merge Two Components
void mergeComponents(int v1, int v2) {
    if (canReach(v1, v2)) {
        printf("V%d and V%d are already in the same component!\n", v1, v2);
        return;
    }
    insertEdge(v1, v2);
    printf("Components merged by connecting V%d and V%d.\n", v1, v2);
}

Separate Components

Remove all edges on a cut-path between two vertex groups to separate them into distinct components.

C — Separate Component by Removing Edge
void separateComponents(int u, int v) {
    int e = searchEdge(u, v);
    if (e == -1) {
        printf("No edge between V%d and V%d to remove!\n", u, v);
        return;
    }
    deleteEdge(e);
    printf("Edge removed. Graph may now be more disconnected.\n");
}

Component Size Analysis

Count how many vertices are in each component and find the largest/smallest.

C — Component Size Analysis
void componentSizeAnalysis() {
    int i, sizes[MAX_V], sCount=0;
    for(i=0;i<V;i++) visited[i]=0;

    for(i=0;i<V;i++){
        if(!visited[i]){
            int cnt=0;
            resetQueue(); enqueue(i); visited[i]=1; cnt++;
            while(!isEmpty()){
                int u=dequeue(),v;
                for(v=0;v<V;v++)
                    if(!visited[v]&&adjacent(u,v))
                    { visited[v]=1; enqueue(v); cnt++; }
            }
            sizes[sCount++]=cnt;
        }
    }
    int maxS=sizes[0], minS=sizes[0];
    for(i=0;i<sCount;i++){
        printf("Component %d: %d vertices\n",i,sizes[i]);
        if(sizes[i]>maxS) maxS=sizes[i];
        if(sizes[i]<minS) minS=sizes[i];
    }
    printf("Largest: %d  Smallest: %d\n",maxS,minS);
}
Path Operations

Find Path

Find any path from source to destination using DFS with parent tracking.

C — Find Path (DFS)
int parent[MAX_V];

void printPath(int dst) {
    if (parent[dst] == -1) { printf("V%d",dst); return; }
    printPath(parent[dst]);
    printf(" → V%d", dst);
}

int findPath(int src, int dst) {
    int i;
    for(i=0;i<V;i++){ visited[i]=0; parent[i]=-1; }
    resetQueue(); enqueue(src); visited[src]=1;
    while(!isEmpty()){
        int u=dequeue(),v;
        if(u==dst){ printPath(dst); printf("\n"); return 1; }
        for(v=0;v<V;v++)
            if(!visited[v]&&adjacent(u,v))
            { visited[v]=1; parent[v]=u; enqueue(v); }
    }
    printf("No path: V%d and V%d are in different components.\n",src,dst);
    return 0;
}

Shortest Path

BFS naturally finds the shortest path (minimum edges) in an unweighted graph. Track distance array.

C — Shortest Path (BFS)
void shortestPath(int src, int dst) {
    int i, dist[MAX_V];
    for(i=0;i<V;i++){ visited[i]=0; parent[i]=-1; dist[i]=-1; }
    resetQueue(); enqueue(src);
    visited[src]=1; dist[src]=0;

    while(!isEmpty()){
        int u=dequeue(),v;
        for(v=0;v<V;v++){
            if(!visited[v]&&adjacent(u,v)){
                visited[v]=1; dist[v]=dist[u]+1;
                parent[v]=u; enqueue(v);
            }
        }
    }
    if(dist[dst]==-1)
        printf("No path (different components).\n");
    else {
        printf("Shortest distance V%d→V%d: %d hops\n",src,dst,dist[dst]);
        printPath(dst); printf("\n");
    }
}

All Paths

C — All Paths (DFS Backtracking)
int path[MAX_V], pathLen=0;

void allPaths(int u, int dst) {
    visited[u]=1; path[pathLen++]=u;
    if(u==dst){
        int k;
        for(k=0;k<pathLen;k++)
            printf(k?" → V%d":"V%d",path[k]);
        printf("\n");
    } else {
        int v;
        for(v=0;v<V;v++)
            if(!visited[v]&&adjacent(u,v))
                allPaths(v, dst);
    }
    pathLen--; visited[u]=0;  /* backtrack */
}

Cycle Operations

Detect Cycle

A cycle exists if during DFS we visit an already-visited vertex that is NOT the parent of the current vertex. Run for every component.

C — Detect Cycle (Undirected)
int hasCycleDFS(int u, int par) {
    visited[u]=1;
    int v;
    for(v=0;v<V;v++){
        if(adjacent(u,v)){
            if(!visited[v]){
                if(hasCycleDFS(v,u)) return 1;
            } else if(v != par) return 1;  /* back edge = cycle! */
        }
    }
    return 0;
}

int detectCycle() {
    int i;
    for(i=0;i<V;i++) visited[i]=0;
    for(i=0;i<V;i++)
        if(!visited[i] && hasCycleDFS(i,-1)){
            printf("Cycle DETECTED in component containing V%d.\n",i);
            return 1;
        }
    printf("No cycle found.\n");
    return 0;
}
🔄 Analogy

Walking in a city: if you find yourself on a street you've already visited (and you didn't just come from it), you've found a cycle — a loop in the road network.

Count Cycles

For an undirected graph: number of cycles = E - V + number of components (using circuit rank formula).

C — Count Cycles (Circuit Rank)
int countCycles() {
    int C = countComponents();
    int cycles = E - V + C;
    /* Circuit Rank (Cyclomatic Number) = E - V + C */
    printf("Edges=%d  Vertices=%d  Components=%d\n", E, V, C);
    printf("Number of independent cycles: %d\n", cycles);
    return cycles;
}

Remove Cycle

To break a cycle, find and remove one back edge. Detect back edge during DFS and delete it from the incidence matrix.

C — Remove Cycle (Remove One Back Edge)
int removeCycleDFS(int u, int par) {
    visited[u]=1;
    int v;
    for(v=0;v<V;v++){
        if(adjacent(u,v)){
            if(!visited[v]){
                if(removeCycleDFS(v,u)) return 1;
            } else if(v!=par){
                /* Back edge found — remove it */
                int e=searchEdge(u,v);
                if(e!=-1){ deleteEdge(e);
                    printf("Removed back edge (V%d,V%d).\n",u,v);
                    return 1;
                }
            }
        }
    }
    return 0;
}
void removeCycle() {
    int i;
    for(i=0;i<V;i++) visited[i]=0;
    for(i=0;i<V;i++) if(!visited[i]) removeCycleDFS(i,-1);
}

Advanced Algorithms

Connected Components Algorithm

The standard algorithm: iterate all vertices, BFS/DFS from each unvisited one, increment component counter. Time: O(V + E).

C — Connected Components Algorithm
/* Complete connected components algorithm */
int connectedComponentsAlgo() {
    int i, numComp = 0;
    for(i=0;i<V;i++) visited[i]=0;

    for(i=0;i<V;i++){
        if(!visited[i]){
            numComp++;
            printf("\n[Component %d]\n", numComp);
            BFS(i);
        }
    }
    printf("\nTotal: %d component(s)\n", numComp);
    return numComp;
}

Union-Find / DSU

Disjoint Set Union (DSU) efficiently tracks which component a vertex belongs to. Supports union (merge components) and find (identify component) in near O(1) with path compression.

C — Union-Find / DSU
int parent_dsu[MAX_V], rank_dsu[MAX_V];

void dsuInit() {
    int i;
    for(i=0;i<V;i++) { parent_dsu[i]=i; rank_dsu[i]=0; }
}

int dsuFind(int x) {
    if(parent_dsu[x] != x)
        parent_dsu[x] = dsuFind(parent_dsu[x]);  /* path compression */
    return parent_dsu[x];
}

void dsuUnion(int a, int b) {
    int ra=dsuFind(a), rb=dsuFind(b);
    if(ra==rb) return;
    if(rank_dsu[ra]<rank_dsu[rb]) parent_dsu[ra]=rb;
    else if(rank_dsu[ra]>rank_dsu[rb]) parent_dsu[rb]=ra;
    else { parent_dsu[rb]=ra; rank_dsu[ra]++; }
}

void dsuBuildFromMatrix() {
    dsuInit();
    int j, i;
    for(j=0;j<E;j++){
        int u=-1, v=-1;
        for(i=0;i<V;i++){
            if(inc[i][j]==1){ if(u==-1) u=i; else v=i; }
        }
        if(u!=-1&&v!=-1) dsuUnion(u,v);
    }
    printf("DSU built. Component roots:\n");
    for(i=0;i<V;i++)
        printf("  V%d → root V%d\n",i,dsuFind(i));
}
🏘️ Analogy

DSU is like an address system. Each person points to their "community leader." To check if two people are neighbors, just compare their leaders. Union = merging two communities under one leader.

Strongly Connected Components (Kosaraju)

For directed graphs: find all SCCs — maximal subgraphs where every vertex can reach every other. Kosaraju's algorithm uses two DFS passes.

C — Kosaraju's SCC Algorithm
/* Kosaraju's SCC for directed incidence matrix */
int finOrder[MAX_V], fIdx=0;

void kosarajuDFS1(int u) {
    visited[u]=1;
    int j, v;
    for(j=0;j<E;j++) {
        if(inc[u][j]==1) {  /* u is source of edge j */
            for(v=0;v<V;v++)
                if(inc[v][j]==-1&&!visited[v]) kosarajuDFS1(v);
        }
    }
    finOrder[fIdx++]=u;
}

void kosarajuDFS2(int u) {
    visited[u]=1; printf("V%d ",u);
    int j, v;
    for(j=0;j<E;j++) {
        if(inc[u][j]==-1) {  /* u is dest — reverse direction */
            for(v=0;v<V;v++)
                if(inc[v][j]==1&&!visited[v]) kosarajuDFS2(v);
        }
    }
}

void kosaraju() {
    int i; fIdx=0;
    for(i=0;i<V;i++) visited[i]=0;
    for(i=0;i<V;i++) if(!visited[i]) kosarajuDFS1(i);
    for(i=0;i<V;i++) visited[i]=0;
    int scc=0;
    while(fIdx>0){
        int u=finOrder[--fIdx];
        if(!visited[u]){
            printf("SCC %d: ",++scc);
            kosarajuDFS2(u); printf("\n");
        }
    }
}

Graph Coloring

Assign minimum colors to vertices so no two adjacent vertices share the same color. Greedy approach — works per component in a disconnected graph.

C — Greedy Graph Coloring
int color[MAX_V];

void greedyColoring() {
    int i, j;
    for(i=0;i<V;i++) color[i]=-1;
    color[0]=0;  /* first vertex gets color 0 */

    int avail[MAX_V];
    for(i=1;i<V;i++){
        for(j=0;j<V;j++) avail[j]=1;  /* all colors available */
        for(j=0;j<V;j++){
            if(adjacent(i,j) && color[j]!=-1)
                avail[color[j]]=0;  /* mark neighbour's color as taken */
        }
        for(j=0;j<V;j++) if(avail[j]){ color[i]=j; break; }
    }
    for(i=0;i<V;i++) printf("V%d → Color %d\n",i,color[i]);
}
🎨 Analogy

Coloring a map so that no two adjacent countries share a color. Disconnected regions can reuse colors freely since they don't touch.

Spanning Forest

A spanning forest is a subgraph that spans all vertices, with exactly one spanning tree per connected component. For disconnected graphs, no single spanning tree exists.

C — Build Spanning Forest (BFS Tree per Component)
/* spanParent[v] = parent of v in the spanning forest (-1 = root) */
int spanParent[MAX_V];

void spanningForest() {
    int i;
    for(i=0;i<V;i++){ visited[i]=0; spanParent[i]=-1; }

    for(i=0;i<V;i++){
        if(!visited[i]){
            /* BFS tree for this component */
            resetQueue(); enqueue(i); visited[i]=1;
            while(!isEmpty()){
                int u=dequeue(), v;
                for(v=0;v<V;v++){
                    if(!visited[v]&&adjacent(u,v)){
                        visited[v]=1;
                        spanParent[v]=u;  /* tree edge recorded */
                        enqueue(v);
                    }
                }
            }
        }
    }
    printf("Spanning Forest Edges:\n");
    for(i=0;i<V;i++)
        if(spanParent[i]!=-1)
            printf("  V%d — V%d\n",spanParent[i],i);
}
