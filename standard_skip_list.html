📖 What is Initialization?

A skip list needs to be set up before use. Think of it like building the frame of a house before adding walls and furniture. During initialization, we:

  1. Decide the maximum number of levels the skip list can have (usually 16 or 32).
  2. Create a special header node (also called "head") that acts as the starting point at every level.
  3. Set all forward pointers of the header to NULL (pointing to nothing, since the list is empty).
  4. Set the current level to 0 (the list starts at level 0 with no real elements).
  5. Set the probability factor P (usually 0.5), which controls how many levels a new node gets.
Why a header node? The header is a dummy node that exists at all levels. It gives us a consistent entry point to start searching or traversing. It doesn't store any real data — it's purely structural.
Max LevelTypically log₂(N) where N is max expected elements. For 65536 items → 16 levels.
Prob Factor PUsually 0.5. Means each node has 50% chance of being promoted to the next level.
Current LevelStarts at 0. Increases as nodes with higher levels are inserted.
NULL PointersAll header forward pointers initialized to NULL — no elements exist yet.

skip_list_init.c 
#include <stdio.h>
#include <stdlib.h>

#define MAX_LEVEL 4      /* Maximum levels in the skip list */
#define PROB      0.5f   /* Probability factor: 50% promotion chance */

/* Each node stores a key and forward pointers for each level */
typedef struct Node {
    int key;                          /* The data key stored in this node */
    struct Node *forward[MAX_LEVEL]; /* Array of forward pointers (one per level) */
} Node;

/* The Skip List itself holds the header and current level */
typedef struct {
    Node  *header;       /* Dummy header node — starting point */
    int    level;        /* Current highest level in use (starts at 0) */
    int    size;         /* Total number of real elements */
} SkipList;

/* Create a single node with a given key */
Node* createNode(int key) {
    Node *n = (Node*) malloc(sizeof(Node));
    if (!n) { fprintf(stderr, "Memory error\n"); exit(1); }
    n->key = key;
    for (int i = 0; i < MAX_LEVEL; i++)
        n->forward[i] = NULL; /* All pointers start as NULL */
    return n;
}

/* Initialize (create) the entire skip list */
SkipList* createSkipList() {
    SkipList *sl = (SkipList*) malloc(sizeof(SkipList));
    sl->header = createNode(-1); /* Header key is -1 (dummy value) */
    sl->level  = 0;              /* Start with level 0 — list is empty */
    sl->size   = 0;              /* No elements yet */
    return sl;
}

int main() {
    SkipList *sl = createSkipList();
    printf("Skip list created!\n");
    printf("Header: %p, Level: %d, Size: %d\n",
           (void*)sl->header, sl->level, sl->size);
    return 0;
}

📖 How Insertion Works

Insertion into a skip list is a 3-phase process:

  1. Find position: Start from the highest level of the header. Move right as long as the next node's key is less than our new key. When blocked, drop one level and try again. Record each "turn" point in an update[] array.
  2. Generate random level: Flip a coin (with probability P) to decide how tall our new node will be. A level-1 node exists only on the bottom; a level-4 node exists on all 4 levels.
  3. Splice in the node: At each level from 0 to randomLevel, insert the new node by pointing its forward pointer to what update[level] was pointing, and making update[level] point to the new node.
The Update Array: As we traverse top-to-bottom to find the insert position, we track the last node we visited at each level in update[]. These are exactly the nodes whose forward pointers need to be updated after insertion.
Handling Duplicates: If the key already exists, you can either update its value or skip insertion. We check this before creating the new node.

skip_list_insert.c 
/* Generate a random level for a new node (max MAX_LEVEL) */
int randomLevel() {
    int lvl = 1;
    /* Keep promoting while coin flip is heads AND below max */
    while (((float)rand() / RAND_MAX) < PROB && lvl < MAX_LEVEL)
        lvl++;
    return lvl;
}

/* Insert a key into the skip list */
void insert(SkipList *sl, int key) {
    Node *update[MAX_LEVEL]; /* Stores the nodes whose forward pointers need updating */
    Node *curr = sl->header;

    /* Step 1: Find the insert position from top level to bottom */
    for (int i = sl->level - 1; i >= 0; i--) {
        /* Move forward while next node's key < new key */
        while (curr->forward[i] != NULL && curr->forward[i]->key < key)
            curr = curr->forward[i];
        update[i] = curr; /* Record last visited node at level i */
    }

    /* Step 2: Check if key already exists at level 0 */
    curr = curr->forward[0];
    if (curr != NULL && curr->key == key) {
        printf("Key %d already exists — skipping duplicate\n", key);
        return;
    }

    /* Step 3: Generate a random level for the new node */
    int newLevel = randomLevel();

    /* Step 4: If new level exceeds current max, extend update[] */
    if (newLevel > sl->level) {
        for (int i = sl->level; i < newLevel; i++)
            update[i] = sl->header; /* New levels connect to header */
        sl->level = newLevel;         /* Raise the skip list's current level */
    }

    /* Step 5: Create the new node */
    Node *newNode = createNode(key);

    /* Step 6: Splice the new node into each level */
    for (int i = 0; i < newLevel; i++) {
        newNode->forward[i]   = update[i]->forward[i]; /* New node points to what update pointed to */
        update[i]->forward[i] = newNode;                  /* Update node now points to new node */
    }

    sl->size++;
    printf("Inserted %d at level %d\n", key, newLevel);
}

📚 Real-Life Analogy: Inserting a Book in a Multi-floor Library

Imagine a library with multiple express floors (levels). The ground floor has every book in order. Higher floors have only a subset for faster browsing. When you add a new book, you first ride the elevator to the top floor and walk forward until the next slot would be past your book's spot (update[]). You record each "I stopped here" position. Then you flip a coin to see how many floors your book should appear on. Finally, you insert your book card at each floor it belongs to, updating the pointers.

🗑️ How Deletion Works

  1. Search with update[]: Traverse top-to-bottom exactly like insertion, building the update array.
  2. Verify target exists: Check at level 0 if the next node equals our key. If not found, report error.
  3. Unlink at each level: For every level the target exists on, set update[i].forward[i] = target.forward[i] — bypassing the node.
  4. Free memory: Call free() on the removed node.
  5. Shrink level if needed: If the highest levels become empty (only header), reduce sl→level.
The key insight: deletion and insertion both build the same update[] array. The difference is that deletion removes pointer links while insertion adds them.

skip_list_delete.c
void deleteKey(SkipList *sl, int key) {
    Node *update[MAX_LEVEL];
    Node *curr = sl->header;

    /* Step 1: Build update[] — same traversal as insertion */
    for (int i = sl->level - 1; i >= 0; i--) {
        while (curr->forward[i] != NULL && curr->forward[i]->key < key)
            curr = curr->forward[i];
        update[i] = curr;
    }

    /* Step 2: The candidate node is at level 0 */
    Node *target = curr->forward[0];

    /* Step 3: Verify target is the correct node */
    if (target == NULL || target->key != key) {
        printf("Key %d not found\n", key);
        return;
    }

    /* Step 4: Remove the node from each level it appears on */
    for (int i = 0; i < sl->level; i++) {
        /* If this level doesn't point to target, no need to go higher */
        if (update[i]->forward[i] != target) break;
        /* Bypass the target: update[i] now skips over target */
        update[i]->forward[i] = target->forward[i];
    }

    /* Step 5: Free the target node's memory */
    free(target);
    sl->size--;

    /* Step 6: Shrink level if top levels are now empty */
    while (sl->level > 1 && sl->header->forward[sl->level - 1] == NULL)
        sl->level--;

    printf("Deleted key %d\n", key);
}

✂️ Analogy: Removing a Book from a Multi-floor Library

When removing a book, you visit each floor where the book appears. On each floor, you find the shelf tag before the book (update[]) and redirect it to point to the book that came after the one you're removing. The removed book is then taken off all shelves and discarded. If the top floors become empty (no more books there), you close those floors — reducing the active level count.

04— Search Operations

Searching for Elements

Search is the core advantage of a skip list. By starting at the highest level and taking "express lanes", we achieve O(log n) average time — much better than a regular linked list's O(n).

🔍 How Search Works

  1. Begin at header, top level: Start with curr = header, level = sl→level - 1.
  2. Move forward if possible: While the next node's key is less than target, advance (curr = curr→forward[i]).
  3. Drop a level: When you can't move forward, decrement i and try again from same position.
  4. Repeat until level 0: Continue until i < 0.
  5. Check result: After the loop, curr→forward[0] is the candidate. If its key equals target → found. Otherwise → not found.
Why O(log n)? Each level roughly halves the nodes you visit. With 4 levels and 16 nodes, you visit at most log₂(16) = 4 nodes per level, for ~16 comparisons instead of 16 sequential comparisons. With 1 million nodes: ~20 comparisons vs 1,000,000!

skip_list_search.c
/* Search for a key — returns the node if found, NULL otherwise */
Node* search(SkipList *sl, int key) {
    Node *curr = sl->header;

    /* Start from the highest level, move down toward level 0 */
    for (int i = sl->level - 1; i >= 0; i--) {
        /* On this level, keep moving right while next key is smaller */
        while (curr->forward[i] != NULL && curr->forward[i]->key < key)
            curr = curr->forward[i]; /* Move right (express lane hop!) */
        /* Can't move right: drop down to the next level */
    }

    /* After loop, curr is just before the target at level 0 */
    curr = curr->forward[0];

    /* Check if this is actually our target */
    if (curr != NULL && curr->key == key) {
        printf("Found: %d\n", curr->key);
        return curr; /* Key found! */
    }

    printf("Key %d not found\n", key);
    return NULL; /* Key does not exist */
}

int main() {
    SkipList *sl = createSkipList();
    insert(sl, 3);  insert(sl, 7);
    insert(sl, 12); insert(sl, 19);

    Node *result = search(sl, 12); /* Found */
    result = search(sl, 50);       /* Not found */
    return 0;
}

To find a city, take the express train as far as possible without overshooting, then switch to the local. ✏️ How Update Works

An update is essentially a search followed by a data mutation. The skip list structure doesn't change; only the value associated with a key is modified.

  1. Run the standard search algorithm to locate the node with the given key.
  2. If the key is not found, report an error or return false.
  3. If found, overwrite the node's value field with the new value.
In practice, nodes often store both a key (for ordering and searching) and a value (the actual data). Update only changes the value — the key remains the same, so the sorted order is preserved.
skip_list_update.c
/* A node with both key and value (for update demo) */
typedef struct KVNode {
    int    key;
    int    value;                    /* The data we want to update */
    struct KVNode *forward[MAX_LEVEL];
} KVNode;

/* Update the value for a given key */
int updateValue(SkipList *sl, int key, int newValue) {
    /* Step 1: Search for the key (same as search algorithm) */
    Node *curr = sl->header;
    for (int i = sl->level - 1; i >= 0; i--)
        while (curr->forward[i] && curr->forward[i]->key < key)
            curr = curr->forward[i];

    curr = curr->forward[0]; /* Candidate node */

    /* Step 2: Validate key existence */
    if (curr == NULL || curr->key != key) {
        printf("Key %d does not exist — cannot update\n", key);
        return 0; /* Failure */
    }

    /* Step 3: Replace old value with new value */
    int oldVal = curr->key; /* In full KV implementation, store value separately */
    curr->key = newValue;    /* For demo: updating the key's associated data */

    printf("Updated key %d: %d → %d\n", key, oldVal, newValue);
    return 1; /* Success */
}

🚶 Traversal Explained

Traversal is the simplest skip list operation — just walk along level 0 from the header to the end. Since all nodes exist at level 0 in sorted order, this gives a complete in-order visit of all elements in O(n) time.

Level-wise display: For debugging or visualization, you can also traverse each level individually. Higher levels show fewer nodes (the "express stations"). This is useful to verify the skip list's structure is correct.
  1. Set curr = header→forward[0] (first real node).
  2. While curr ≠ NULL: process curr, then curr = curr→forward[0].
  3. For level-wise: repeat for i from 0 to sl→level-1.

void traverseAll(SkipList *sl) {

void displayLevelwise(SkipList *sl) {
    printf("=== Skip List Structure ===\n");
    for (int i = sl->level - 1; i >= 0; i--) {
        printf("Level %d: HEAD", i);
        Node *curr = sl->header->forward[i];
        while (curr != NULL) {
            printf(" → %d", curr->key);
            curr = curr->forward[i];
        }

void traverseLevel(SkipList *sl, int level) {

🏗️ Level Management Explained

The skip list tracks its current level — the highest level that has at least one node. This avoids wasting time checking empty upper levels during searches.

Increase: When a new node's randomLevel exceeds the current level, we extend the header's update[] entries and raise sl→level. Done during insertion.
Decrease: After deletion, if the top level of the header points to NULL, it's empty. We shrink sl→level until the top level is non-empty (or back to 1 minimum).

/* Called inside insert() when new node's level > current level */
void increaseLevel(SkipList *sl, Node *update[], int newLevel) {
    if (newLevel > sl->level) {
        for (int i = sl->level; i < newLevel; i++)
            update[i] = sl->header; /* New levels: update starts at header */
        sl->level = newLevel;
        printf("Level increased to %d\n", newLevel);
    }
}

/* Called inside deleteKey() after removing a node */
void decreaseLevel(SkipList *sl) {
    while (sl->level > 1 &&
           sl->header->forward[sl->level - 1] == NULL) {
        sl->level--; /* Remove empty top level */
    }
    printf("Level decreased to %d\n", sl->level);
}

/* Check if a specific level is empty */
int isLevelEmpty(SkipList *sl, int level) {
    return sl->header->forward[level] == NULL;
}

/* Print the height distribution of all nodes */
void checkLevelBalance(SkipList *sl) {
    for (int i = sl->level - 1; i >= 0; i--) {
        int count = 0;
        Node *c = sl->header->forward[i];
        while (c) { count++; c = c->forward[i]; }
        printf("Level %d: %d node(s)\n", i, count);
    }
}

🎲 The Randomization Algorithm

When a new node is inserted, we don't know in advance how many levels it should span. Instead of computing the optimal placement (expensive), we flip a biased coin:

  1. Start at level 1. The new node always exists at level 0 (ground floor).
  2. Generate a random float between 0 and 1.
  3. If it's less than P (e.g., 0.5), promote to next level and repeat.
  4. Stop when the coin fails OR we hit MAX_LEVEL.
Expected distribution with P=0.5: 50% of nodes are level 1 only, 25% reach level 2, 12.5% reach level 3, 6.25% reach level 4, etc. This geometric distribution is what gives skip lists their O(log n) expected performance.
P = 0.5Classic choice. Level i has ~n/2ⁱ nodes.
P = 0.25Faster, fewer levels per node. Redis uses P=0.25.
MAX_LEVELHard cap. Prevents rare runaway coin flips from creating absurd levels.

skip_list_random.c
#include <stdlib.h>
#include <time.h>

#define MAX_LEVEL 16
#define PROB      0.5f

/* Seed the random number generator once at program start */
void initRandom() {
    srand((unsigned int) time(NULL));
}

/* Coin flip: returns 1 (heads) with probability PROB */
int coinFlip() {
    return ((float)rand() / RAND_MAX) < PROB;
}

/* Generate random level for a new node */
int randomLevel() {
    int level = 1; /* All nodes start at level 1 (exists at level 0) */

    /* Keep flipping: heads = promote, tails = stop */
    while (coinFlip() && level < MAX_LEVEL)
        level++;

    return level; /* This node will span 'level' levels */
}

/* Demo: show the distribution of random levels */
void showDistribution() {
    int counts[MAX_LEVEL] = {0};
    int trials = 1000;

    for (int i = 0; i < trials; i++)
        counts[randomLevel() - 1]++;

    printf("Level distribution over %d trials:\n", trials);
    for (int i = 0; i < MAX_LEVEL; i++) {
        if (counts[i] == 0) break;
        printf("Level %2d: %4d nodes (~%.1f%%)\n",
               i + 1, counts[i], 100.0f * counts[i] / trials);
    }
}

🪙 Analogy: Building Floors by Coin Flip

Imagine you're building a tower. You always build the ground floor. Then you flip a coin — heads means add another floor. Keep flipping heads? Keep building. Tails? Stop. Most towers stay short (1-2 floors). A few get tall. Very few become skyscrapers. This natural distribution means there are always fewer "express stops" at higher levels, exactly what efficient skip-list searching needs.

🔗 How Forward Pointers Work

Each node in a skip list has an array of forward pointers — one per level. forward[0] points to the next node at level 0 (the immediate neighbor), forward[1] skips over some nodes at level 1, and so on.

Key pointer operations:
• Insert: newNode→forward[i] = update[i]→forward[i]; then update[i]→forward[i] = newNode;
• Delete: update[i]→forward[i] = target→forward[i];
• Init: All pointers set to NULL when node is created.

The forward pointer array is the key difference between a skip list and a regular singly-linked list. It's what enables multi-level traversal without extra overhead per hop.

skip_list_pointers.c
/* Initialize all forward pointers of a new node to NULL */
void initPointerArray(Node *node) {
    for (int i = 0; i < MAX_LEVEL; i++)
        node->forward[i] = NULL;
}

/* Update pointer links during insertion (splice in newNode) */
void updatePointerLinks(Node *update[], Node *newNode, int level) {
    for (int i = 0; i < level; i++) {
        /* Step A: newNode points forward to what update[i] was pointing */
        newNode->forward[i]   = update[i]->forward[i];
        /* Step B: update[i] now points to newNode (splice in) */
        update[i]->forward[i] = newNode;
    }
}

/* Redirect pointers during deletion (bypass target node) */
void redirectForwardRefs(Node *update[], Node *target, int maxLevel) {
    for (int i = 0; i < maxLevel; i++) {
        if (update[i]->forward[i] != target) break; /* Target not on this level */
        /* Skip over target: update[i] now points to what target pointed to */
        update[i]->forward[i] = target->forward[i];
    }
}

/* Invalidate (NULL out) all pointers of a node before freeing */
void removeInvalidLinks(Node *node) {
    for (int i = 0; i < MAX_LEVEL; i++)
        node->forward[i] = NULL; /* Prevent dangling pointers */
    free(node);
}

🔗 Analogy: Chain of Hyperlinks

Think of each node as a web page with multiple hyperlinks on it — one link to the "immediate next page" (level 0), one link skipping 4 pages ahead (level 1), one skipping 16 pages (level 2), etc. When you add a new page, you update the surrounding pages' links. When you remove a page, you redirect its predecessor's links to bypass it. The links are the forward pointers.

/* Remove all elements (free memory) but keep the header */
void clearSkipList(SkipList *sl) {
    Node *curr = sl->header->forward[0];
    while (curr != NULL) {
        Node *next = curr->forward[0];
        free(curr); /* Free current node */
        curr = next;
    }
    /* Reinitialize header: all forward pointers to NULL */
    for (int i = 0; i < MAX_LEVEL; i++)
        sl->header->forward[i] = NULL;
    sl->level = 0; /* Reset level back to empty state */
    sl->size  = 0;
    printf("Skip list cleared.\n");
}

/* Completely destroy skip list and free all memory */
void destroySkipList(SkipList *sl) {
    clearSkipList(sl);
    free(sl->header);
    free(sl);
}

💾 Memory Management in C

C doesn't have garbage collection. Every node you create with malloc() must be explicitly freed with free() when no longer needed.

Memory Leak: Forgetting to free() a node means that memory block is lost forever (until program ends). In long-running programs this causes RAM to fill up and crash.
Dangling Pointer: Using a pointer after free()ing the memory it points to. This causes undefined behavior — usually a crash or corrupted data.
malloc()Allocates heap memory. Returns NULL on failure — always check!
free()Returns memory to OS. Set pointer to NULL after freeing.
Node sizesizeof(Node) = sizeof(int) + MAX_LEVEL × sizeof(pointer).

#include <stdlib.h>
#include <stdio.h>

/* Safe malloc: allocates memory and checks for failure */
void* safeMalloc(size_t size) {
    void *ptr = malloc(size);
    if (ptr == NULL) {
        fprintf(stderr, "Fatal: out of memory\n");
        exit(EXIT_FAILURE);
    }
    return ptr;
}

/* Create a new node and allocate its memory safely */
Node* createNode(int key) {
    Node *node = (Node*) safeMalloc(sizeof(Node));
    node->key = key;
    for (int i = 0; i < MAX_LEVEL; i++)
        node->forward[i] = NULL;
    return node;
}

/* Free a single node safely */
void freeNode(Node **nodePtr) {
    if (*nodePtr != NULL) {
        free(*nodePtr);
        *nodePtr = NULL; /* Prevent dangling pointer */
    }
}

/* Free the entire skip list — prevent memory leak */
void freeSkipList(SkipList **slPtr) {
    if (*slPtr == NULL) return;

    Node *curr = (*slPtr)->header;
    while (curr != NULL) {
        Node *next = curr->forward[0];
        free(curr);  /* Free each node including header */
        curr = next;
    }
    free(*slPtr);    /* Free the SkipList struct itself */
    *slPtr = NULL;   /* Null the caller's pointer */
    printf("All memory freed. No leaks.\n");
}

int main() {
    SkipList *sl = createSkipList();
    insert(sl, 10); insert(sl, 20);
    /* Always free when done! */
    freeSkipList(&sl); /* sl is NULL after this */
    return 0;
}

float averageHeight(SkipList *sl) {
    int totalHeight = 0, nodeCount = 0;
    for (int i = 0; i < sl->level; i++) {
        Node *c = sl->header->forward[i];
        while (c) { totalHeight++; c = c->forward[i]; }

    nodeCount = sl->size;

/* Calculate theoretical search complexity */
void analyzeComplexity(SkipList *sl) {
    int n = sl->size;
    if (n == 0) { printf("Empty list\n"); return; }
    double log2n = 0;
    int tmp = n;
    while (tmp > 1) { log2n++; tmp /= 2; }
    printf("Nodes: %d | Expected search: O(log2 %d) ≈ %.0f comparisons\n",
           n, n, log2n * 2);
    printf("Actual levels: %d | Avg height: %.2f\n",
           sl->level, averageHeight(sl));
}

void evaluateDistribution(SkipList *sl) {
    printf("Level distribution:\n");
    for (int i = sl->level - 1; i >= 0; i--) {
        int cnt = 0;
        Node *c = sl->header->forward[i];
        while (c) { cnt++; c = c->forward[i]; }
        printf("  L%d: %3d nodes [expected ~%d]\n",
               i, cnt, sl->size >> i); /* n >> i == n / 2^i */

🔄 Iterator Pattern

An iterator wraps a pointer to the current node. You call hasNext() to check if more elements exist, and next() to advance and retrieve the current element.

Why iterators? They separate the "how to traverse" from "what to do with each element". Your code that uses the skip list doesn't need to know about forward pointers — it just calls next() and gets values.
  1. Create iterator: Set curr = header→forward[0] (first real element).
  2. hasNext(): Return curr != NULL.
  3. next(): Save curr→key, advance curr = curr→forward[0], return saved key.
  4. reset(): Set curr back to header→forward[0].
skip_list_iterator.c
/* Iterator struct: holds current position in the skip list */
typedef struct {
    Node      *curr;   /* Pointer to current node */
    SkipList  *list;   /* Reference to the skip list */
} Iterator;

/* Create an iterator starting at the first element */
Iterator createIterator(SkipList *sl) {
    Iterator it;
    it.list = sl;
    it.curr = sl->header->forward[0]; /* Start at first real node */
    return it;
}

/* Check if more elements exist */
int hasNext(Iterator *it) {
    return it->curr != NULL;
}

/* Advance and return current key */
int next(Iterator *it) {
    int key = it->curr->key;
    it->curr = it->curr->forward[0]; /* Move to next at bottom level */
    return key;
}

/* Reset iterator to the beginning */
void resetIterator(Iterator *it) {
    it->curr = it->list->header->forward[0];
}

/* Usage example */
int main() {
    SkipList *sl = createSkipList();
    insert(sl, 5); insert(sl, 3); insert(sl, 8);

    Iterator it = createIterator(sl);
    printf("All elements: ");
    while (hasNext(&it))
        printf("%d ", next(&it)); /* Prints: 3 5 8 (sorted) */
    printf("\n");

    resetIterator(&it);
    printf("Second pass: %d\n", next(&it)); /* Prints: 3 */
    return 0;
}

An iterator is like a TV remote. The remote has a "current channel" pointer. Pressing Next Channel (next()) advances to the next one. You can check if there are more channels (hasNext()). Pressing Home (reset()) takes you back to channel 1. You use the remote without knowing how the cable box internally stores channels.

⚡ Merge & Split Explained

Merge combines two sorted skip lists into one sorted skip list. The simplest approach: iterate list B and insert each element into list A. For optimized merge: walk both lists simultaneously (like merge sort's merge step) and link nodes directly.

Merge complexity: Simple approach: O(m log(m+n)) where m = size of B. Optimized walk: O(m+n) but requires rebuilding levels.
Split divides a skip list at a pivot key K into: left (all keys < K) and right (all keys ≥ K). Walk level 0, reconnect pointers at the split point. Rebuild upper level connections.
  1. Find split point: Search for key K using update[] array — these are the exact nodes to disconnect.
  2. Create right list: New header's forward[i] = update[i]→forward[i] (what comes after the split).
  3. Terminate left list: update[i]→forward[i] = NULL at each level.
  4. Rebuild levels: Shrink both lists' level counts appropriately.
skip_list_merge_split.c
/* Simple merge: insert all elements of list B into list A */
void mergeSkipLists(SkipList *A, SkipList *B) {
    Node *curr = B->header->forward[0]; /* Walk B at level 0 */
    while (curr != NULL) {
        insert(A, curr->key);           /* Insert each B node into A */
        curr = curr->forward[0];
    }
    printf("Merged: A now has %d elements\n", A->size);
}

/* Split: divide skip list at pivotKey into left (< pivot) and right (>= pivot) */
SkipList* splitSkipList(SkipList *sl, int pivotKey) {
    Node *update[MAX_LEVEL];
    Node *curr = sl->header;

    /* Step 1: Build update array at split point */
    for (int i = sl->level - 1; i >= 0; i--) {
        while (curr->forward[i] && curr->forward[i]->key < pivotKey)
            curr = curr->forward[i];
        update[i] = curr;
    }

    /* Step 2: Create right skip list */
    SkipList *right = createSkipList();
    right->level = sl->level;

    /* Step 3: Transfer right half to new list */
    for (int i = 0; i < sl->level; i++) {
        right->header->forward[i] = update[i]->forward[i];
        update[i]->forward[i] = NULL; /* Left list ends here */
    }

    /* Step 4: Recalculate sizes and levels */
    sl->size = countNodes(sl);
    right->size = countNodes(right);
    decreaseLevel(sl);
    decreaseLevel(right);

    printf("Split at %d: left=%d, right=%d\n",
           pivotKey, sl->size, right->size);
    return right;
}

🔀 Analogy: Merging / Splitting a Sorted Card Deck

Merge: You have two sorted card decks. Pick the smaller top card from either deck and place it on the merged pile — repeat until both decks are empty. The result is one sorted deck.
Split: Find card #K and cut the deck there. Everything to the left stays; everything from K onward becomes a new deck. Two sorted decks emerge from one.

/* Find the minimum key — always the first element at level 0 */
int findMin(SkipList *sl) {
    if (isEmpty(sl)) { printf("List is empty\n"); return -1; }
    return sl->header->forward[0]->key; /* O(1) — first node */
}

/* Find the maximum key — walk level 0 to last node */
int findMax(SkipList *sl) {
    if (isEmpty(sl)) { printf("List is empty\n"); return -1; }
    Node *curr = sl->header;
    /* Use highest level for faster traversal to end */
    for (int i = sl->level - 1; i >= 0; i--)
        while (curr->forward[i]) curr = curr->forward[i];
    return curr->key;
}

/* Range query: return all keys in [lo, hi] */
void rangeQuery(SkipList *sl, int lo, int hi) {
    Node *curr = sl->header;
    /* Step 1: Find first node with key >= lo (O(log n)) */
    for (int i = sl->level - 1; i >= 0; i--)
        while (curr->forward[i] && curr->forward[i]->key < lo)
            curr = curr->forward[i];
    curr = curr->forward[0];
    /* Step 2: Walk level 0 collecting keys <= hi (O(k)) */
    printf("Range [%d, %d]: ", lo, hi);
    while (curr && curr->key <= hi) {
        printf("%d ", curr->key);
        curr = curr->forward[0];
    }
    printf("\n");
}

/* Nearest neighbor: find key closest to target */
int nearestNeighbor(SkipList *sl, int target) {
    Node *curr = sl->header;
    for (int i = sl->level - 1; i >= 0; i--)
        while (curr->forward[i] && curr->forward[i]->key < target)
            curr = curr->forward[i];
    /* curr is just before target, curr->forward[0] is at or after target */
    Node *before = curr;                    /* key < target */
    Node *after  = curr->forward[0];      /* key >= target */
    if (!after) return before->key;
    if (before == sl->header) return after->key;
    int diffBefore = target - before->key;
    int diffAfter  = after->key  - target;
    return (diffBefore <= diffAfter) ? before->key : after->key;
}

int main() {
    SkipList *sl = createSkipList();
    int keys[] = {3, 7, 12, 19, 26, 33};
    for (int i = 0; i < 6; i++) insert(sl, keys[i]);

    printf("Min: %d\n", findMin(sl));         /* 3  */
    printf("Max: %d\n", findMax(sl));         /* 33 */
    rangeQuery(sl, 7, 26);                   /* 7 12 19 26 */
    printf("Nearest to 15: %d\n", nearestNeighbor(sl, 15)); /* 12 */
    return 0;
}

Imagine a street with numbered houses in sorted order. Find Min = check house #1. Find Max = check the last house. Range Query = "list all houses between #10 and #50" — walk from #10 to #50.
