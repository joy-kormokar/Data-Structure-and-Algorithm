Array-Based Singly Linked List — How It Works
Core structure, initialization, node allocation
▼

💡 What is an Array-Based Linked List?

Instead of using malloc() and real memory pointers, we use a fixed-size array of structs. Each struct stores data and a next index (an integer pointing to the next element in the array). This simulates linked list behavior without dynamic memory.

📦 The Node Struct

Each node has two fields:
data — the stored value
next — index of next node (−1 = NULL)

🗂️ The Array

We declare list[MAX] — an array that holds all nodes. Unused slots are tracked via a "free list" starting at freeHead.

🔑 head Index

head = -1 means the list is empty. Otherwise it stores the array index of the first node.

♻️ Free List

Available slots are chained together. When we delete a node, its slot returns to the free list for reuse.

Why use array-based? It avoids memory fragmentation, is cache-friendly, and works well in embedded systems where malloc is unavailable. Indices replace pointers — same logic, simpler syntax.

  • Embedded Systems — no heap allocation needed
  • OS Free Block Lists — tracking free memory blocks

linked_list_setup.c
#include <stdio.h>

#define MAX 100   /* Maximum number of nodes */

/* === Node Structure === */
struct Node {
    int data;   /* value stored in node   */
    int next;   /* index of next node     */
                /* next = -1 means NULL   */
};

/* === Global Array and Pointers === */
struct Node list[MAX];  /* array holding all nodes */
int head;               /* index of first node     */
int freeHead;           /* index of first free slot*/

/* =============================================
   INITIALIZE the linked list
   - Links all slots into a "free list"
   - head = -1 (empty list)
   ============================================= */
void init() {
    int i;
    /* Link all slots together as free slots */
    for (i = 0; i < MAX - 1; i++) {
        list[i].next = i + 1;  /* each slot points to next */
    }
    list[MAX - 1].next = -1;   /* last slot has no next    */

    freeHead = 0;  /* first free slot is index 0 */
    head = -1;     /* list is empty              */
}

/* =============================================
   ALLOCATE a new node with given data
   Returns: index of the new node, or -1 if full
   ============================================= */
int allocNode(int data) {
    int idx;

    if (freeHead == -1) {
        printf("List is full!\n");
        return -1;   /* no free slots available */
    }

    idx = freeHead;             /* take the first free slot    */
    freeHead = list[freeHead].next; /* advance freeHead        */

    list[idx].data = data;      /* store the data              */
    list[idx].next = -1;        /* this node points to nothing */

    return idx;  /* return the index of new node */
}

/* =============================================
   FREE a node back to the free list
   ============================================= */
void freeNode(int idx) {
    list[idx].next = freeHead;  /* point to current free list */
    freeHead = idx;             /* this slot is now first free*/
}

/* === MAIN to test === */
int main() {
    init();
    printf("Linked list initialized!\n");
    printf("head = %d (empty)\n", head);
    printf("freeHead = %d\n", freeHead);
    return 0;
}

Step-by-Step Explanation

1
We declare struct Node list[MAX] — an array of 100 node structs. Each has data and next (an integer index).
2
init() chains all slots together: slot 0→1→2→…→99→-1. This creates the "free list" so we know which slots are available.
3
allocNode(data) takes the first free slot, updates freeHead to the next free, stores data, and returns the slot index.
4
freeNode(idx) returns a used slot back by prepending it to the free list — perfect for deletion.

insert_beginning.c
/* Insert a new node at the BEGINNING of the list */
void insertAtBeginning(int data) {
    int newIdx;

    /* Step 1: Get a free slot for the new node */
    newIdx = allocNode(data);
    if (newIdx == -1) return;  /* list is full */

    /* Step 2: New node's next = current head */
    list[newIdx].next = head;

    /* Step 3: Update head to point to new node */
    head = newIdx;

    printf("Inserted %d at beginning.\n", data);
}

/* === Example usage === */
int main() {
    init();
    insertAtBeginning(30);   /* list: 30 */
    insertAtBeginning(20);   /* list: 20 → 30 */
    insertAtBeginning(10);   /* list: 10 → 20 → 30 */
    return 0;
}

Step-by-Step

1
Call allocNode(data) to get a free array slot. It returns the index of the new slot.
2
Set list[newIdx].next = head — the new node now points to whatever was the first node.
3
Update head = newIdx — now the list starts at the new node. Done in O(1)!

insert_end.c
/* Insert a new node at the END of the list */
void insertAtEnd(int data) {
    int newIdx, curr;

    /* Step 1: Allocate a new node */
    newIdx = allocNode(data);
    if (newIdx == -1) return;

    /* Step 2: If list is empty, new node becomes head */
    if (head == -1) {
        head = newIdx;
        printf("Inserted %d (first node).\n", data);
        return;
    }

    /* Step 3: Traverse to the last node */
    curr = head;
    while (list[curr].next != -1) {
        curr = list[curr].next;  /* move to next node */
    }

    /* Step 4: Link last node to new node */
    list[curr].next = newIdx;

    printf("Inserted %d at end.\n", data);
}

/* === Example === */
int main() {
    init();
    insertAtEnd(10);   /* list: 10 */
    insertAtEnd(20);   /* list: 10 → 20 */
    insertAtEnd(30);   /* list: 10 → 20 → 30 */
    return 0;
}

Step-by-Step

1
Allocate new node with allocNode(data).
2
If head == -1, the list is empty — set head = newIdx and return.
3
Traverse using curr = list[curr].next until list[curr].next == -1 — that's the last node.
4
Set list[curr].next = newIdx to attach the new node at the end.

💡 What is it?

Insert a node at position k (1-based). We traverse to node at position k−1, then splice the new node between it and the next.

⏱️ Time Complexity: O(n)

Edge cases to handle:
• Position 1 → same as insert at beginning
• Position > length → invalid, insert at end or reject
• Empty list at position 1 → just add as head

insert_position.c
/* Insert at a given position (1-based) */
void insertAtPosition(int data, int pos) {
    int newIdx, curr, i;

    if (pos < 1) {
        printf("Invalid position!\n");
        return;
    }

    /* Position 1: same as insert at beginning */
    if (pos == 1) {
        insertAtBeginning(data);
        return;
    }

    /* Allocate new node */
    newIdx = allocNode(data);
    if (newIdx == -1) return;

    /* Traverse to node just BEFORE the target position */
    curr = head;
    for (i = 1; i < pos - 1; i++) {
        if (curr == -1) {
            printf("Position out of range!\n");
            freeNode(newIdx);
            return;
        }
        curr = list[curr].next;
    }

    if (curr == -1) {
        printf("Position out of range!\n");
        freeNode(newIdx);
        return;
    }

    /* Splice: new node → next, then prev → new node */
    list[newIdx].next = list[curr].next;
    list[curr].next   = newIdx;

    printf("Inserted %d at position %d.\n", data, pos);
}

/* === Example === */
int main() {
    init();
    insertAtEnd(10);
    insertAtEnd(20);
    insertAtEnd(30);
    insertAtEnd(40);
    insertAtPosition(99, 3); /* [10,20,99,30,40] */
    return 0;
}

Step-by-Step

1
Handle pos == 1 as a special case — use insertAtBeginning().
2
Traverse the list stopping at node pos-1 using a for loop counter.
3
Set list[newIdx].next = list[curr].next — new node points to the old next.
4
Set list[curr].next = newIdx — previous node now points to new node. Insertion complete!

💡 What is it?

Remove the first node. Save the head index, move head to the next node, then free the old head slot back to the free list.

⏱️ Time Complexity: O(1)

Always check: If head == -1, the list is empty — nothing to delete!

delete_beginning.c
/* Delete node from the BEGINNING */
void deleteFromBeginning() {
    int temp;

    /* Check if list is empty */
    if (head == -1) {
        printf("List is empty! Nothing to delete.\n");
        return;
    }

    /* Save old head index */
    temp = head;

    /* Move head to next node */
    head = list[head].next;

    /* Return old head slot to free list */
    freeNode(temp);

    printf("Deleted node from beginning.\n");
}

/* === Example === */
int main() {
    init();
    insertAtEnd(10);
    insertAtEnd(20);
    insertAtEnd(30);
    deleteFromBeginning(); /* removes 10, list: 20 → 30 */
    return 0;
}

Step-by-Step

1
If head == -1, the list is empty — print message and return.
2
Save temp = head so we know which slot to free later.
3
Update head = list[head].next to skip over the deleted node.
4
Call freeNode(temp) to return the slot to the free list for reuse.

delete_end.c
/* Delete node from the END */
void deleteFromEnd() {
    int curr, prev;

    if (head == -1) {
        printf("List is empty!\n");
        return;
    }

    /* Only one node: free it and reset head */
    if (list[head].next == -1) {
        freeNode(head);
        head = -1;
        printf("Deleted last node.\n");
        return;
    }

    /* Traverse to second-to-last node */
    prev = -1;
    curr = head;
    while (list[curr].next != -1) {
        prev = curr;
        curr = list[curr].next;
    }

    /* Disconnect last node */
    list[prev].next = -1;
    freeNode(curr);

    printf("Deleted node from end.\n");
}

/* === Example === */
int main() {
    init();
    insertAtEnd(10);
    insertAtEnd(20);
    insertAtEnd(30);
    deleteFromEnd(); /* removes 30, list: 10 → 20 */
    return 0;
}

Step-by-Step

1
If list[head].next == -1, there's only one node — free it and set head = -1.
2
Traverse: keep both prev and curr. Stop when list[curr].next == -1.
3
Set list[prev].next = -1 — second-to-last node now points to nothing.
4
Call freeNode(curr) to return the deleted slot to the free list.

06 — Deletion from Specific Position
O(n) — Traverse to position k-1
▼

💡 What is it?

Remove the node at position k (1-based). Traverse to position k−1, bypass node k, free its slot.

⏱️ Time Complexity: O(n)

delete_position.c
/* Delete node at a given position (1-based) */
void deleteAtPosition(int pos) {
    int curr, prev, i;

    if (head == -1) {
        printf("List is empty!\n");
        return;
    }
    if (pos < 1) {
        printf("Invalid position!\n");
        return;
    }

    /* Position 1: same as delete from beginning */
    if (pos == 1) {
        deleteFromBeginning();
        return;
    }

    /* Traverse to node at position pos */
    prev = -1;
    curr = head;
    for (i = 1; i < pos; i++) {
        if (curr == -1) {
            printf("Position %d not found!\n", pos);
            return;
        }
        prev = curr;
        curr = list[curr].next;
    }

    if (curr == -1) {
        printf("Position %d not found!\n", pos);
        return;
    }

    /* Bypass node at position */
    list[prev].next = list[curr].next;
    freeNode(curr);

    printf("Deleted node at position %d.\n", pos);
}

/* === Example === */
int main() {
    init();
    insertAtEnd(10); insertAtEnd(20);
    insertAtEnd(30); insertAtEnd(40);
    deleteAtPosition(2); /* removes 20, list: 10,30,40 */
    return 0;
}

Step-by-Step

1
Handle pos == 1 using deleteFromBeginning().
2
Use two pointers prev and curr. Loop until we reach position pos.
3
Set list[prev].next = list[curr].next — skip over the deleted node.
4
Free the deleted slot with freeNode(curr).

07 — Deletion by Value
O(n) — Search then delete
▼

💡 What is it?

Search for the first node whose data matches a given value, then remove it by relinking its neighbors.

⏱️ Time Complexity: O(n)

Variation: You can extend this to delete all occurrences of a value by not stopping at the first match.
delete_by_value.c
/* Delete first node with given value */
void deleteByValue(int val) {
    int curr, prev;

    if (head == -1) {
        printf("List is empty!\n");
        return;
    }

    /* Check if head node has the value */
    if (list[head].data == val) {
        deleteFromBeginning();
        return;
    }

    /* Traverse searching for value */
    prev = head;
    curr = list[head].next;

    while (curr != -1) {
        if (list[curr].data == val) {
            /* Found it! Bypass and free */
            list[prev].next = list[curr].next;
            freeNode(curr);
            printf("Deleted node with value %d.\n", val);
            return;
        }
        prev = curr;
        curr = list[curr].next;
    }

    printf("Value %d not found in list.\n", val);
}

/* === Example === */
int main() {
    init();
    insertAtEnd(10); insertAtEnd(20);
    insertAtEnd(30); insertAtEnd(20);
    deleteByValue(20); /* removes first 20 */
    return 0;
}

Step-by-Step

1
If head node's data matches, call deleteFromBeginning() and return.
2
Walk through list with prev and curr checking list[curr].data == val.
3
When found: list[prev].next = list[curr].next — bypass the found node, then free it.
4
If the loop ends without finding the value, print "not found".

08 — Traversal (Print Full List)
O(n) — Visit every node once
▼

💡 What is it?

Start at head, visit each node by following next indices, print or process each node's data until we reach -1.

⏱️ Time Complexity: O(n)   Space: O(1)

🌍 Use Cases

  • Printing all elements
  • Computing sum/average
  • Finding max/min value
  • Applying an operation to every node

traversal.c
/* Print all nodes in the linked list */
void traverse() {
    int curr;

    if (head == -1) {
        printf("List is empty!\n");
        return;
    }

    printf("List: ");
    curr = head;

    /* Keep going until we reach end (-1) */
    while (curr != -1) {
        printf("%d", list[curr].data);

        /* Print arrow if not last node */
        if (list[curr].next != -1)
            printf(" -> ");

        curr = list[curr].next;  /* move to next */
    }

    printf(" -> NULL\n");
}

/* === Example === */
int main() {
    init();
    insertAtEnd(10);
    insertAtEnd(20);
    insertAtEnd(30);
    traverse();  /* Output: List: 10 -> 20 -> 30 -> NULL */
    return 0;
}

Step-by-Step

1
Start with curr = head. If head == -1, list is empty.
2
In the while loop, print list[curr].data at each iteration.
3
Update curr = list[curr].next to move forward.
4
Loop ends when curr == -1 — we've reached the end of the list.
SEARCH
09 — Searching (Find Node by Value)
O(n) — Linear search through list
▼

💡 What is it?

Traverse the list checking each node's data until the target is found or the list ends. Returns the position (1-based) or -1 if not found.

CaseTime
Best (found at head)O(1)
AverageO(n/2)
Worst (not found)O(n)
searching.c
/* Search for a value; returns position (1-based) or -1 */
int search(int val) {
    int curr, pos;

    curr = head;
    pos  = 1;

    while (curr != -1) {
        if (list[curr].data == val) {
            printf("Value %d found at position %d.\n", val, pos);
            return pos;
        }
        curr = list[curr].next;
        pos++;
    }

    printf("Value %d not found.\n", val);
    return -1;
}

/* === Example === */
int main() {
    init();
    insertAtEnd(10); insertAtEnd(20); insertAtEnd(30);
    search(20);  /* Output: Value 20 found at position 2. */
    search(99);  /* Output: Value 99 not found. */
    return 0;
}

Step-by-Step

1
Start curr = head and pos = 1.
2
At each node, compare list[curr].data == val. If match, return pos.
3
If no match, advance curr = list[curr].next and increment pos.
4
If loop ends without finding value, return -1.
UPDATE
10 — Updating (Modify Node Data)
O(n) — Search then update in-place
▼

💡 What is it?

Find a node by its position or by its old value, then change its data field to the new value. No relinking needed — only the data changes.

⏱️ Time Complexity: O(n)

updating.c
/* Update node data at given position (1-based) */
void updateAtPosition(int pos, int newData) {
    int curr, i;

    if (head == -1) { printf("List is empty!\n"); return; }
    if (pos < 1)    { printf("Invalid position!\n"); return; }

    curr = head;
    for (i = 1; i < pos; i++) {
        if (curr == -1) {
            printf("Position %d not found!\n", pos);
            return;
        }
        curr = list[curr].next;
    }

    if (curr == -1) { printf("Position not found!\n"); return; }

    printf("Updated position %d: %d → %d\n",
           pos, list[curr].data, newData);
    list[curr].data = newData;  /* simply overwrite the data */
}

/* Update by old value */
void updateByValue(int oldVal, int newVal) {
    int curr = head;
    while (curr != -1) {
        if (list[curr].data == oldVal) {
            list[curr].data = newVal;
            printf("Updated %d → %d\n", oldVal, newVal);
            return;
        }
        curr = list[curr].next;
    }
    printf("Value %d not found.\n", oldVal);
}

/* === Example === */
int main() {
    init();
    insertAtEnd(10); insertAtEnd(20); insertAtEnd(30);
    updateAtPosition(2, 99); /* changes 20 to 99 */
    updateByValue(30, 55);   /* changes 30 to 55 */
    traverse();
    return 0;
}

Step-by-Step

1
Traverse to the target position or search for the old value.
2
Simply set list[curr].data = newData — no relinking required since we don't change structure.
UTILITY
11 — Counting Nodes
O(n) — Traverse and increment counter
▼

💡 What is it?

Walk through the entire list, incrementing a count variable at each node. Return the total.

⏱️ Time Complexity: O(n)

Optimization: Maintain a size variable that you increment on insert and decrement on delete to get count in O(1).
count_nodes.c
/* Count and return total number of nodes */
int countNodes() {
    int curr, count;

    curr  = head;
    count = 0;

    while (curr != -1) {
        count++;                 /* increment for each node */
        curr = list[curr].next;  /* move to next            */
    }

    printf("Total nodes: %d\n", count);
    return count;
}

/* === Example === */
int main() {
    init();
    insertAtEnd(10);
    insertAtEnd(20);
    insertAtEnd(30);
    countNodes();  /* Output: Total nodes: 3 */
    return 0;
}

Step-by-Step

1
Initialize count = 0 and curr = head.
2
In each iteration, do count++ then curr = list[curr].next.
3
When curr == -1, loop ends. Return count.
SORT
12 — Sorting (Bubble Sort on List)
O(n²) — Swap data values between nodes
▼

💡 What is it?

We apply Bubble Sort on the linked list. In each pass, we compare adjacent nodes. If they're out of order, we swap their data values (not the nodes themselves — no relinking needed!).

Array-based advantage: Since we just swap data fields (integers), no index relinking is needed — it's simpler than pointer-based swaps.
CaseTime
Best (already sorted)O(n)
AverageO(n²)

/* Sort list in ascending order using Bubble Sort */
void sortList() {
    int i, j, temp, swapped;

    if (head == -1 || list[head].next == -1) return;

    /* Outer loop: repeated passes */
    i = head;
    while (i != -1) {

        swapped = 0;  /* flag to detect if any swap happened */

        /* Inner loop: compare adjacent pairs */
        j = head;
        while (list[j].next != -1) {
            int nextIdx = list[j].next;

            /* If current data > next data, swap the data */
            if (list[j].data > list[nextIdx].data) {
                temp             = list[j].data;
                list[j].data     = list[nextIdx].data;
                list[nextIdx].data = temp;
                swapped = 1;
            }
            j = list[j].next;
        }

        /* If no swaps happened, list is sorted */
        if (swapped == 0) break;

        i = list[i].next;
    }

    printf("List sorted in ascending order.\n");
}

    return 0;
}

/* Reverse the linked list in-place */
void reverseList() {
    int prev, curr, nextNode;

    if (head == -1 || list[head].next == -1) return;

    prev = -1;        /* previous node index (starts as NULL) */
    curr = head;      /* start at head                        */

    while (curr != -1) {
        nextNode = list[curr].next;  /* save next before overwriting */
        list[curr].next = prev;      /* flip: point to previous      */
        prev = curr;                 /* move prev forward            */
        curr = nextNode;             /* move curr forward            */
    }

    head = prev;  /* prev is now the new head (old last node) */
    printf("List reversed.\n");
}

/* === Example === */

    init();
    insertAtEnd(10); insertAtEnd(20); insertAtEnd(30);
    traverse();      /* 10 -> 20 -> 30 -> NULL */
    reverseList();
    traverse();      /* 30 -> 20 -> 10 -> NULL */
    return 0;
}

Step-by-Step

1
Before changing curr.next, save it: nextNode = list[curr].next (or we lose the rest of the list).
2
Flip: list[curr].next = prev — this node now points backwards.
3
Advance both: prev = curr, curr = nextNode.
4
After the loop, prev is at the old last node — set head = prev.

14 — Rotation by K Positions
O(n) — Make circular then break at right point
▼

💡 What is it?

Rotate the list left by k positions. The first k nodes move to the end. Strategy: find the k-th node, make the list circular temporarily, then break at the right point.

Example: [1→2→3→4→5] rotated left by 2 → [3→4→5→1→2]

⏱️ Time: O(n)   Space: O(1)

rotation.c
/* Rotate list LEFT by k positions */
void rotateLeft(int k) {
    int curr, i, n, kthNode, lastNode;

    if (head == -1 || k <= 0) return;

    /* Step 1: Count length */
    n = countNodes();
    k = k % n;  /* normalize k (avoid full rotations) */
    if (k == 0) return;

    /* Step 2: Find the k-th node (new last node after rotation) */
    curr = head;
    for (i = 1; i < k; i++) {
        curr = list[curr].next;
    }
    kthNode = curr;  /* this becomes the new tail */

    /* Step 3: Find the last node */
    lastNode = kthNode;
    while (list[lastNode].next != -1) {
        lastNode = list[lastNode].next;
    }

    /* Step 4: Make list circular temporarily */
    list[lastNode].next = head;

    /* Step 5: New head is node after kthNode */
    head = list[kthNode].next;

    /* Step 6: Break the circle */
    list[kthNode].next = -1;

    printf("List rotated left by %d.\n", k);
}

/* === Example === */
int main() {
    init();
    insertAtEnd(1); insertAtEnd(2); insertAtEnd(3);
    insertAtEnd(4); insertAtEnd(5);
    traverse();        /* 1 -> 2 -> 3 -> 4 -> 5 */
    rotateLeft(2);
    traverse();        /* 3 -> 4 -> 5 -> 1 -> 2 */
    return 0;
}

Step-by-Step

1
Normalize: k = k % n to handle k larger than list size.
2
Traverse to the k-th node — this becomes the new tail after rotation.
3
Find the last node, then set list[last].next = head to make it circular.
4
Update head = list[kthNode].next then break: list[kthNode].next = -1.

15 — Reverse in Groups of K
O(n) — Reverse k nodes at a time
▼

💡 What is it?

Divide the list into groups of k nodes. Reverse each group individually, then connect the reversed groups together.

Example: [1→2→3→4→5→6] with k=2 → [2→1→4→3→6→5]

⏱️ Time: O(n)   Space: O(1)

reverse_groups.c
/* Reverse nodes in groups of size k */
int reverseGroup(int startIdx, int k) {
    int prev, curr, nextNode, count;

    prev = -1;
    curr = startIdx;
    count = 0;

    /* Reverse k nodes */
    while (curr != -1 && count < k) {
        nextNode = list[curr].next;
        list[curr].next = prev;
        prev = curr;
        curr = nextNode;
        count++;
    }
    /* prev = new head of this reversed group
       curr = start of remaining list          */
    return prev;  /* return new head of this group */
}

void reverseInGroups(int k) {
    int curr, prevGroupTail, groupHead, newHead;
    int isFirstGroup;

    if (head == -1 || k <= 1) return;

    curr          = head;
    prevGroupTail = -1;
    isFirstGroup  = 1;

    while (curr != -1) {
        int groupStart = curr;
        int i, temp = curr;

        /* Find the start of next group */
        for (i = 0; i < k && temp != -1; i++) {
            curr = list[temp].next;
            temp = curr;
        }
        /* curr now points to start of next group */

        /* Get the new head after reversing this group */
        groupHead = reverseInGroupsHelper(groupStart, k);

        if (isFirstGroup) {
            head = groupHead;
            isFirstGroup = 0;
        } else {
            list[prevGroupTail].next = groupHead;
        }

        prevGroupTail = groupStart; /* old start = new tail */
        list[prevGroupTail].next = curr; /* connect to rest */
    }
    printf("List reversed in groups of %d.\n", k);
}

/* Helper: reverse k nodes starting from startIdx */
int reverseInGroupsHelper(int startIdx, int k) {
    int prev = -1, curr = startIdx, nextNode, count = 0;
    while (curr != -1 && count < k) {
        nextNode = list[curr].next;
        list[curr].next = prev;
        prev = curr;
        curr = nextNode;
        count++;
    }
    return prev;
}

int main() {
    init();
    insertAtEnd(1); insertAtEnd(2); insertAtEnd(3);
    insertAtEnd(4); insertAtEnd(5); insertAtEnd(6);
    traverse();             /* 1 -> 2 -> 3 -> 4 -> 5 -> 6 */
    reverseInGroups(2);
    traverse();             /* 2 -> 1 -> 4 -> 3 -> 6 -> 5 */
    return 0;
}

Step-by-Step

1
For each group of k nodes, apply the same reverse logic (prev/curr/nextNode technique).
2
After reversing a group, the old start becomes the new tail of that group.
3
Connect the previous group's tail to the new group's head.
4
Repeat until all groups are processed. First group's new head becomes the overall head.

/* Merge two sorted lists (head1 and head2) into one sorted list.
   Returns: index of new merged list head */
int mergeSorted(int head1, int head2) {
    int mergedHead, tail, curr1, curr2;

    /* If either list is empty, return the other */
    if (head1 == -1) return head2;
    if (head2 == -1) return head1;

    /* Initialize merged list with the smaller first element */
    if (list[head1].data <= list[head2].data) {
        mergedHead = head1;
        curr1 = list[head1].next;
        curr2 = head2;
    } else {
        mergedHead = head2;
        curr1 = head1;
        curr2 = list[head2].next;
    }

    tail = mergedHead;

    /* Compare and link nodes from both lists */
    while (curr1 != -1 && curr2 != -1) {
        if (list[curr1].data <= list[curr2].data) {
            list[tail].next = curr1;
            curr1 = list[curr1].next;
        } else {
            list[tail].next = curr2;
            curr2 = list[curr2].next;
        }
        tail = list[tail].next;
    }

    /* Attach remaining nodes of whichever list is left */
    if (curr1 != -1) list[tail].next = curr1;
    else             list[tail].next = curr2;

    return mergedHead;
}

    /* Build list2: 2 -> 4 -> 6 */

    mergedHead = mergeSorted(head1, head2);

💡 What is it?

Split a list into two halves. Use the slow-and-fast pointer technique to find the middle, then disconnect at that point.

⏱️ Time: O(n)   Space: O(1)

Used in: Merge Sort — recursively split then merge back sorted.

/* Split list into two halves.
   head1Out and head2Out are output parameters (indices). */
void splitList(int srcHead, int *head1Out, int *head2Out) {
    int slow, fast;

    if (srcHead == -1) {
        *head1Out = -1;
        *head2Out = -1;
        return;
    }

    /* Only one node: first half has it, second is empty */
    if (list[srcHead].next == -1) {
        *head1Out = srcHead;
        *head2Out = -1;
        return;
    }

    slow = srcHead;
    fast = list[srcHead].next;

    /* Fast moves 2 steps, slow moves 1 step */
    while (fast != -1) {
        fast = list[fast].next;
        if (fast != -1) {
            fast = list[fast].next;
            slow = list[slow].next;
        }
    }

    /* Slow is now at the midpoint */
    *head1Out = srcHead;          /* first half starts here  */
    *head2Out = list[slow].next;  /* second half starts here */
    list[slow].next = -1;         /* break the link          */

    printf("List split into two halves.\n");
}

/* === Example === */
int main() {
    int h1, h2;
    init();
    insertAtEnd(1); insertAtEnd(2);
    insertAtEnd(3); insertAtEnd(4); insertAtEnd(5);
    splitList(head, &h1, &h2);
    /* h1: 1->2->3, h2: 4->5 */
    return 0;
}

Step-by-Step

1
Use slow and fast pointers: fast = list[srcHead].next (one step ahead of slow).
2
Move fast 2 steps, slow 1 step each iteration. When fast reaches end, slow is at midpoint.
3
Second half starts at list[slow].next. Break by setting list[slow].next = -1.

COPY
20 — Copying a List (Deep Copy)
O(n) — Create an independent duplicate
▼

💡 What is it?

Create a completely independent duplicate of a list. New nodes are allocated for every node. Changes to the copy don't affect the original.

Deep Copy vs Shallow Copy: Deep copy allocates new nodes. Shallow copy would just copy the head index — both lists would share the same nodes (dangerous!). Always use deep copy.

⏱️ Time: O(n)   Space: O(n)

copying.c
/* Deep copy of a linked list.
   Returns: head index of the new copy */
int deepCopy(int srcHead) {
    int curr, newHead, newIdx, tail;

    if (srcHead == -1) return -1;

    /* Copy first node to initialize new list */
    newHead = allocNode(list[srcHead].data);
    if (newHead == -1) return -1;

    tail = newHead;
    curr = list[srcHead].next;

    /* Copy remaining nodes */
    while (curr != -1) {
        newIdx = allocNode(list[curr].data);
        if (newIdx == -1) break;

        list[tail].next = newIdx;  /* link to new list */
        tail = newIdx;             /* advance tail      */
        curr = list[curr].next;    /* advance source    */
    }

    printf("List copied successfully.\n");
    return newHead;
}

/* === Example === */
int main() {
    int copyHead;
    init();
    insertAtEnd(10);
    insertAtEnd(20);
    insertAtEnd(30);

    copyHead = deepCopy(head);

    /* Original list unchanged */
    head = head;
    traverse();       /* 10 -> 20 -> 30 */

    /* Copy is independent */
    head = copyHead;
    traverse();       /* 10 -> 20 -> 30 */
    return 0;
}

Step-by-Step

1
Copy the first (source) node using allocNode(). This becomes newHead.
2
Traverse source. For each node, allocate a fresh slot and copy data.
3
Link new node to the copy list via tail. Advance tail.
4
Return newHead — a completely independent duplicate.

22 — Removing Duplicates
O(n²) unsorted | O(n) sorted — Eliminate repeated values
▼

💡 What is it?

Remove all nodes that have the same value as a previous node. Two approaches:

  • Sorted list: Compare adjacent nodes — O(n)
  • Unsorted list: For each node, scan rest of list for duplicates — O(n²)

⏱️ Sorted: O(n)   Unsorted: O(n²)

remove_duplicates.c
/* Remove duplicates from a SORTED list - O(n) */
void removeDuplicatesSorted() {
    int curr, nextIdx;

    if (head == -1) return;

    curr = head;

    while (list[curr].next != -1) {
        nextIdx = list[curr].next;

        if (list[curr].data == list[nextIdx].data) {
            /* Same value: skip and free next node */
            list[curr].next = list[nextIdx].next;
            freeNode(nextIdx);
        } else {
            /* Different value: move forward */
            curr = list[curr].next;
        }
    }
    printf("Duplicates removed.\n");
}

/* Remove duplicates from an UNSORTED list - O(n^2) */
void removeDuplicatesUnsorted() {
    int curr, runner, prev;

    curr = head;

    while (curr != -1) {
        runner = curr;

        /* For each curr, scan rest of list for same value */
        while (list[runner].next != -1) {
            int runNext = list[runner].next;
            if (list[runNext].data == list[curr].data) {
                /* Found duplicate: remove runNext */
                list[runner].next = list[runNext].next;
                freeNode(runNext);
            } else {
                runner = list[runner].next;
            }
        }
        curr = list[curr].next;
    }
    printf("Duplicates removed (unsorted).\n");
}

/* === Example === */
int main() {
    init();
    insertAtEnd(1); insertAtEnd(1); insertAtEnd(2);
    insertAtEnd(3); insertAtEnd(3); insertAtEnd(3);
    traverse();              /* 1 -> 1 -> 2 -> 3 -> 3 -> 3 */
    removeDuplicatesSorted();
    traverse();              /* 1 -> 2 -> 3 */
    return 0;
}

Step-by-Step (Sorted)

1
Start at curr = head. Check if curr and curr.next have same data.
2
If same: bypass next node (curr.next = next.next) and free the duplicate.
3
If different: just advance curr = curr.next. Repeat until end.

💡 What is it?

Detect if a linked list has a cycle (a node's next points back to a previous node). Uses Floyd's Cycle Detection Algorithm (also called the "tortoise and hare" algorithm).

  • Finding start of cycle in O(n)

/* Detect loop using Floyd's Cycle Detection Algorithm */
int detectLoop() {
    int slow, fast;

    if (head == -1) return 0;  /* empty list, no loop */

    slow = head;
    fast = head;

    /* Slow: 1 step, Fast: 2 steps */
    while (fast != -1 && list[fast].next != -1) {
        slow = list[slow].next;         /* 1 step */
        fast = list[fast].next;
        fast = list[fast].next;         /* 2nd step */

        if (slow == fast) {
            /* They met! Loop exists */
            printf("Loop DETECTED in the list!\n");
            return 1;
        }
    }

    /* Fast reached end without meeting slow: no loop */
    printf("No loop found.\n");
    return 0;
}

/* Create a loop for testing (node at pos points to nodeAt) */
void createLoop(int fromPos, int toPos) {
    int from, to, i;

    from = head; to = head;
    for (i = 1; i < fromPos; i++) from = list[from].next;
    for (i = 1; i < toPos; i++)   to   = list[to].next;

    list[from].next = to;  /* create the cycle */
    printf("Loop created: node%d -> node%d\n", fromPos, toPos);
}

/* === Example === */
int main() {
    init();
    insertAtEnd(1); insertAtEnd(2);
    insertAtEnd(3); insertAtEnd(4);
    detectLoop();       /* No loop found */
    createLoop(4, 2);   /* node 4 points back to node 2 */
    detectLoop();       /* Loop DETECTED */
    return 0;
}

Step-by-Step

1
Start both slow and fast at head.
2
Each iteration: slow moves 1 step, fast moves 2 steps.
3
If slow == fast at any point → loop detected! Return 1.
4
If fast reaches -1 (or fast.next == -1) → no loop. Return 0.
FLATTEN
24 — Flattening a Multilevel List
O(n) — Convert nested structure to single-level
▼

💡 What is it?

A multilevel list has nodes where some nodes also have a child pointer pointing to another sub-list. Flattening merges all levels into a single linear list.

Approach: When we encounter a node with a child list, splice the child list between the current node and its next. Continue traversal through the flattened portion.

⏱️ Time: O(n) where n = total nodes across all levels

🌍 Use Cases

  • Flattening nested menus or UI trees
  • Processing hierarchical XML/JSON into flat arrays
  • LeetCode problem: Flatten a Multilevel Doubly Linked List
// Multilevel → Flat
Level 1:
[1] → [2] → [3] → [4]
↓ child
[10] → [11] → [12]
After flattening:
[1] → [2] → [10] → [11] → [12] → [3] → [4]
flattening.c
/* Extended Node structure for multilevel list */
struct NodeML {
    int  data;
    int  next;   /* index of next sibling */
    int  child;  /* index of child list head (-1 if none) */
};

struct NodeML mlist[MAX];  /* multilevel array */
int mhead = -1;
int mfree = 0;

void initML() {
    int i;
    for (i = 0; i < MAX - 1; i++) {
        mlist[i].next  = i + 1;
        mlist[i].child = -1;
    }
    mlist[MAX-1].next  = -1;
    mlist[MAX-1].child = -1;
    mfree = 0;
    mhead = -1;
}

/* Flatten the multilevel list starting from startHead */
int flatten(int startHead) {
    int curr, childHead, lastOfChild;

    curr = startHead;

    while (curr != -1) {
        /* If this node has a child list */
        if (mlist[curr].child != -1) {
            childHead = mlist[curr].child;

            /* Find the tail of the child list */
            lastOfChild = childHead;
            while (mlist[lastOfChild].next != -1) {
                lastOfChild = mlist[lastOfChild].next;
            }

            /* Splice: curr.next → child list → curr's old next */
            mlist[lastOfChild].next = mlist[curr].next;
            mlist[curr].next        = childHead;
            mlist[curr].child       = -1;  /* remove child link */
        }

        curr = mlist[curr].next;  /* move forward */
    }

    printf("List flattened.\n");
    return startHead;
}

/* === Example (conceptual setup) === */
int main() {
    initML();
    /* Set up multilevel structure manually */
    /* Level1: node0(1) → node1(2) → node2(3) */
    /* node1 has child: node3(10) → node4(11)  */
    /* After flatten: 1 → 2 → 10 → 11 → 3     */
    mhead = flatten(mhead);
    return 0;
}

Step-by-Step

1
Our extended struct has next (sibling) and child (sub-list head). Both are indices, -1 means none.
2
Traverse main list. When a node has child != -1, we need to flatten that child.
3
Find the tail of the child list, then: childTail.next = curr.next and curr.next = childHead.
4
Clear curr.child = -1 and continue. The child list is now part of the main level and will be traversed automatically.
