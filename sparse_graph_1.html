Introduction to Graphs & Sparse Graphs

📖 What is a Graph?

A graph is a data structure made of vertices (nodes) connected by edges (links). It models relationships between objects.

  • V = set of vertices (nodes)
  • E = set of edges (connections between nodes)
  • Notation: G = (V, E)

📖 What is a Sparse Graph?

A graph is called sparse when the number of edges is much less than the maximum possible edges.

  • Max edges in undirected graph: V×(V−1)/2
  • Max edges in directed graph: V×(V−1)
  • Rule of thumb: E ≪ V² → sparse. E ≈ V² → dense.
  • Formally sparse when: E = O(V)

🌍 Real-Life Analogy

City Road Map: A city has 1000 intersections (vertices). If every intersection connected to every other, there'd be ~500,000 roads. In reality, there are maybe 2000–3000 roads — that's a sparse graph! Most roads connect only nearby intersections.

Adjacency Matrix Representation

📖 Adjacency Matrix

An adjacency matrix is a 2D array of size V×V where matrix[i][j] = 1 if there's an edge from vertex i to vertex j, and 0 otherwise.

  • Space: O(V²) — even if edges are few
  • Edge check: O(1) — very fast!
  • For sparse graphs: wastes memory (mostly zeros)
  • Symmetric for undirected graphs

C — Adjacency Matrix Structure
// Sparse Graph using Adjacency Matrix in C
// MAX_V = maximum number of vertices

#define MAX_V 10

typedef struct {
    int adj[MAX_V][MAX_V];  // 2D matrix: 0 or 1
    int numVertices;        // current vertex count
    int numEdges;           // current edge count
} Graph;

/* Initialize an empty graph */
void initGraph(Graph *g, int v) {
    g->numVertices = v;
    g->numEdges = 0;
    for (int i = 0; i < v; i++)
        for (int j = 0; j < v; j++)
            g->adj[i][j] = 0;  // all zeros = no edges
}

🌍 Real-Life Analogy

Friendship Table: Imagine a classroom of 5 students. Draw a table with all names on both rows and columns. Put a ✓ where two students are friends. Most cells will be empty (0) — that's an adjacency matrix of a sparse social graph!

03
Insert & Delete Vertex Easy

📖 Insert Vertex

Adding a new vertex increases numVertices by 1. The new row and column in the matrix are initialized to 0 (no edges yet).

  • Time: O(V) — clear new row/col
  • Space: +1 row and +1 column in matrix

📖 Delete Vertex

Remove a vertex by shifting all rows and columns after it, reducing numVertices. All edges connected to it are also removed.

  • Time: O(V²) — shift rows and cols
  • All edges of that vertex are lost
// Insert & Delete Vertex Animation
C — Insert Vertex
void insertVertex(Graph *g) {
    if (g->numVertices >= MAX_V) {
        printf("Graph is full! Cannot insert vertex.\n");
        return;
    }
    int v = g->numVertices;  // new vertex index
    for (int i = 0; i <= v; i++) {
        g->adj[v][i] = 0;  // new row: all zero
        g->adj[i][v] = 0;  // new col: all zero
    }
    g->numVertices++;
    printf("Vertex %d inserted.\n", v);
}
C — Delete Vertex
void deleteVertex(Graph *g, int v) {
    if (v < 0 || v >= g->numVertices) {
        printf("Invalid vertex!\n");
        return;
    }
    // Shift rows up: remove row v
    for (int i = v; i < g->numVertices - 1; i++)
        for (int j = 0; j < g->numVertices; j++)
            g->adj[i][j] = g->adj[i+1][j];
    // Shift columns left: remove col v
    for (int i = 0; i < g->numVertices - 1; i++)
        for (int j = v; j < g->numVertices - 1; j++)
            g->adj[i][j] = g->adj[i][j+1];
    g->numVertices--;
    printf("Vertex %d deleted.\n", v);
}

🌍 Analogy

Add/Remove a member from a WhatsApp group: Adding a person = insert vertex (they can now have connections). Removing = delete vertex (all their message threads disappear).

📖 Search Vertex

Check if a vertex exists by verifying its index is within [0, numVertices-1]. In labeled graphs, search by name/value in a label array.

📖 Update Vertex

Update a vertex's label or data. The adjacency matrix structure itself doesn't change — only the vertex's metadata (stored in a separate array) is updated.

        printf("Invalid vertex index!\n");
        return;

C — Insert & Delete Edge (Undirected)
/* Insert Edge — O(1) */
void insertEdge(Graph *g, int u, int v) {
    if (u < 0 || v < 0 || u >= g->numVertices || v >= g->numVertices) {
        printf("Invalid vertices!\n");
        return;
    }
    if (g->adj[u][v] == 1) {
        printf("Edge already exists!\n");
        return;
    }
    g->adj[u][v] = 1;  // directed: u → v
    g->adj[v][u] = 1;  // undirected: also v → u
    g->numEdges++;
    printf("Edge (%d-%d) inserted.\n", u, v);
}

/* Delete Edge — O(1) */
void deleteEdge(Graph *g, int u, int v) {
    if (g->adj[u][v] == 0) {
        printf("Edge does not exist!\n");
        return;
    }
    g->adj[u][v] = 0;
    g->adj[v][u] = 0;
    g->numEdges--;
    printf("Edge (%d-%d) deleted.\n", u, v);
}

Search & Update Edge Easy

📖 Search Edge

Check if edge (u,v) exists: simply look at adj[u][v]. If it's 1, edge exists. Time: O(1) — best feature of adjacency matrix!

📖 Update Edge (Weighted)

For weighted graphs, the matrix stores weights instead of 0/1. Update: set adj[u][v] = newWeight. Use 0 or -1 for "no edge".

C — Search & Update Edge
/* Search Edge — O(1) — best feature! */
int searchEdge(Graph *g, int u, int v) {
    if (u < 0 || v < 0 || u >= g->numVertices || v >= g->numVertices)
        return -1;  // invalid
    return g->adj[u][v];  // 1=exists, 0=no edge
}

/* Update Edge Weight (for weighted graph) */
/* Use int adj[][] instead of 0/1 */
void updateEdge(Graph *g, int u, int v, int weight) {
    if (g->adj[u][v] == 0) {
        printf("Edge doesn't exist. Insert first.\n");
        return;
    }
    g->adj[u][v] = weight;
    g->adj[v][u] = weight;  // for undirected
    printf("Edge (%d-%d) weight updated to %d\n", u, v, weight);
}

/* Usage example */
int main() {
    Graph g;
    initGraph(&g, 5);
    insertEdge(&g, 0, 2);
    if (searchEdge(&g, 0, 2))
        printf("Edge (0-2) EXISTS!\n");
    else
        printf("Edge (0-2) NOT found.\n");
    return 0;
}

Breadth First Search (BFS) Medium

📖 BFS Explained

BFS visits nodes level by level — first all neighbors of start, then their neighbors, and so on. It uses a Queue (FIFO) data structure.

  • Start: enqueue the start vertex, mark visited
  • Loop: dequeue vertex, visit all unvisited neighbors → enqueue them
  • Finds shortest path in unweighted graphs
  • Time: O(V²) with adjacency matrix
// BFS Step-by-Step Visualizer
Queue: [ ]
C — BFS with Queue
/* BFS — Breadth First Search */
void BFS(Graph *g, int start) {
    int visited[MAX_V] = {0};  // track visited nodes
    int queue[MAX_V];           // simple array queue
    int front = 0, rear = 0;

    /* Step 1: enqueue start, mark visited */
    queue[rear++] = start;
    visited[start] = 1;
    printf("BFS Order: ");

    while (front < rear) {
        /* Step 2: dequeue front vertex */
        int curr = queue[front++];
        printf("%d ", curr);  // visit node

        /* Step 3: enqueue all unvisited neighbors */
        for (int j = 0; j < g->numVertices; j++) {
            if (g->adj[curr][j] == 1 && !visited[j]) {
                queue[rear++] = j;
                visited[j] = 1;  // mark before enqueue!
            }
        }
    }
    printf("\n");
}

🌍 Analogy

Ripples in Water: Drop a stone (start node). Ripples spread outward in all directions equally — first ring = level 1 neighbors, second ring = level 2. That's BFS! Also like spreading news: you tell your friends, they tell their friends.

Depth First Search (DFS) Medium

📖 DFS Explained

DFS goes as deep as possible before backtracking. It uses a Stack (or recursion).

  • Recursive DFS: simple and elegant — call stack acts as the stack
  • Iterative DFS: use explicit stack array
  • Backtracking: when no unvisited neighbor → go back
  • Time: O(V²) with adjacency matrix
// DFS Step-by-Step Visualizer
Stack: [ ]
C — Recursive DFS
int visited[MAX_V];  // global visited array

/* Recursive DFS — elegant solution */
void DFS_Recursive(Graph *g, int v) {
    visited[v] = 1;
    printf("%d ", v);  // visit node

    for (int j = 0; j < g->numVertices; j++) {
        if (g->adj[v][j] == 1 && !visited[j]) {
            DFS_Recursive(g, j);  // go deeper!
        }
        // backtrack happens automatically here
    }
}

void DFS(Graph *g, int start) {
    memset(visited, 0, sizeof(visited));
    printf("DFS Order: ");
    DFS_Recursive(g, start);
    printf("\n");
}
C — Iterative DFS (Explicit Stack)
/* Iterative DFS using Stack Array */
void DFS_Stack(Graph *g, int start) {
    int visited[MAX_V] = {0};
    int stack[MAX_V];
    int top = -1;

    stack[++top] = start;  // push start
    printf("DFS (Stack) Order: ");

    while (top >= 0) {
        int curr = stack[top--];  // pop
        if (visited[curr]) continue;

        visited[curr] = 1;
        printf("%d ", curr);  // visit

        /* Push neighbors (reverse for same order as recursive) */
        for (int j = g->numVertices-1; j >= 0; j--) {
            if (g->adj[curr][j] == 1 && !visited[j])
                stack[++top] = j;
        }
    }
    printf("\n");
}

🌍 Analogy

Maze Exploration: DFS = always take the first available path, go as far as possible. Hit a dead end? Backtrack and try another path. Like solving a maze by always going straight until you can't.

C — Sparse Analysis Functions
/* Check if graph is sparse (density < 25%) */
int isSparse(Graph *g) {
    int maxEdges = g->numVertices * (g->numVertices - 1) / 2;
    if (maxEdges == 0) return 1;
    float density = (float)g->numEdges / maxEdges;
    return density < 0.25;  // sparse if < 25% full
}

/* Calculate edge density as percentage */
float edgeDensity(Graph *g) {
    int maxEdges = g->numVertices * (g->numVertices - 1) / 2;
    if (maxEdges == 0) return 0.0;
    return (float)g->numEdges / maxEdges * 100;
}

/* Compare edge ratio */
void compareEdgeRatio(Graph *g) {
    int maxE = g->numVertices * (g->numVertices - 1) / 2;
    int missing = maxE - g->numEdges;
    printf("Actual edges: %d / %d max\n", g->numEdges, maxE);
    printf("Missing edges: %d\n", missing);
    printf("Density: %.1f%%\n", edgeDensity(g));
    printf("Type: %s\n", isSparse(g) ? "SPARSE" : "DENSE");
}

/* Missing Edge Detection */
void findMissingEdges(Graph *g) {
    printf("Missing edges:\n");
    for (int i = 0; i < g->numVertices; i++)
        for (int j = i+1; j < g->numVertices; j++)
            if (g->adj[i][j] == 0)
                printf("  (%d, %d)\n", i, j);
}

/* Graph Compression Analysis */
void compressionAnalysis(Graph *g) {
    int matrixSize = g->numVertices * g->numVertices;
    int listSize = g->numVertices + 2 * g->numEdges;
    printf("Matrix storage: %d cells\n", matrixSize);
    printf("List storage: %d cells\n", listSize);
    printf("Savings with adj list: %d cells\n", matrixSize - listSize);
}

🌍 Analogy

Parking Lot: A 100-spot parking lot (max edges). If only 8 cars are parked (actual edges), the lot is 8% full = sparse! Edge density tells you how "full" your graph is.

Connectivity Operations Medium

📖 Connected Graph

A graph is connected if there's a path between every pair of vertices. Use BFS/DFS from any vertex — if all vertices are visited, it's connected.

📖 Connected Components

Sub-graphs where all vertices can reach each other. A disconnected graph has 2+ components. Run DFS from every unvisited node.

C — All Connectivity Operations
/* DFS helper — marks all reachable from v */
void dfsHelper(Graph *g, int v, int *vis) {
    vis[v] = 1;
    for (int j = 0; j < g->numVertices; j++)
        if (g->adj[v][j] && !vis[j])
            dfsHelper(g, j, vis);
}

/* Check if graph is connected */
int isConnected(Graph *g) {
    int vis[MAX_V] = {0};
    dfsHelper(g, 0, vis);
    for (int i = 0; i < g->numVertices; i++)
        if (!vis[i]) return 0;  // unreachable vertex!
    return 1;
}

/* Count connected components */
int countComponents(Graph *g) {
    int vis[MAX_V] = {0};
    int components = 0;
    for (int i = 0; i < g->numVertices; i++) {
        if (!vis[i]) {
            dfsHelper(g, i, vis);
            components++;  // new component found
        }
    }
    return components;
}

/* Reachability check — can we go from src to dst? */
int canReach(Graph *g, int src, int dst) {
    int vis[MAX_V] = {0};
    dfsHelper(g, src, vis);
    return vis[dst];
}

/* Detect isolated vertices (degree 0) */
void findIsolated(Graph *g) {
    printf("Isolated vertices: ");
    for (int i = 0; i < g->numVertices; i++) {
        int deg = 0;
        for (int j = 0; j < g->numVertices; j++)
            deg += g->adj[i][j];
        if (deg == 0) printf("%d ", i);
    }
    printf("\n");
}

/* Weak Connectivity (treat directed as undirected) */
int isWeaklyConnected(Graph *g) {
    Graph undirected = *g;
    /* Make symmetric — add both directions */
    for (int i = 0; i < g->numVertices; i++)
        for (int j = 0; j < g->numVertices; j++)
            if (g->adj[i][j]) undirected.adj[j][i] = 1;
    return isConnected(&undirected);
}

🌍 Analogy

Island Network: Each island = vertex, bridges = edges. Connected graph = you can travel between any two islands. Components = groups of islands connected among themselves but not to others. Isolated vertex = an island with no bridges!

Path Operations Medium

📖 Path Operations Overview

  • Find Path: DFS with parent tracking — find any path src→dst
  • Shortest Path: BFS gives shortest path in unweighted graphs
  • All Paths: Backtracking DFS — explore every possible route
  • Path Length: Count edges in the path
C — Path Operations (Find + Shortest + All)
/* Find ONE path from src to dst using DFS */
int path[MAX_V], pathLen = 0;
int vis2[MAX_V] = {0};

int findPath(Graph *g, int curr, int dst) {
    vis2[curr] = 1;
    path[pathLen++] = curr;
    if (curr == dst) return 1;  // found!
    for (int j = 0; j < g->numVertices; j++) {
        if (g->adj[curr][j] && !vis2[j])
            if (findPath(g, j, dst)) return 1;
    }
    pathLen--;  // backtrack
    return 0;
}

/* Shortest Path using BFS — O(V²) */
void shortestPath(Graph *g, int src, int dst) {
    int vis[MAX_V]={0}, parent[MAX_V], dist[MAX_V];
    int queue[MAX_V], front=0, rear=0;
    for(int i=0;i<MAX_V;i++) parent[i]=-1, dist[i]=-1;

    queue[rear++]=src; vis[src]=1; dist[src]=0;
    while(front < rear) {
        int u=queue[front++];
        if(u==dst) break;
        for(int j=0;j<g->numVertices;j++)
            if(g->adj[u][j]&&!vis[j]){
                vis[j]=1; parent[j]=u;
                dist[j]=dist[u]+1;
                queue[rear++]=j;
            }
    }
    if(dist[dst]==-1){printf("No path!\n");return;}
    /* Reconstruct path by following parents */
    int p[MAX_V],k=0,cur=dst;
    while(cur!=-1){p[k++]=cur;cur=parent[cur];}
    printf("Shortest (%d hops): ",dist[dst]);
    for(int i=k-1;i>=0;i--) printf("%d%s",p[i],i?"→":"\n");
}

/* All Paths — backtracking */
void allPaths(Graph *g, int curr, int dst,
              int *vis, int *p, int len) {
    vis[curr]=1; p[len++]=curr;
    if(curr==dst){
        for(int i=0;i<len;i++) printf("%d%s",p[i],i<len-1?"→":"\n");
    } else
        for(int j=0;j<g->numVertices;j++)
            if(g->adj[curr][j]&&!vis[j])
                allPaths(g,j,dst,vis,p,len);
    vis[curr]=0;  // backtrack — unmark!
}

🌍 Analogy

Google Maps: Find Path = any route from A to B. Shortest Path = fastest route. All Paths = every possible route (like when Maps shows 3 alternatives). Path Length = distance in km.

Cycle Operations Medium

📖 Cycles in Graphs

  • Cycle: A path that starts and ends at the same vertex
  • Detect Cycle: DFS — if we visit an already-visited non-parent node, there's a cycle
  • Tree: Connected graph with NO cycles. Exactly V-1 edges.
  • Forest: Multiple trees (disconnected, no cycles)
C — Cycle Detection, Tree & Forest Validation
/* Detect Cycle using DFS (undirected graph) */
int hasCycleUtil(Graph *g, int v, int *vis, int parent) {
    vis[v] = 1;
    for (int j = 0; j < g->numVertices; j++) {
        if (!g->adj[v][j]) continue;
        if (!vis[j]) {
            if (hasCycleUtil(g, j, vis, v)) return 1;
        } else if (j != parent) {
            return 1;  // back edge = CYCLE!
        }
    }
    return 0;
}

int hasCycle(Graph *g) {
    int vis[MAX_V] = {0};
    for (int i = 0; i < g->numVertices; i++)
        if (!vis[i] && hasCycleUtil(g, i, vis, -1))
            return 1;
    return 0;
}

/* Check if graph is a Tree */
/* Tree = connected + no cycles + V-1 edges */
int isTree(Graph *g) {
    if (g->numEdges != g->numVertices - 1) return 0;
    return isConnected(g) && !hasCycle(g);
}

/* Forest = no cycles (may be disconnected) */
int isForest(Graph *g) {
    return !hasCycle(g);
}

/* Count cycles (approximate — by component) */
int countCycles(Graph *g) {
    /* Cycle count = E - V + Components */
    return g->numEdges - g->numVertices + countComponents(g);
}

🌍 Analogy

Family Tree: A proper family tree has NO cycles — you can't be your own ancestor! If a cycle is detected, something's wrong. A forest = multiple separate family trees.

Degree Operations Easy

📖 Degree of a Vertex

The degree of a vertex = number of edges connected to it = count of 1s in its row (or column) in the adjacency matrix.

  • Sum of all degrees = 2 × number of edges (Handshaking Lemma)
  • In sparse graphs, most vertices have low degree
// Degree Distribution Visualizer
C — All Degree Operations
/* Degree of one vertex — O(V) */
int findDegree(Graph *g, int v) {
    int deg = 0;
    for (int j = 0; j < g->numVertices; j++)
        deg += g->adj[v][j];  // count 1s in row v
    return deg;
}

/* Maximum degree vertex */
int maxDegree(Graph *g) {
    int maxD = 0;
    for (int i = 0; i < g->numVertices; i++) {
        int d = findDegree(g, i);
        if (d > maxD) maxD = d;
    }
    return maxD;
}

/* Minimum degree vertex */
int minDegree(Graph *g) {
    int minD = g->numVertices;
    for (int i = 0; i < g->numVertices; i++) {
        int d = findDegree(g, i);
        if (d < minD) minD = d;
    }
    return minD;
}

/* Average degree = 2*E / V */
float avgDegree(Graph *g) {
    return (float)(2 * g->numEdges) / g->numVertices;
}

/* Degree Distribution — count vertices per degree */
void degreeDistribution(Graph *g) {
    int dist[MAX_V] = {0};
    for (int i = 0; i < g->numVertices; i++)
        dist[findDegree(g, i)]++;
    printf("Degree Distribution:\n");
    for (int d = 0; d < g->numVertices; d++)
        if (dist[d] > 0)
            printf("  Degree %d: %d vertices\n", d, dist[d]);
}

📖 Graph Size vs Order

  • Order: Number of vertices (|V|)
  • Size: Number of edges (|E|)
  • Density Check: E / max_possible_E

📖 Sparse/Dense Classification

  • Sparse: E ≈ O(V) — few edges
  • Dense: E ≈ O(V²) — many edges
  • Tree: E = V-1, connected, acyclic
  • Forest: E < V, acyclic (multi-tree)

📖 Minimum Spanning Tree (MST)

A tree that connects all vertices with minimum total edge weight. Used in network design (cheapest cable layout, etc.).

  • Kruskal's Algorithm: Sort edges by weight, add edge if it doesn't form a cycle (uses Union-Find)
  • Uses Union-Find / DSU: Disjoint Set Union to detect cycles efficiently
C — Kruskal's MST Algorithm
/* Edge structure for weighted graph */
typedef struct {
    int u, v, weight;
} Edge;

Edge edges[MAX_V * MAX_V];
int edgeCount = 0;

/* Extract all edges from adjacency matrix */
void extractEdges(Graph *g) {
    edgeCount = 0;
    for (int i = 0; i < g->numVertices; i++)
        for (int j = i+1; j < g->numVertices; j++)
            if (g->adj[i][j])
                edges[edgeCount++] = (Edge){i, j, g->adj[i][j]};
}

/* Compare for qsort */
int cmpEdge(const void *a, const void *b) {
    return ((Edge*)a)->weight - ((Edge*)b)->weight;
}

int parent[MAX_V];
int find(int x){return parent[x]==x?x:parent[x]=find(parent[x]);}
void unite(int x,int y){parent[find(x)]=find(y);}

/* Kruskal's MST — O(E log E) */
void kruskalMST(Graph *g) {
    extractEdges(g);
    qsort(edges, edgeCount, sizeof(Edge), cmpEdge);
    for(int i=0;i<g->numVertices;i++) parent[i]=i;

    printf("MST Edges:\n");
    int mstWeight=0, mstEdges=0;
    for(int i=0;i<edgeCount&&mstEdges<g->numVertices-1;i++){
        int u=edges[i].u, v=edges[i].v;
        if(find(u) != find(v)){   // no cycle!
            unite(u, v);
            printf("  (%d-%d) weight=%d\n",u,v,edges[i].weight);
            mstWeight += edges[i].weight;
            mstEdges++;
        }
    }
    printf("Total MST weight: %d\n", mstWeight);
}
C — Union-Find / DSU (Disjoint Set Union)
/* DSU with Path Compression + Union by Rank */
typedef struct {
    int parent[MAX_V];
    int rank[MAX_V];
} DSU;

void dsuInit(DSU *d, int n) {
    for(int i=0;i<n;i++) d->parent[i]=i, d->rank[i]=0;
}

/* Find with Path Compression — nearly O(1) */
int dsuFind(DSU *d, int x) {
    if(d->parent[x] != x)
        d->parent[x] = dsuFind(d, d->parent[x]);  // compress!
    return d->parent[x];
}

/* Union by Rank */
void dsuUnion(DSU *d, int x, int y) {
    int px=dsuFind(d,x), py=dsuFind(d,y);
    if(px==py) return;  // already in same set
    if(d->rank[px] < d->rank[py]) { int t=px;px=py;py=t; }
    d->parent[py] = px;
    if(d->rank[px]==d->rank[py]) d->rank[px]++;
}

/* Check if two vertices are in same component */
int dsuConnected(DSU *d, int x, int y) {
    return dsuFind(d,x) == dsuFind(d,y);
}
C — Graph Compression (CSR Format)
/* CSR = Compressed Sparse Row — efficient sparse storage */
/* Instead of V×V matrix, store only existing edges */

typedef struct {
    int rowPtr[MAX_V + 1]; // start index of each row
    int colIdx[MAX_V*MAX_V]; // column indices of edges
    int numV, numE;
} CSRGraph;

/* Convert adjacency matrix → CSR */
void toCSR(Graph *g, CSRGraph *csr) {
    csr->numV = g->numVertices;
    csr->numE = 0;
    csr->rowPtr[0] = 0;
    for(int i=0;i<g->numVertices;i++){
        for(int j=0;j<g->numVertices;j++)
            if(g->adj[i][j])
                csr->colIdx[csr->numE++] = j;
        csr->rowPtr[i+1] = csr->numE;
    }
    printf("Matrix cells: %d → CSR cells: %d (saved %d)\n",
        g->numVertices * g->numVertices,
        g->numVertices + 1 + csr->numE,
        g->numVertices*g->numVertices - (g->numVertices+1+csr->numE));
}
C — Efficient Traversal for Sparse Graphs
/* For sparse graphs, skip zero entries in traversal */
/* Build neighbor list on-the-fly from adj matrix */

void getNeighbors(Graph *g, int v, int *nbrs, int *cnt) {
    *cnt = 0;
    for(int j=0;j<g->numVertices;j++)
        if(g->adj[v][j]) nbrs[(*cnt)++] = j;
}

/* Efficient BFS — uses neighbor list to skip zeros */
void efficientBFS(Graph *g, int start) {
    int vis[MAX_V]={0}, q[MAX_V], f=0,r=0;
    int nbrs[MAX_V], cnt;
    q[r++]=start; vis[start]=1;
    while(f<r){
        int u=q[f++];
        printf("%d ",u);
        getNeighbors(g,u,nbrs,&cnt);
        for(int i=0;i<cnt;i++)
            if(!vis[nbrs[i]]){
                vis[nbrs[i]]=1;
                q[r++]=nbrs[i];
            }
    }
}

DSU: Like sorting people into groups — quick check if two people are in the same group. CSR Compression: Instead of saving all 100 seats in a sparse exam hall, just write down which seats are occupied.

void clearGraph(Graph *g) {
    for(int i=0;i<g->numVertices;i++)
        for(int j=0;j<g->numVertices;j++)
            g->adj[i][j] = 0;
    g->numEdges = 0;
    printf("Graph cleared. Vertices: %d, Edges: 0\n", g->numVertices);

/* Deep copy: src → dst */
void copyGraph(Graph *dst, Graph *src) {
    dst->numVertices = src->numVertices;
    dst->numEdges    = src->numEdges;
    for(int i=0;i<src->numVertices;i++)
        for(int j=0;j<src->numVertices;j++)
            dst->adj[i][j] = src->adj[i][j];
    printf("Graph copied. V=%d E=%d\n",dst->numVertices,dst->numEdges);
}
