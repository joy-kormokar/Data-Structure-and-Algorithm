Introduction to Singly Linked List ▼
What is a Linked List?

A Linked List is a linear data structure where elements (called nodes) are stored in memory and connected using pointers. Unlike arrays, nodes are NOT stored in contiguous memory — each node can be anywhere in memory, and they are chained together via pointers.

Key Insight: Think of a linked list like a treasure hunt — each clue (node) tells you where the next clue (next node) is. The last clue points to nothing (NULL).
Node Structure

Every node has two parts:

  • Data — the actual value stored (int, char, etc.)
  • Next pointer — address of the next node (or NULL if last)
A Single Node: ┌────────┬──────────┐ │ data │ next │ │ (10) │ ──────► │──► (points to next node) └────────┴──────────┘ A Singly Linked List with 3 nodes: head │ ▼ ┌────┬───┐ ┌────┬───┐ ┌────┬──────┐ │ 10 │ ● │────►│ 20 │ ● │────►│ 30 │ NULL │ └────┴───┘ └────┴───┘ └────┴──────┘ Node 1 Node 2 Node 3 (tail)
C Structure Definition
C
#include <stdio.h>
#include <stdlib.h>

/* Define the structure of a Node */
struct Node {
    int data;               /* stores the value */
    struct Node* next;      /* pointer to next node */
};

/* Helper: create a new node */
struct Node* createNode(int value) {
    struct Node* newNode = (struct Node*)malloc(sizeof(struct Node));
    newNode->data = value;   /* set data */
    newNode->next = NULL;    /* next points to nothing yet */
    return newNode;
}

Insertion ▼

Insertion means adding a new node to the linked list. There are three places you can insert: at the beginning, at the end, or at a specific position.

Insert at Beginning — O(1)

The new node becomes the new head. It points to the old head. This is the fastest insertion!

BEFORE: head ──► [10│●]──►[20│●]──►[30│NULL] Step 1: Create new node with value 5 newNode = [5│NULL] Step 2: newNode→next = head (point to old head) newNode = [5│●]──────────►[10│●]──►[20│●]──►[30│NULL] Step 3: head = newNode (update head) head ──► [5│●]──►[10│●]──►[20│●]──►[30│NULL] AFTER: List is now: 5 → 10 → 20 → 30 → NULL
C
/* Insert a new node at the beginning */
struct Node* insertAtBeginning(struct Node* head, int value) {
    /* Step 1: Create the new node */
    struct Node* newNode = createNode(value);

    /* Step 2: New node points to the current head */
    newNode->next = head;

    /* Step 3: Update head to new node */
    head = newNode;

    return head;  /* return updated head */
}

Insert at End — O(n)

We traverse to the last node (where next == NULL), then link the new node there.

BEFORE: head ──► [10│●]──►[20│●]──►[30│NULL] Step 1: Traverse to last node (30) temp ──────────────────────►[30│NULL] ← found (next is NULL) Step 2: Create newNode(40) newNode = [40│NULL] Step 3: temp→next = newNode [30│●]────────────────────►[40│NULL] AFTER: head ──► [10│●]──►[20│●]──►[30│●]──►[40│NULL]
C
/* Insert a new node at the end */
struct Node* insertAtEnd(struct Node* head, int value) {
    struct Node* newNode = createNode(value);

    /* If list is empty, new node becomes head */
    if (head == NULL) return newNode;

    /* Traverse to the last node */
    struct Node* temp = head;
    while (temp->next != NULL) {
        temp = temp->next;
    }

    /* Link last node to new node */
    temp->next = newNode;

    return head;
}

Insert at Specific Position — O(n)

Traverse to position (pos - 1), then insert the new node between that node and the next.

Insert value=25 at position 3 BEFORE: head ──►[10│●]──►[20│●]──────────────►[30│NULL] pos=1 pos=2 pos=3 Step 1: Traverse to position 2 (pos-1) temp stops at [20│●] Step 2: newNode→next = temp→next [25│●]─────────────────────────────►[30│NULL] Step 3: temp→next = newNode [20│●]──►[25│●]──►[30│NULL] AFTER: head ──►[10│●]──►[20│●]──►[25│●]──►[30│NULL]
C
/* Insert at a specific position (1-indexed) */
struct Node* insertAtPosition(struct Node* head, int value, int pos) {
    struct Node* newNode = createNode(value);

    /* Insert at beginning if pos is 1 */
    if (pos == 1) {
        newNode->next = head;
        return newNode;
    }

    /* Traverse to the node BEFORE the position */
    struct Node* temp = head;
    int i;
    for (i = 1; i < pos - 1 && temp != NULL; i++) {
        temp = temp->next;
    }

    if (temp == NULL) {
        printf("Position out of range!\n");
        return head;
    }

    /* Wire the new node in between */
    newNode->next = temp->next;
    temp->next = newNode;

    return head;
}

Deletion ▼

Deletion removes a node from the list and frees its memory. You must always relink the previous node to skip the deleted node.

Delete from Beginning — O(1)

BEFORE: head ──►[10│●]──►[20│●]──►[30│NULL] Step 1: Save head in temp Step 2: Move head to head→next Step 3: Free temp (original head) AFTER: head ──►[20│●]──►[30│NULL] [10] freed ✓
C
struct Node* deleteFromBeginning(struct Node* head) {
    if (head == NULL) {
        printf("List is empty!\n");
        return NULL;
    }
    struct Node* temp = head;  /* save current head */
    head = head->next;          /* move head forward */
    free(temp);                  /* free old head */
    return head;
}

Delete from End — O(n)

BEFORE: head──►[10│●]──►[20│●]──►[30│NULL] Traverse: find second-to-last node [20] prev=20, current=30 (current→next == NULL → found!) prev→next = NULL then free(current) AFTER: head──►[10│●]──►[20│NULL] [30] freed ✓
C
struct Node* deleteFromEnd(struct Node* head) {
    if (head == NULL) return NULL;

    /* Only one node */
    if (head->next == NULL) {
        free(head);
        return NULL;
    }

    /* Find second-to-last node */
    struct Node* prev = NULL;
    struct Node* curr = head;
    while (curr->next != NULL) {
        prev = curr;
        curr = curr->next;
    }
    prev->next = NULL;
    free(curr);
    return head;
}

Delete at Position — O(n)

C
struct Node* deleteAtPosition(struct Node* head, int pos) {
    if (head == NULL) return NULL;

    if (pos == 1) {
        struct Node* temp = head;
        head = head->next;
        free(temp);
        return head;
    }

    struct Node* prev = NULL;
    struct Node* curr = head;
    int i;
    for (i = 1; i < pos && curr != NULL; i++) {
        prev = curr;
        curr = curr->next;
    }
    if (curr == NULL) { printf("Invalid position!\n"); return head; }

    prev->next = curr->next;  /* skip over curr */
    free(curr);
    return head;
}

Delete by Value — O(n)

Find the first node with the given value and remove it.

C
struct Node* deleteByValue(struct Node* head, int value) {
    if (head == NULL) return NULL;

    /* If head itself has the value */
    if (head->data == value) {
        struct Node* temp = head;
        head = head->next;
        free(temp);
        return head;
    }

    struct Node* prev = head;
    struct Node* curr = head->next;

    while (curr != NULL) {
        if (curr->data == value) {
            prev->next = curr->next;
            free(curr);
            return head;
        }
        prev = curr;
        curr = curr->next;
    }
    printf("Value %d not found!\n", value);
    return head;
}
⚠ Always free memory! After deletion, call free() on the removed node. Forgetting to do so causes a memory leak.

Traversal ▼

Traversal means visiting every node in the list from head to tail. We use a temporary pointer that hops from node to node until it hits NULL.

head │ ▼ [10│●]──►[20│●]──►[30│●]──►[40│NULL] ▲ temp (starts here) Iteration 1: print 10, temp = temp→next Iteration 2: print 20, temp = temp→next Iteration 3: print 30, temp = temp→next Iteration 4: print 40, temp = temp→next = NULL Loop ends! Output: 10 → 20 → 30 → 40 → NULL
C
/* Print all nodes in the list */
void printList(struct Node* head) {
    if (head == NULL) {
        printf("List is empty!\n");
        return;
    }

    struct Node* temp = head;  /* start from head */

    while (temp != NULL) {
        printf("%d → ", temp->data);
        temp = temp->next;  /* move to next node */
    }

    printf("NULL\n");
}
Why temp and not head? Never move head directly — if you do, you lose access to the beginning of the list! Always use a copy pointer like temp.

Searching ▼

Searching finds a node with a given value. Since the list is not indexed, we do a linear search — checking each node one by one.

Search for value = 30 temp=[10] → 10≠30 → move next temp=[20] → 20≠30 → move next temp=[30] → 30==30 → FOUND at position 3! ✓ Search for value = 99 temp=[10]→[20]→[30]→[40]→NULL → NOT FOUND ✗
C
/* Returns position (1-indexed) or -1 if not found */
int searchNode(struct Node* head, int value) {
    struct Node* temp = head;
    int position = 1;

    while (temp != NULL) {
        if (temp->data == value) {
            printf("Found %d at position %d\n", value, position);
            return position;
        }
        temp = temp->next;
        position++;
    }

    printf("%d not found in list\n", value);
    return -1;
}
CaseTime
Best case (first node matches)O(1)
Worst case (last node or not found)O(n)
Average caseO(n/2) = O(n)

Updating ▼

Updating modifies the data of an existing node — either by position or by finding a specific value.

C
/* Update node at given position with new value */
void updateAtPosition(struct Node* head, int pos, int newValue) {
    struct Node* temp = head;
    int i;

    /* Traverse to the desired position */
    for (i = 1; i < pos && temp != NULL; i++) {
        temp = temp->next;
    }

    if (temp == NULL) {
        printf("Position out of range!\n");
        return;
    }

    temp->data = newValue;  /* simply overwrite the data */
    printf("Updated position %d to %d\n", pos, newValue);
}

/* Update first occurrence of old value to new value */
void updateByValue(struct Node* head, int oldVal, int newVal) {
    struct Node* temp = head;
    while (temp != NULL) {
        if (temp->data == oldVal) {
            temp->data = newVal;
            return;
        }
        temp = temp->next;
    }
    printf("Value %d not found!\n", oldVal);
}

Counting Nodes ▼

Count the total number of nodes by traversing the entire list and incrementing a counter for each node visited.

[10│●]──►[20│●]──►[30│●]──►[40│NULL] ↑ ↑ ↑ ↑ count=1 count=2 count=3 temp=NULL → stop return 3
C
/* Count total nodes in the list */
int countNodes(struct Node* head) {
    int count = 0;
    struct Node* temp = head;

    while (temp != NULL) {
        count++;
        temp = temp->next;
    }
    return count;
}

/* Recursive version */
int countRecursive(struct Node* head) {
    if (head == NULL) return 0;
    return 1 + countRecursive(head->next);
}

Sorting ▼

Sorting arranges nodes in order. The simplest approach for linked lists is Bubble Sort — repeatedly swap adjacent nodes' data if they are in the wrong order.

Bubble Sort — Pass 1 on list: 30→10→20→NULL Compare [30,10]: 30>10 → swap data → 10→30→20→NULL Compare [30,20]: 30>20 → swap data → 10→20→30→NULL No more pairs. Pass 1 done. Pass 2: Compare [10,20]: 10<20 → no swap Compare [20,30]: 20<30 → no swap No swaps → List is sorted! Result: 10→20→30→NULL ✓
C
/* Bubble Sort on linked list (swaps data, not nodes) */
void bubbleSort(struct Node* head) {
    int swapped;
    struct Node* ptr;
    struct Node* lastSorted = NULL;

    if (head == NULL) return;

    do {
        swapped = 0;
        ptr = head;

        while (ptr->next != lastSorted) {
            if (ptr->data > ptr->next->data) {
                /* Swap data values */
                int temp = ptr->data;
                ptr->data = ptr->next->data;
                ptr->next->data = temp;
                swapped = 1;
            }
            ptr = ptr->next;
        }
        lastSorted = ptr;
    } while (swapped);
}
CaseTimeSpace
Best (already sorted)O(n)O(1)
Worst / AverageO(n²)O(1)
For large lists, prefer Merge Sort on linked lists — it achieves O(n log n) and works efficiently because merging linked lists is pointer-based (no extra array needed).

Reversing ▼

Reversing changes the direction of all next pointers so the last node becomes the head. We use three pointers: prev, curr, and next.

BEFORE: NULL ◄──[10│●]──►[20│●]──►[30│●]──►NULL head Initial: prev=NULL, curr=head(10), next=NULL Step 1 (curr=10): next = curr→next (=20) curr→next = prev (=NULL) ← flip! prev = curr (=10), curr = next (=20) Step 2 (curr=20): next=30, curr→next=10, prev=20, curr=30 Step 3 (curr=30): next=NULL, curr→next=20, prev=30, curr=NULL Loop ends (curr==NULL). head = prev = 30 AFTER: NULL ◄──[10│●]◄──[20│●]◄──[30│●] head (new head)
C
/* Reverse the linked list iteratively */
struct Node* reverseList(struct Node* head) {
    struct Node* prev = NULL;
    struct Node* curr = head;
    struct Node* next = NULL;

    while (curr != NULL) {
        next = curr->next;   /* save next node */
        curr->next = prev;    /* flip the pointer */
        prev = curr;           /* move prev forward */
        curr = next;           /* move curr forward */
    }

    return prev;  /* prev is now the new head */
}
MetricValue
Time ComplexityO(n)
Space ComplexityO(1) — in-place

Merging Two Sorted Lists ▼

Merge two already sorted linked lists into a single sorted list. We compare the heads of both lists and pick the smaller one at each step.

List1: [1│●]──►[3│●]──►[5│NULL] List2: [2│●]──►[4│●]──►[6│NULL] Step 1: 1 < 2 → take 1 from List1 result: 1 Step 2: 3 > 2 → take 2 from List2 result: 1→2 Step 3: 3 < 4 → take 3 from List1 result: 1→2→3 Step 4: 5 > 4 → take 4 from List2 result: 1→2→3→4 Step 5: 5 < 6 → take 5 from List1 result: 1→2→3→4→5 Step 6: List1 empty → attach rest of List2 Result: 1→2→3→4→5→6→NULL ✓
C
/* Merge two sorted linked lists */
struct Node* mergeSorted(struct Node* l1, struct Node* l2) {
    /* dummy node makes it easier to build result */
    struct Node dummy;
    dummy.next = NULL;
    struct Node* tail = &dummy;

    while (l1 != NULL && l2 != NULL) {
        if (l1->data <= l2->data) {
            tail->next = l1;
            l1 = l1->next;
        } else {
            tail->next = l2;
            l2 = l2->next;
        }
        tail = tail->next;
    }

    /* Attach remaining nodes */
    tail->next = (l1 != NULL) ? l1 : l2;

    return dummy.next;  /* skip dummy, return actual head */
}

Loop Detection (Floyd's Algorithm) ▼

A loop (cycle) occurs when a node's next pointer points back to a previous node, causing infinite traversal. Floyd's Cycle Detection (Tortoise and Hare) uses two pointers — one slow (moves 1 step) and one fast (moves 2 steps). If they ever meet, there's a loop.

List with a loop: ┌──────────┐ ▼ │ head──►[1│●]──►[2│●]──►[3│●]──►[4│●]──►[5│●]──┘ slow moves: 1 step at a time fast moves: 2 steps at a time Round 1: slow=2, fast=3 Round 2: slow=3, fast=5 Round 3: slow=4, fast=4 ← THEY MET! → Loop exists ✓ List without a loop: fast will reach NULL before meeting slow → No loop
C
/* Floyd's Cycle Detection Algorithm */
int detectLoop(struct Node* head) {
    struct Node* slow = head;  /* tortoise: 1 step */
    struct Node* fast = head;  /* hare: 2 steps */

    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;           /* move 1 step */
        fast = fast->next->next;     /* move 2 steps */

        if (slow == fast) {
            printf("Loop detected!\n");
            return 1;  /* loop found */
        }
    }
    printf("No loop found.\n");
    return 0;
}

/* Remove the loop once detected */
void removeLoop(struct Node* head) {
    struct Node* slow = head, *fast = head;
    while (fast && fast->next) {
        slow = slow->next;
        fast = fast->next->next;
        if (slow == fast) break;
    }
    /* Move slow to head, find loop start */
    slow = head;
    while (slow->next != fast->next) {
        slow = slow->next;
        fast = fast->next;
    }
    fast->next = NULL;  /* break the loop */
}
Why does it work? If a loop exists, the fast pointer "laps" the slow pointer inside the loop — they must collide. If no loop, fast hits NULL first.

Concatenation ▼

Concatenation joins two lists end-to-end. Find the tail of the first list and point it to the head of the second list.

List1: head1──►[1│●]──►[2│●]──►[3│NULL] List2: head2──►[4│●]──►[5│●]──►[6│NULL] Traverse List1 to find tail: [3│NULL] Set tail→next = head2 Result: [1│●]──►[2│●]──►[3│●]──►[4│●]──►[5│●]──►[6│NULL]
C
/* Concatenate list2 at the end of list1 */
struct Node* concatenate(struct Node* head1, struct Node* head2) {
    if (head1 == NULL) return head2;
    if (head2 == NULL) return head1;

    /* Find the last node of list1 */
    struct Node* temp = head1;
    while (temp->next != NULL) {
        temp = temp->next;
    }

    temp->next = head2;  /* link tail of list1 to head of list2 */
    return head1;
}

Splitting ▼

Splitting divides a list into two halves. We use the slow/fast pointer trick to find the midpoint, then break the link at the middle.

Original: [1│●]──►[2│●]──►[3│●]──►[4│●]──►[5│NULL] Use slow/fast to find mid: slow stops at node 3 (middle for 5 nodes) Break link: slow→next = NULL First half: [1│●]──►[2│●]──►[3│NULL] Second half: [4│●]──►[5│NULL]
C
/* Split list into two halves */
void splitList(struct Node* head,
               struct Node** front,
               struct Node** back) {
    struct Node* slow = head;
    struct Node* fast = head->next;

    /* fast moves 2x faster → slow ends at midpoint */
    while (fast != NULL) {
        fast = fast->next;
        if (fast != NULL) {
            slow = slow->next;
            fast = fast->next;
        }
    }

    *front = head;       /* first half starts at head */
    *back  = slow->next;  /* second half starts after mid */
    slow->next = NULL;    /* break the link at midpoint */
}

Copying (Shallow & Deep) ▼
Shallow Copy — copies only the pointer to the head (both lists share the same nodes). Modifying one affects the other.

Deep Copy — creates brand new nodes with the same data. Each list is independent.
Original: head1──►[10│●]──►[20│●]──►[30│NULL] SHALLOW COPY (dangerous): head2 = head1 → both point to SAME nodes! head2──►[10│●]──►[20│●]──►[30│NULL] ↑ (same memory!) DEEP COPY (safe — new nodes): head2──►[10'│●]──►[20'│●]──►[30'│NULL] (new) (new) (new)
C
/* Deep copy: creates completely new independent list */
struct Node* deepCopy(struct Node* head) {
    if (head == NULL) return NULL;

    /* Create new head */
    struct Node* newHead = createNode(head->data);
    struct Node* copyTail = newHead;
    struct Node* orig = head->next;

    while (orig != NULL) {
        /* Create a new node for each original node */
        struct Node* newNode = createNode(orig->data);
        copyTail->next = newNode;
        copyTail = newNode;
        orig = orig->next;
    }

    return newHead;
}

Finding Middle Element ▼

The Slow & Fast Pointer technique finds the middle node in a single pass — no need to count total nodes first!

Key Insight: When fast pointer reaches the end (NULL), slow pointer is exactly at the middle — because fast travels 2x the speed of slow.
List: [1]──►[2]──►[3]──►[4]──►[5]──►NULL Start: slow=1, fast=1 Move 1: slow=2, fast=3 Move 2: slow=3, fast=5 Move 3: fast→next=NULL → stop! slow = 3 → MIDDLE ELEMENT ✓ Even length list: [1]──►[2]──►[3]──►[4]──►NULL Move 1: slow=2, fast=3 Move 2: slow=3, fast=NULL → stop! slow = 3 → second middle (first: 2, second: 3)
C
/* Find middle node using slow & fast pointer */
struct Node* findMiddle(struct Node* head) {
    if (head == NULL) return NULL;

    struct Node* slow = head;
    struct Node* fast = head;

    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;           /* 1 step */
        fast = fast->next->next;     /* 2 steps */
    }

    printf("Middle element: %d\n", slow->data);
    return slow;
}
ApproachTimePasses
Count then findO(n)2 passes
Slow & Fast pointerO(n)1 pass ✓

Removing Duplicates ▼

Remove all nodes that have duplicate data. If the list is sorted, duplicates are adjacent — easy to remove in O(n). For an unsorted list we need O(n²).

Sorted list with duplicates: [1│●]──►[1│●]──►[2│●]──►[3│●]──►[3│●]──►[4│NULL] curr=[1], curr→next=[1]: same! → delete next [1] curr=[1], curr→next=[2]: diff → move curr curr=[2], curr→next=[3]: diff → move curr curr=[3], curr→next=[3]: same! → delete next [3] curr=[3], curr→next=[4]: diff → move curr Result: [1]──►[2]──►[3]──►[4]──►NULL ✓
C
/* Remove duplicates from SORTED list — O(n) */
void removeDuplicatesSorted(struct Node* head) {
    struct Node* curr = head;

    while (curr != NULL && curr->next != NULL) {
        if (curr->data == curr->next->data) {
            /* Next node is a duplicate → skip and free it */
            struct Node* dup = curr->next;
            curr->next = dup->next;
            free(dup);
        } else {
            curr = curr->next;  /* no dup, move forward */
        }
    }
}

/* Remove duplicates from UNSORTED list — O(n²) */
void removeDuplicatesUnsorted(struct Node* head) {
    struct Node* curr = head;
    while (curr != NULL) {
        struct Node* runner = curr;
        while (runner->next != NULL) {
            if (runner->next->data == curr->data) {
                struct Node* dup = runner->next;
                runner->next = dup->next;
                free(dup);
            } else {
                runner = runner->next;
            }
        }
        curr = curr->next;
    }
}

Rotation ▼

Rotation shifts all nodes by k positions. To rotate left by k: the (k+1)th node becomes the new head, and the original tail connects to the old head.

Rotate LEFT by k=2: Original: [1]──►[2]──►[3]──►[4]──►[5]──►NULL Step 1: Find length = 5. k = k % 5 = 2 Step 2: Traverse to node at position (5-2)=3: that's node [3] Step 3: new_head = [3]→next = [4] Step 4: [3]→next = NULL (break old link) Step 5: Walk to tail of second part: tail=[5] Step 6: tail→next = old head (=[1]) Result: [3]──►[4]──►[5]──►[1]──►[2]──►NULL ✓
C
/* Rotate list left by k positions */
struct Node* rotateLeft(struct Node* head, int k) {
    if (head == NULL || k == 0) return head;

    /* Find length and tail */
    int len = 1;
    struct Node* tail = head;
    while (tail->next != NULL) {
        tail = tail->next;
        len++;
    }

    k = k % len;  /* handle k > length */
    if (k == 0) return head;

    /* Find the (len-k)th node — it becomes new tail */
    struct Node* curr = head;
    int i;
    for (i = 1; i < len - k; i++) {
        curr = curr->next;
    }

    struct Node* newHead = curr->next;
    curr->next = NULL;    /* break the link */
    tail->next = head;    /* attach old head to old tail */

    return newHead;
}

Intersection of Two Lists ▼

Two lists intersect if they share a common node (same memory address, not just same value). The classic trick uses the length difference.

List1: [1│●]──►[2│●]──────────────────┐ ▼ [8│●]──►[9│NULL] ▲ List2: [4│●]──►[5│●]──►[6│●]────────┘ List1 length = 4, List2 length = 5 Difference = 1 → advance List2 by 1 step Now both start at same distance from intersection. Move both pointers one step at a time. When they point to the SAME node address → INTERSECTION FOUND at [8] ✓
C
/* Find intersection node of two linked lists */
struct Node* getIntersection(struct Node* h1, struct Node* h2) {
    int len1 = countNodes(h1);
    int len2 = countNodes(h2);
    int diff = len1 - len2;
    int i;

    /* Advance the longer list by the difference */
    if (diff > 0) {
        for (i = 0; i < diff; i++) h1 = h1->next;
    } else {
        for (i = 0; i < -diff; i++) h2 = h2->next;
    }

    /* Move both together until they meet */
    while (h1 != h2) {
        h1 = h1->next;
        h2 = h2->next;
    }

    return h1;  /* NULL if no intersection, else common node */
}

Reverse in Groups (k-sized blocks) ▼

Reverse every group of k consecutive nodes. Process each group independently and connect them together.

List: [1]──►[2]──►[3]──►[4]──►[5]──►[6]──►NULL k=3 Group 1 (1,2,3): Reverse → [3]──►[2]──►[1] Group 2 (4,5,6): Reverse → [6]──►[5]──►[4] Connect: [1]→next = head of Group 2 Result: [3]──►[2]──►[1]──►[6]──►[5]──►[4]──►NULL ✓
C
/* Reverse linked list in groups of k */
struct Node* reverseInGroups(struct Node* head, int k) {
    struct Node* curr = head;
    struct Node* prev = NULL;
    struct Node* next = NULL;
    int count = 0;

    /* Reverse k nodes */
    while (curr != NULL && count < k) {
        next  = curr->next;
        curr->next = prev;
        prev  = curr;
        curr  = next;
        count++;
    }

    /* head is now the tail of the reversed group */
    /* Recursively reverse the rest and attach */
    if (next != NULL) {
        head->next = reverseInGroups(next, k);
    }

    return prev;  /* prev is the new head of this group */
}

Flattening a Multilevel List ▼

A multilevel linked list has nodes that may have a child pointer pointing to another linked list. Flattening merges all levels into a single singly linked list.

Multilevel structure: Level 1: [1│●]──►[2│●]──►[3│NULL] │ ▼ child Level 2: [4│●]──►[5│NULL] │ ▼ child Level 3: [6│NULL] Flattening strategy: when a node has a child, insert the child list right after that node, then continue processing. Result: [1]──►[2]──►[4]──►[6]──►[5]──►[3]──►NULL ✓
For a simple singly linked list with a child pointer, we need a modified node structure shown below.
C
/* Multilevel node needs both next and child */
struct MNode {
    int data;
    struct MNode* next;
    struct MNode* child;
};

/* Flatten multilevel list */
struct MNode* flattenList(struct MNode* head) {
    if (head == NULL) return NULL;

    struct MNode* curr = head;

    while (curr != NULL) {
        if (curr->child != NULL) {
            /* Save the rest of current level */
            struct MNode* nextNode = curr->next;

            /* Insert child list right after curr */
            curr->next = curr->child;
            curr->child = NULL;

            /* Find tail of child list */
            struct MNode* tail = curr->next;
            while (tail->next != NULL) {
                tail = tail->next;
            }

            /* Attach the saved next back */
            tail->next = nextNode;
        }
        curr = curr->next;
    }
    return head;
}
