01 · Introduction
What is a Sparse Graph?
Understanding the fundamentals of graphs and why sparsity matters in algorithm design.
🌐
Graph Fundamentals
Theory & Core Concepts
A Graph is a collection of vertices (nodes) connected by edges (links). It is one of the most powerful data structures in computer science, used to model real-world relationships.

A graph G = (V, E) where V = set of vertices and E = set of edges.

A graph is called Sparse when the number of edges is much less than the maximum possible edges. Formally: |E| ≪ |V|². In contrast, a Dense graph has edges close to |V|².

Key Rule: For a graph with V vertices, the maximum edges are V*(V-1)/2 (undirected) or V*(V-1) (directed). A sparse graph has far fewer edges than this maximum.
Sparse Condition
E ≈ O(V)
Edges proportional to vertices
Dense Condition
E ≈ O(V²)

Adj. List

🌍 Real-World Analogy
Think of Facebook friend connections. There are 3 billion users (vertices), but each person has on average only 200–300 friends (edges). Most people are NOT friends with most other people. This is a perfect sparse graph! Using an adjacency matrix would waste billions of GB of memory storing "not connected" for most pairs.

02 · Representation
Adjacency List Representation
The most memory-efficient way to store a sparse graph using linked lists in C.
📋
Adjacency List Structure
Core Data Structure in C
An Adjacency List stores a graph as an array of linked lists. Each index of the array represents a vertex, and the linked list at that index contains all vertices connected to it.

Structure: Array[V] where each element is a pointer to a linked list of neighbors.

For a sparse graph with V=6, E=5: Adjacency list uses ~(V + 2E) = 16 units. An adjacency matrix would use V² = 36 units — much more wasteful!

C · Adjacency List Structure 
/* Core structure for Adjacency List Sparse Graph */

#include <stdio.h>
#include <stdlib.h>

#define MAX_VERTICES 100

/* Node in the adjacency linked list */
struct Node {
    int vertex;            /* connected vertex number */
    int weight;            /* edge weight (0 if unweighted) */
    struct Node* next;    /* pointer to next neighbor */
};

/* Graph structure */
struct Graph {
    int numVertices;                          /* total vertices */
    int numEdges;                             /* total edges */
    struct Node* adjList[MAX_VERTICES];   /* array of lists */
};

/* Create a new graph */
struct Graph* createGraph(int v) {
    struct Graph* g = (struct Graph*) malloc(sizeof(struct Graph));
    g->numVertices = v;
    g->numEdges = 0;
    for (int i = 0; i < v; i++)
        g->adjList[i] = NULL;   /* empty list for each vertex */
    return g;
}

/* Create a new node */
struct Node* createNode(int v, int w) {
    struct Node* n = (struct Node*) malloc(sizeof(struct Node));
    n->vertex = v;
    n->weight = w;
    n->next = NULL;
    return n;
}

03 · Vertex Operations
Insert, Delete & Search Vertex
Fundamental operations to manage vertices in a sparse graph.
🔵
Vertex Insert / Delete / Search
Core Vertex Management
🔵 Interactive Vertex Manager

Insert Vertex: Add a new vertex to the graph by incrementing numVertices and initializing adjList[newV] = NULL. Time: O(1)

Delete Vertex: Remove a vertex and all edges connected to it. Must scan ALL adjacency lists to remove references. Time: O(V+E)

Search Vertex: Check if a vertex number is valid (0 ≤ v < numVertices). Time: O(1)
Real-world: Deleting a user from a social network — you must also remove all their friend connections (edges)!
C · Vertex Insert / Delete / Search 
/* ---- INSERT VERTEX ---- */
void insertVertex(struct Graph* g) {
    if (g->numVertices >= MAX_VERTICES) {
        printf("Graph is full!\n");
        return;
    }
    g->adjList[g->numVertices] = NULL; /* new empty list */
    g->numVertices++;
    printf("Vertex %d inserted.\n", g->numVertices - 1);
}

/* ---- DELETE VERTEX ---- */
void deleteVertex(struct Graph* g, int v) {
    if (v < 0 || v >= g->numVertices) {
        printf("Vertex not found!\n"); return;
    }
    /* Free the vertex's own adjacency list */
    struct Node* curr = g->adjList[v];
    while (curr) {
        struct Node* tmp = curr;
        curr = curr->next;
        free(tmp);
    }
    /* Remove edges pointing TO v in other lists */
    for (int i = 0; i < g->numVertices; i++) {
        if (i == v) continue;
        struct Node** ptr = &g->adjList[i];
        while (*ptr) {
            if ((*ptr)->vertex == v) {
                struct Node* del = *ptr;
                *ptr = (*ptr)->next;
                free(del);
            } else ptr = &(*ptr)->next;
        }
    }
    /* Shift remaining vertices down */
    for (int i = v; i < g->numVertices - 1; i++)
        g->adjList[i] = g->adjList[i+1];
    g->numVertices--;
}

/* ---- SEARCH VERTEX ---- */
int searchVertex(struct Graph* g, int v) {
    return (v >= 0 && v < g->numVertices) ? 1 : 0;
}

Think of vertices as cities and edges as roads. Inserting a vertex is like founding a new city — it exists but has no roads yet. Deleting a vertex is like destroying a city — you must also close all roads leading to it. Searching is simply checking if a city exists on the map.

05 · Edge Operations
Insert, Delete & Search Edge
Edges define connections between vertices. Mastering edge operations is core to graph manipulation.
🔗
Edge Insert / Delete / Search
Managing Graph Connections
🔗 Edge Operation Visualizer

Insert Edge (u → v): Create a new node with vertex=v and prepend it to adjList[u]. For undirected graphs, also add u to adjList[v]. Time: O(1)

Delete Edge (u → v): Traverse adjList[u] to find and remove node with vertex=v. Time: O(degree(u))

Search Edge (u → v): Scan adjList[u] for vertex v. Time: O(degree(u))
For undirected graphs, every insert/delete must be done in BOTH directions (u→v and v→u).
C · Edge Insert / Delete / Search 
/* ---- INSERT EDGE (undirected) ---- */
void insertEdge(struct Graph* g, int u, int v, int w) {
    if (!searchVertex(g,u) || !searchVertex(g,v)) {
        printf("Invalid vertices!\n"); return;
    }
    /* Add v to u's list */
    struct Node* newNode = createNode(v, w);
    newNode->next = g->adjList[u];
    g->adjList[u] = newNode;

    /* Add u to v's list (undirected) */
    newNode = createNode(u, w);
    newNode->next = g->adjList[v];
    g->adjList[v] = newNode;

    g->numEdges++;
    printf("Edge %d--%d added (weight=%d)\n", u, v, w);
}

/* ---- DELETE EDGE ---- */
void deleteEdge(struct Graph* g, int u, int v) {
    /* Helper: remove v from u's adjacency list */
    struct Node** ptr = &g->adjList[u];
    while (*ptr && (*ptr)->vertex != v)
        ptr = &(*ptr)->next;
    if (*ptr) {
        struct Node* del = *ptr;
        *ptr = (*ptr)->next;
        free(del);
    }
    /* Also remove u from v's list (undirected) */
    ptr = &g->adjList[v];
    while (*ptr && (*ptr)->vertex != u)
        ptr = &(*ptr)->next;
    if (*ptr) {
        struct Node* del = *ptr;
        *ptr = (*ptr)->next;
        free(del);
        g->numEdges--;
    }
}

/* ---- SEARCH EDGE ---- */
int searchEdge(struct Graph* g, int u, int v) {
    struct Node* curr = g->adjList[u];
    while (curr) {
        if (curr->vertex == v) return 1; /* found! */
        curr = curr->next;
    }
    return 0; /* not found */
}
✈️ Airline Routes Analogy
Inserting an edge is like adding a new direct flight route between two cities. Deleting an edge means canceling that route permanently. Searching an edge is asking "Is there a direct flight from Delhi to Tokyo?" — you check the departure city's list of destinations.

07 · Traversal Operations
Breadth First Search (BFS)
Explore a graph level by level using a queue. The foundational algorithm for shortest paths and connectivity.
🌊
BFS — Queue-Based Traversal
Visit nodes level by level
BFS starts at a source node and explores all neighbors at the current depth before moving deeper. It uses a Queue (FIFO) to track nodes to visit.

Steps: 1) Enqueue start node, mark visited. 2) Dequeue front. 3) Visit all unvisited neighbors, enqueue them. 4) Repeat until queue is empty.

BFS always finds the shortest path (in terms of number of edges) in an unweighted graph.
Time: O(V+E) Space: O(V) Use: Shortest Path, Level Order

C · BFS with Queue 
/* ---- BFS IMPLEMENTATION ---- */
void BFS(struct Graph* g, int start) {
    int visited[MAX_VERTICES] = {0}; /* all unvisited */
    int queue[MAX_VERTICES];
    int front = 0, rear = 0;

    /* Step 1: Initialize - enqueue start node */
    visited[start] = 1;
    queue[rear++] = start;
    printf("BFS: ");

    while (front < rear) {
        /* Step 2: Dequeue front node */
        int curr = queue[front++];
        printf("%d ", curr); /* Visit node */

        /* Step 3: Traverse adjacent nodes */
        struct Node* adj = g->adjList[curr];
        while (adj) {
            if (!visited[adj->vertex]) {
                visited[adj->vertex] = 1; /* Mark visited */
                queue[rear++] = adj->vertex; /* Enqueue */
            }
            adj = adj->next;
        }
    }
    printf("\n");
}
/* For graph: 0-1, 0-2, 1-3, 2-4
   BFS from 0: 0 1 2 3 4  (level by level) */
🎂 Birthday Party Analogy
Imagine you're at a party and want to meet everyone. You start with yourself (node 0), then meet all your direct friends (level 1), then all their friends you haven't met (level 2), and so on. You never go deep into one friend group before meeting all your immediate friends first. That's exactly BFS!
08 · Traversal Operations
Depth First Search (DFS)
Explore as deep as possible along each branch before backtracking. Uses recursion or an explicit stack.
🌀
DFS — Recursive & Stack-Based
Dive deep, then backtrack
DFS explores as deep as possible along each path before backtracking. It uses either recursion (call stack) or an explicit stack.

Steps: 1) Visit node, mark visited. 2) Recursively visit each unvisited neighbor. 3) Backtrack when no unvisited neighbors remain.

DFS is used for: cycle detection, topological sort, finding connected components, solving mazes.
Time: O(V+E) Space: O(V) Use: Cycles, Topo Sort, Maze

C · DFS Recursive & Stack 
/* ---- RECURSIVE DFS ---- */
void DFSUtil(struct Graph* g, int v, int* visited) {
    visited[v] = 1;
    printf("%d ", v); /* Visit node */

    struct Node* adj = g->adjList[v];
    while (adj) {
        if (!visited[adj->vertex])
            DFSUtil(g, adj->vertex, visited); /* Recurse */
        adj = adj->next;
    }
    /* Backtracking happens here automatically */
}

void DFS(struct Graph* g, int start) {
    int visited[MAX_VERTICES] = {0};
    printf("DFS: ");
    DFSUtil(g, start, visited);
    printf("\n");
}

/* ---- STACK-BASED DFS (Iterative) ---- */
void DFS_Stack(struct Graph* g, int start) {
    int visited[MAX_VERTICES] = {0};
    int stack[MAX_VERTICES];
    int top = -1;

    stack[++top] = start;
    printf("DFS(Stack): ");

    while (top >= 0) {
        int v = stack[top--]; /* Pop */
        if (!visited[v]) {
            visited[v] = 1;
            printf("%d ", v);
            struct Node* adj = g->adjList[v];
            while (adj) {
                if (!visited[adj->vertex])
                    stack[++top] = adj->vertex; /* Push */
                adj = adj->next;
            }
        }
    }
    printf("\n");
}
🌳 Maze Solving Analogy
DFS is like solving a maze by always turning left at every junction. You go as deep as possible until you hit a dead end, then backtrack to the last junction and try the next path. Recursive DFS uses the program's own call stack for backtracking — just like your memory of where you came from in the maze.
09 · Sparse Analysis Operations
Density, Sparsity & Compression
Measure and validate how sparse a graph is — crucial for choosing the right algorithms.
📉
Sparse Analysis Suite
Check, Analyze, Validate, Compress
Edge Density = (2 × |E|) / (|V| × (|V|-1)) for undirected graphs. Ranges from 0 (no edges) to 1 (complete graph).

Sparse threshold: Density < 0.1 (less than 10% of possible edges exist).

Missing Edge Detection: Find pairs (u,v) where no edge exists — useful for recommendation systems.

Graph Compression: Sparse graphs can be compressed using CSR (Compressed Sparse Row) format, saving significant memory.
A road network of 10,000 intersections has ~30,000 roads — density of only 0.0006! Extremely sparse, adjacency list is perfect.

C · Sparse Analysis Operations 
/* ---- CHECK IF GRAPH IS SPARSE ---- */
int checkSparse(struct Graph* g) {
    int v = g->numVertices;
    int maxEdges = v * (v - 1) / 2; /* undirected */
    if (maxEdges == 0) return 1; /* 0 or 1 vertex */
    return (g->numEdges * 10 < maxEdges); /* density < 10% */
}

/* ---- EDGE DENSITY ---- */
double edgeDensity(struct Graph* g) {
    int v = g->numVertices;
    int maxEdges = v * (v - 1) / 2;
    if (maxEdges == 0) return 0.0;
    return (double) g->numEdges / maxEdges;
}

/* ---- COMPARE EDGE RATIO ---- */
void compareEdgeRatio(struct Graph* g) {
    int v = g->numVertices;
    int actual = g->numEdges;
    int maxE   = v * (v-1) / 2;
    printf("Actual edges: %d / Max possible: %d\n", actual, maxE);
    printf("Density: %.4f (%.1f%%)\n",
        (double)actual/maxE, (double)actual/maxE*100);
    printf("Graph is: %s\n", checkSparse(g) ? "SPARSE" : "DENSE");
}

/* ---- MISSING EDGE DETECTION ---- */
void findMissingEdges(struct Graph* g) {
    printf("Missing edges:\n");
    for (int i = 0; i < g->numVertices; i++)
        for (int j = i+1; j < g->numVertices; j++)
            if (!searchEdge(g, i, j))
                printf("  %d -- %d\n", i, j);
}

/* ---- SPARSE VALIDATION ---- */
void sparseValidation(struct Graph* g) {
    int v = g->numVertices;
    int e = g->numEdges;
    /* Sparse: edges should be O(V), not O(V^2) */
    if (e <= 2*v)
        printf("VERY SPARSE: E <= 2V\n");
    else if (checkSparse(g))
        printf("SPARSE: density < 10%%\n");
    else
        printf("DENSE graph detected.\n");
}
🕸️ Spider Web Analogy
A perfect spider web has every point connected to every other — that's a dense graph. A real web (sparse graph) only connects nearby points, saving silk (memory). The internet is an even better example: billions of websites (vertices) but each site only links to a tiny fraction — making it an extraordinarily sparse graph.
10 · Connectivity Operations
Connected Components & Reachability
🔌
Connectivity Suite
Connected? Components? Isolated? Reachable?
Check Connected: A graph is connected if every vertex is reachable from every other. Use DFS/BFS from node 0 — if all V nodes are visited, it's connected.

Find Components: Run DFS on each unvisited vertex — each DFS run discovers one connected component.

Isolated Vertices: A vertex with degree 0 (no edges). Check if adjList[v] == NULL.

Weak Connectivity: For directed graphs, check connectivity by ignoring edge direction.
Finding connected components is crucial for network analysis — e.g., detecting isolated subnetworks in a computer network.

C · Connectivity Operations 
/* ---- CHECK CONNECTED GRAPH ---- */
int isConnected(struct Graph* g) {
    int visited[MAX_VERTICES] = {0};
    DFSUtil(g, 0, visited);
    for (int i = 0; i < g->numVertices; i++)
        if (!visited[i]) return 0; /* unreachable node */
    return 1; /* all nodes visited */
}

/* ---- FIND CONNECTED COMPONENTS ---- */
int findComponents(struct Graph* g) {
    int visited[MAX_VERTICES] = {0};
    int count = 0;
    for (int i = 0; i < g->numVertices; i++) {
        if (!visited[i]) {
            printf("Component %d: ", ++count);
            DFSUtil(g, i, visited); /* DFS explores one component */
            printf("\n");
        }
    }
    printf("Total components: %d\n", count);
    return count;
}

/* ---- REACHABILITY CHECK ---- */
int canReach(struct Graph* g, int src, int dest) {
    int visited[MAX_VERTICES] = {0};
    DFSUtil(g, src, visited);
    return visited[dest];
}

/* ---- DETECT ISOLATED VERTICES ---- */
void findIsolated(struct Graph* g) {
    printf("Isolated vertices: ");
    for (int i = 0; i < g->numVertices; i++)
        if (g->adjList[i] == NULL)
            printf("%d ", i);
    printf("\n");
}

/* ---- WEAK CONNECTIVITY (directed graph) ---- */
int isWeaklyConnected(struct Graph* g) {
    /* Build undirected copy and check connectivity */
    struct Graph* undirected = createGraph(g->numVertices);
    for (int i = 0; i < g->numVertices; i++) {
        struct Node* curr = g->adjList[i];
        while (curr) {
            insertEdge(undirected, i, curr->vertex, 0);
            curr = curr->next;
        }
    }
    return isConnected(undirected);
}
🏝️ Island Network Analogy
Think of vertices as islands and edges as bridges. A connected graph means you can travel from any island to any other island using bridges. Finding components is like discovering how many separate island clusters exist. Isolated vertices are islands with no bridges at all — completely cut off.
11 · Path Operations
Find Path, Shortest Path & All Paths
🛤️
Path Operations Suite
Find Path: Find any path between two vertices using DFS. Returns True/False and the path taken.

Shortest Path (BFS): BFS naturally finds shortest paths in unweighted graphs. Track parent array to reconstruct path.

All Paths: DFS with backtracking — explore all possibilities. Exponential time, only for small graphs.

Path Length Count: Count the number of edges in the found path.
BFS gives shortest path by hops. For weighted shortest path, use Dijkstra's algorithm.
C · Path Operations 
/* ---- FIND ANY PATH (DFS) ---- */
int findPath(struct Graph* g, int src, int dest,
             int* visited, int* path, int pathLen) {
    visited[src] = 1;
    path[pathLen] = src;
    pathLen++;
    if (src == dest) {
        for (int i = 0; i < pathLen; i++)
            printf("%d%s", path[i], i<pathLen-1 ? "→" : "\n");
        return 1;
    }
    struct Node* adj = g->adjList[src];
    while (adj) {
        if (!visited[adj->vertex])
            if (findPath(g, adj->vertex, dest, visited, path, pathLen))
                return 1;
        adj = adj->next;
    }
    return 0; /* no path found */
}

/* ---- SHORTEST PATH (BFS) ---- */
void shortestPath(struct Graph* g, int src, int dest) {
    int visited[MAX_VERTICES] = {0};
    int parent[MAX_VERTICES];
    int dist[MAX_VERTICES];
    int queue[MAX_VERTICES];
    int front = 0, rear = 0;
    for(int i=0;i<MAX_VERTICES;i++){parent[i]=-1;dist[i]=-1;}

    visited[src]=1; dist[src]=0;
    queue[rear++] = src;

    while (front < rear) {
        int v = queue[front++];
        struct Node* adj = g->adjList[v];
        while (adj) {
            if (!visited[adj->vertex]) {
                visited[adj->vertex] = 1;
                dist[adj->vertex] = dist[v] + 1;
                parent[adj->vertex] = v;
                queue[rear++] = adj->vertex;
            }
            adj = adj->next;
        }
    }
    if (dist[dest] == -1) { printf("No path!\n"); return; }
    printf("Shortest distance: %d hops\n", dist[dest]);
    /* Reconstruct path using parent array */
    int path[MAX_VERTICES], len = 0;
    for(int v=dest; v!=-1; v=parent[v]) path[len++]=v;
    printf("Path: ");
    for(int i=len-1; i>=0; i--)
        printf("%d%s", path[i], i?"→":"\n");
}

/* ---- ALL PATHS (DFS with backtracking) ---- */
void allPaths(struct Graph* g, int src, int dest,
              int* visited, int* path, int pathLen) {
    visited[src] = 1;
    path[pathLen++] = src;
    if (src == dest) {
        for(int i=0;i<pathLen;i++)
            printf("%d%s",path[i],i<pathLen-1?"→":"\n");
    } else {
        struct Node* adj = g->adjList[src];
        while (adj) {
            if (!visited[adj->vertex])
                allPaths(g, adj->vertex, dest, visited, path, pathLen);
            adj = adj->next;
        }
    }
    visited[src] = 0; /* BACKTRACK */
}
🗺️ GPS Navigation Analogy
Find Path is like GPS finding ANY route to your destination. Shortest Path is GPS finding the route with fewest traffic lights (hops). All Paths is like listing every possible route — useful when you want to compare options but gets overwhelming in large cities (exponential time)!
12 · Cycle Operations
Detect Cycles, Count, Tree & Forest
🔄
Cycle Detection Suite
Detect Cycle: During DFS, if we reach an already-visited vertex that is NOT the parent, a cycle exists.

Tree Structure: A tree is a connected graph with exactly V-1 edges and NO cycles. Check: connected AND numEdges == numVertices-1.

Forest: Multiple disconnected trees. Check: numEdges == numVertices - numComponents.

Cycle detection is critical for: deadlock detection in OS, dependency resolution, circuit analysis.
C · Cycle Detection & Tree/Forest Validation 
/* ---- DETECT CYCLE (undirected) ---- */
int cycleUtil(struct Graph* g, int v, int* vis, int parent) {
    vis[v] = 1;
    struct Node* adj = g->adjList[v];
    while (adj) {
        if (!vis[adj->vertex]) {
            if (cycleUtil(g, adj->vertex, vis, v))
                return 1;
        } else if (adj->vertex != parent) {
            return 1; /* Back edge = CYCLE FOUND! */
        }
        adj = adj->next;
    }
    return 0;
}

int detectCycle(struct Graph* g) {
    int vis[MAX_VERTICES] = {0};
    for (int i = 0; i < g->numVertices; i++)
        if (!vis[i])
            if (cycleUtil(g, i, vis, -1)) return 1;
    return 0;
}

/* ---- CHECK TREE ---- */
int isTree(struct Graph* g) {
    /* Tree = connected + no cycle + E == V-1 */
    if (g->numEdges != g->numVertices - 1) return 0;
    return isConnected(g) && !detectCycle(g);
}

/* ---- FOREST VALIDATION ---- */
int isForest(struct Graph* g) {
    /* Forest = no cycles (may be disconnected) */
    /* E must equal V - number_of_components */
    if (detectCycle(g)) return 0;
    int components = findComponents(g);
    return g->numEdges == g->numVertices - components;
}

/* ---- COUNT CYCLES (via DFS) ---- */
int countCycles = 0;
void countCyclesUtil(struct Graph* g, int v,
                       int* visited, int* recStack) {
    visited[v] = recStack[v] = 1;
    struct Node* adj = g->adjList[v];
    while (adj) {
        if (!visited[adj->vertex])
            countCyclesUtil(g, adj->vertex, visited, recStack);
        else if (recStack[adj->vertex])
            countCycles++;
        adj = adj->next;
    }
    recStack[v] = 0;
}
🚦 Traffic Deadlock Analogy
A cycle in a dependency graph is like a traffic deadlock: Car A waits for Car B, Car B waits for Car C, Car C waits for Car A — nobody can move! Operating systems use cycle detection in resource allocation graphs to detect and break these deadlocks.
13 · Degree Operations
Find, Max, Min, Average Degree
📐
Degree Operations
Measure vertex connectivity
The degree of a vertex is the number of edges connected to it. In an adjacency list, it's simply the length of that vertex's linked list.

Degree Sum Theorem: Sum of all degrees = 2 × |E| (each edge contributes to 2 vertices).

Average Degree: 2|E| / |V|. For sparse graphs, this is close to a small constant.
In a sparse graph, average degree ≈ 2 (like trees) to a small constant. Dense graphs have average degree ≈ V.

/* ---- FIND DEGREE OF A VERTEX ---- */
int findDegree(struct Graph* g, int v) {
    int count = 0;
    struct Node* curr = g->adjList[v];
    while (curr) { count++; curr = curr->next; }
    return count;
}

/* ---- MAXIMUM DEGREE ---- */
int maxDegree(struct Graph* g) {
    int maxD = 0;
    for (int i = 0; i < g->numVertices; i++) {
        int d = findDegree(g, i);
        if (d > maxD) maxD = d;
    }
    return maxD;
}

/* ---- MINIMUM DEGREE ---- */
int minDegree(struct Graph* g) {
    int minD = g->numVertices; /* start with max possible */
    for (int i = 0; i < g->numVertices; i++) {
        int d = findDegree(g, i);
        if (d < minD) minD = d;
    }
    return minD;
}

/* ---- AVERAGE DEGREE ---- */
double averageDegree(struct Graph* g) {
    /* Sum of degrees = 2 * |E| */
    return (double) 2 * g->numEdges / g->numVertices;
}

/* ---- DEGREE DISTRIBUTION ---- */
void degreeDistribution(struct Graph* g) {
    int dist[MAX_VERTICES] = {0};
    for (int i = 0; i < g->numVertices; i++)
        dist[findDegree(g, i)]++;
    printf("Degree Distribution:\n");
    for (int d = 0; d < g->numVertices; d++)
        if (dist[d] > 0)
            printf("  Degree %d: %d vertices\n", d, dist[d]);
    printf("Max: %d | Min: %d | Avg: %.2f\n",
        maxDegree(g), minDegree(g), averageDegree(g));
}

15 · Advanced Algorithms
MST, Kruskal, Union-Find & Compression
Production-level algorithms that work beautifully on sparse graphs.
⚡
Minimum Spanning Tree — Kruskal's Algorithm
A Minimum Spanning Tree (MST) connects all vertices with minimum total edge weight and no cycles. It has exactly V-1 edges.

Kruskal's Algorithm: Sort all edges by weight. Add the smallest edge if it doesn't create a cycle (use Union-Find to check). Repeat until V-1 edges added.

Union-Find (DSU): Data structure that efficiently tracks which vertices belong to the same component. Supports union and find operations in near O(1) with path compression.
Kruskal's runs in O(E log E) which is perfect for sparse graphs where E ≈ V. Much faster than Prim's on sparse graphs!
Time: O(E log E) Space: O(V+E) Use: Network Design, Clustering

C · Kruskal's MST + Union-Find 
/* ---- UNION-FIND (Disjoint Set Union) ---- */
int parent[MAX_VERTICES], rank_[MAX_VERTICES];

void makeSet(int v) {
    parent[v] = v; rank_[v] = 0;
}

int find(int v) { /* Path compression */
    if (parent[v] != v)
        parent[v] = find(parent[v]);
    return parent[v];
}

void unite(int u, int v) { /* Union by rank */
    int pu = find(u), pv = find(v);
    if (pu == pv) return;
    if (rank_[pu] < rank_[pv]) { int tmp=pu; pu=pv; pv=tmp; }
    parent[pv] = pu;
    if (rank_[pu] == rank_[pv]) rank_[pu]++;
}

/* ---- KRUSKAL'S MST ---- */
struct Edge { int u, v, w; };

int cmpEdge(const void* a, const void* b) {
    return ((struct Edge*)a)->w - ((struct Edge*)b)->w;
}

void kruskalMST(struct Graph* g) {
    struct Edge edges[MAX_VERTICES*MAX_VERTICES];
    int edgeCount = 0;

    /* Collect all edges */
    for (int i = 0; i < g->numVertices; i++) {
        struct Node* curr = g->adjList[i];
        while (curr) {
            if (curr->vertex > i) {
                edges[edgeCount++] = (struct Edge){i, curr->vertex, curr->weight};
            }
            curr = curr->next;
        }
    }

    /* Sort edges by weight */
    qsort(edges, edgeCount, sizeof(struct Edge), cmpEdge);

    /* Initialize Union-Find */
    for (int i = 0; i < g->numVertices; i++) makeSet(i);

    printf("MST Edges:\n");
    int totalWeight = 0, mstEdges = 0;

    for (int i = 0; i < edgeCount && mstEdges < g->numVertices-1; i++) {
        int u=edges[i].u, v=edges[i].v, w=edges[i].w;
        if (find(u) != find(v)) { /* No cycle */
            unite(u, v);
            printf("  %d -- %d (weight %d)\n", u, v, w);
            totalWeight += w;
            mstEdges++;
        }
    }
    printf("Total MST weight: %d\n", totalWeight);
}

/* ---- GRAPH COMPRESSION (CSR format) ---- */
void compressToCSR(struct Graph* g,
                   int* rowPtr, int* colIdx, int* weights) {
    /* CSR: Compressed Sparse Row format */
    int idx = 0;
    rowPtr[0] = 0;
    for (int i = 0; i < g->numVertices; i++) {
        struct Node* curr = g->adjList[i];
        while (curr) {
            colIdx[idx]  = curr->vertex;
            weights[idx] = curr->weight;
            idx++; curr = curr->next;
        }
        rowPtr[i+1] = idx;
    }
    /* To access neighbors of v: rowPtr[v]..rowPtr[v+1]-1 */
}
💡 Electrical Grid Analogy
MST is like connecting all cities with electrical cables using minimum total wire length. You pick the cheapest cable first, connect cities, and skip any cable that would create a loop (since electricity already flows that way). Kruskal's algorithm does exactly this — greedy shortest-cable-first strategy.

/* ---- CLEAR GRAPH (free all memory) ---- */
void clearGraph(struct Graph* g) {
    for (int i = 0; i < g->numVertices; i++) {
        struct Node* curr = g->adjList[i];
        while (curr) {
            struct Node* tmp = curr;
            curr = curr->next;
            free(tmp); /* free each node */
        }
        g->adjList[i] = NULL;
    }
    g->numVertices = 0;
    g->numEdges    = 0;
    printf("Graph cleared.\n");
}

/* ---- COPY GRAPH ---- */
struct Graph* copyGraph(struct Graph* src) {
    struct Graph* dest = createGraph(src->numVertices);
    dest->numEdges = src->numEdges;
    for (int i = 0; i < src->numVertices; i++) {
        /* Reverse copy to maintain order */
        struct Node* curr = src->adjList[i];
        struct Node** tail = &dest->adjList[i];
        while (curr) {
            *tail = createNode(curr->vertex, curr->weight);
            tail = &(*tail)->next;
            curr = curr->next;
        }
    }
    return dest;
}

/* ---- DISPLAY GRAPH (adjacency list) ---- */

            curr = curr->next;
        }
        printf(" NULL\n");
    }
    printf("V=%d, E=%d\n", g->numVertices, g->numEdges);
}
