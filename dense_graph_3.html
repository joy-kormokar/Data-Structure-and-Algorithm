What is a Dense Graph?
Understand graphs, density, and why the incidence matrix is the perfect fit for dense graphs.
Graph Basics
A graph is a collection of vertices (nodes) connected by edges. Think of a city map — cities are vertices, and roads connecting them are edges.

A graph with V vertices can have at most V×(V-1)/2 edges (undirected) or V×(V-1) edges (directed).

A graph is called dense when the number of edges is close to the maximum possible. Specifically, if E ≈ V², we call it dense. If E ≈ V, it's sparse.
🌍 Real-Life Analogy
Imagine a group of 5 friends where almost everyone knows everyone else. This is a dense social graph. Compare this to a tree where each person only knows 2-3 others — that's sparse.

Sparse Graph

Few edges. E ≈ V. Like a linked list or tree. Best stored with adjacency lists.

Dense Graph

Many edges. E ≈ V². Like a social network or road network. Best stored with matrices.

Incidence Matrix Representation
How we store a dense graph as a 2D array where rows = vertices, columns = edges.
How the Incidence Matrix Works
The Incidence Matrix is a 2D array of size V × E (vertices × edges).

For an undirected graph: cell [v][e] = 1 if vertex v is connected by edge e, else 0.
For a directed graph: cell [v][e] = 1 if edge e leaves vertex v (source), -1 if edge e enters vertex v (destination), else 0.
🚂 Real-Life Analogy
Think of train stations (vertices) and routes (edges). The incidence matrix is like a schedule board: rows = stations, columns = routes. A "1" means this station is part of that route.

C Incidence Matrix — Data Structure Definition 
/* Dense Graph using Incidence Matrix */
#define MAX_V 10   /* max vertices */
#define MAX_E 50   /* max edges    */

int incMatrix[MAX_V][MAX_E]; /* 2D array: rows=vertices, cols=edges */
int V = 0;  /* current number of vertices */
int E = 0;  /* current number of edges    */

/* Initialize: fill matrix with zeros */
void initGraph() {
    int i, j;
    for (i = 0; i < MAX_V; i++)
        for (j = 0; j < MAX_E; j++)
            incMatrix[i][j] = 0;
    V = 0; E = 0;
}

Vertex Operations — Insert & Delete
Insert Vertex
Inserting a vertex means adding a new row in the incidence matrix and initializing all cells in that row to 0 (no edges yet). We simply increment the vertex count V.
🏙️ Analogy
Building a new city — it exists on the map but has no roads yet.
CInsert Vertex
void insertVertex() {
    if (V >= MAX_V) {
        printf("Error: Max vertices reached!\n");
        return;
    }
    int j;
    for (j = 0; j < E; j++)
        incMatrix[V][j] = 0;  /* new vertex has no edges */
    V++;
    printf("Vertex %d inserted. Total: %d\n", V-1, V);
}
Delete Vertex
Deleting a vertex removes its row from the matrix. All edges involving that vertex must also be removed (any column where that vertex had 1 or -1). Then shift remaining rows up.
CDelete Vertex
void deleteVertex(int v) {
    if (v < 0 || v >= V) { printf("Invalid vertex!\n"); return; }

    int i, j, col = 0;
    /* Remove columns (edges) where vertex v is incident */
    while (col < E) {
        if (incMatrix[v][col] != 0) {
            /* Shift all columns left to fill gap */
            for (i = 0; i < V; i++)
                for (j = col; j < E-1; j++)
                    incMatrix[i][j] = incMatrix[i][j+1];
            E--;  /* one less edge */
        } else col++;
    }
    /* Shift rows up to remove vertex v */
    for (i = v; i < V-1; i++)
        for (j = 0; j < E; j++)
            incMatrix[i][j] = incMatrix[i+1][j];
    V--;
    printf("Vertex %d deleted.\n", v);
}

Edge Operations — Insert & Delete
Insert Edge
Inserting an edge between vertex u and vertex v means:
• Adding a new column to the incidence matrix
• Setting incMatrix[u][E] = 1 and incMatrix[v][E] = 1 (undirected)
• For directed: incMatrix[u][E] = 1 (source), incMatrix[v][E] = -1 (destination)
• Incrementing E (edge count)
🛣️ Analogy
Building a new road between two cities on the map.
Edge Insertion Demo
CInsert Undirected Edge
void insertEdge(int u, int v) {
    if (u < 0 || u >= V || v < 0 || v >= V) {
        printf("Invalid vertices!\n"); return;
    }
    if (E >= MAX_E) { printf("Max edges reached!\n"); return; }

    /* Set both endpoints to 1 for undirected edge */
    incMatrix[u][E] = 1;
    incMatrix[v][E] = 1;
    E++;  /* new column added */
    printf("Edge e%d inserted: V%d -- V%d\n", E-1, u, v);
}

/* For directed edge: u -> v */
void insertDirectedEdge(int u, int v) {
    if (E >= MAX_E) { printf("Max edges reached!\n"); return; }
    incMatrix[u][E] = 1;   /* +1 for source */
    incMatrix[v][E] = -1;  /* -1 for destination */
    E++;
}
Delete Edge
Find the column index for the edge e, then shift all columns to the right of e one position left. Decrement E.
CDelete Edge by Index
void deleteEdge(int e) {
    if (e < 0 || e >= E) { printf("Invalid edge!\n"); return; }
    int i, j;
    /* Shift columns left starting from edge e */
    for (i = 0; i < V; i++)
        for (j = e; j < E-1; j++)
            incMatrix[i][j] = incMatrix[i][j+1];
    E--;
    printf("Edge e%d deleted.\n", e);
}

Edge Operations — Search & Update
Search Edge
To find if an edge exists between u and v, scan all columns. A column where both u and v have non-zero values is that edge.
CSearch Edge
int searchEdge(int u, int v) {
    int j;
    for (j = 0; j < E; j++) {
        if (incMatrix[u][j] != 0 && incMatrix[v][j] != 0) {
            printf("Edge between V%d and V%d: e%d\n", u, v, j);
            return j;   /* return edge index */
        }
    }
    printf("No edge between V%d and V%d\n", u, v);
    return -1;
}

/* Update edge: change which vertices it connects */
void updateEdge(int e, int newU, int newV) {
    if (e < 0 || e >= E) { printf("Invalid edge!\n"); return; }
    int i;
    for (i = 0; i < V; i++) incMatrix[i][e] = 0; /* clear old */
    incMatrix[newU][e] = 1;
    incMatrix[newV][e] = 1;
    printf("Edge e%d updated: V%d -- V%d\n", e, newU, newV);
}

Intermediate
Breadth First Search (BFS)
Visit all vertices level by level, starting from a source. Uses a queue.
BFS Theory & Steps
BFS explores the graph layer by layer. Starting from vertex s, we visit all neighbors first, then neighbors of neighbors, and so on. It uses a Queue (FIFO).

With incidence matrix: for each vertex u, scan all edges (columns). If incMatrix[u][j] != 0, find the other vertex in that column — that's a neighbor.
💧 Analogy
BFS is like a water ripple from a stone thrown in a pond — it spreads outward one ring at a time.
1
Initialize: Create a visited[] array (all false), create an empty queue, mark start vertex as visited, add to queue.
2
Visit Node: Dequeue front vertex u, print/process it.
3
Traverse Adjacent: Scan all edge columns. If incMatrix[u][e] != 0, find vertex w in that column where w ≠ u. If not visited, mark visited and enqueue w.
4
Repeat until queue is empty.

VISIT ORDER
CBFS using Incidence Matrix
void BFS(int start) {
    int visited[MAX_V] = {0};
    int queue[MAX_V];
    int front = 0, rear = 0;  /* queue indices */

    /* Queue Initialization */
    visited[start] = 1;
    queue[rear++] = start;

    printf("BFS from V%d: ", start);

    while (front < rear) {
        int u = queue[front++];  /* dequeue */
        printf("V%d ", u);           /* Visit Node */

        /* Traverse Adjacent Nodes via incidence matrix */
        int e;
        for (e = 0; e < E; e++) {
            if (incMatrix[u][e] != 0) {  /* u is in edge e */
                int w;
                for (w = 0; w < V; w++) {  /* find other vertex */
                    if (w != u && incMatrix[w][e] != 0) {
                        if (!visited[w]) {
                            visited[w] = 1;      /* Mark Visited */
                            queue[rear++] = w;    /* enqueue */
                        }
                    }
                }
            }
        }
    }
    printf("\n");
}
Intermediate
Depth First Search (DFS)
Explore as deep as possible before backtracking. Two versions: recursive and stack-based.
DFS — Recursive & Stack
DFS goes as deep as possible along a path before backtracking. Like navigating a maze — go forward until a dead end, then backtrack.

Recursive DFS uses the call stack implicitly. Stack DFS uses an explicit stack data structure.
🏔️ Analogy
DFS is like hiking — you pick one trail and follow it all the way to the end before coming back to try another trail.

CRecursive DFS + Stack-based DFS
int visited[MAX_V];  /* global visited array */

/* Recursive DFS */
void DFS_Recursive(int u) {
    visited[u] = 1;         /* Visit Node */
    printf("V%d ", u);
    int e, w;
    for (e = 0; e < E; e++) {
        if (incMatrix[u][e] != 0) {
            for (w = 0; w < V; w++) {
                if (w != u && incMatrix[w][e] != 0 && !visited[w])
                    DFS_Recursive(w);  /* Backtracking via recursion */
            }
        }
    }
}

/* Stack-based DFS */
void DFS_Stack(int start) {
    int vis[MAX_V] = {0};
    int stack[MAX_V], top = -1;
    stack[++top] = start;
    printf("DFS (stack) from V%d: ", start);
    while (top >= 0) {
        int u = stack[top--];  /* pop */
        if (vis[u]) continue;
        vis[u] = 1;
        printf("V%d ", u);
        int e, w;
        for (e = E-1; e >= 0; e--)
            if (incMatrix[u][e] != 0)
                for (w = 0; w < V; w++)
                    if (w != u && incMatrix[w][e] != 0 && !vis[w])
                        stack[++top] = w;  /* push neighbor */
    }
    printf("\n");
}

Dense Analysis Operations
Algorithms to measure and analyze how dense a graph is.
Density Calculation
Graph Density = (actual edges) / (maximum possible edges)

For undirected: max edges = V×(V-1)/2
For directed: max edges = V×(V-1)

Density ranges from 0.0 (no edges) to 1.0 (complete graph).
Typically, if density > 0.5, the graph is considered dense.
Density Calculator
CDensity Calculation & Dense Graph Check
/* Calculate density (0.0 to 1.0) */
float densityCalc() {
    if (V <= 1) return 0.0;
    float maxEdges = (V * (V - 1)) / 2.0;  /* undirected */
    float density = E / maxEdges;
    printf("Density: %.2f\n", density);
    return density;
}

/* Check if graph is dense (density > 0.5) */
int checkDenseGraph() {
    float d = densityCalc();
    if (d > 0.5) {
        printf("Graph is DENSE (density=%.2f)\n", d);
        return 1;
    }
    printf("Graph is SPARSE (density=%.2f)\n", d);
    return 0;
}

/* Edge Saturation Analysis */
void edgeSaturationAnalysis() {
    int maxPossible = V * (V-1) / 2;
    int missing = maxPossible - E;
    printf("Max possible: %d, Current: %d, Missing: %d\n",
           maxPossible, E, missing);
}

/* Compare Edge Ratio: directed vs undirected */
void compareEdgeRatio() {
    float undirRatio = E / (V*(V-1)/2.0);
    float dirRatio   = E / (float)(V*(V-1));
    printf("Undirected ratio: %.2f | Directed ratio: %.2f\n",
           undirRatio, dirRatio);
}

/* Detect High Connectivity: vertex with all neighbors */
void detectHighConnectivity() {
    int i, e, cnt;
    for (i = 0; i < V; i++) {
        cnt = 0;
        for (e = 0; e < E; e++)
            if (incMatrix[i][e] != 0) cnt++;
        printf("V%d degree: %d", i, cnt);
        if (cnt == V-1) printf(" [FULLY CONNECTED]");
        printf("\n");
    }
}
Intermediate
Connectivity Operations
Check whether all vertices are reachable from any starting point.
Connected Graph Check & Components
A graph is connected if every vertex is reachable from every other vertex. We use BFS/DFS from vertex 0 — if all V vertices are visited, the graph is connected.

Strong Connectivity (directed): every vertex is reachable from every other vertex in both directions.

Connected Components: groups of vertices where each group is internally connected but not connected to others.
🌐 Analogy
Like checking if all rooms in a building are reachable from the entrance. If yes, it's connected. If some rooms are isolated, those form separate "components".
CConnectivity Check + Connected Components
/* BFS helper: returns count of visited nodes */
int bfsCount(int start, int vis[]) {
    int queue[MAX_V], front=0, rear=0, count=0;
    vis[start] = 1; queue[rear++] = start;
    while (front < rear) {
        int u = queue[front++]; count++;
        int e, w;
        for (e=0; e<E; e++)
            if (incMatrix[u][e]!=0)
                for (w=0; w<V; w++)
                    if (w!=u && incMatrix[w][e]!=0 && !vis[w]) {
                        vis[w]=1; queue[rear++]=w;
                    }
    }
    return count;
}

/* Check Connected Graph */
int checkConnected() {
    int vis[MAX_V] = {0};
    int cnt = bfsCount(0, vis);
    if (cnt == V) { printf("Graph is CONNECTED\n"); return 1; }
    printf("Graph is DISCONNECTED\n");
    return 0;
}

/* Find Connected Components */
void connectedComponents() {
    int vis[MAX_V] = {0}, comp = 0, i;
    for (i = 0; i < V; i++) {
        if (!vis[i]) {
            comp++;
            printf("Component %d: ", comp);
            bfsCount(i, vis);  /* visit component */
            printf("\n");
        }
    }
    printf("Total components: %d\n", comp);
}

/* Reachability: can we get from u to v? */
int reachabilityCheck(int u, int v) {
    int vis[MAX_V] = {0};
    bfsCount(u, vis);
    if (vis[v]) { printf("V%d is reachable from V%d\n", v, u); return 1; }
    printf("V%d is NOT reachable from V%d\n", v, u);
    return 0;
}

Path Operations
Finding paths, shortest paths, all paths, and path lengths.
Find Path, Shortest Path, All Paths
A path is a sequence of vertices connected by edges with no repeated vertices.
Shortest path (unweighted) = path with fewest edges — found by BFS.
All paths uses DFS with backtracking.
🗺️ Analogy
Finding how to get from city A to city B: BFS gives you the fewest stops, DFS explores every possible route.
CShortest Path (BFS) + All Paths (DFS)
/* Shortest path using BFS (unweighted) */
int shortestPath(int src, int dst) {
    int dist[MAX_V], vis[MAX_V];
    int queue[MAX_V], front=0, rear=0, i;
    for (i=0;i<V;i++){dist[i]=-1;vis[i]=0;}
    vis[src]=1; dist[src]=0; queue[rear++]=src;
    while(front<rear){
        int u=queue[front++], e, w;
        for(e=0;e<E;e++)
            if(incMatrix[u][e]!=0)
                for(w=0;w<V;w++)
                    if(w!=u&&incMatrix[w][e]!=0&&!vis[w]){
                        vis[w]=1; dist[w]=dist[u]+1;
                        queue[rear++]=w;
                    }
    }
    printf("Shortest path V%d->V%d: %d edges\n", src, dst, dist[dst]);
    return dist[dst];
}

/* All Paths using DFS with backtracking */
int pathArr[MAX_V], pathLen;
void findAllPaths(int u, int dst, int vis[]) {
    vis[u]=1;
    pathArr[pathLen++]=u;
    if(u==dst){
        int i;
        for(i=0;i<pathLen;i++) printf("V%d%s",pathArr[i],i==pathLen-1?"\n":"->");
    } else {
        int e, w;
        for(e=0;e<E;e++)
            if(incMatrix[u][e]!=0)
                for(w=0;w<V;w++)
                    if(w!=u&&incMatrix[w][e]!=0&&!vis[w])
                        findAllPaths(w,dst,vis);
    }
    pathLen--;
    vis[u]=0;  /* backtrack */
}

/* Path Length Count */
int pathLengthCount(int src, int dst) {
    return shortestPath(src, dst);
}

Cycle Operations
Detect, count cycles, and check Hamiltonian & Euler cycles.
Cycle Detection, Euler & Hamiltonian
A cycle is a path where the first and last vertex are the same.

Euler Cycle: visits every edge exactly once and returns to start. Exists if all vertices have even degree.
Hamiltonian Cycle: visits every vertex exactly once and returns to start. Checking requires backtracking (NP-complete).
🚴 Analogy
Euler Cycle: a postman delivering mail on every street exactly once. Hamiltonian Cycle: a salesman visiting every city exactly once.
CCycle Detection + Euler Check + Hamiltonian Check
/* Detect cycle using DFS */
int detectCycleDFS(int u, int parent, int vis[]) {
    vis[u] = 1;
    int e, w;
    for (e=0; e<E; e++) {
        if (incMatrix[u][e] != 0) {
            for (w=0; w<V; w++) {
                if (w!=u && incMatrix[w][e]!=0) {
                    if (!vis[w]) {
                        if (detectCycleDFS(w, u, vis)) return 1;
                    } else if (w != parent) return 1; /* back edge */
                }
            }
        }
    }
    return 0;
}

/* Euler Cycle: all vertices must have even degree */
int eulerCycleCheck() {
    int i, e, deg;
    for (i=0; i<V; i++) {
        deg = 0;
        for (e=0; e<E; e++)
            if (incMatrix[i][e] != 0) deg++;
        if (deg % 2 != 0) {
            printf("No Euler Cycle (V%d has odd degree %d)\n", i, deg);
            return 0;
        }
    }
    printf("Euler Cycle EXISTS\n"); return 1;
}

/* Hamiltonian Cycle check via backtracking */
int hamPath[MAX_V];
int hamiltonianHelper(int pos, int vis[]) {
    if (pos == V) {
        /* check if last connects back to start */
        int e;
        for (e=0; e<E; e++)
            if (incMatrix[hamPath[pos-1]][e]!=0 &&
                incMatrix[hamPath[0]][e]!=0) return 1;
        return 0;
    }
    int w, e;
    for (w=1; w<V; w++) {
        if (!vis[w]) {
            for (e=0; e<E; e++) {
                if (incMatrix[hamPath[pos-1]][e]!=0&&incMatrix[w][e]!=0) {
                    vis[w]=1; hamPath[pos]=w;
                    if (hamiltonianHelper(pos+1,vis)) return 1;
                    vis[w]=0;
                }
            }
        }
    }
    return 0;
}

Degree Operations
Degree = number of edges incident to a vertex. Key metric for graph analysis.
Degree — Find, Min, Max, Average, Distribution
The degree of a vertex is the count of edges it touches. In the incidence matrix, it's simply the count of non-zero values in that vertex's row.

Sum of all degrees = 2 × E (Handshaking Lemma).
Degree Distribution
CAll Degree Operations
/* Get degree of vertex v */
int findDegree(int v) {
    int e, cnt = 0;
    for (e=0; e<E; e++)
        if (incMatrix[v][e] != 0) cnt++;

    return cnt;
}

/* Maximum Degree */
int maxDegree() {
    int i, maxD = 0, maxV = 0;
    for (i=0; i<V; i++) {
        int d = findDegree(i);
        if (d > maxD) { maxD=d; maxV=i; }
    }
    printf("Max degree: V%d = %d\n", maxV, maxD);
    return maxD;
}

/* Minimum Degree */
int minDegree() {
    int i, minD = V, minV = 0;
    for (i=0; i<V; i++) {
        int d = findDegree(i);
        if (d < minD) { minD=d; minV=i; }
    }
    printf("Min degree: V%d = %d\n", minV, minD);
    return minD;
}

/* Average Degree */
float avgDegree() {
    float avg = (2.0 * E) / V;  /* Handshaking Lemma */
    printf("Average degree: %.2f\n", avg);

    printf("Degree : Count\n");
    for (i=0; i<V; i++)
        if (dist[i]>0) printf("  %d    :  %d\n", i, dist[i]);
}

/* Clique Detection: check if subset is a clique */
int isClique(int subset[], int size) {
    int i, j;
    for (i=0; i<size; i++) {
        for (j=i+1; j<size; j++) {
            if (searchEdge(subset[i], subset[j]) == -1) {
                printf("Not a clique\n"); return 0;
            }
        }
    }
    printf("Is a clique of size %d\n", size);
    return 1;
}
Advanced
Floyd-Warshall Algorithm
Find shortest paths between ALL pairs of vertices in O(V³).
All-Pairs Shortest Path
Floyd-Warshall computes the shortest distance between every pair of vertices. It builds a distance matrix and tries each vertex as an intermediate node.

Key formula: dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j]) for all intermediate k.

Time Complexity: O(V³) — best for dense graphs since they already use O(V²) space.
✈️ Analogy
Finding the cheapest flight between every pair of cities, considering all possible layovers.
Floyd-Warshall Step Visualizer
CFloyd-Warshall Algorithm
#define INF 99999

int dist[MAX_V][MAX_V];

/* Build dist matrix from incidence matrix */
void buildDistMatrix() {
    int i, j, e;
    for(i=0;i<V;i++) for(j=0;j<V;j++)
        dist[i][j] = (i==j) ? 0 : INF;
    /* For each edge, set dist[u][v] = 1 (unweighted) */
    for(e=0;e<E;e++){
        int u=-1, v=-1;
        for(i=0;i<V;i++)
            if(incMatrix[i][e]!=0){
                if(u==-1) u=i; else v=i;
            }
        if(u!=-1&&v!=-1){ dist[u][v]=1; dist[v][u]=1; }
    }
}

/* Floyd-Warshall main algorithm */
void floydWarshall() {
    buildDistMatrix();
    int k, i, j;
    for(k=0;k<V;k++)        /* try each vertex as intermediate */
        for(i=0;i<V;i++)
            for(j=0;j<V;j++)
                if(dist[i][k]!=INF && dist[k][j]!=INF)
                    if(dist[i][k]+dist[k][j] < dist[i][j])
                        dist[i][j] = dist[i][k]+dist[k][j];
    /* Print distance matrix */
    int x, y;
    printf("All-Pairs Shortest Distances:\n");
    for(x=0;x<V;x++){
        for(y=0;y<V;y++)
            if(dist[x][y]==INF) printf("%4s","INF");
            else printf("%4d",dist[x][y]);
        printf("\n");
    }
}
Advanced
Prim's Algorithm — Minimum Spanning Tree
Find the tree that connects all vertices with minimum total edge weight.
Prim's MST
Prim's algorithm builds a Minimum Spanning Tree (MST) by greedily adding the cheapest edge that connects an unvisited vertex to the tree.

1. Start from any vertex. Mark it in the MST.
2. Among all edges from MST to non-MST vertices, pick the minimum weight one.
3. Add that vertex to MST. Repeat until all vertices are included.
💡 Analogy
Laying cable to connect all houses in a neighborhood with minimum total cable length.
CPrim's Algorithm (Incidence Matrix, Weighted)
int edgeWeight[MAX_E];  /* weight for each edge */

void primMST() {
    int inMST[MAX_V] = {0};
    int mstEdges = 0, totalCost = 0;
    inMST[0] = 1;  /* start from vertex 0 */

    printf("Prim's MST edges:\n");
    while (mstEdges < V-1) {
        int bestEdge = -1, bestW = INF;
        int bestU, bestV, e, i, j;

        /* Find minimum weight edge from MST to non-MST */
        for (e=0; e<E; e++) {
            int u=-1, v=-1;
            for (i=0; i<V; i++)
                if (incMatrix[i][e]!=0){
                    if(u==-1) u=i; else v=i;
                }
            if (u!=-1&&v!=-1&&inMST[u]!=inMST[v]) {
                if (edgeWeight[e] < bestW) {
                    bestW=edgeWeight[e]; bestEdge=e;
                    bestU=u; bestV=v;
                }
            }
        }
        if (bestEdge==-1) break;
        inMST[bestU]=inMST[bestV]=1;
        totalCost += bestW;
        printf("  V%d -- V%d (weight %d)\n", bestU, bestV, bestW);
        mstEdges++;
    }
    printf("MST Total Cost: %d\n", totalCost);
}
Advanced
Graph Coloring
Assign colors to vertices so no two adjacent vertices share the same color.
Greedy Graph Coloring
Graph coloring assigns a color to each vertex such that no two connected vertices have the same color. The minimum number of colors needed is the chromatic number χ(G).

For dense graphs, χ can be high. The greedy algorithm picks the smallest available color for each vertex.
📅 Analogy
Scheduling exams: courses that share students can't be at the same time. Each time slot is a "color".
Graph Coloring Visualizer
CGreedy Graph Coloring
void graphColoring() {
    int color[MAX_V];
    int i, e, w, c;
    for (i=0; i<V; i++) color[i] = -1;  /* uncolored */
    color[0] = 0;  /* first vertex gets color 0 */

    for (i=1; i<V; i++) {
        int used[MAX_V] = {0};  /* colors used by neighbors */

        /* Find all neighbors via incidence matrix */
        for (e=0; e<E; e++) {
            if (incMatrix[i][e] != 0) {
                for (w=0; w<V; w++) {
                    if (w!=i && incMatrix[w][e]!=0 && color[w]!=-1)
                        used[color[w]] = 1;
                }
            }
        }
        /* Assign smallest unused color */
        for (c=0; used[c]; c++);
        color[i] = c;
        printf("V%d -> Color %d\n", i, c);
    }
    /* Find chromatic number */
    int maxColor = 0;
    for (i=0; i<V; i++) if(color[i]>maxColor) maxColor=color[i];
    printf("Chromatic number = %d\n", maxColor+1);
}
Advanced
Hamiltonian Algorithms
Find a cycle or path that visits every vertex exactly once.
Hamiltonian Path & Cycle — Backtracking
A Hamiltonian Path visits every vertex exactly once. A Hamiltonian Cycle does this and returns to the starting vertex.

This is NP-complete — no polynomial algorithm is known. We use backtracking: try each vertex, recurse, and undo if it doesn't work.
🏃 Analogy
A traveling salesman visiting every city exactly once and returning home. Every possible route is explored until a valid one is found.
CHamiltonian Cycle — Complete Backtracking Solution
int hamCycle[MAX_V + 1];

/* Check if vertex w can be added at position pos */
int isSafe(int w, int pos) {
    /* Check edge exists from previous vertex to w */
    int e;
    int found = 0;
    for (e=0; e<E; e++)
        if(incMatrix[hamCycle[pos-1]][e]!=0&&incMatrix[w][e]!=0)
            found = 1;
    if (!found) return 0;

    /* Check w not already visited */
    int i;
    for (i=0; i<pos; i++)
        if (hamCycle[i] == w) return 0;
    return 1;
}

int hamCycleUtil(int pos) {
    if (pos == V) {
        /* Check edge back to start */
        int e;
        for (e=0; e<E; e++)
            if(incMatrix[hamCycle[pos-1]][e]!=0&&incMatrix[hamCycle[0]][e]!=0)
                return 1;
        return 0;
    }
    int v;
    for (v=1; v<V; v++) {
        if (isSafe(v, pos)) {
            hamCycle[pos] = v;
            if (hamCycleUtil(pos+1)) return 1;
            hamCycle[pos] = -1;   /* backtrack */
        }
    }
    return 0;
}

void findHamiltonianCycle() {
    int i;
    for(i=0;i<V+1;i++) hamCycle[i]=-1;
    hamCycle[0] = 0;
    if (!hamCycleUtil(1))
        printf("No Hamiltonian Cycle\n");
    else {
        printf("Hamiltonian Cycle: ");
        for(i=0;i<V;i++) printf("V%d->",hamCycle[i]);
        printf("V%d\n",hamCycle[0]);
    }
}
Advanced
Clique Algorithms
Find maximum cliques — subsets of vertices all connected to each other.
Maximum Clique via Backtracking (Bron-Kerbosch)
A clique is a complete subgraph — every pair of vertices is connected. The maximum clique is the largest such group.

Finding the maximum clique is NP-complete. We use the Bron-Kerbosch algorithm with backtracking.
👥 Analogy
Finding the largest group of mutual friends in a social network — where everyone in the group knows everyone else.
CBron-Kerbosch Clique Algorithm
int maxCliqueSet[MAX_V], maxCliqueSize = 0;

/* Check edge between u and v */
int hasEdge(int u, int v) {
    int e;
    for(e=0;e<E;e++)
        if(incMatrix[u][e]!=0&&incMatrix[v][e]!=0) return 1;
    return 0;
}

/* Bron-Kerbosch: R=current clique, P=candidates, X=excluded */
void bronKerbosch(int R[], int Rsize,
                    int P[], int Psize,
                    int X[], int Xsize) {
    if (Psize==0 && Xsize==0) {
        if (Rsize > maxCliqueSize) {
            maxCliqueSize = Rsize;
            int i;
            for(i=0;i<Rsize;i++) maxCliqueSet[i]=R[i];
        }
        return;
    }
    int i, j, v;
    for (i=0; i<Psize; i++) {
        v = P[i];
        int newR[MAX_V], newP[MAX_V], newX[MAX_V];
        int nR=Rsize, nP=0, nX=0;
        for(j=0;j<Rsize;j++) newR[j]=R[j];
        newR[nR++]=v;
        for(j=0;j<Psize;j++)
            if(hasEdge(v,P[j])) newP[nP++]=P[j];
        for(j=0;j<Xsize;j++)
            if(hasEdge(v,X[j])) newX[nX++]=X[j];
        bronKerbosch(newR,nR,newP,nP,newX,nX);
        X[Xsize++]=v;
    }
}

void findMaxClique() {
    int R[MAX_V], P[MAX_V], X[MAX_V], i;
    for(i=0;i<V;i++) P[i]=i;
    bronKerbosch(R,0,P,V,X,0);
    printf("Max Clique (size %d): ", maxCliqueSize);
    for(i=0;i<maxCliqueSize;i++) printf("V%d ",maxCliqueSet[i]);
    printf("\n");
}
