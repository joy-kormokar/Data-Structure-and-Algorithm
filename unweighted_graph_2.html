Foundation · Chapter 0

What is an Unweighted Graph?

A graph is a collection of vertices (nodes) connected by edges (lines). In an unweighted graph, every edge simply means "these two nodes are connected" — no cost or distance is attached.

🔵
Vertex (Node)
A single entity — like a city, person, or webpage.
➡️
Edge (Link)
A connection between two vertices. Can be directed or undirected.
⚖️
Unweighted
All edges are equal — no distance, cost, or priority assigned.
🔁
Undirected
If A→B exists, then B→A also exists. Like a two-way road.
🌍 Real-life Analogy

Think of Facebook friendships: you and your friend are vertices. The friendship is an edge. Since friendships are mutual (unweighted, undirected), if you're friends with someone, they're friends with you too — and the friendship has no "score" or "weight".

Key Terminology

  • Adjacent vertices: Two vertices connected by an edge.
  • Degree of a vertex: Number of edges connected to it.
  • Path: A sequence of vertices connected by edges.
  • Cycle: A path that starts and ends at the same vertex.
  • Connected graph: Every vertex can reach every other vertex.
Foundation · Chapter 1

Adjacency List Representation

An adjacency list stores a graph as an array of linked lists. Each index i holds a list of all vertices adjacent to vertex i. It's memory-efficient for sparse graphs (few edges).

💾
Memory: O(V + E)
Only stores existing edges. Sparse graphs use very little memory.
⚡
Edge Check: O(degree)
To check if edge (u,v) exists, scan u's list. Fast for low-degree nodes.
🔗
Uses Linked Lists
Each vertex's neighbor list is a dynamic linked list — grows as edges are added.
📒 Real-life Analogy

Imagine a class attendance register. Each student (vertex) has a list of classmates they sit next to (edges). Instead of a full grid showing all possible pairs, you just list who's next to whom — that's an adjacency list!

Data Structures in C

C — Graph Structure
/* ─── Node in the adjacency linked list ─── */
struct Node {
    int vertex;           /* which neighbor vertex */
    struct Node* next;   /* pointer to next neighbor */
};

/* ─── Graph: array of adjacency lists ─── */
struct Graph {
    int numVertices;      /* total number of vertices */
    struct Node** adjList; /* array of linked list heads */
    char names[50][20];  /* optional: vertex labels */
};

/* ─── Create a new graph with V vertices ─── */
struct Graph* createGraph(int V) {
    struct Graph* g = (struct Graph*) malloc(sizeof(struct Graph));
    g->numVertices = V;
    g->adjList = (struct Node**) malloc(V * sizeof(struct Node*));
    for (int i = 0; i < V; i++)
        g->adjList[i] = NULL; /* start with empty lists */
    return g;
}

Building a new city on a map. The city exists but has no roads yet — we just register it so roads can be added later.

C — Insert Vertex
/* Insert a new vertex into the graph */
void insertVertex(struct Graph** g, char* name) {
    int V = (*g)->numVertices;

    /* Reallocate adjacency list array with one more slot */
    (*g)->adjList = (struct Node**) realloc(
        (*g)->adjList, (V+1) * sizeof(struct Node*)
    );

    (*g)->adjList[V] = NULL; /* new vertex has no edges */
    strcpy((*g)->names[V], name);
    (*g)->numVertices++;

    printf("Vertex '%s' inserted at index %d\n", name, V);
}

Vertex Operations · 2.2

Delete Vertex

Deleting a vertex requires: (1) freeing its adjacency list, (2) removing all edges pointing to it from other vertices, and (3) shifting the array. It's more complex than insertion!

💥 Analogy

Demolishing a city: you must also remove all roads that led to it, not just erase it from the map.

C — Delete Vertex
/* Helper: remove vertex 'target' from a linked list */
struct Node* removeFromList(struct Node* head, int target) {
    struct Node* prev = NULL;
    struct Node* curr = head;
    while (curr) {
        if (curr->vertex == target) {
            if (prev) prev->next = curr->next;
            else head = curr->next;
            free(curr);
            return head;
        }
        prev = curr; curr = curr->next;
    }
    return head;
}

/* Delete vertex at index 'v' */
void deleteVertex(struct Graph* g, int v) {
    int V = g->numVertices;

    /* 1. Free the vertex's own adjacency list */
    struct Node* curr = g->adjList[v];
    while (curr) {
        struct Node* temp = curr->next;
        free(curr); curr = temp;
    }

    /* 2. Remove 'v' from all other adjacency lists */
    /*    Also fix indices > v (they shift down by 1) */
    for (int i = 0; i < V; i++) {
        if (i == v) continue;
        g->adjList[i] = removeFromList(g->adjList[i], v);
        /* Decrement indices > v in the remaining list */
        struct Node* n = g->adjList[i];
        while (n) { if (n->vertex > v) n->vertex--; n = n->next; }
    }

    /* 3. Shift adjList array left */
    for (int i = v; i < V-1; i++)
        g->adjList[i] = g->adjList[i+1];
    g->numVertices--;
    printf("Vertex %d deleted.\n", v);
}

Insert Edge

Adding an edge between vertex u and vertex v: create a new node with value v and prepend it to adjList[u]. For undirected graphs, also add u to adjList[v].

🛣️ Analogy

Building a road between two cities. If it's a two-way road (undirected), you add the road to both cities' connection lists.

C — Insert Edge (Undirected)
/* Create a new list node */
struct Node* newNode(int vertex) {
    struct Node* node =
        (struct Node*) malloc(sizeof(struct Node));
    node->vertex = vertex;
    node->next   = NULL;
    return node;
}

/* Insert undirected edge between u and v */
void insertEdge(struct Graph* g, int u, int v) {
    if (u < 0 || v < 0 || u >= g->numVertices || v >= g->numVertices) {
        printf("Error: Invalid vertex!\n");
        return;
    }

    /* Add v to u's adjacency list (prepend = O(1)) */
    struct Node* n1 = newNode(v);
    n1->next = g->adjList[u];
    g->adjList[u] = n1;

    /* Add u to v's adjacency list (undirected) */
    struct Node* n2 = newNode(u);
    n2->next = g->adjList[v];
    g->adjList[v] = n2;

    printf("Edge (%d–%d) inserted.\n", u, v);
}
OperationComplexity
Insert EdgeO(1) — prepend to list
Edge Operations · 3.2

Delete Edge

Removing an edge between u and v requires traversing the adjacency list to find and remove the node.

C — Delete Edge
/* Helper: remove vertex 'target' from list starting at head */
struct Node* deleteFromList(struct Node* head, int target) {
    struct Node* prev = NULL;
    struct Node* curr = head;
    while (curr != NULL) {
        if (curr->vertex == target) {
            if (prev == NULL) head = curr->next;
            else             prev->next = curr->next;
            free(curr);
            return head;
        }
        prev = curr;
        curr = curr->next;
    }
    return head; /* not found, unchanged */
}

/* Delete edge between u and v (undirected) */
void deleteEdge(struct Graph* g, int u, int v) {
    g->adjList[u] = deleteFromList(g->adjList[u], v);
    g->adjList[v] = deleteFromList(g->adjList[v], u);
    printf("Edge (%d–%d) deleted.\n", u, v);
}
OperationComplexity
Delete EdgeO(E) worst case — must scan the list
Edge Operations · 3.3

Search Edge

Check if an edge exists between two vertices by scanning the adjacency list of one of them.

C — Search Edge
/* Returns 1 if edge (u,v) exists, else 0 */
int searchEdge(struct Graph* g, int u, int v) {
    struct Node* curr = g->adjList[u];
    while (curr != NULL) {
        if (curr->vertex == v) {
            printf("Edge (%d–%d) EXISTS.\n", u, v);
            return 1;
        }
        curr = curr->next;
    }
    printf("Edge (%d–%d) does NOT exist.\n", u, v);
    return 0;
}
Edge Operations · 3.4

Count Edges

To count edges, sum up all adjacency list lengths. For an undirected graph, divide by 2 (each edge is counted twice).

C — Count Edges
/* Count total edges in undirected graph */
int countEdges(struct Graph* g) {
    int total = 0;
    for (int i = 0; i < g->numVertices; i++) {
        struct Node* curr = g->adjList[i];
        while (curr != NULL) {
            total++;
            curr = curr->next;
        }
    }
    /* Each undirected edge is counted twice */
    total = total / 2;
    printf("Total edges: %d\n", total);
    return total;
}

Traversal Operations · 4.1

Breadth First Search (BFS)

BFS explores the graph level by level — first all neighbors of the start node, then their neighbors, and so on. It uses a queue (FIFO). BFS finds the shortest path in unweighted graphs.

1
Start at a source vertex. Mark it visited. Add it to the queue.
2
Dequeue a vertex. Process it (print/record).
3
For each unvisited neighbor: mark visited, enqueue it.
4
Repeat until queue is empty.
🌊 Analogy

Spreading a ripple in water: the wave expands uniformly in all directions, layer by layer. BFS is like this — it visits everything at distance 1 before distance 2.

C — BFS with Queue
#include <stdio.h>
#include <stdlib.h>

/* Simple queue for BFS */
int queue[100], front = 0, rear = 0;
void enqueue(int v) { queue[rear++] = v; }
int  dequeue()      { return queue[front++]; }
int  isEmpty()      { return front == rear; }

/* BFS from source vertex 'start' */
void BFS(struct Graph* g, int start) {
    int visited[100] = {0};  /* all unvisited at start */
    front = rear = 0;        /* reset queue */

    /* Step 1: Visit start node */
    visited[start] = 1;
    enqueue(start);
    printf("BFS Order: ");

    while (!isEmpty()) {
        int v = dequeue();     /* Step 2: Dequeue */
        printf("%s ", g->names[v]);

        /* Step 3: Enqueue unvisited neighbors */
        struct Node* curr = g->adjList[v];
        while (curr != NULL) {
            if (!visited[curr->vertex]) {
                visited[curr->vertex] = 1;
                enqueue(curr->vertex);
            }
            curr = curr->next;
        }
    }
    printf("\n");
}
ComplexityValue
TimeO(V + E)
Space (queue)O(V)
Traversal Operations · 4.2

Depth First Search (DFS)

DFS explores as deep as possible before backtracking. It uses a stack (explicit or call stack via recursion). DFS is great for cycle detection, path finding, and topological sort.

1
Visit the current vertex. Mark it visited.
2
For each unvisited neighbor, recursively call DFS.
3
Backtrack when no unvisited neighbors remain.
🕳️ Analogy

Exploring a cave system: you always go as far down one tunnel as possible before turning back and trying another branch. You never miss anything, but you go deep first.

C — Recursive DFS
/* Recursive DFS helper */
void DFSHelper(struct Graph* g, int v, int* visited) {
    visited[v] = 1;
    printf("%s ", g->names[v]);  /* process vertex */

    struct Node* curr = g->adjList[v];
    while (curr != NULL) {
        if (!visited[curr->vertex]) {
            DFSHelper(g, curr->vertex, visited);  /* go deeper */
        }
        curr = curr->next;
    }
    /* implicit backtrack when function returns */
}

/* DFS entry point */
void DFS(struct Graph* g, int start) {
    int visited[100] = {0};
    printf("DFS Order: ");
    DFSHelper(g, start, visited);
    printf("\n");
}

/* ─── Iterative DFS using explicit stack ─── */
void DFS_Iterative(struct Graph* g, int start) {
    int visited[100] = {0};
    int stack[100], top = -1;

    stack[++top] = start;  /* push start */

    while (top >= 0) {
        int v = stack[top--];  /* pop */
        if (visited[v]) continue;
        visited[v] = 1;
        printf("%s ", g->names[v]);

        struct Node* curr = g->adjList[v];
        while (curr) {
            if (!visited[curr->vertex])
                stack[++top] = curr->vertex;
            curr = curr->next;
        }
    }
}
ComplexityValue
TimeO(V + E)
Space (stack)O(V)

C — Find Any Path (DFS)
/* DFS-based path finder — stores path in pathArr[] */
int findPathDFS(struct Graph* g, int src, int dest,
               int* visited, int* path, int pathIdx) {
    visited[src] = 1;
    path[pathIdx] = src;

    if (src == dest) {
        printf("Path: ");
        for (int i = 0; i <= pathIdx; i++)
            printf("%d ", path[i]);
        printf("\n");
        return 1;  /* path found! */
    }

    struct Node* curr = g->adjList[src];
    while (curr) {
        if (!visited[curr->vertex]) {
            if (findPathDFS(g, curr->vertex, dest,
                            visited, path, pathIdx+1))
                return 1;
        }
        curr = curr->next;
    }
    return 0;  /* backtrack */
}
Path Operations · 5.2

Shortest Path

In an unweighted graph, BFS gives the shortest path (minimum number of edges). We store the parent of each vertex and trace back from destination to source.

C — Shortest Path using BFS
/* Find shortest path from src to dest using BFS */
void shortestPath(struct Graph* g, int src, int dest) {
    int visited[100] = {0};
    int parent[100];
    int queue[100], front = 0, rear = 0;

    for (int i = 0; i < g->numVertices; i++)
        parent[i] = -1;  /* -1 means no parent yet */

    visited[src] = 1;
    queue[rear++] = src;

    while (front != rear) {
        int v = queue[front++];
        if (v == dest) break;  /* destination reached */

        struct Node* curr = g->adjList[v];
        while (curr) {
            if (!visited[curr->vertex]) {
                visited[curr->vertex] = 1;
                parent[curr->vertex] = v; /* record parent */
                queue[rear++] = curr->vertex;
            }
            curr = curr->next;
        }
    }

    /* Trace back from dest to src using parent[] */
    if (!visited[dest]) {
        printf("No path from %d to %d\n", src, dest);
        return;
    }
    int path[100], pathLen = 0;
    for (int v = dest; v != -1; v = parent[v])
        path[pathLen++] = v;
    printf("Shortest path (length=%d): ", pathLen-1);
    for (int i = pathLen-1; i >= 0; i--)
        printf("%d ", path[i]);
    printf("\n");
}
Path Operations · 5.3

All Paths

Find every possible path from source to destination using backtracking DFS. Mark a vertex visited when entering, unmark when backtracking.

C — All Paths (Backtracking DFS)
/* Print all paths from src to dest */
void allPaths(struct Graph* g, int src, int dest,
              int* visited, int* path, int pathIdx) {
    visited[src] = 1;
    path[pathIdx] = src;

    if (src == dest) {
        for (int i = 0; i <= pathIdx; i++)
            printf("%d ", path[i]);
        printf("\n");
    } else {
        struct Node* curr = g->adjList[src];
        while (curr) {
            if (!visited[curr->vertex])
                allPaths(g, curr->vertex, dest,
                          visited, path, pathIdx+1);
            curr = curr->next;
        }
    }
    /* BACKTRACK: unmark so other paths can use this vertex */
    visited[src] = 0;
}
⚠️ All-paths can be exponential O(V!) in the worst case. Only use on small graphs or with depth limits.
Path Operations · 5.4

Reachability Check

Is vertex dest reachable from vertex src? Run BFS/DFS from src — if dest gets visited, it's reachable.

C — Reachability Check
/* Returns 1 if dest is reachable from src */
int isReachable(struct Graph* g, int src, int dest) {
    int visited[100] = {0};
    int queue[100], front = 0, rear = 0;

    visited[src] = 1;
    queue[rear++] = src;

    while (front != rear) {
        int v = queue[front++];
        if (v == dest) { printf("REACHABLE\n"); return 1; }
        struct Node* curr = g->adjList[v];
        while (curr) {
            if (!visited[curr->vertex]) {
                visited[curr->vertex] = 1;
                queue[rear++] = curr->vertex;
            }
            curr = curr->next;
        }
    }
    printf("NOT reachable\n");
    return 0;
}
Path Operations · 5.5

Path Length Count

Count the number of edges in the shortest path from source to destination. BFS naturally gives this as each level = 1 step.

C — Path Length (BFS Distance)
/* Returns shortest distance from src to dest (-1 if unreachable) */
int pathLength(struct Graph* g, int src, int dest) {
    int dist[100], visited[100] = {0};
    int queue[100], front = 0, rear = 0;

    for (int i = 0; i < g->numVertices; i++) dist[i] = -1;

    dist[src] = 0;  /* distance from src to itself = 0 */
    visited[src] = 1;
    queue[rear++] = src;

    while (front != rear) {
        int v = queue[front++];
        struct Node* curr = g->adjList[v];
        while (curr) {
            if (!visited[curr->vertex]) {
                visited[curr->vertex] = 1;
                dist[curr->vertex] = dist[v] + 1; /* +1 edge */
                queue[rear++] = curr->vertex;
            }
            curr = curr->next;
        }
    }
    printf("Distance from %d to %d: %d\n", src, dest, dist[dest]);
    return dist[dest];
}
Connectivity Operations · 6.1

Check Connected Graph

A graph is connected if every vertex can reach every other vertex. Start BFS/DFS from vertex 0 — if all vertices get visited, the graph is connected.

🕸️ Analogy

A phone network is connected if you can call anyone from anyone, directly or through others. If some phones are isolated, the network is disconnected.

C — Check Connected Graph
/* Returns 1 if graph is connected, else 0 */
int isConnected(struct Graph* g) {
    int visited[100] = {0};
    int visitCount = 0;
    int queue[100], front = 0, rear = 0;

    visited[0] = 1;
    queue[rear++] = 0;  /* start BFS from vertex 0 */

    while (front != rear) {
        int v = queue[front++];
        visitCount++;
        struct Node* curr = g->adjList[v];
        while (curr) {
            if (!visited[curr->vertex]) {
                visited[curr->vertex] = 1;
                queue[rear++] = curr->vertex;
            }
            curr = curr->next;
        }
    }
    /* If all vertices were visited, graph is connected */
    if (visitCount == g->numVertices) {
        printf("Graph IS connected.\n");
        return 1;
    }
    printf("Graph is NOT connected.\n");
    return 0;
}
Connectivity Operations · 6.2

Find Connected Components

A connected component is a maximal set of vertices where every pair is connected. Run BFS/DFS from each unvisited vertex to find all components.

C — Find Connected Components
/* Find and print all connected components */
void findComponents(struct Graph* g) {
    int visited[100] = {0};
    int compNum = 0;

    for (int i = 0; i < g->numVertices; i++) {
        if (!visited[i]) {
            compNum++;
            printf("Component %d: ", compNum);

            /* BFS from unvisited vertex i */
            int queue[100], front = 0, rear = 0;
            visited[i] = 1;
            queue[rear++] = i;
            while (front != rear) {
                int v = queue[front++];

Detect Cycle

A cycle exists if we find a visited vertex that is not the parent of the current vertex during DFS. For undirected graphs, we track the parent to avoid false positives.

🔄 Analogy

Walking through a city: if you end up back where you started without retracing a road, there's a cycle. Cycles are important for detecting circular dependencies or infinite loops in networks.

C — Cycle Detection (DFS, Undirected)
/* DFS helper — returns 1 if cycle found */
int cycleHelper(struct Graph* g, int v,
                int* visited, int parent) {
    visited[v] = 1;
    struct Node* curr = g->adjList[v];

    while (curr) {
        int u = curr->vertex;
        if (!visited[u]) {
            /* Recurse on unvisited neighbor */
            if (cycleHelper(g, u, visited, v))
                return 1;
        } else if (u != parent) {
            /* Visited AND not the parent → CYCLE! */
            return 1;
        }
        curr = curr->next;
    }
    return 0;
}

/* Check if graph has a cycle */
int hasCycle(struct Graph* g) {
    int visited[100] = {0};
    for (int i = 0; i < g->numVertices; i++) {
        if (!visited[i]) {
            if (cycleHelper(g, i, visited, -1)) {
                printf("Cycle DETECTED!\n");
                return 1;
            }
        }
    }
    printf("No cycle found.\n");
    return 0;
}

Count Cycles

For an undirected graph, the number of independent cycles (cyclomatic complexity) = E - V + C, where C is the number of connected components.

C — Count Cycles (Formula)
/* Count cycles using formula: cycles = E - V + C */
int countCycles(struct Graph* g) {
    int V = g->numVertices;
    int E = countEdges(g);   /* count edges */

    /* Count connected components using BFS */
    int visited[100] = {0};
    int C = 0;
    for (int i = 0; i < V; i++) {
        if (!visited[i]) {
            C++;
            /* BFS to mark all vertices in this component */
            int q[100], f = 0, r = 0;
            visited[i] = 1; q[r++] = i;
            while (f != r) {
                int v = q[f++];
                struct Node* n = g->adjList[v];
                while (n) { if(!visited[n->vertex]){visited[n->vertex]=1;q[r++]=n->vertex;} n=n->next;}
            }
        }
    }
    int cycles = E - V + C;
    printf("Cycles = E(%d) - V(%d) + C(%d) = %d\n", E, V, C, cycles);
    return cycles;
}

Average Degree

Average degree = (2 × E) / V. Since sum of all degrees = 2E (Handshaking Lemma), dividing by V gives the average.

C — Average Degree
/* Average degree = 2E / V */
float averageDegree(struct Graph* g) {
    int totalDeg = 0;
    for (int i = 0; i < g->numVertices; i++) {
        struct Node* curr = g->adjList[i];
        while (curr) { totalDeg++; curr = curr->next; }
    }
    /* totalDeg counts each edge twice (undirected), so it equals 2E */
    float avg = (float)totalDeg / g->numVertices;
    printf("Average degree: %.2f\n", avg);
    return avg;
}

Bipartite Check

A graph is bipartite if its vertices can be split into two groups (sets) such that every edge connects a vertex from one group to the other — no edges within the same group. We verify this using 2-coloring with BFS.

👫 Analogy

A dating app graph: users on one side, matches on the other. Every match (edge) goes between a user and a potential match — never between two users directly. That's a bipartite graph!

C — Bipartite Check (BFS 2-Coloring)
/* Check if graph is bipartite using 2-coloring */
int isBipartite(struct Graph* g) {
    int color[100];
    for (int i = 0; i < g->numVertices; i++) color[i] = -1;

    for (int start = 0; start < g->numVertices; start++) {
        if (color[start] != -1) continue;

        /* BFS with 2-coloring */
        int queue[100], front = 0, rear = 0;
        color[start] = 0;  /* assign color 0 to start */
        queue[rear++] = start;

        while (front != rear) {
            int v = queue[front++];
            struct Node* curr = g->adjList[v];
            while (curr) {
                int u = curr->vertex;
                if (color[u] == -1) {
                    /* Assign opposite color */
                    color[u] = 1 - color[v];
                    queue[rear++] = u;
                } else if (color[u] == color[v]) {
                    /* Same color on adjacent vertices → NOT bipartite */
                    printf("NOT bipartite (conflict at %d–%d)\n",v,u);
                    return 0;
                }
                curr = curr->next;
            }
        }
    }
    printf("Graph IS bipartite.\n");
    return 1;
}

Graph Coloring Basics

Graph coloring assigns colors to vertices so that no two adjacent vertices share the same color. The minimum number of colors needed is the chromatic number χ(G). Bipartite graphs have χ = 2 (if non-trivial).

🟦
1-colorable
Only possible for graphs with no edges (empty graph).
🟦🟨
2-colorable
Bipartite graphs. No odd-length cycles.
🟦🟨🟥
3-colorable
Planar graphs are always ≤4 colorable (Four Color Theorem).
C — Greedy Graph Coloring
/* Greedy coloring — assigns minimum colors (not always optimal) */
void greedyColoring(struct Graph* g) {
    int color[100];
    int available[100];

    color[0] = 0;  /* assign color 0 to first vertex */
    for (int i = 1; i < g->numVertices; i++) color[i] = -1;
    for (int i = 0; i < g->numVertices; i++) available[i] = 0;

    for (int v = 1; v < g->numVertices; v++) {
        /* Mark colors used by neighbors as unavailable */
        struct Node* curr = g->adjList[v];
        while (curr) {
            if (color[curr->vertex] != -1)
                available[color[curr->vertex]] = 1;
            curr = curr->next;
        }
        /* Find the first available color */
        int c;
        for (c = 0; c < g->numVertices; c++)
            if (!available[c]) break;
        color[v] = c;

        /* Reset available[] for next vertex */
        curr = g->adjList[v];
        while (curr) {
            if (color[curr->vertex] != -1)
                available[color[curr->vertex]] = 0;
            curr = curr->next;
        }
    }
    /* Print result */
    printf("Vertex Colors:\n");
    for (int i = 0; i < g->numVertices; i++)
        printf("  Vertex %d: Color %d\n", i, color[i]);
}

Structural Operations · 10.2

Density Check — Sparse vs Dense

Density = actual edges / maximum possible edges. Maximum edges in an undirected graph = V×(V-1)/2. Density ranges from 0 (no edges) to 1 (complete graph).

  • Sparse graph: Density close to 0. Few edges. Adjacency list is efficient here.
  • Dense graph: Density close to 1. Many edges. Adjacency matrix may be better.
C — Density Calculation
/* Density = E / (V*(V-1)/2) for undirected graph */
float density(struct Graph* g) {
    int V = g->numVertices;
    int E = countEdges(g);
    int maxEdges = V * (V - 1) / 2;

    if (maxEdges == 0) { printf("Empty graph!\n"); return 0; }

    float d = (float)E / maxEdges;
    printf("Density: %.3f (%s)\n", d, d < 0.5 ? "Sparse" : "Dense");
    return d;
}

Complete Graph Check

A complete graph Kn has every possible edge — every vertex connects to every other. It has V×(V-1)/2 edges and every vertex has degree V-1.

🤝 Analogy

A fully connected conference room: everyone shakes hands with everyone else. If there are 5 people, that's 5×4/2 = 10 handshakes total.

int isComplete(struct Graph* g) {
    int V = g->numVertices;
    int expectedEdges = V * (V - 1) / 2;
    int E = countEdges(g);

    if (E == expectedEdges) {
        printf("Graph is COMPLETE (K%d)\n", V);
        return 1;
    }
    printf("Graph is NOT complete (%d/%d edges)\n", E, expectedEdges);
    return 0;
}

Clear Graph & Copy Graph

Clear: Remove all edges and vertices, freeing memory. Copy: Create a deep copy with all adjacency lists duplicated.

C — Clear and Copy Graph
/* Clear all edges and free memory */
void clearGraph(struct Graph* g) {
    for (int i = 0; i < g->numVertices; i++) {
        struct Node* curr = g->adjList[i];
        while (curr) {
            struct Node* temp = curr->next;
            free(curr);
            curr = temp;
        }
        g->adjList[i] = NULL;
    }
    g->numVertices = 0;
    printf("Graph cleared.\n");
}

/* Deep copy of a graph */
struct Graph* copyGraph(struct Graph* src) {
    struct Graph* dest = createGraph(src->numVertices);
    for (int i = 0; i < src->numVertices; i++) {
        strcpy(dest->names[i], src->names[i]);
        /* Copy adjacency list in reverse to preserve order */
        struct Node* curr = src->adjList[i];
        while (curr) {
            struct Node* n = newNode(curr->vertex);
            n->next = dest->adjList[i];
            dest->adjList[i] = n;
            curr = curr->next;
        }
    }
    printf("Graph copied successfully.\n");
    return dest;
}

#include <stdio.h>
#include <stdlib.h>
#include <string.h>

struct Node { int vertex; struct Node* next; };
struct Graph { int V; struct Node** list; };

struct Graph* create(int V) {
    struct Graph* g = malloc(sizeof(struct Graph));
    g->V = V;
    g->list = malloc(V * sizeof(struct Node*));
    for (int i = 0; i < V; i++) g->list[i] = NULL;

}

void addEdge(struct Graph* g, int u, int v) {
    struct Node* n = malloc(sizeof(struct Node));
    n->vertex = v; n->next = g->list[u]; g->list[u] = n;
    n = malloc(sizeof(struct Node));
    n->vertex = u; n->next = g->list[v]; g->list[v] = n;
}
